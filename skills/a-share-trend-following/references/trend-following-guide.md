# 趋势跟踪策略指南 (Trend Following Guide)

## 一、趋势跟踪理论基础

趋势跟踪（Trend Following）是最经典的量化交易策略之一，核心假设：
- 市场价格存在持续性（momentum），趋势一旦形成，倾向于延续
- 不预测方向，而是跟随已确认的趋势
- 依靠少数大趋势的盈利覆盖多次小亏损（"截断亏损，让利润奔跑"）

### 理论支撑
- **动量效应 (Momentum Effect)**: Jegadeesh & Titman (1993) 证实 3-12 个月动量在全球市场普遍存在
- **行为金融学**: 投资者对信息的反应不足（underreaction）和过度反应（overreaction）导致趋势延续
- **A股特征**: 散户占比高，趋势延续性在某些阶段更强，但政策干预可导致趋势突变

## 二、核心趋势指标

### 2.1 移动平均线交叉 (Moving Average Crossover)

**原理**: 短期均线上穿/下穿长期均线产生买入/卖出信号。

**类型**:
- **SMA (Simple Moving Average)**: 简单平均，对所有数据等权。公式: SMA(n) = sum(close, n) / n
- **EMA (Exponential Moving Average)**: 指数加权，近期数据权重更大。公式: EMA(t) = close(t) * k + EMA(t-1) * (1-k), k = 2/(n+1)

**常用参数组合**:
| 组合 | 适用场景 | 特点 |
|------|---------|------|
| 5/20 | 短线趋势 | 灵敏但假信号多 |
| 10/30 | 中短线 | 平衡 |
| 20/60 | 中线趋势 | 经典组合，适合A股 |
| 50/200 | 长线趋势 | 即"金叉/死叉"，信号少但可靠 |

**信号规则**:
- 金叉（Golden Cross）: 短期MA上穿长期MA → 买入
- 死叉（Death Cross）: 短期MA下穿长期MA → 卖出
- 可增加过滤条件：交叉时价格需在长期MA上方一定比例

### 2.2 ADX/DMI (Average Directional Index / Directional Movement Index)

**原理**: Welles Wilder 发明，量化趋势强度（不区分方向）。

**计算步骤**:
1. 计算 +DM (Positive Directional Movement) 和 -DM (Negative Directional Movement)
   - +DM = max(high(t) - high(t-1), 0)（当 +DM > -DM 时取值，否则为0）
   - -DM = max(low(t-1) - low(t), 0)（当 -DM > +DM 时取值，否则为0）
2. 计算 True Range (TR) = max(high-low, |high-prev_close|, |low-prev_close|)
3. +DI = 100 * smoothed(+DM, n) / smoothed(TR, n)
4. -DI = 100 * smoothed(-DM, n) / smoothed(TR, n)
5. DX = 100 * |+DI - -DI| / (+DI + -DI)
6. ADX = smoothed(DX, n)，默认 n=14

**解读**:
| ADX 值 | 趋势强度 | 策略建议 |
|--------|---------|---------|
| 0-15 | 极弱/无趋势 | 避免趋势策略 |
| 15-20 | 弱趋势 | 谨慎 |
| 20-25 | 趋势形成中 | 关注信号 |
| 25-50 | 强趋势 | 积极跟随 |
| 50-75 | 极强趋势 | 注意超买超卖 |
| >75 | 罕见极端 | 可能接近反转 |

**方向判定**: +DI > -DI → 上升趋势; -DI > +DI → 下降趋势

### 2.3 Donchian Channel (唐奇安通道)

**原理**: Richard Donchian 发明，海龟交易法的核心工具。

**计算**:
- Upper Band = max(high, n期)
- Lower Band = min(low, n期)
- Middle = (Upper + Lower) / 2

**信号规则**:
- 价格突破上轨 → 买入信号
- 价格跌破下轨 → 卖出信号
- 常用参数: 突破 n=20（入场），n=10（出场）

### 2.4 ATR Trailing Stop (ATR 跟踪止损)

**原理**: 基于 Average True Range 设置动态止损位，随趋势推进逐步上移。

**计算**:
- ATR(n) = smoothed(TR, n)，默认 n=14
- 多头止损位 = highest_close - multiplier * ATR
- 止损位只升不降（多头时），确保跟踪趋势

**参数选择**:
| multiplier | 特点 |
|-----------|------|
| 1.5 | 较紧，适合短线 |
| 2.0 | 经典，平衡灵敏与稳健 |
| 3.0 | 较宽，适合大趋势 |

## 三、海龟交易法则 (Turtle Trading Rules) — A股适配版

原版海龟法则由 Richard Dennis 设计，以下为A股适配：

### 入场规则
- **System 1**: 价格突破 20 日 Donchian 上轨 → 买入（前一次突破盈利则跳过）
- **System 2**: 价格突破 55 日 Donchian 上轨 → 买入（不跳过）
- **A股适配**: T+1 限制下，信号日收盘确认，次日开盘执行

### 仓位管理
- 1 Unit = 账户净值 * 1% / (ATR * 每手股数)
- 最多加仓 4 Units，每次价格上涨 0.5*ATR 加仓一次
- **A股适配**: A股最小交易单位 100 股，需向下取整到 100 的倍数

### 止损规则
- 每 Unit 最大亏损 = 2 * ATR
- 总仓位亏损不超过账户 2%
- **A股适配**: 跌停板时无法止损，实际亏损可能超过预设

### 出场规则
- System 1: 价格跌破 10 日 Donchian 下轨 → 卖出
- System 2: 价格跌破 20 日 Donchian 下轨 → 卖出

## 四、趋势状态分类 (Trend Regime Classification)

综合多指标判定当前市场处于何种趋势状态：

### 分类矩阵
| 状态 | ADX | MA关系 | 价格位置 | 典型特征 |
|------|-----|--------|---------|---------|
| 强上升 | >25 | 短>长且发散 | MA上方 | 量价齐升，回调浅 |
| 弱上升 | <25 | 短>长但收敛 | MA附近 | 上涨乏力，可能转震荡 |
| 震荡 | <20 | 缠绕 | MA附近反复穿越 | 无方向，趋势策略失效 |
| 弱下降 | <25 | 短<长但收敛 | MA附近 | 下跌减缓，可能筑底 |
| 强下降 | >25 | 短<长且发散 | MA下方 | 放量下跌，反弹弱 |

### 多周期共振
- 周线趋势确定大方向
- 日线趋势确定操作方向
- 周线与日线同向 → 高概率趋势信号

## 五、A股特殊考量

### 5.1 T+1 制度
- 买入当日不可卖出，趋势信号需考虑隔日执行
- 日内反转无法止损，增加单笔风险
- 回测时必须用次日开盘价作为成交价

### 5.2 涨跌停板
- 主板 ±10%，科创板/创业板 ±20%，ST 股 ±5%，北交所 ±30%
- 涨停时可能无法买入（封板），跌停时可能无法卖出
- 连续涨跌停会导致信号滞后执行
- 回测时需检测涨跌停状态，模拟真实成交

### 5.3 停牌与复牌
- 重大事项停牌期间无法交易
- 复牌后可能大幅跳空（gap），ATR 骤升
- 建议：停牌股不纳入趋势策略，或复牌后等待 ATR 恢复正常再入场

### 5.4 交易成本
| 费用项 | 比例 | 说明 |
|--------|------|------|
| 佣金 | ~0.025% | 双边收取，各券商不同 |
| 印花税 | 0.05% | 仅卖出收取（2024年起） |
| 过户费 | 0.001% | 双边收取 |
| 单次往返总成本 | ~0.101% | 频繁交易会显著侵蚀收益 |

### 5.5 市场微观结构
- 集合竞价（9:15-9:25, 14:57-15:00）可能产生异常价格
- 建议使用收盘价或 VWAP 作为信号触发价格
- 量能配合：突破时成交量应放大（至少 1.5 倍于 20 日均量）

## 六、策略评估指标

| 指标 | 公式/说明 | 优秀标准 |
|------|----------|---------|
| 年化收益率 | (总收益)^(252/交易天数) - 1 | >15% |
| Sharpe Ratio | (年化收益-无风险)/年化波动率 | >1.0 |
| 最大回撤 | max(peak - trough) / peak | <25% |
| 胜率 | 盈利次数/总交易次数 | >40%（趋势策略通常偏低） |
| 盈亏比 | 平均盈利/平均亏损 | >2.0 |
| Profit Factor | 总盈利/总亏损 | >1.5 |
| Calmar Ratio | 年化收益/最大回撤 | >1.0 |

趋势策略的特征：胜率通常较低（30-45%），但依靠高盈亏比实现整体盈利。
