# 补查与核保映射指引

本文件用于将常规体检异常转换为可执行的补问问题、补件建议和核保关注点。

## 一、补问设计原则

补问应聚焦以下四个目标：

1. 明确异常项目
2. 明确是否持续存在
3. 明确是否已复查
4. 明确是否已有专科诊断或处理

## 二、常用补问模板

### 指标异常补问

- 请补充该异常指标是否曾复查、最近一次复查时间及复查结果。
- 请说明医生是否给出明确诊断、随访建议或治疗意见。

### 影像或功能检查补问

- 请补充心电图、胸片、彩超等异常项目的具体结论及是否已进一步检查。

### 系统性风险补问

- 请说明是否存在相关既往病史、长期用药或专科随访情况。

## 三、补件建议口径

示例：

- 建议补充最近一次复查报告，以确认异常指标是否持续存在。
- 建议补充专项检查结果，以核实体检结论中的器官异常提示。
- 建议补充门诊记录或健康说明，以支持异常结论的进一步判断。
