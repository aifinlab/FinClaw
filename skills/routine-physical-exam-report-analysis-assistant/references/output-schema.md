# 常规体检解析标准输出结构

本文件用于统一常规体检报告解析输出结构，确保结果可直接用于核保审查、风险筛查和资料流转。

## 一、体检基本信息

建议字段：

- 体检人基本信息：
- 体检日期：
- 体检机构：
- 材料来源或文本类型：
- 主要检查项目范围：

## 二、体检解析结论摘要

建议输出 1 至 3 条：

- 是否存在明确异常指标
- 是否存在系统性风险或复查建议
- 是否建议补问、补件或人工审核

## 三、异常指标与异常结论提取

重点覆盖：

- 明确异常指标
- 报告异常结论
- 提示语与复查建议

## 四、边界异常与系统性风险识别

建议按风险主题组织：

- 边界异常
- 代谢风险
- 肝功能风险
- 肾功能风险
- 心血管风险
- 呼吸系统风险

## 五、重点检查项目梳理

建议覆盖：

- 血压与 BMI
- 血糖与血脂
- 肝肾功能
- 血常规与尿常规
- 心电图、胸片、彩超等

## 六、核保关注点与风险提示

写法建议采用“已知事实 + 风险提示/判断边界”。

## 七、建议补充核实的问题或资料

问题应能直接用于业务人员、核保人员继续补问。

## 八、后续处理建议

可组合输出以下动作：

- 建议补充说明
- 建议补件
- 建议重点人工审核
- 建议结合问卷、病史或专项检查进一步判断
- 当前可进入下一环节核保审查

必须避免写成正式核保结论。
