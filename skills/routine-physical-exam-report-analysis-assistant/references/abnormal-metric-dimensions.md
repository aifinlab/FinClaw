# 常规体检异常指标识别维度与口径

本文件用于统一常规体检报告分析时的异常识别维度、标注方法和判断边界。

## 一、基本信息维度

优先核对以下信息是否齐备：

- 年龄
- 性别
- 体检日期
- 体检机构
- 报告来源
- 主要项目范围

如以上任一项缺失且会影响判断，应在体检基本信息或后续处理建议中说明。

## 二、重点异常提取口径

重点提取以下内容：

- 血压、BMI
- 血糖、血脂
- 肝肾功能
- 血常规、尿常规
- 心电图
- 胸片
- 腹部彩超及常规超声

## 三、异常识别口径

以下情况应优先列入异常指标与异常结论提取：

- 明确标记异常、高于或低于参考范围
- 报告结论明确提示异常
- 医生建议复查、进一步检查或健康管理
- 同类指标多项偏离参考范围

## 四、边界异常识别口径

以下情况应列为边界异常或需持续关注：

- 接近参考范围上限或下限
- 报告虽未写异常，但结论提示关注
- 轻度偏离但与其他指标共同异常

处理原则：

- 不把边界异常直接解释为明确疾病诊断
- 不把边界异常直接视为完全正常
- 应结合其他指标和复查建议综合提示
