# 偿债能力诊断检查清单（快用版）

## 1. 到期结构与“硬到期”识别

- 未来 3/6/12 个月到期本金与利息分别是多少？是否集中在单月/单季？
- 渠道拆分：银行贷款、非银/信托、供应链金融、票据、债券、租赁
- “硬到期”识别：能否滚续？是否存在压降/抽贷/授信到期不续作信号？

## 2. 覆盖能力三件套

- 经营现金流覆盖：现金流能否覆盖利息？覆盖本金需要什么前提？
- 现金头寸与受限资金：可动用现金 vs 保证金/冻结/受限资金
- 备用来源：已批未用授信、资产处置、股东/集团支持、增信安排

## 3. 触发点与加速到期风险

- 交叉违约条款、重大不利变化条款、担保代偿、诉讼冻结
- 经营侧触发：关键客户流失、订单崩塌、回款链条断裂
- 金融侧触发：票据保证金飙升、融资利率显著抬升、融资渠道收缩

## 4. 缓释条件（把风险翻译成动作）

- 资金归集/回款监管/收支两条线
- 补充抵质押、保证金比例、担保方式与反担保
- 债务重组计划：展期条件、还款安排、里程碑与约束条款
