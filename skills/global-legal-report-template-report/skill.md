# 法律条文/公司报告模板化报告 Skill

## 数据来源
本 Skill 仅面向公网可查数据，优先使用官方或准官方来源：
1. 国家法律法规数据库：检索法律、行政法规、司法解释等公开文本。
2. 中国证监会证券期货法规数据库：检索证券期货监管规则、规范性文件与部分条款查询页面。
3. 上海证券交易所/深圳证券交易所规则页面：检索上市、交易、持续监管、信息披露等业务规则。
4. 巨潮资讯等上市公司信息披露入口：获取上市公司公告、定期报告或临时报告页面/PDF。
5. 用户额外提供的公开 URL：只要可以从公网直接访问，脚本即可抓取并处理。

## 功能
- 读取公开网页或 PDF 文档正文。
- 面向“法律条文”与“公司报告”两类文本做自动抽取与清洗。
- 输出固定栏目，包括文件概况、关键事实、监管要点、影响判断、待核事项和引用片段，便于复核与归档。
- 支持命令行执行，可嵌入到外部规则引擎、批处理或工作流中。

## 使用示例
安装依赖：
```bash
pip install -r requirements.txt
```

执行示例：
```bash
python script/generate_template_report.py --source-type report --url "https://www.cninfo.com.cn/..." --company "示例上市公司" --template compliance_brief --output template_report.md
```

输出文件默认写入当前工作目录，例如：`template_report.md`。

## 交易说明
- 本 Skill 仅用于公开信息检索、文本整理和辅助分析。
- 结果不构成投资建议、证券买卖建议、收益承诺、法律意见或审计结论。
- 若将输出用于投资、合规、信息披露、诉讼或客服正式答复，必须由人工复核原文与最新版本。
- 上市公司公告、交易所规则和监管法规可能更新、修订或被废止，使用前应再次核验来源页面的时效性。
