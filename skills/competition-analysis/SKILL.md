# 同业竞争梳理助手

## 功能概述

本技能用于梳理控股股东、实际控制人控制的其他企业，识别是否存在同业竞争，并协助制定解决方案。

## 适用场景

- IPO 申报前同业竞争核查
- 控股股东资产梳理
- 同业竞争解决方案制定
- 反馈回复准备

## 核心梳理内容

### 一、同业竞争认定标准

#### 1. 竞争方范围

- [ ] 控股股东
- [ ] 实际控制人及其近亲属
- [ ] 控股股东、实际控制人控制的其他企业
- [ ] 董事、高管及其近亲属控制的企业

#### 2. 同业认定

**产品/服务相同或相似：**
- [ ] 产品名称相同
- [ ] 产品功能相同
- [ ] 目标客户相同
- [ ] 销售区域重叠

**业务替代性：**
- [ ] 可相互替代
- [ ] 存在竞争关系
- [ ] 影响独立性

### 二、核查方法

#### 1. 关联方排查

| 核查对象 | 核查方式 | 核查内容 |
|----------|----------|----------|
| 控股股东 | 工商查询、访谈 | 控制的企业清单 |
| 实际控制人 | 访谈、承诺函 | 投资情况、兼职情况 |
| 董监高 | 调查问卷 | 对外投资、兼职 |

#### 2. 业务比对

| 比对维度 | 发行人 | 竞争方 | 是否同业 |
|----------|--------|--------|----------|
| 主营业务 | | | |
| 主要产品 | | | |
| 目标客户 | | | |
| 销售区域 | | | |
| 核心技术 | | | |

#### 3. 财务数据分析

- 竞争方相关业务收入占比
- 竞争方相关业务毛利占比
- 与发行人业务占比对比

### 三、解决方案

#### 1. 业务整合

- [ ] 将竞争业务注入发行人
- [ ] 发行人收购竞争方股权
- [ ] 竞争方将业务转让给无关联第三方

#### 2. 业务调整

- [ ] 竞争方变更经营范围
- [ ] 竞争方停止竞争业务
- [ ] 划分业务区域/客户

#### 3. 承诺措施

- [ ] 避免同业竞争承诺
- [ ] 优先受让权承诺
- [ ] 违约赔偿承诺

### 四、信息披露

- [ ] 竞争方基本情况
- [ ] 竞争业务情况
- [ ] 对发行人的影响
- [ ] 解决方案及进展
- [ ] 相关承诺

## 使用流程

### 步骤 1：收集关联方信息

收集控股股东、实际控制人控制的企业清单及业务信息

### 步骤 2：运行梳理脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\10. 同业竞争梳理助手"
python scripts/analyze_competition.py --input <关联方数据> --output <分析结果路径>
```

### 步骤 3：查看分析结果

分析报告将包含：
- 关联方业务梳理
- 同业竞争识别
- 解决方案建议

### 步骤 4：制定解决方案

根据分析结果制定并实施解决方案

## 输出示例

```markdown
## 同业竞争分析报告

### 关联方概况
- 控股股东控制企业：8 家
- 实际控制人控制企业：12 家
- 已排查企业：20 家

### 同业竞争识别

| 竞争方 | 主营业务 | 与发行人关系 | 是否同业 |
|--------|----------|--------------|----------|
| A 公司 | XX 产品 | 控股股东控制 | 是 ⚠️ |
| B 公司 | YY 服务 | 实际控制人控制 | 否 |
| C 公司 | ZZ 产品 | 控股股东控制 | 相似 ⚠️ |

### 影响分析

- 竞争方 A 公司 2024 年收入：5,000 万元
- 占发行人同类业务收入比例：15%
- 对发行人影响：中等

### 解决方案

1. **A 公司同业竞争**
   - 方案：发行人收购 A 公司 100% 股权
   - 进展：已签署收购协议
   - 预计完成：2026-06-30

2. **C 公司业务相似**
   - 方案：C 公司变更经营范围，停止竞争业务
   - 进展：已启动变更程序
   - 预计完成：2026-05-31

### 承诺措施

控股股东、实际控制人已出具《避免同业竞争承诺函》

### 结论

同业竞争问题已制定解决方案，预计申报前完成整改。
```
