# A股行业研究 (a-share-sector)

## 数据源

```bash
SCRIPTS="$SKILLS_ROOT/cn-stock-data/scripts"

# 行业内主要公司行情（市值、PE、PB 等）
python "$SCRIPTS/cn_stock_data.py" quote --code [CODE1],[CODE2],...

# 各公司财务指标
python "$SCRIPTS/cn_stock_data.py" finance --code [CODE]

# 各公司近 1 年日线（行业趋势对比）
python "$SCRIPTS/cn_stock_data.py" kline --code [CODE] --freq daily --start [1年前日期]

# 北向资金（行业层面资金偏好）
python "$SCRIPTS/cn_stock_data.py" north_flow
```

补充：通过 web 搜索获取行业政策、市场规模、产业链结构、行业新闻。

## Workflow

### Step 1: 明确研究范围

- 确定行业/板块（用户指定或通过 web 搜索确认申万/中信行业分类）
- 选取行业内代表性上市公司（龙头 + 二线，通常 8-15 家）
- 选择标准：市值排名靠前、业务代表性强、避免 ST 公司

### Step 2: 行业全景

通过 web 搜索获取：

- 市场规模与增速（TAM、近 3 年 CAGR）
- 行业生命周期位置（导入期/成长期/成熟期/衰退期）
- 产业链结构（上游原材料 → 中游制造 → 下游应用/消费）
- 政策环境（近期重大政策、监管动向）
- 行业核心驱动力（需求端 vs 供给端）

### Step 3: 竞争格局

通过 cn-stock-data 获取各公司数据，构建竞争格局表：

| 公司 | 市值(亿) | 收入(亿) | 市占率(%) | 毛利率(%) | ROE(%) | 核心优势 |
|------|---------|---------|----------|----------|--------|---------|
| ...  | ...     | ...     | ...      | ...      | ...    | ...     |

分析要点：
- 行业集中度（CR3/CR5/HHI）
- 竞争壁垒（品牌/规模/技术/渠道/牌照）
- 近年格局变化趋势

### Step 4: 财务对比

从 cn-stock-data 获取并对比行业内公司的关键财务指标：

核心对比表：

| 公司 | 营收增速(%) | 净利润增速(%) | 毛利率(%) | 净利率(%) | ROE(%) | 资产负债率(%) | PE(TTM) | PB |
|------|-----------|-------------|----------|----------|--------|-------------|---------|-----|
| ...  | ...       | ...         | ...      | ...      | ...    | ...         | ...     | ... |
| **行业中位数** | ... | ... | ... | ... | ... | ... | ... | ... |

标注行业中位数，识别优于/弱于中位数的公司。

### Step 5: 估值分析

- 行业整体估值水平（PE/PB 中位数、分位数）
- 估值历史分位（当前处于近 3/5 年的什么位置 — 需 web 搜索）
- 个股估值分散度
- 行业 vs 大盘估值溢价/折价

### Step 6: 输出

根据风格要求输出（默认 formal）。参考 `references/sector-template.md` 中的模板。

## 风格说明

| 维度 | formal（券商行业报告） | brief（个人行业笔记） |
|------|-------------------|-------------------|
| 篇幅 | 5-10 页 | 1-2 页 |
| 行业全景 | 完整（市场规模+产业链+政策+驱动力） | 精简（1-2 段核心逻辑） |
| 竞争格局表 | 完整（10+ 公司，含市占率） | 精简（Top 5 龙头） |
| 财务对比 | 10+ 列指标 | 5-6 列核心指标 |
| 估值 | 详细（含历史分位） | 当前水平 + 一句判断 |
| 投资建议 | 客观描述机会与风险 | 可加个人观点 |
| 免责声明 | 需要 | 不需要 |

## 关键规则

1. **数据+逻辑驱动**：所有观点必须有数据支撑，不做空泛定性判断
2. **行业分类标准**：优先使用申万行业分类（一级→二级→三级），web 搜索确认
3. **时效性标注**：标注数据截止日期，行业数据标注来源
4. **不给具体买卖建议**：formal 模式只分析行业趋势和估值位置，不推荐个股
5. **与其他 skill 联动**：报告末尾可建议用 a-share-comps 做更细致的估值对标，或用 a-share-stock-screen 在该行业内筛选
