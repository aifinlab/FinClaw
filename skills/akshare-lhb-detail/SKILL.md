## 功能列表

### 1. 每日龙虎榜
- **功能描述**: 获取当日龙虎榜数据
- **输入参数**: 日期（可选，默认今日）
- **输出格式**: Markdown表格
- **数据源**: AkShare-交易所
- **数据时效**: 日频（收盘后）
- **使用示例**:
  ```bash
  python scripts/lhb_daily.py
  python scripts/lhb_daily.py --date 20260318
  ```

### 2. 个股龙虎榜历史
- **功能描述**: 查询特定股票的历史龙虎榜记录
- **输入参数**: 股票代码
- **输出格式**: Markdown表格
- **数据源**: AkShare
- **数据时效**: 历史数据
- **使用示例**:
  ```bash
  python scripts/lhb_stock.py --code 600519
  ```

### 3. 营业部追踪
- **功能描述**: 追踪特定营业部的交易动向
- **输入参数**: 营业部名称
- **输出格式**: Markdown表格
- **数据源**: AkShare
- **数据时效**: 日频
- **使用示例**:
  ```bash
  python scripts/lhb_dealer.py --name "中信证券上海溧阳路"
  ```

### 4. 机构席位分析
- **功能描述**: 分析机构专用席位的买卖动向
- **输入参数**: 时间范围
- **输出格式**: Markdown表格
- **数据源**: AkShare
- **数据时效**: 日频
- **使用示例**:
  ```bash
  python scripts/lhb_institution.py --days 30
  ```

## 龙虎榜解读

### 席位类型
| 类型 | 说明 | 信号意义 |
|:---|:---|:---|
| **机构专用** | 公募基金、保险、QFII | 中长期资金，信号较稳 |
| **游资营业部** | 知名游资席位 | 短线资金，波动大 |
| **普通营业部** | 散户/小机构 | 参考意义有限 |

### 买卖方向解读
| 情形 | 解读 |
|:---|:---|
| 机构大买 | 看好中长期，可能上涨 |
| 机构大卖 | 看空或调仓，可能下跌 |
| 游资大买 | 短线炒作，波动加大 |
| 游资大卖 | 获利了结，短期承压 |
| 机构+游资同买 | 强烈看多信号 |
| 机构买+游资卖 | 分歧，观望 |
