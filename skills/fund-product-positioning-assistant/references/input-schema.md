# 输入字段建议

## 最低必需材料

- 产品合同与募集材料
- 策略与基准信息
- 目标客群和渠道定位
- 同类产品对照

## 可选增强材料

- 销售反馈
- 存量客户画像
- 公司产品线规划

## 建议输入结构

~~~json
{
  "task_name": "产品定位分析助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
