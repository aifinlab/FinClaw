# 输出结构建议

建议最终输出包含以下模块：

- 定位摘要
- 目标客群与渠道
- 差异化卖点
- 冲突与风险
- 优化建议

## 建议结构化结果

~~~json
{
  "task_name": "产品定位分析助手",
  "summary": "",
  "key_findings": [],
  "risks": [],
  "missing_information": [],
  "recommended_actions": [],
  "appendix": {}
}
~~~

## 表达要求

- 先写结论摘要，再写证据链
- 区分已确认事实、解释性判断和待补充事项
- 对外材料相关内容必须保留复核边界
