# Global Investor Suitability Checker

## 概述

这是一个基于**公网可查数据**的 Skill，用于对**某家上市企业对应股票的投资者适当性**进行校验。Skill 先识别企业所属板块（主板 / 科创板 / 创业板 / 北交所），再结合公开法律法规与交易所规则，对输入的投资者画像进行适当性判断，并输出结论、缺口项与法律依据。

示例企业默认使用：**中芯国际（688981.SH）**。该公司在上海证券交易所科创板上市，适合作为科创板适当性校验样例。

---

## 数据来源

本 Skill 仅使用公网可查、可复核的信息：

1. **中国证监会（CSRC）**
   - 《证券期货投资者适当性管理办法》
   - 用于通用适当性原则、普通/专业投资者划分、经营机构适当性义务等法律依据

2. **上海证券交易所（SSE）**
   - 科创板投资者适当性页面/规则摘要
   - 用于科创板股票交易门槛与风险揭示要求

3. **深圳证券交易所（SZSE）**
   - 《深圳证券交易所创业板投资者适当性管理实施办法（2020年修订）》
   - 用于创业板交易门槛与风险揭示要求

4. **北京证券交易所（BSE）**
   - 北交所公开发行与上市相关公开披露材料、公开规则口径
   - 用于北交所“合格投资者/适当性”规则映射（如需生产部署，建议定期复核官方最新规则）

5. **上市公司所属交易所公开页面**
   - 例如上交所/深交所/北交所公司详情页
   - 用于识别证券代码、上市板块、上市日期、证券简称等基础事实

## 功能

### 1) 企业板块识别
根据股票代码与交易所公开页面，识别目标企业属于：
- 上交所主板
- 上交所科创板
- 深交所主板
- 深交所创业板
- 北交所

### 2) 法规查询与规则映射
将不同板块映射到公开可查的适当性规则，例如：
- **科创板**：个人投资者通常需满足开通前 20 个交易日日均资产不低于 50 万元、参与证券交易满 24 个月，并签署风险揭示书。
- **创业板**：个人投资者通常需满足开通前 20 个交易日日均资产不低于 10 万元、参与证券交易满 24 个月，并签署风险揭示书。
- **主板**：重点核验风险揭示、风险承受能力匹配与普通/专业投资者一般性适当性要求。
- **北交所**：重点核验合格投资者门槛与风险揭示要求。

### 3) 投资者画像校验
支持根据投资者画像进行适当性校验。输入字段包括：
- investor_type：`individual` / `institution`
- avg_assets_20d：申请权限开通前 20 个交易日日均资产（人民币）
- trading_experience_months：证券交易经验（月）
- signed_risk_disclosure：是否已签署风险揭示书
- risk_tolerance：`low` / `medium` / `high`
- is_professional_investor：是否已被认定为专业投资者

### 4) 输出可解释结论
输出内容包括：
- 适当性结论：`PASS` / `FAIL` / `REVIEW`
- 不满足项列表
- 规则依据列表
- 风险提示
- 适配的板块与证券信息

## 使用示例

### 示例 1：对中芯国际（科创板）做适当性校验

```bash
python script/run_demo.py \
  --symbol 688981 \
  --investor-type individual \
  --avg-assets-20d 600000 \
  --trading-experience-months 36 \
  --signed-risk-disclosure true \
  --risk-tolerance high
```

预期：
- 识别为 **科创板**
- 若资产、经验、风险揭示满足要求，则返回 `PASS`

### 示例 2：资产不足导致不通过

```bash
python script/run_demo.py \
  --symbol 688981 \
  --investor-type individual \
  --avg-assets-20d 120000 \
  --trading-experience-months 36 \
  --signed-risk-disclosure true \
  --risk-tolerance high
```

预期：
- 输出 `FAIL`
- 明确提示“科创板 20 个交易日日均资产门槛未达到 50 万元”

### 示例 3：创业板校验

```bash
python script/run_demo.py \
  --symbol 300750 \
  --investor-type individual \
  --avg-assets-20d 120000 \
  --trading-experience-months 30 \
  --signed-risk-disclosure true \
  --risk-tolerance medium
```

预期：
- 识别为 **创业板**
- 满足创业板典型门槛时返回 `PASS`

## 交易说明

1. **本 Skill 只做规则校验，不直接下单，不构成投资建议。**
2. **适当性通过 ≠ 投资一定合适。** 适当性仅表示从公开规则角度看，投资者画像与某类证券/交易权限的门槛较为匹配。
3. 实际业务中，券商/经营机构还会结合：
   - 投资者分类
   - 风险测评结果
   - 产品或服务风险等级
   - 冷静期/回访/双录等内部流程
4. 生产环境使用前，应定期复核最新法律法规与交易所业务规则。
5. 对北交所、可转债、权限转签、机构客户等复杂情形，建议由合规/法务做二次复核。
