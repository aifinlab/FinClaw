# 单一证券风险评估

## 一、风险评估维度

### 1.1 基本面风险
| 因素 | 评估要点 |
|------|----------|
| 估值 | PE、PB 是否合理 |
| 盈利 | 盈利是否稳定 |
| 成长 | 成长是否可持续 |
| 财务 | 财务是否健康 |

### 1.2 技术面风险
| 因素 | 评估要点 |
|------|----------|
| 趋势 | 上升/下降/盘整 |
| 位置 | 高位/低位 |
| 成交量 | 放量/缩量 |
| 指标 | 超买/超卖 |

### 1.3 流动性风险
| 因素 | 评估要点 |
|------|----------|
| 日均成交 | >1 亿/5000 万 -1 亿/<5000 万 |
| 换手率 | 高/中/低 |
| 买卖价差 | 大/中/小 |
| 冲击成本 | 高/中/低 |

## 二、风险等级划分

| 等级 | 特征 | 建议 |
|------|------|------|
| 低风险 | 蓝筹股、盈利稳定、流动性好 | 可适度配置 |
| 中风险 | 成长股、盈利波动、流动性中 | 控制仓位 |
| 高风险 | 题材股、盈利不稳定、流动性差 | 谨慎参与 |
| 极高风险 | ST 股、亏损股、流动性差 | 避免参与 |

## 三、风险预警信号

### 3.1 基本面预警
- 业绩大幅下滑
- 财务造假嫌疑
- 高管频繁离职
- 重大诉讼仲裁
- 核心资产纠纷

### 3.2 技术面预警
- 跌破重要支撑
- 连续大跌
- 放量下跌
- 技术指标背离

### 3.3 流动性预警
- 日均成交大幅下降
- 换手率持续低迷
- 买卖价差扩大
- 停牌风险

## 四、风险应对措施

### 4.1 降低仓位
- 减仓至安全水平
- 分批减仓
- 设置止损

### 4.2 对冲风险
- 融券对冲
- 期权对冲
- 股指期货对冲

### 4.3 及时止损
- 设置止损位
- 严格执行
- 避免情绪化

## 五、单一证券持仓建议

| 风险等级 | 最大持仓占比 | 止损位 |
|----------|--------------|--------|
| 低风险 | <30% | -15% |
| 中风险 | <20% | -10% |
| 高风险 | <10% | -8% |
| 极高风险 | 0% | - |
