# 问询问题分类体系

## 一、一级分类

| 分类代码 | 分类名称 | 说明 |
|----------|----------|------|
| B | 业务类 | 商业模式、市场、竞争、技术等 |
| F | 财务类 | 收入、成本、利润、资产、负债等 |
| L | 法律类 | 股权、资产、合同、诉讼、合规等 |
| M | 募投类 | 项目必要性、可行性、效益等 |
| I | 信息披露类 | 披露完整性、风险因素等 |
| Q | 发行条件类 | 主体资格、独立性、规范性等 |

## 二、二级分类

### 2.1 业务类 (B)
| 代码 | 分类 | 典型问题 |
|------|------|----------|
| B01 | 商业模式 | 盈利模式、收入来源 |
| B02 | 市场竞争 | 市场份额、竞争对手 |
| B03 | 客户供应商 | 集中度、依赖性 |
| B04 | 技术实力 | 核心技术、专利、研发 |
| B05 | 业务合规 | 资质、许可、处罚 |

### 2.2 财务类 (F)
| 代码 | 分类 | 典型问题 |
|------|------|----------|
| F01 | 收入确认 | 收入确认政策、时点 |
| F02 | 成本核算 | 成本、毛利率 |
| F03 | 资产质量 | 应收、存货、减值 |
| F04 | 关联交易 | 关联方、公允性 |
| F05 | 财务内控 | 内控、不规范情形 |

### 2.3 法律类 (L)
| 代码 | 分类 | 典型问题 |
|------|------|----------|
| L01 | 历史沿革 | 股权变更、出资 |
| L02 | 资产权属 | 房产、土地、专利 |
| L03 | 重大合同 | 合同履约、风险 |
| L04 | 诉讼仲裁 | 未决诉讼、纠纷 |
| L05 | 业务资质 | 资质、许可 |

## 三、问题标签体系

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| 核心问题 | 影响发行条件 | 重点回复 |
| 财务数据 | 涉及财务数据 | 需会计师核查 |
| 法律事项 | 涉及法律事项 | 需律师核查 |
| 业务数据 | 涉及业务数据 | 需保荐人核查 |
| 风险揭示 | 涉及风险 | 需充分披露 |

## 四、优先级分类

| 优先级 | 标准 | 响应时间 |
|--------|------|----------|
| 高 | 涉及发行条件、重大风险 | 24 小时内 |
| 中 | 重要事项、需详细论证 | 48 小时内 |
| 低 | 一般事项、补充说明 | 72 小时内 |
