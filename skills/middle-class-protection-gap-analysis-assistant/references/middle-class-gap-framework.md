# 中产客户保障缺口识别框架

## 一、分析主线

围绕以下主线展开：

1. 家庭现金流是否具备抗风险能力
2. 主要收入承担者是否有足够的责任保障
3. 重大疾病和医疗支出是否会冲击当前生活方式
4. 长期目标是否建立在基础保障已打底的前提下
5. 家庭资产增长与保障配置是否同步

## 二、中产客户画像维度

### 1. 家庭成员结构

- 家庭人数
- 夫妻结构
- 子女人数与年龄阶段
- 是否需要赡养老人
- 是否存在单亲、再婚、多代同堂等特殊结构

### 2. 家庭阶段

- 新婚起步期
- 育儿成长阶段
- 事业上升与责任高峰期
- 资产积累阶段
- 临退休过渡期

### 3. 财务责任维度

- 家庭总收入及主要来源
- 单收入或双收入结构
- 房贷、车贷、消费贷、经营负债
- 子女教育责任
- 老人赡养责任
- 家庭生活成本与品质维持压力
- 养老储备压力

## 三、保障盘点维度

### 1. 家庭收入保障

重点看：

- 家庭主要收入承担者是否有足够的身故责任保障
- 收入中断后家庭能否维持现有生活方式
- 是否过度依赖工资收入、社保或团险

### 2. 健康保障层次

重点看：

- 重疾保障是否兼顾治疗费与收入补偿
- 医疗保障是否能覆盖大额住院、特药和补充医疗需求
- 是否存在“有一点，但层次不够”的情况

### 3. 资产安全与长期稳定

重点看：

- 家庭资产积累是否伴随相应的责任保障
- 是否过早偏向储蓄型安排，忽视风险型保障
- 是否存在资产增长快于保障配置的情况

## 四、七大缺口识别逻辑

### 1. 家庭收入保障缺口

出现以下任一情况，可优先提示缺口：

- 主要收入承担者缺少寿险或责任保障明显偏弱
- 房贷、教育、赡养责任较重，但家庭现金流保护不足
- 保障无法支撑现有生活成本和家庭稳定需求

### 2. 重疾保障缺口

出现以下任一情况，可提示缺口：

- 主要经济支柱没有重疾保障或保额明显偏弱
- 重疾保障只覆盖治疗费用，未体现收入补偿
- 家庭处于责任高峰期，但康复期现金流保障不足

### 3. 医疗保障缺口

出现以下任一情况，可提示缺口：

- 无百万医疗或住院医疗安排
- 仅有医保和团险，商业医疗补充不足
- 生活品质要求较高，但医疗保障层级偏低

### 4. 意外保障缺口

出现以下任一情况，可提示缺口：

- 家庭支柱没有独立意外保障
- 经常出行、驾驶、差旅或高频商务活动，但保障薄弱
- 家庭成员日常意外保障基础覆盖不足

### 5. 子女责任保障缺口

出现以下任一情况，可提示缺口：

- 子女教育责任较重，但承担责任的成人保障未打底
- 子女缺少基础医疗和意外保障
- 过早做教育储备，挤占基础保障预算

### 6. 养老或长期规划缺口

出现以下任一情况，可提示缺口：

- 家庭长期目标明确，但缺乏稳定的保障承接逻辑
- 已开始储蓄或养老安排，但基础健康与责任保障不完整
- 长期规划目标与当前预算承接能力不匹配

### 7. 资产安全与责任匹配缺口

出现以下任一情况，可提示缺口：

- 家庭资产在增长，但核心风险仍主要靠自留
- 资产配置和保险配置脱节
- 对生活稳定性要求高，但风险隔离安排不足

## 五、结论写法建议

每个缺口结论尽量包含以下元素：

- 当前状态
- 缺口判断
- 主要依据
- 风险影响
- 是否需要进一步确认

推荐句式：

- 基于当前收入与负债结构，家庭现金流高度依赖主要经济支柱，但已知身故责任保障信息不足，初步判断家庭收入保障存在明显缺口。
- 基于现有保单配置和中产家庭生活成本压力，客户虽具备部分健康保障，但重疾后的收入补偿与康复期支出承接能力仍偏弱。
- 基于当前保障顺序，家庭更适合先补足核心风险保障，再考虑教育、养老和资产稳定类长期安排。
