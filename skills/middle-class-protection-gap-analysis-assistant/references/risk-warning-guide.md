# 中产客户保障分析风险提示与人工复核边界

## 一、常见风险提示方向

### 1. 现金流安全保障不足

常用表达：

- 家庭收入水平虽具备一定基础，但对主要经济支柱依赖较强，一旦发生身故、重疾或长期失能，家庭现金流和生活品质可能明显承压。

### 2. 医疗与重疾层次错配

常用表达：

- 当前已知保障中，医疗费用补偿和重疾收入补偿之间衔接不足，难以完整承接中产家庭的健康风险冲击。

### 3. 责任与保障不匹配

常用表达：

- 家庭承担房贷、教育和赡养等多重责任，但现有保障强度与责任规模不完全匹配。

### 4. 保障顺序偏移

常用表达：

- 当前保障结构更偏向储蓄或长期安排，基础风险保障尚未完全打底，存在顺序错位问题。

### 5. 资产增长快于保障配置

常用表达：

- 家庭资产和生活标准在提升，但风险转移安排未同步跟进，整体抗风险能力可能被高估。

## 二、人工复核触发条件

出现以下任一情况，必须明确提示人工复核：

- 家庭年收入、负债、预算信息缺失较多
- 已有保单保额、期限、责任信息不清
- 需要做精确保额测算
- 需要直接下具体产品购买结论
- 涉及高保额寿险、长期年金、资产安全安排、税务法律问题
- 客户明显属于高净值复杂规划场景

## 三、标准边界表述

推荐直接复用以下句式：

- 以下分析仅用于中产客户保障诊断和缺口识别，不构成最终产品购买、承保、授信、投资、税务或合规结论。
- 涉及具体产品配置、保额测算、长期缴费安排和高风险金融决策的内容，需结合完整资料由人工进一步复核。
- 若客户已进入高净值复杂规划场景，当前分析仅覆盖中产家庭保障结构，不延展至家族传承、税务筹划或家族信托安排。
