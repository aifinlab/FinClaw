# 输入字段建议

## 最低必需材料

- 客户持仓和交易行为
- 客户分层与风险等级
- 产品标签和营销主题
- 近期市场或活动事件

## 可选增强材料

- 渠道触达记录
- 历史成交偏好
- 活动资源位信息

## 建议输入结构

~~~json
{
  "task_name": "销售机会识别助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
