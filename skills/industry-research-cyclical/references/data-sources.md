# 周期行业数据源清单

## 官方数据

| 数据源 | 网址 | 数据类型 |
|--------|------|----------|
| 统计局 | www.stats.gov.cn | 产量、价格指数 |
| 海关总署 | www.customs.gov.cn | 进出口数据 |
| 发改委 | www.ndrc.gov.cn | 价格监测、产能政策 |

## 行业协会

| 协会 | 数据类型 |
|------|----------|
| 钢铁工业协会 | 钢铁产量、库存 |
| 煤炭工业协会 | 煤炭产量、库存 |
| 有色金属工业协会 | 有色产量、库存 |
| 石油和化学工业联合会 | 化工产量、价格 |
