# IPO 材料初审助手 - 募投项目版

## 功能概述

本技能用于检查 IPO 募投项目的合规性、可行性及信息披露的完整性，确保募投项目符合监管要求，具备实施的可行性。

## 适用场景

- 募投项目立项阶段
- 招股书募投章节撰写
- 申报材料提交前检查
- 问询回复准备

## 核心检查内容

### 一、募投项目基本信息

| 检查项 | 要求 | 检查方法 |
|--------|------|----------|
| 项目数量 | 原则上不超过 5 个 | 统计项目数量 |
| 项目类型 | 与主营业务相关 | 比对业务范围 |
| 投资总额 | 合理、可承受 | 财务分析 |
| 建设周期 | 明确、合理 | 进度计划检查 |

### 二、合规性检查

#### 1. 产业政策合规

- [ ] 是否符合国家产业政策
- [ ] 是否属于限制/禁止类行业
- [ ] 是否符合行业发展规划

#### 2. 项目审批/备案

- [ ] 项目备案/核准文件
- [ ] 环评批复文件
- [ ] 用地预审意见
- [ ] 能评批复（如适用）
- [ ] 安评批复（如适用）

#### 3. 土地合规

- [ ] 用地性质符合规划
- [ ] 土地使用权取得方式合法
- [ ] 土地费用计入投资总额

### 三、可行性分析

#### 1. 市场可行性

- [ ] 市场容量分析
- [ ] 竞争格局分析
- [ ] 市场开拓计划

#### 2. 技术可行性

- [ ] 技术来源及先进性
- [ ] 技术团队能力
- [ ] 技术风险分析

#### 3. 财务可行性

- [ ] 投资回收期
- [ ] 内部收益率（IRR）
- [ ] 净现值（NPV）
- [ ] 敏感性分析

### 四、信息披露检查

#### 招股书披露内容

- [ ] 项目基本情况
- [ ] 投资概算
- [ ] 建设进度
- [ ] 效益分析
- [ ] 风险分析

## 使用流程

### 步骤 1：准备项目资料

收集募投项目相关资料：
- 可行性研究报告
- 项目备案/核准文件
- 环评批复
- 用地文件

### 步骤 2：运行检查脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\4. IPO 材料初审助手 - 募投项目版"
python scripts/check_investment_projects.py --input <项目资料路径> --output <检查结果路径>
```

### 步骤 3：查看检查结果

检查报告将包含：
- 合规性问题
- 可行性分析
- 信息披露完整性

### 步骤 4：整改完善

根据检查结果补充材料或完善披露

## 常见问题

1. **项目与主业不相关**：需调整项目或说明协同效应
2. **审批手续不全**：需补办相关审批
3. **投资金额过大**：需说明合理性和消化能力
4. **效益预测过于乐观**：需调整预测或增加风险提示
