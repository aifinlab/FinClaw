# 高频微观结构/盘口深度分析助手参考指南

## 一、微观结构理论

### 1.1 经典模型
- Kyle (1985): 知情交易者通过订单流传递信息，做市商通过 Lambda 调整价格
- Glosten-Milgrom (1985): 买卖价差源于逆向选择成本
- Easley-O'Hara PIN 模型: 基于信息事件概率和知情/非知情交易者到达率

### 1.2 A股微结构特征
- T+1 制度：日内无法卖出当日买入股票，影响日内流动性供给
- 涨跌停板 (±10%/±20%): 极端行情下流动性枯竭
- 散户占比高 (交易量 70%+): 噪声交易多，PIN 基准偏低
- 集合竞价机制: 9:15-9:25 开盘集合竞价，14:57-15:00 收盘集合竞价


## 二、实用指标计算

### 2.1 OIR 计算
- OIR = (Σ Bid_Volume - Σ Ask_Volume) / (Σ Bid_Volume + Σ Ask_Volume)
- 取 5 档或 10 档加权，越靠近中间价权重越高
- OIR > 0 买盘占优，OIR < 0 卖盘占优，|OIR| > 0.3 为显著不平衡

### 2.2 VPIN 计算
- 将成交量等分为 N 个 bucket (如每 bucket = 日均量/50)
- 每个 bucket 内用 BVC 方法分类买卖主导
- VPIN = Σ|V_buy - V_sell| / (N * V_bucket)
- VPIN 上升预示波动率即将增大
