# IPO 材料初审助手 - 文件清单版

## 功能概述

本技能用于对 IPO 申报材料进行初步审查，重点检查文件清单的完整性。根据证监会、交易所的最新要求，系统性地核对企业提交的文件是否齐全，识别缺失文件并生成补正建议。

## 适用场景

- IPO 项目申报前的材料自查
- 保荐机构内部质量控制
- 律师事务所文件齐备性检查
- 会计师事务所申报材料核对

## 使用流程

### 步骤 1：准备材料清单

收集企业提交的所有文件，建立文件清单表格，包含：
- 文件名称
- 文件类型
- 提交日期
- 版本号
- 页码范围

### 步骤 2：运行检查脚本

```bash
# 进入技能目录
cd "C:\Users\51994\Desktop\skills\证券 2\1. IPO 材料初审助手 - 文件清单版"

# 运行检查脚本
python scripts/check_file_list.py --input <文件清单 Excel 路径> --output <检查结果路径>
```

### 步骤 3：查看检查结果

检查报告将包含：
- ✅ 已提供的文件
- ❌ 缺失的文件
- ⚠️ 需要补充的文件
- 📝 备注说明

### 步骤 4：生成补正建议

根据缺失文件清单，生成详细的补正建议，包括：
- 缺失文件名称
- 文件要求说明
- 获取途径建议
- 预计办理时间

## 注意事项

1. **文件版本**：确保所有文件为最新版本，特别是公司章程、营业执照等可能变更的文件
2. **文件有效期**：注意部分文件的有效期，如审计报告有效期为 6 个月
3. **文件盖章**：所有文件需加盖相应公章，复印件需注明"与原件一致"
4. **文件扫描**：电子版文件应清晰可读，建议采用 PDF 格式
5. **保密处理**：涉及商业秘密的内容可按规定申请豁免披露
