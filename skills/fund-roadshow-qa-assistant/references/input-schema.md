# 输入字段建议

## 最低必需材料

- 产品资料与经理资料
- 路演主题和目标客群
- 历史高频问题
- 合规限制

## 可选增强材料

- 市场热点
- 近期回撤或事件
- 渠道关注点

## 建议输入结构

~~~json
{
  "task_name": "路演问答助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
