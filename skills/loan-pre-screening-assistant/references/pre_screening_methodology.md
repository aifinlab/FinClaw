# 个贷预审方法说明

个贷预审的核心目的，是在正式审批前快速识别“是否具备继续推进的基础条件”。因此，分析重点不是追求绝对精确，而是优先识别：

1. 是否满足最低资料要求
2. 是否存在明显硬性障碍
3. 是否存在高优先级风险信号
4. 是否需要补件或人工重点复核

建议从以下顺序展开：

一、先检查资料是否具备可判断性。若关键信息严重缺失，应先停止结论输出，转为补件。

二、再看基础准入与身份真实性。若年龄、证件、居住信息、联系方式存在异常，要优先标注。

三、然后审看职业收入与资产负债。这里重点看稳定性、连续性与相互印证关系，而不是单看某一个数值。

四、最后看征信和外部风险。若存在较多逾期、查询异常、授信利用率偏高或涉诉等，应显著影响预审等级。

五、输出结论时，必须把“已知、未知、冲突、建议”四类内容拆开写。
