# 政信主体尽调指标说明

## 核心指标

- `debt_to_revenue`：总债务/财政收入。用于衡量债务负担。
- `debt_to_fiscal_expenditure`：总债务/财政支出。用于观察财政承压。
- `debt_service_coverage`：偿债保障倍数（DSCR）。
- `cash_to_short_debt`：现金储备/短期债务。衡量短期流动性。
- `self_sufficiency_ratio`：财政自给率。
- `revenue_growth`：财政收入同比增速。

## 文本信号

- 政策支持关键词：用于识别增信与协调能力信号。
- 风险事件关键词：用于识别舆情与信用负面事件。

## 使用边界

- 指标来源可能存在口径差异，需结合原始报表核验。
- 自动评分仅用于尽调初筛，不可替代人工决策。
