# 信号回测参考指南
## 核心指标
| 指标 | 公式 | 合格线 |
|------|------|--------|
| 年化收益 | (终值/初值)^(252/天数)-1 | >15% |
| 夏普比率 | (R-Rf)/σ年化 | >1.5 |
| 最大回撤 | max(peak-trough)/peak | <25% |
| 胜率 | 盈利次数/总次数 | >45% |
| 盈亏比 | 平均盈利/平均亏损 | >1.5 |
| Calmar | 年化收益/最大回撤 | >1.0 |

## A股交易成本
- 佣金: 万2.5(双边)
- 印花税: 千1(卖出)
- 合计: 约0.15%单边

## 过拟合检测
- 样本内 vs 样本外夏普差 < 50%
- 参数空间热力图平滑（非尖峰）
- 信号数量足够（>30次）
- 多标的验证一致性
