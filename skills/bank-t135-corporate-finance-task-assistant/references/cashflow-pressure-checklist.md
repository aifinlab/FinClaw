# 现金流压力诊断检查清单（快用版）

## 1. 先把“现象”说清楚（口径与基线）

- 经营现金流净额是否为负？持续多少期？是季节性还是趋势性？
- 口径是否变化：合并/单体、是否剔除票据/保理、报表重分类、一次性项目
- 对比基线：同比/环比/预算/行业平均（如有）

## 2. 利润与现金流是否背离（先抓最大矛盾）

- 净利润为正但经营现金流为负：重点看营运资本占用与非现金收益
- 净利润下滑且经营现金流下滑：同时关注盈利能力与周转
- 经营现金流为正但仍“缺钱”：重点看投资现金流/筹资现金流与到期结构

## 3. 现金被谁占用（营运资本三件套）

- 应收：账龄拉长、回款集中度、票据结构、退货与折扣
- 存货：库龄、跌价、在制品与产成品结构、产销匹配
- 预付/其他应收：预付异常、关联方占用、往来款长期挂账

## 4. 现金被什么挤出（投资/筹资/表外）

- CAPEX：扩产/技改/在建工程资金占用与完工进度
- 对外支出：分红/抽资、对外投资、股权收购、对外借款
- 表外与或有：对外担保、代偿、诉讼保全、受限资金上升

## 5. 最后落到行动

- 优先补件：哪 3 份材料最能验证关键假设（应收明细/合同回款条款/到期表）
- 访谈路径：财务总监 -> 销售负责人 -> 采购/生产 -> 资金/融资
- 监测指标：回款率、应收账龄迁徙、存货周转、受限资金、短债覆盖
