# 现金流压力常见红旗（用于预警与升级）

- **利润为正、经营现金流持续为负**，且应收/存货/其他应收明显上升，提示利润质量与占用风险。
- **应收账龄整体后移**、期后回款弱、客户集中度上升，提示回款链条脆弱。
- **预付/其他应收异常增加**且交易对手疑似关联方，提示资金占用或利益输送风险。
- **受限资金显著上升**（保证金、冻结、保全等），提示资金调度空间收缩。
- **短债到期集中且续贷不确定**，提示流动性危机触发概率上升。
- **CAPEX/在建工程投入高强度**但产能去化/订单不足，提示“扩张挤兑现金”的结构性风险。
- **票据/保理/应收融资异常放大**且基础回款恶化，提示“用金融工具拖延现金压力”的倾向。

升级建议（满足任一条可考虑升级）：

- 出现代偿/冻结/诉讼保全等硬事件
- 资金链依赖短期滚续，且关键融资渠道出现收缩迹象
- 存量授信出现逾期、欠息、展期频发或压降要求
