# 尽调问题清单助手 - 再融资版

## 功能概述

本技能用于生成再融资（定增、配股、可转债等）尽职调查问题清单，重点关注前次募集资金使用、本次募资必要性及可行性。

## 适用场景

- 上市公司再融资项目
- 定增/配股/可转债尽调
- 募资必要性论证
- 反馈回复准备

## 尽调问题清单框架

### 一、公司基本情况

1. 公司基本信息
   - [ ] 公司设立及上市情况
   - [ ] 股权结构及控制关系
   - [ ] 控股股东及实际控制人
   - [ ] 子公司及参股公司

2. 规范运作
   - [ ] 三会运作情况
   - [ ] 内部控制情况
   - [ ] 信息披露情况
   - [ ] 承诺履行情况

### 二、前次募集资金使用

3. 前次募资基本情况
   - [ ] 募资时间、方式、金额
   - [ ] 募投项目情况
   - [ ] 募资到账时间
   - [ ] 验资情况

4. 前次募资使用进度
   - [ ] 投资进度
   - [ ] 建设进度
   - [ ] 达产情况
   - [ ] 效益实现情况

5. 前次募资变更情况
   - [ ] 用途变更
   - [ ] 实施方式变更
   - [ ] 实施地点变更
   - [ ] 决策程序

6. 前次募资结余情况
   - [ ] 结余金额
   - [ ] 结余原因
   - [ ] 后续安排

### 三、本次募集资金必要性

7. 融资必要性
   - [ ] 融资背景
   - [ ] 融资目的
   - [ ] 融资规模合理性
   - [ ] 融资时机

8. 业务发展规划
   - [ ] 发展战略
   - [ ] 经营目标
   - [ ] 业务规划
   - [ ] 产能规划

9. 资金需求分析
   - [ ] 资金缺口
   - [ ] 融资渠道
   - [ ] 融资成本
   - [ ] 股权融资优势

### 四、本次募投项目

10. 项目基本情况
    - [ ] 项目背景
    - [ ] 项目内容
    - [ ] 投资估算
    - [ ] 建设周期

11. 项目可行性
    - [ ] 市场可行性

12. 项目合规性

    - [ ] 其他审批

13. 项目效益
    - [ ] 收入预测
    - [ ] 成本预测
    - [ ] 利润预测
    - [ ] 投资回报

### 五、财务情况

14. 财务状况
    - [ ] 资产负债情况
    - [ ] 盈利能力

15. 财务合规
    - [ ] 会计政策

    - [ ] 成本核算

### 六、法律合规

16. 合规情况

17. 担保情况

    - [ ] 发行对象范围
    - [ ] 发行对象资格
    - [ ] 认购意向
    - [ ] 锁定期安排

    - [ ] 定价方式

## 使用流程

### 步骤 1：选择融资类型

选择再融资类型（定增/配股/可转债）

### 步骤 2：生成问题清单

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\15. 尽调问题清单助手 - 再融资版"
python scripts/generate_refinance_dd.py --type <融资类型> --output <输出路径>
```

### 步骤 3：查看问题清单

问题清单将按领域分类，包含具体问题

### 步骤 4：开展尽调

根据问题清单开展尽职调查
