# 回购公告解读输出模板

## 快速解读（200-300 字）

```markdown
【XX 公司回购公告解读】

回购金额：XX-XX 亿，占总市值 XX%

回购价格：上限 XX 元，较现价溢价 XX%

回购目的：[注销/股权激励/员工持股]

回购期限：XX 个月

回购力度：[大力度/中等/小力度]

投资判断：[利好程度 + 简要理由]
```
