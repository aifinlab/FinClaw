# 募资文件对比助手 - 招股书版

## 功能概述

本技能用于对比招股说明书中募集资金运用章节与可行性研究报告、备案文件等其他文件的一致性，确保信息披露准确。

## 适用场景

- 招股书募投章节撰写
- 申报材料提交前核对
- 反馈回复后的更新检查
- 多版本文件一致性验证

## 使用流程

### 步骤 1：准备对比文件

收集招股书、可研报告、备案文件等

### 步骤 2：提取关键数据

从各文件中提取募投项目关键数据

### 步骤 3：运行对比脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\11. 募资文件对比助手 - 招股书版"
python scripts/compare_prospectus.py --input <数据文件> --output <对比结果路径>
```

### 步骤 4：查看对比结果

对比报告将列出：
- 一致的项目
- 存在差异的项目
- 差异原因分析

### 步骤 5：核实修正

对差异项进行核实并修正
