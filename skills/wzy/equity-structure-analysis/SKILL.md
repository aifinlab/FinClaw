# 股权结构梳理助手

## 功能概述

本技能用于梳理公司股权结构，穿透核查最终持有人，识别股权代持、特殊权利安排等潜在问题，确保股权清晰稳定。

## 适用场景

- IPO 申报前股权核查
- 股东穿透核查
- 代持情况排查
- 特殊权利安排清理

## 核心梳理内容

### 一、股权结构图谱

```
                    实际控制人
                         │
                    控股股东
                         │
                    发行人
                   /   │   \
              股东 A  股东 B  股东 C
                │       │       │
            穿透核查  穿透核查  穿透核查
```

### 二、股东分类核查

#### 1. 自然人股东

- [ ] 身份信息核实
- [ ] 出资来源核查
- [ ] 关联关系核查
- [ ] 代持情况核查

#### 2. 法人股东

- [ ] 工商登记信息
- [ ] 股权结构穿透
- [ ] 实际控制人认定
- [ ] 关联关系核查

#### 3. 合伙企业股东

- [ ] 合伙协议
- [ ] 合伙人信息
- [ ] GP/LP 身份
- [ ] 穿透核查

#### 4. 资管产品/基金股东

- [ ] 产品备案信息
- [ ] 管理人信息
- [ ] 投资人信息
- [ ] 穿透核查

### 三、特殊事项核查

#### 1. 股权代持

**核查要点：**
- [ ] 代持协议
- [ ] 代持原因
- [ ] 代持解除
- [ ] 还原情况

**风险信号：**
- 股东身份与出资能力不匹配
- 股东与公司业务无关
- 股权转让价格异常
- 股东不参与公司治理

#### 2. 特殊权利安排

**核查要点：**
- [ ] 对赌协议
- [ ] 优先权条款
- [ ] 回购条款
- [ ] 一票否决权

**清理要求：**
- 申报前必须清理
- 清理协议签署
- 不存在恢复条款

#### 3. 员工持股平台

**核查要点：**
- [ ] 设立合规性
- [ ] 入股价格公允
- [ ] 锁定期安排
- [ ] 信息披露

### 四、股权稳定性分析

- 控股股东持股比例
- 实际控制人控制力
- 一致行动关系
- 锁定期安排

## 使用流程

### 步骤 1：收集股东资料

收集所有股东的身份信息、出资证明、协议文件等

### 步骤 2：绘制股权结构图

按层级绘制完整的股权结构图

### 步骤 3：运行梳理脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\8. 股权结构梳理助手"
python scripts/analyze_equity.py --input <股东数据> --output <分析结果路径>
```

### 步骤 4：查看分析结果

分析报告将包含：
- 股权结构图谱
- 股东穿透情况
- 潜在问题识别

### 步骤 5：整改规范

根据分析结果清理代持、解除特殊权利安排
