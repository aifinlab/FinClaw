# 市场波动陪伴内容模板库

## 大跌场景模板

### 单日大跌 (>2%)

**标题**: 市场波动提醒

**正文**:
```
今日市场大幅调整，沪深 300 下跌 X%，主要受 [原因] 影响。

理解大家的焦虑，市场波动是常态，历史上类似情况多次出现。

长期来看，[核心理念]。

建议：
1. 保持冷静，不要盲目操作
2. 检视持仓，优质资产值得坚守
3. 有闲钱可考虑分批布局

市场波动期，我们与您一起坚守。
```

### 连续下跌 (3 日累计>5%)

**标题**: 致投资者的一封信

**正文**:
```
最近市场连续调整，很多朋友感到焦虑，这很正常。

📊 市场情况
- 近 X 日累计下跌 X%
- 主要受 [因素] 影响
- 当前估值处于历史 [分位]

📚 历史参考
- 20XX 年类似情况：下跌 X% 后，X 个月内修复
- 20XX 年：[另一个案例]

💡 我们的建议
1. 检视持仓逻辑是否变化
2. 避免情绪化操作
3. 闲钱可分批布局

🌟 长期理念
投资是认知的变现，波动是收益的来源。
```

## 大涨场景模板

### 单日大涨 (>2%)

**标题**: 市场快速上涨提醒

**正文**:
```
今日市场大幅上涨 X%，恭喜持仓的朋友！

理性看待：
- 上涨受 [原因] 推动
- 短期涨幅较大，注意波动风险
- 不建议追高加仓

建议：
1. 持仓的继续持有
2. 想加仓的等待回调
3. 浮盈较多的可考虑部分止盈
```

## 常见下跌原因库

| 原因类型 | 说明 | 话术要点 |
|---------|------|---------|
| 政策利空 | 监管政策收紧 | "政策影响短期，长期逻辑未变" |
| 外围拖累 | 美股/港股下跌 | "A 股有独立逻辑，情绪影响为主" |
| 经济数据 | 数据不及预期 | "单月数据不改长期趋势" |
| 资金面 | 北向资金流出 | "资金流动正常，不必过度解读" |
| 技术调整 | 涨多回调 | "健康调整，有利于后续上涨" |
| 行业事件 | 黑天鹅事件 | "区分行业与个股，优质公司受影响有限" |

### 正常波动范围

- 单日涨跌±2%: 常见，无需过度反应
- 周度涨跌±5%: 正常波动范围
- 月度涨跌±10%: 较大波动，需关注
- 年度涨跌±20%: 正常年度波动

## 使用指南

1. **选择合适的模板** - 根据市场情况选择大跌/大涨模板
2. **填充具体数据** - 填入当日涨跌幅、原因等
3. **调整语气** - 根据客户群体调整语言风格
4. **添加个性化** - 加入客户持仓相关分析
5. **及时发送** - 市场波动后尽快发送
