# 募资文件对比助手 - 问询回复版

## 功能概述

本技能用于对比审核问询回复中关于募投项目的相关回复与招股说明书的一致性，确保信息披露前后一致。

## 适用场景

- 问询回复撰写
- 回复提交前核对
- 多轮问询回复一致性检查
- 招股书更新后的同步检查

## 使用流程

### 步骤 1：准备对比文件

收集招股书、问询回复等文件

### 步骤 2：提取关键信息

从各文件中提取募投项目相关信息

### 步骤 3：运行对比脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\12. 募资文件对比助手 - 问询回复版"
python scripts/compare_inquiry_response.py --input <数据文件> --output <对比结果路径>
```

### 步骤 4：查看对比结果

对比报告将列出：
- 一致的信息
- 存在差异的信息
- 差异原因及处理建议

### 步骤 5：核实修正

对差异项进行核实并修正
