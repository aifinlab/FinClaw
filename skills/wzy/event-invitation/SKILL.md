# 活动邀约文案助手

**核心定位**：设计客户活动邀约文案和话术，提高活动报名率和到场率。

**适用场景**：线下沙龙、投资策略会、客户答谢会、产品路演、投资者教育

---

## 活动类型

| 类型 | 目的 | 目标客户 | 规模 |
|------|------|---------|------|
| 投资策略会 | 传递观点、促成配置 | 高净值客户 | 20-50 人 |
| 产品路演 | 产品介绍、促成购买 | 意向客户 | 30-100 人 |
| 客户答谢会 | 维护关系、增强粘性 | 核心客户 | 50-200 人 |
| 投资者教育 | 知识普及、建立信任 | 大众客户 | 50-100 人 |
| 小型沙龙 | 深度交流、建立信任 | 重点客户 | 10-20 人 |

### 模板 B：电话邀约话术
```
【电话邀约话术】

【开场】
"您好 XX 总，我是 XX，最近还好吗？"

【寒暄】
"[简单寒暄]"

【来意】
"今天给您打电话，主要是有个活动想邀请您参加。"

【活动介绍】
"是 [活动名称]，时间是 [时间]，地点在 [地点]。"

【亮点传递】
"这次活动的亮点是 [亮点 1]、[亮点 2]。"

【邀约】
"觉得挺适合您的，给您留了个名额。"

【确认】
"您看您有时间参加吗？"

【异议处理】
- "没时间" → "理解您忙。不过这次机会难得，要不我让助理把资料发您看看？"
- "再考虑" → "好的，不过名额有限，您尽快决定哈。"

【结尾】
"好的，那我等您回复。有任何问题随时联系我。"
```

## 邀约技巧

### 文案要点
1. **标题吸引** - 用亮点吸引注意
2. **信息清晰** - 时间地点明确
3. **价值传递** - 说明参加收获
4. **紧迫感** - 名额有限、限时报名

### 电话要点
1. **先问候** - 建立良好氛围
2. **简洁介绍** - 1 分钟内说清
3. **强调价值** - 为什么值得参加
4. **促成报名** - 明确报名动作

### 提高报名率
1. **精准邀请** - 邀请合适客户
2. **价值传递** - 清晰说明收获
3. **稀缺性** - 名额有限
4. **便捷报名** - 简化报名流程

### 提高到场率
1. **多次提醒** - 活动前多次提醒
2. **确认意向** - 电话确认参加
3. **提供便利** - 交通指引、停车信息
4. **违约成本** - 收取定金或积分

## 高频场景专项

### 场景 1：投资策略会邀约
**目标客户**：高净值客户

**文案要点**：
- 强调嘉宾分量
- 突出独家观点
- VIP 专属邀请

**话术**：
> "XX 总，这次我们请到了 [嘉宾]，会分享 [独家观点]。这种机会不多，特意给您留了 VIP 席位。"

### 场景 2：产品路演邀约
**目标客户**：意向客户

**文案要点**：
- 产品介绍
- 限时优惠
- 现场答疑

**话术**：
> "XX 总，最近有个新产品发行，这次路演会详细介绍。现场还有专属优惠，有兴趣来听听吗？"

### 场景 3：客户答谢会邀约
**目标客户**：核心客户

**文案要点**：
- 感谢支持
- 回馈活动
- 高端体验

**话术**：
> "XX 总，感谢您一直以来的支持。我们准备了客户答谢会，有 [活动内容]，期待您的光临。"

### 场景 4：投资者教育活动
**目标客户**：大众客户

**文案要点**：
- 知识普及
- 免费参与
- 实用干货

**话术**：
> "XX 总，我们有个投资者教育活动，免费参加。会讲 [实用知识]，对投资很有帮助。有兴趣吗？"

## 活动预热

### 预热内容
1. **嘉宾剧透** - 提前透露嘉宾信息
2. **亮点预告** - 预告活动亮点
3. **倒计时** - 活动倒计时提醒
4. **报名播报** - 播报报名进度

### 预热节奏
- T-7 天：正式邀请
- T-5 天：嘉宾剧透
- T-3 天：亮点预告
- T-1 天：倒计时提醒

---

## 使用建议

1. **精准邀请** - 邀请合适的客户
2. **价值传递** - 清晰说明参加收获
3. **多次跟进** - 不轻易放弃
4. **数据追踪** - 追踪报名率、到场率
