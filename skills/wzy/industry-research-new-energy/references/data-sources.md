# 新能源行业数据源清单

## 官方数据

| 数据源 | 网址 | 数据类型 |
|--------|------|----------|
| 国家能源局 | www.nea.gov.cn | 装机数据、发电数据 |
| 工信部 | www.miit.gov.cn | 新能源汽车产销 |
| 中电联 | www.cec.org.cn | 电力工业统计 |

## 行业协会

| 协会 | 数据类型 |
|------|----------|
| 光伏业协会 | 光伏产业链价格、排产 |
| 风能协会 | 风电招标、装机 |
| 汽车工业协会 | 新能源车销量 |
| 化学与物理电源行业协会 | 锂电池产量 |

## 第三方数据

| 机构 | 数据类型 |
|------|----------|
| BloombergNEF | 全球新能源数据 |
| 鑫椤资讯 | 锂电产业链价格 |
| 我的钢铁网 | 光伏产业链价格 |
| S&P Global | 大宗商品价格 |

## 上市公司数据

| 数据源 | 数据类型 |
|--------|----------|
