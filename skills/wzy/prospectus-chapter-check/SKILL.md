# 招股书章节检查助手

## 功能概述

本技能用于全面检查招股说明书各章节的内容质量，确保信息披露完整、准确、合规，符合监管要求。

## 适用场景

- 招股书撰写过程中的质量检查
- 申报材料提交前的最终审核
- 反馈回复后的更新检查
- 多版本招股书的差异比对

## 检查内容框架

### 第一节 封面、书脊、目录、释义

**检查要点：**
- [ ] 封面信息完整（发行人名称、股票类型、发行股数、日期等）
- [ ] 书脊信息准确
- [ ] 目录与正文一致
- [ ] 释义清晰、准确

### 第二节 概览

**检查要点：**
- [ ] 发行人简介准确
- [ ] 控股股东及实际控制人披露完整
- [ ] 主要财务数据准确（与审计报告一致）
- [ ] 本次发行情况明确
- [ ] 募集资金用途清晰

### 第三节 本次发行概况

**检查要点：**
- [ ] 股票种类、面值准确
- [ ] 发行股数、定价方式明确
- [ ] 发行方式、对象符合规定
- [ ] 承销方式、费用披露完整

### 第四节 风险因素

**检查要点：**
- [ ] 风险分类清晰
- [ ] 风险揭示充分、有针对性
- [ ] 按重要性排序
- [ ] 避免模板化表述

### 第五节 发行人基本情况

**检查要点：**
- [ ] 基本信息准确（设立时间、注册资本等）
- [ ] 历史沿革清晰、完整
- [ ] 股权结构披露准确
- [ ] 控股股东及实际控制人认定准确
- [ ] 子公司及参股公司披露完整
- [ ] 员工情况披露准确

### 第六节 业务与技术

**检查要点：**
- [ ] 主营业务清晰
- [ ] 行业基本情况准确
- [ ] 竞争地位披露客观
- [ ] 产品/服务描述准确
- [ ] 业务模式清晰
- [ ] 采购、销售情况披露完整
- [ ] 资产情况披露准确
- [ ] 核心技术及研发能力披露充分

### 第七节 同业竞争与关联交易

**检查要点：**
- [ ] 同业竞争情况披露完整
- [ ] 关联方认定准确
- [ ] 关联交易披露完整、准确
- [ ] 规范措施有效

### 第八节 董事、监事、高级管理人员与核心技术人员

**检查要点：**
- [ ] 简历完整、准确
- [ ] 薪酬情况披露
- [ ] 兼职情况披露
- [ ] 持股情况披露

### 第九节 公司治理

**检查要点：**
- [ ] 三会制度健全
- [ ] 独立董事制度完善
- [ ] 内部控制有效

### 第十节 财务会计信息

**检查要点：**
- [ ] 财务报表完整、准确
- [ ] 会计政策披露充分
- [ ] 财务指标计算准确
- [ ] 与非经常性损益一致

### 第十一节 管理层讨论与分析

**检查要点：**
- [ ] 财务状况分析深入
- [ ] 盈利能力分析充分
- [ ] 现金流量分析准确
- [ ] 重大会计政策/估计分析充分

### 第十二节 业务发展目标

**检查要点：**
- [ ] 发展战略清晰
- [ ] 经营目标可行
- [ ] 发展计划具体
- [ ] 风险提示充分

### 第十三节 募集资金运用

**检查要点：**
- [ ] 募集资金总额明确
- [ ] 投资项目可行
- [ ] 效益分析合理
- [ ] 风险分析充分

### 第十四节 股利分配政策

**检查要点：**
- [ ] 现行政策披露
- [ ] 历史分红情况准确
- [ ] 发行后政策明确
- [ ] 回报规划合理

### 第十五节 其他重要事项

**检查要点：**
- [ ] 重大合同披露完整
- [ ] 对外担保披露
- [ ] 诉讼仲裁披露
- [ ] 行政处罚披露

### 第十六节 有关声明

**检查要点：**
- [ ] 各方声明完整
- [ ] 签字盖章齐全

### 第十七节 备查文件

**检查要点：**
- [ ] 备查文件清单完整
- [ ] 查阅方式明确

## 使用流程

### 步骤 1：准备招股书文本

将招股书转换为可分析的文本格式

### 步骤 2：运行检查脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\6. 招股书章节检查助手"
python scripts/check_prospectus.py --input <招股书文本> --output <检查结果路径>
```

### 步骤 3：查看检查结果

检查报告将列出各章节的检查结果及问题

### 步骤 4：整改完善

根据检查结果修改完善招股书

## 输出示例

```markdown
## 招股书章节检查报告

### 检查概况
- 检查章节：17 节
- 通过章节：14 节
- 待完善章节：3 节

### 待完善章节

1. **第六节 业务与技术**
   - 问题：缺少核心技术及研发小节
   - 建议：补充核心技术来源、先进性及研发投入情况

2. **第十一节 管理层讨论与分析**
   - 问题：现金流量分析不充分
   - 建议：补充经营活动现金流与净利润差异分析

3. **第十三节 募集资金运用**
   - 问题：效益分析假设条件不充分
   - 建议：补充效益测算的关键假设及敏感性分析
```
