# 医药行业数据源清单

## 官方数据

| 数据源 | 网址 | 数据类型 |
|--------|------|----------|
| NMPA 药监局 | www.nmpa.gov.cn | 药品/器械审批信息 |
| 医保局 | www.nhsa.gov.cn | 医保目录、集采文件 |
| CDE 药审中心 | www.cde.org.cn | 临床试验登记、审评进度 |
| 统计局 | www.stats.gov.cn | 医药工业数据 |

## 第三方数据库

| 数据库 | 数据类型 | 备注 |
|--------|----------|------|
| 医药魔方 | 药品销售、竞争格局 | 付费 |
| 米内网 | 医院/药店终端销售 | 付费 |
| Insight 数据库 | 全球研发管线 | 付费 |
| 药智网 | 药品注册、审评 | 付费 |

## 上市公司数据

| 数据源 | 数据类型 |
|--------|----------|
| 同花顺 iFinD | 财务数据、研发费用 |
| 万得 Wind | 财务数据、估值 |
| 公司年报 | 研发进展、商业化数据 |
| 公司公告 | 临床进展、获批公告 |

## 常用数据调用示例

### 药品审批信息
```
来源：NMPA/CDE
指标：受理号、审评状态、获批时间
```

### 医保谈判结果
```
来源：医保局
指标：谈判药品、支付标准、降幅
```

### 集采中标信息
```
来源：联采办
指标：中标企业、中标价、降幅
```
