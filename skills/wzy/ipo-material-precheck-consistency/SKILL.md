# IPO 材料初审助手 - 财务一致性版

## 功能概述

本技能用于检查 IPO 申报材料中各文件之间的财务数据是否一致，包括招股说明书、审计报告、法律意见书、问询回复等文件中的关键财务指标。

## 适用场景

- 申报材料提交前的数据核对
- 反馈回复后的数据更新检查
- 多版本文件的数据一致性验证
- 财务数据异常波动分析

## 核心检查项目

### 一、基础财务数据

| 数据项 | 招股书 | 审计报告 | 法律意见书 | 问询回复 |
|--------|--------|----------|------------|----------|
| 营业收入 | ✓ | ✓ | - | ✓ |
| 净利润 | ✓ | ✓ | - | ✓ |
| 扣非净利润 | ✓ | ✓ | - | ✓ |
| 资产总额 | ✓ | ✓ | ✓ | ✓ |
| 负债总额 | ✓ | ✓ | ✓ | ✓ |
| 所有者权益 | ✓ | ✓ | ✓ | ✓ |
| 经营活动现金流 | ✓ | ✓ | - | ✓ |

### 二、财务比率指标

| 指标 | 招股书 | 审计报告 | 一致性要求 |
|------|--------|----------|------------|
| 毛利率 | ✓ | - | 各期一致 |
| 净利率 | ✓ | - | 各期一致 |

### 三、分部数据

- 分产品收入
- 分地区收入

- 前五大客户销售占比

## 使用流程

### 步骤 1：准备数据文件

提取各文件中的财务数据，整理为 Excel 或 JSON 格式

### 步骤 2：运行检查脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\3. IPO 材料初审助手 - 财务一致性版"
python scripts/check_financial_consistency.py --input <数据文件路径> --output <检查结果路径>
```

### 步骤 3：查看不一致项

检查报告将列出所有数据不一致的项目及差异金额/比例

### 步骤 4：核实并修正

对不一致项进行核实，确定原因并修正

## 常见不一致原因

1. **统计口径差异**：合并范围调整、会计政策变更
2. **时间差异**：数据截止日不同
3. **四舍五入**：小数位保留不同
4. **笔误**：录入错误、单位错误
5. **版本差异**：使用了不同版本的文件
