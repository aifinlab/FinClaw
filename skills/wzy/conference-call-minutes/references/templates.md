```markdown

- 参会人员：[高管姓名 + 职务]
- 记录人：[姓名]

- 要点 3

- [同上]

| 指标 | 数值 | 同比 | 环比 | 预期 |
|------|------|------|------|------|
| | | | | |

## 三、核心观点
- [观点 1]
- [观点 2]
- [观点 3]

## 四、问答环节

### Q1: [问题]
**A**: [回答要点]

### Q2: [问题]
**A**: [回答要点]

## 五、超预期信息
- [超预期点 1]
- [超预期点 2]

## 六、新增信息
- [新增信息 1]
- [新增信息 2]

## 七、投资要点
- [要点 1]
- [要点 2]

整理：[姓名]  审核：[姓名]
```
