# 尽调问题清单助手-IPO 版

## 功能概述

本技能用于生成 IPO 尽职调查问题清单，覆盖业务、财务、法律、公司治理等各领域，帮助系统性地开展尽职调查工作。

## 适用场景

- IPO 项目立项阶段
- 尽职调查启动阶段
- 中介机构协调会
- 反馈回复准备

## 尽调问题清单框架

### 一、公司基本情况

1. 公司设立及历史沿革

2. 股权结构

## 使用流程

### 步骤 1：选择尽调范围

根据项目情况选择需要尽调的领域

### 步骤 2：生成问题清单

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\14. 尽调问题清单助手-IPO 版"
python scripts/generate_dd_questions.py --industry <行业> --output <输出路径>
```

### 步骤 3：查看问题清单

问题清单将按领域分类，包含具体问题

### 步骤 4：开展尽调

根据问题清单开展尽职调查，收集资料并核实
