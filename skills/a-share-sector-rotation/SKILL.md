# A股行业轮动策略分析

## 数据获取

使用 cn-stock-data skill 获取以下数据：
1. **宏观指数 kline**（近60个交易日）：沪深300(sh000300)、中证1000(sh000852)、上证50(sh000016)、创业板指(sz399006)、中证全指(sh000985)
2. **行业指数 kline**（近20个交易日）：申万一级行业指数（银行/非银/地产/消费/医药/科技/新能源/周期/公用事业等）
3. **web搜索**补充：最新PMI、社融、CPI/PPI、利率动态、行业政策

## 分析流程（5步）

### Step 1: 经济周期定位
- 综合PMI趋势（荣枯线50上下）、社融增速、CPI/PPI剪刀差、利率方向
- 判断当前处于：**复苏 / 扩张 / 过热 / 衰退** 哪个阶段
- 参考 references/sector-rotation-guide.md 中修正版美林时钟映射

### Step 2: 风格判断
- **大小盘**：计算沪深300/中证1000比值近20日趋势（上升=大盘占优，下降=小盘占优）
- **价值成长**：计算上证50/创业板指比值近20日趋势（上升=价值占优，下降=成长占优）
- 给出当前主导风格和切换信号

### Step 3: 行业强弱排名
- 计算各行业指数近5日、10日、20日涨跌幅
- 按动量排名，标注：强势行业（前5）、弱势行业（后5）、加速行业（短期排名显著提升）
- 识别资金从哪些行业流出、流入哪些行业

### Step 4: 轮动信号识别
- **动量延续信号**：强势行业成交额放大+涨幅加速 → 趋势可能延续
- **均值回归信号**：弱势行业出现放量止跌+基本面改善 → 可能反转
- **切换预警**：领涨行业滞涨缩量 + 滞后行业开始异动 → 轮动可能发生
- 结合经济周期判断哪类信号更可靠

### Step 5: 输出结论

```

```

## 注意事项
- 行业指数数据可能有延迟，注意标注数据截至日期
- 经济周期判断需结合多个指标交叉验证，单一指标不可靠
- 轮动策略适合中期（2-4周）视角，不适用于日内交易
- A股行业轮动受政策影响极大，需关注最新政策导向
