# 核保摘要维度参考

以下内容用于帮助组织整案摘要，不替代正式核保意见。

## 基本信息维度

- 产品名称、险种类型、申请对象、年龄、性别
- 当前所处环节，例如初审、复核、升级审核前
- 材料来源类型，例如问卷、OCR、病历、系统导出字段

## 投保与告知维度

- 健康告知中的重点异常、既往史、治疗史
- 职业信息中的高风险作业、兼职或特殊暴露
- 生活习惯中的吸烟、饮酒、特殊运动或高风险爱好

## 医疗与病史维度

- 体检异常指标和对应检查结论
- 门诊、住院、手术、病理、影像、复查资料
- 疾病时间线、治疗经过、恢复情况、随访情况

## 核保处理维度

- 已识别的风险主题
- 已追问和已补件情况
- 当前仍未解决的问题和判断盲区
