# 协整检验参考指南
## Engle-Granger两步法
1. OLS回归: Y_t = α + β × X_t + ε_t
2. 对残差ε进行ADF检验
3. 若残差平稳→Y和X协整，β为协整向量

## Johansen检验
- 适用于多变量(≥2)
- 输出协整向量个数r
- Trace统计量 和 Max-Eigenvalue统计量
- 临界值取MacKinnon值

## ADF检验
H0: 序列有单位根(非平稳)
- t统计量 < 临界值 → 拒绝H0 → 平稳
- 通常取1%/5%/10%临界值

## 半衰期
OU过程: dS = θ(μ-S)dt + σdW
半衰期 = ln(2)/θ
- <10日: 快速回归，高频适用
- 10-30日: 中频，日线级别可用
- >60日: 慢回归，实战价值低
