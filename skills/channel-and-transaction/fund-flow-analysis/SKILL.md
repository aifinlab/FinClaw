# 资金流向分析助手

## 1. 业务问题

银行需要掌握资金在客户、账户、机构、渠道和产品间的流动方向与结构，以便识别资金来源集中度、流出流向特征和关键迁移路径。资金流向分析有助于发现客户资金搬家、同业竞争流失、产品间内部迁移等关键经营信号。

核心问题包括：
- 资金主要从哪里流入？流向哪里？
- 流入流出的对手方集中度如何？
- 是否存在明显的资金迁移路径或搬家特征？
- 资金流向在时间维度上是否出现结构性变化？

## 2. 适用场景

- 客户或客群资金来源与去向分析
- 存款资金流入流出结构分析
- 分支机构间资金流动格局梳理
- 产品间资金内部迁移识别（如活期转理财、定期转他行）
- 对公客户资金链路追踪
- 资金流向异常信号初步识别

## 3. 输入

| 字段 | 说明 | 是否必需 |
|------|------|----------|
| 分析时段 | 日/周/月/季/自定义区间 | 是 |
| 分析对象 | 客户/账户/客群/机构/产品 | 是 |
| 交易流水数据 | 含交易方向、对手方、金额、时间等 | 是 |
| 渠道范围 | 全渠道或指定渠道 | 否 |
| 金额阈值 | 关注的最低金额门槛 | 否 |
| 对比基准 | 同比/环比/指定基准期 | 否 |

## 4. 输出

- 资金流入流出总量与净流向概览（结论先行）
- 主要流入来源TOP-N与流出去向TOP-N
- 对手方集中度分析
- 资金迁移路径图谱（产品间、机构间、渠道间）
- 时间维度的流向结构变化趋势
- 异常流向信号标注与初步判断

## 5. 默认分析维度

- **流向维度**：流入/流出/净流向
- **对手方维度**：行内/行外、同业/非银、具体对手机构
- **渠道维度**：手机银行、网银、柜面、快捷支付等
- **产品维度**：活期、定期、理财、基金、贷款还款等
- **时间维度**：日/周/月趋势、时段分布
- **路径维度**：资金流转链路、多跳路径
- **机构维度**：总行/分行/支行/网点

## 6. 默认分析框架

1. **总览结论**：先给出资金流向的整体判断（净流入/净流出、主要方向）
2. **流入分析**：主要流入来源、金额占比、集中度
3. **流出分析**：主要流出去向、金额占比、集中度
4. **对手方分析**：核心对手方识别与交易频次、金额排名
5. **路径分析**：识别关键迁移路径（如"活期→理财→赎回→转他行"）
6. **趋势分析**：流向结构的时间演变，标注结构性变化节点
7. **异常识别**：标注异常集中流出、突发大额流向变化等信号
8. **经营建议**：基于流向特征的初步经营策略参考

## 7. 安全边界

- 不编造数据；不夸大结论；缺失信息标注"未获取"或"待核实"
- 不替代正式审批、正式风控、正式合规、正式报送、正式处置、正式责任认定、正式管理决策
- 不替代正式技术排障、正式故障定位、正式处置决策、正式责任认定
- 资金流向分析为数据统计层面的结构描述，不替代反洗钱或合规部门的正式判断
- 异常流向信号仅供参考，不构成可疑交易认定
