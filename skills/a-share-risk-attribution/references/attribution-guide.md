# 绩效归因参考指南

## 一、Brinson-Fachler 行业归因模型

### 1.1 基本思想

将组合相对基准的超额收益分解为三个效应：
- **配置效应 (Allocation Effect)**：行业权重偏离带来的收益
- **选股效应 (Selection Effect)**：行业内个股选择带来的收益
- **交互效应 (Interaction Effect)**：配置与选股的交叉影响

### 1.2 单期 Brinson-Fachler 公式

设第 i 个行业：
- Wp,i = 组合在行业 i 的权重
- Wb,i = 基准在行业 i 的权重
- Rp,i = 组合在行业 i 的收益率
- Rb,i = 基准在行业 i 的收益率
- Rb = 基准总收益率

**配置效应**:
```
AA_i = (Wp,i - Wb,i) × (Rb,i - Rb)
```
含义：超配了表现好的行业（或低配了表现差的行业）产生正贡献。

**选股效应**:
```
SS_i = Wb,i × (Rp,i - Rb,i)
```
含义：在某行业内，组合持仓收益优于该行业基准收益时产生正贡献。

**交互效应**:
```
II_i = (Wp,i - Wb,i) × (Rp,i - Rb,i)
```
含义：超配了选股表现好的行业（双重叠加效应）。

**超额收益分解**:
```
Rp - Rb = Σ(AA_i) + Σ(SS_i) + Σ(II_i)
```

### 1.3 Brinson-Hood-Beebower (BHB) 变体

与 Brinson-Fachler 的区别在于配置效应的计算：
```
AA_i(BHB) = (Wp,i - Wb,i) × Rb,i
```
BHB 模型不减去基准总收益 Rb，因此配置效应包含了更多噪音。实务中推荐使用 Brinson-Fachler。

## 二、多期归因 — Carino 链接法

### 2.1 问题

单期归因满足可加性：超额收益 = 配置 + 选股 + 交互。但多期时，由于复利效应，各期归因效应简单相加不等于总期间超额收益。

### 2.2 Carino 链接因子

设 T 个子期间，第 t 期：
- rp,t = 组合收益率
- rb,t = 基准收益率

总期间收益：
```
Rp = Π(1 + rp,t) - 1
Rb = Π(1 + rb,t) - 1
```

Carino 链接系数：
```
k_t = ln(1 + rp,t) - ln(1 + rb,t)
      ─────────────────────────────
           rp,t - rb,t

K = ln(1 + Rp) - ln(1 + Rb)
    ─────────────────────────
         Rp - Rb
```

调整后的第 t 期归因效应：
```
调整后效应_t = (k_t / K) × 原始效应_t
```

调整后各期效应之和 = 总期间超额收益（精确成立）。

### 2.3 实务建议

- 归因区间 ≤ 1 个季度：单期归因即可
- 归因区间 > 1 个季度：按月拆分 + Carino 链接法
- 归因区间 > 1 年：按季度拆分 + Carino 链接法

## 三、风格因子归因（回归法）

### 3.1 模型

将组合超额收益对因子收益做截面回归：
```
Rp - Rf = α + β1·F1 + β2·F2 + ... + βk·Fk + ε
```

其中：
- Rf = 无风险利率（可用 Shibor 或国债收益率）
- Fi = 第 i 个因子的收益率
- βi = 组合对第 i 个因子的暴露（敏感度）
- α = 无法被因子解释的超额收益（个股 alpha）
- ε = 残差

### 3.2 A股常用因子及构建方法

| 因子 | 代理变量 | 构建方法 |
|------|---------|---------|
| 市场 (MKT) | 市场超额收益 | 沪深300收益率 - 无风险利率 |
| 规模 (SMB) | 市值 | 小市值组合收益 - 大市值组合收益 |
| 价值 (HML) | BP (账面市值比) | 高BP组合收益 - 低BP组合收益 |
| 动量 (MOM) | 过去收益率 | 过去3-6月高收益组合 - 低收益组合 |
| 波动率 (VOL) | 日收益率标准差 | 低波组合收益 - 高波组合收益 |
| 质量 (QMJ) | ROE | 高ROE组合收益 - 低ROE组合收益 |
| 流动性 (LIQ) | 换手率 | 低换手组合收益 - 高换手组合收益 |

### 3.3 简化实操方法

在数据有限时，可采用**持仓加权暴露法**：
1. 计算组合各股的因子暴露值（如 PE、市值、ROE 等）
2. 按权重加权得到组合的因子暴露
3. 同样计算基准的因子暴露
4. 主动暴露 = 组合暴露 - 基准暴露
5. 因子贡献 ≈ 主动暴露 × 因子收益率

此方法避免了回归法对数据量的要求，适合快速归因。

### 3.4 因子贡献分解

```
总超额收益 = Σ(主动暴露_i × 因子收益_i) + α(个股选择)
```

1. 与组合的投资范围匹配（大盘/中小盘/全市场）

3. 可投资、可复制（ETF 可交易的指数更佳）

## 五、归因结果解读

### 5.1 配置效应为正
- 说明投资经理在行业配置上有能力
- 正确判断了行业轮动方向
- 超配了强势行业，低配了弱势行业

### 5.2 选股效应为正
- 说明投资经理在个股选择上有能力
- 在各行业内选到了优于行业平均的个股
- 体现了自下而上的研究深度

### 5.3 交互效应解读
- 通常较小，可合并到配置或选股效应中
- 为正：超配行业的选股也做得好（能力一致性）
- 为负：超配行业但选股不佳（配置对但选错了股）

### 5.4 投资能力评估

| 指标 | 计算方法 | 判断标准 |
|------|---------|---------|
| 配置能力 | 配置效应 / 超额收益 | > 50% → 配置驱动型 |
| 选股能力 | 选股效应 / 超额收益 | > 50% → 选股驱动型 |
| 信息比率 IR | 超额收益 / 跟踪误差 | > 0.5 良好, > 1.0 优秀 |
| 配置胜率 | 配置效应为正的月份占比 | > 60% 有配置能力 |
| 选股胜率 | 选股效应为正的月份占比 | > 60% 有选股能力 |

### 5.5 常见归因模式

1. **均衡型**：配置和选股效应均为正，说明投资经理能力全面
2. **配置驱动型**：配置效应显著为正，选股效应接近零，擅长行业轮动
3. **选股驱动型**：选股效应显著为正，配置效应接近零，擅长精选个股
4. **运气型**：交互效应大于配置+选股，超额收益来源不稳定
5. **能力不足型**：配置和选股效应均为负，需要检视投资流程
