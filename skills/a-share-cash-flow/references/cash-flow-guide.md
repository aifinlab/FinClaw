# 现金流分析指南

## 一、现金流三大类解读

### 1. 经营活动现金流（Operating Cash Flow, OCF）
反映企业**主营业务造血能力**，是最核心的现金流。

**关键科目：**
- 销售商品、提供劳务收到的现金 → 对应营收
- 支付给职工以及为职工支付的现金 → 人力成本
- 支付的各项税费 → 税负水平
- 经营活动现金流净额 → 主业真实盈利能力

**健康标志：**
- 经营现金流净额 > 净利润（净现比 > 1）
- 持续为正且稳定增长
- 与营收/净利润增长趋势一致

**警示信号：**
- 经营现金流持续 < 净利润 → 利润质量可疑
- 经营现金流为负但净利润为正 → 严重背离，需深入排查
- 大幅波动 → 业务不稳定或存在人为调节

### 2. 投资活动现金流（Investing Cash Flow, ICF）
反映企业**扩张/收缩意愿和资本配置方向**。

**关键科目：**
- 购建固定资产、无形资产支付的现金 → 资本开支（Capex）
- 取得子公司及其他营业单位支付的现金 → 并购扩张
- 处置固定资产/子公司收回的现金 → 资产瘦身
- 投资支付/收回的现金 → 理财/证券投资

**解读要点：**
- 投资现金流为负是正常的（说明在扩张）
- 关键看资本开支的**回报率**和**合理性**
- 大量购买理财产品 → 可能缺乏好的投资项目

### 3. 筹资活动现金流（Financing Cash Flow, FCF）
反映企业**融资策略和股东回报**。

**关键科目：**
- 吸收投资收到的现金 → 股权融资/增发
- 取得借款收到的现金 → 债务融资
- 偿还债务支付的现金 → 还债压力
- 分配股利、利润支付的现金 → 股东回报

**解读要点：**
- 筹资现金流持续为正 → 依赖外部输血
- 筹资现金流为负 → 可能在还债或分红（需区分）
- 频繁增发/借债 + 经营CF不佳 → 危险信号

## 二、自由现金流（FCF）计算方法

### 基础公式
```
FCF = 经营活动现金流净额 - 资本开支
```

其中：
- 资本开支 = 购建固定资产、无形资产和其他长期资产支付的现金

### 衍生指标
```
FCF Yield = FCF / 总市值          （类比股息率，>3% 为佳）
FCF 转化率 = FCF / 净利润         （>70% 为优质）
FCF 增长率 = (本期FCF - 上期FCF) / |上期FCF|
每股 FCF = FCF / 总股本
```

### FCF 质量分级
| 等级 | FCF 特征 | 典型表现 |
|------|---------|---------|
| A（优秀） | 持续正且增长 | FCF转化率>80%，FCF Yield>4% |
| B（良好） | 多数期为正 | FCF转化率50-80%，偶有波动 |
| C（一般） | 正负交替 | 受资本开支周期影响大 |
| D（较差） | 持续为负 | 烧钱阶段，需关注融资能力 |

## 三、核心比率标准

### 收现比（销售收现比率）
```
收现比 = 销售商品、提供劳务收到的现金 / 营业收入
```
| 收现比 | 评价 | 含义 |
|--------|------|------|
| > 1.17 | 优秀 | 不仅回款好，还收到了预收款（含增值税因素） |
| 1.0-1.17 | 良好 | 回款正常，考虑增值税后基本匹配 |
| 0.8-1.0 | 一般 | 有一定应收账款积压 |
| < 0.8 | 较差 | 回款困难，需警惕坏账风险 |

> 注：收现比天然应 > 1，因为收到的现金含增值税（一般13%），而营收不含税。

### 净现比（经营现金流/净利润）
```
净现比 = 经营活动现金流净额 / 净利润
```
| 净现比 | 评价 | 含义 |
|--------|------|------|
| > 1.5 | 优秀 | 现金流远超利润，盈利含金量极高 |
| 1.0-1.5 | 良好 | 利润基本都能变成现金 |
| 0.5-1.0 | 一般 | 部分利润沉淀在应收/存货中 |
| < 0.5 | 较差 | 利润含金量低，需警惕 |
| < 0 | 危险 | 盈利但现金在流出，可能有重大问题 |

## 四、现金流与利润背离的警示意义

### 常见背离场景

| 场景 | 表现 | 可能原因 | 风险等级 |
|------|------|---------|---------|
| 利润增 + 经营CF降 | 增收不增现 | 应收激增、放宽信用 | 高 |
| 利润增 + 经营CF负 | 赚钱但没钱 | 可能虚增收入 | 极高 |
| 利润降 + 经营CF增 | 利润差但回款好 | 压缩库存、收紧信用 | 低（可能是好事） |
| 利润负 + 经营CF正 | 亏损但有现金 | 折旧大但业务健康 | 中低 |

### 重点排查项
1. **应收账款增速 >> 营收增速** → 可能虚增收入或放宽信用
2. **存货异常增长** → 可能滞销或虚增资产
3. **经营CF靠压应付账款** → 占用上游资金，不可持续
4. **投资收益占利润比重大** → 非经常性，不可持续
5. **大额资产减值前利润很好** → 前期可能隐藏问题

## 五、不同行业现金流特征

### 轻资产行业（互联网、软件、消费品牌）
- 资本开支低，FCF 通常较高
- 收现比高（预付模式，如 SaaS）
- 投资现金流主要是并购和投资理财
- **关注点**：营收增长是否带来 OCF 同步增长

### 重资产行业（制造业、钢铁、化工、电力）
- 资本开支大，FCF 可能周期性为负
- 折旧摊销金额大，经营CF一般 > 净利润
- 投资现金流长期为负（持续建设产能）
- **关注点**：资本开支回报率、产能利用率、折旧政策

### 周期行业（有色、煤炭、航运）
- 现金流随周期大幅波动
- 景气周期：经营CF暴增，FCF大增
- 低谷周期：经营CF骤降，可能需要融资度日
- **关注点**：跨周期平均 FCF，而非单期数据

### 金融行业（银行、保险、证券）
- 现金流量表结构与一般企业不同
- 经营活动包含大量金融资产买卖
- 不适用传统 FCF 计算方法
- **关注点**：改用 RORAC、经济利润等指标

### 房地产行业
- 经营CF受拿地/预售节奏影响极大
- 存货（土地+在建工程）占用大量现金
- 筹资CF依赖度高
- **关注点**：回款率、在手现金/短期债务

### 高增长/初创企业
- 经营CF可能为负（烧钱换增长）
- 投资CF大额为负（扩张）
- 筹资CF持续为正（依赖融资）
- **关注点**：现金消耗速率（Cash Burn Rate）、融资能力

## 六、常见现金流操纵手段

### 1. 美化经营现金流
- **延迟付款给供应商**：短期提升经营CF，但损害供应链关系
- **加速催收应收账款**：季末/年末突击回款，过后又放出去
- **将经营支出资本化**：如研发费用资本化，把经营流出移到投资流出
- **保理/票据贴现**：应收账款出表，经营CF看起来好

### 2. 美化自由现金流
- **延缓资本开支**：短期 FCF 好看，但牺牲长期竞争力
- **售后回租**：固定资产卖出（投资流入）再租回（经营流出），FCF被高估
- **调整资本开支分类**：将维护性资本开支归入扩张性，低估真实 capex

### 3. 识别方法
- 连续多期对比，看经营CF是否只在年末/季末突增
- 对比应付账款周转天数变化
- 检查资本化支出占研发总支出比例是否异常上升
- 关注经营CF与净利润的长期偏离度
