# 沟通边界参考

## 可以解释的内容

- 核保结论在业务上的一般含义
- 与标准承保相比的主要差异
- 对责任范围、保费、承保时点和后续流程的常见影响
- 当前资料下可执行的后续动作，例如补件、复议、延期后再看、责任调整

## 不能直接下结论的内容

- 未经明确依据支持的医学诊断
- 未经条款或正式决定确认的责任结论
- 未经授权的费率形成逻辑、内部评分规则或风控模型结果
- 对未来复议结果、延期后结论或再次投保结果的确定性承诺
- 正式法律意见、监管口径或投诉处理结论

## 建议表达方式

- 优先说“从当前核保结论看”
- 优先说“通常意味着”或“本次可理解为”
- 边界不清时说“需结合正式条款/核保决定进一步确认”
- 资料不足时说“现有材料下只能作保守解释”

## 避免表达方式

- 避免说“公司认定你有某疾病”
- 避免说“以后肯定不能买”
- 避免说“这部分一定赔不了”，除非除外责任表述和条款边界已明确
- 避免说“补完资料就一定能通过”
- 避免说“只是流程问题，不影响结果”，除非已有明确依据
