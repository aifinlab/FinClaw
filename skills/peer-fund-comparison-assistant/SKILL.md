# 技能说明

本技能用于支持基金业务中的同类分析工作，重点解决「对同类基金进行业绩、风险、费率、规模、风格和卖点差异的结构化比较。」这类任务。输出默认使用中文，要求结构清楚、证据边界明确，并能被投研、产品、销售、投顾或合规团队直接复用。

# 本任务重点关注的问题

- 目标产品相对同类真正的优势和短板是什么
- 差异来自策略、费率、风格还是渠道定位
- 哪些比较维度适合对外表达
- 当前是否存在被替代或被误卖的风险

# 适用范围

- 目标产品与同类产品横向对比
- 竞品卖点和短板识别
- 渠道培训中的产品差异化说明
- 产品线梳理与替代关系判断

# 不适用范围

- 没有统一口径却要求精细排名
- 把对比结果直接当成销售承诺
- 同类样本选择明显失真

# 输入要求

优先输入包括：

- 目标产品与同类产品清单
- 收益、回撤、波动和规模数据
- 费率、持有期和渠道规则
- 产品定位和基准

可选增强输入包括：

- 风格标签与持仓特征
- 经理信息
- 渠道反馈

当输入不完整时，必须先列出缺失材料，再基于现有信息输出阶段性判断，不得编造未确认事实。

# 核心分析框架

## 一、先判断产品角色和所处阶段

- 明确产品定位、目标客群、渠道场景和所处生命周期。
- 区分内部管理讨论和对外表达口径。
- 看清产品在线路中的分工，而不是只看单一指标。

## 二、围绕产品事实和协同问题展开

- 整理正式披露材料、内部口径、渠道反馈和运营数据。
- 识别定位、标签、材料、同类竞争和生命周期中的关键矛盾。
- 先找到真正影响经营和沟通的问题。

## 三、输出能落地的判断

- 建议要带动作和优先级。
- 区分短期修补、中期优化和长期定位问题。
- 对外可见内容要保留复核边界。

# 推荐工作流

1. 先明确产品所处阶段、目标客群、定位约束和业务目标。
2. 整理产品资料、同类对照、渠道反馈和运营数据，建立统一口径。
3. 识别产品定位、生命周期、标签、材料或运维流程中的关键问题。
4. 给出可执行的优化建议、信息补充清单和后续动作。
5. 输出时同时保留适用边界，避免把内部分析包装成外部承诺。

# 关键判断视角

- 产品定位、客群与渠道适配度
- 同类差异化、阶段性表现与运营状态
- 材料、标签、卖点和内部口径的一致性
- 待补充数据、流程依赖和落地优先级

# 输出要求

建议输出至少覆盖以下模块：

- 对比摘要
- 关键指标矩阵
- 差异化结论
- 适用客群建议
- 风险提示

若任务涉及对客材料、话术或解释文案，必须同时输出「不能直接对外使用的前提说明」或「需正式复核」的提醒。

# 输出约束

- 不得制造超出合同和正式披露的新卖点。
- 不得把内部讨论口径直接转成外部宣传。
- 不得跳过定位冲突和渠道适配问题。

# 质量检查清单

- 是否先界定了任务边界、时间窗口、样本范围和使用场景。
- 是否把已确认事实、解释性判断和待核验事项分开表达。
- 是否给出了证据链，而不是只写结论标签。
- 是否保留了风险提示、限制说明和后续动作。
- 若内容可能进入对客或外部场景，是否保留了复核边界。
- 当前 skill 特别要盯住的判断视角：
- 产品定位、客群与渠道适配度
- 同类差异化、阶段性表现与运营状态
- 材料、标签、卖点和内部口径的一致性
- 待补充数据、流程依赖和落地优先级
