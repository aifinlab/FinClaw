# 输入字段建议

## 最低必需材料

- 目标产品与同类产品清单
- 收益、回撤、波动和规模数据
- 费率、持有期和渠道规则
- 产品定位和基准

## 可选增强材料

- 风格标签与持仓特征
- 经理信息
- 渠道反馈

## 建议输入结构

~~~json
{
  "task_name": "同类基金对比助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
