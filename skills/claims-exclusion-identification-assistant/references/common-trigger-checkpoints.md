# 常见除外责任触发点检查清单

## 时间与病史类

- 等待期内出险、发病、确诊或治疗。
- 投保前已存在症状、检查异常、诊疗记录或长期管理线索。
- 既往症、先天性疾病、遗传性疾病、精神心理类疾病是否可能命中除外定义。

## 行为与事故类

- 自杀、自伤、故意行为。
- 违法犯罪、斗殴、拘捕、刑事案件。
- 酒驾毒驾、药物滥用、无证驾驶。
- 高风险运动、高风险作业、特殊职业活动。

## 医疗与费用类

- 非医学必要治疗、保健性质治疗、预防性治疗。
- 美容整形、矫形、减重、生殖辅助、妊娠生育相关情形。
- 非约定医院就医、不合规医疗机构、超出合同约定项目。

## 其他常见边界

- 事故原因与疾病原因混杂，免责方向不清。
- 医疗必要性、行为性质、事故归因或病史归因存在争议。
- 条款中的定义和案件事实表达并不完全同义。
