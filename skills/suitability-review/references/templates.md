# 适当性审核输出模板

## 审核报告模板

```markdown
# 【研报适当性审核报告】

## 报告信息
- 报告标题：[标题]
- 分析师：[姓名]
- 报告类型：[公司/行业/策略]
- 提交时间：[时间]

## 一、合规检查

### 1.1 标题合规
- 检查结果：[通过/需修改]
- 问题描述：[如有]
- 修改建议：[建议]

### 1.2 评级合规
- 检查结果：[通过/需修改]
- 评级依据：[是否有依据]

### 1.3 数据合规
- 检查结果：[通过/需修改]
- 数据来源：[是否标注]

### 1.4 表述合规
- 检查结果：[通过/需修改]
- 问题表述：[列出问题]

### 1.5 利益冲突
- 检查结果：[通过/需披露]
- 持仓情况：[是否披露]

## 二、风险披露

### 2.1 风险完整性
- 检查结果：[充分/不充分]

### 2.2 风险具体性
- 检查结果：[具体/空泛]

### 2.3 风险显著性
- 检查结果：[显著/不显著]

## 三、适当性匹配

### 3.1 投资者类型
- 适用投资者：[专业/普通]
- 风险等级：[R1-R5]

### 3.2 适当性警示
- 检查结果：[有/无]

## 四、免责声明

### 4.1 标准免责声明
- 检查结果：[有/无]

### 4.2 分析师声明
- 检查结果：[有/无]

## 五、审核结论

### 综合结论
- [ ] 通过，可发布
- [ ] 修改后通过
- [ ] 不通过，需重大修改

### 修改清单
| 序号 | 问题 | 修改建议 | 优先级 |
|------|------|----------|--------|
| 1 | | | 高/中/低 |

### 审核人
- 审核人：[姓名]
- 审核时间：[时间]
```

## 合规检查清单

- [ ] 标题不含夸大词汇
- [ ] 评级有依据支撑
- [ ] 数据来源标注
- [ ] 无误导性表述
- [ ] 风险披露完整
- [ ] 免责声明完整
