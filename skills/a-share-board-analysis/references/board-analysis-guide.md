# 涨跌停板分析参考指南

## 1. 涨跌停制度

### 1.1 各板块涨跌幅限制

| 板块 | 涨跌幅限制 | 备注 |
|------|-----------|------|
| 主板（沪深） | ±10% | 最常见，大部分 A 股适用 |
| 科创板（688开头） | ±20% | 2019年7月开板，上市前5日不设涨跌幅限制 |
| 创业板（300开头） | ±20% | 2020年8月注册制改革后调整 |
| 北交所（8开头） | ±30% | 2021年11月开市 |
| ST / *ST 股 | ±5% | 被特别处理的股票，风险警示 |
| 新股上市首日 | 主板44%/科创板创业板不限 | 主板首日涨幅上限44%，跌幅下限36% |

> 涨停价 = 前收盘价 ×（1 + 涨幅限制），结果四舍五入到分。

### 1.2 涨停/跌停机制

- **涨停**：股价触及当日最高涨幅限制价位，买单排队但无人卖出
- **跌停**：股价触及当日最低跌幅限制价位，卖单排队但无人买入
- **一字涨停/跌停**：开盘即涨停/跌停，全天无交易机会（或极少量成交）
- **T字涨停**：开盘涨停，盘中打开后再封回涨停，留下下影线

## 2. 连板术语

| 术语 | 含义 |
|------|------|
| 首板（一板） | 第一个涨停板，连板的起点 |
| 二板（二连板） | 连续第二个涨停 |
| 三板（三连板） | 连续第三个涨停 |
| 高标 / 最高标 | 当前市场连板最多的股票（龙头地位） |
| 空间板 | 最高连板股的高度，代表市场情绪的"天花板" |
| 梯队 | 按连板天数分层的股票群体（如"三板梯队有5只"） |
| 断板 | 连板中断，未能继续涨停 |
| 反包 | 断板次日再次涨停，"反包涨停" |

### 2.1 连板梯队结构

典型的健康市场连板梯队呈金字塔型：

```
      [最高标]          ← 1-2只（如7连板龙头）
     [次高标]           ← 2-3只（5-6连板）
    [中位梯队]          ← 5-10只（3-4连板）
   [二板梯队]           ← 10-20只（2连板）
  [首板梯队]            ← 30-60只（当日新涨停）
```

- 梯队完整 = 市场情绪好，接力意愿强
- 高标断板 + 低位首板多 = 轮动加速、情绪切换
- 全线缩量 = 情绪退潮

## 3. 炸板

### 3.1 炸板含义

"炸板"指股票盘中触及涨停价后被打开，未能维持涨停至收盘。

### 3.2 炸板率计算

```
炸板率 = 炸板股数量 / (涨停股数量 + 炸板股数量) × 100%
```

| 炸板率 | 市场含义 |
|--------|---------|
| < 20% | 封板坚决，市场做多情绪强 |
| 20%-40% | 正常水平，有分歧但可接受 |
| 40%-60% | 分歧较大，追高风险增加 |
| > 60% | 封板意愿极弱，不宜追涨停 |

### 3.3 炸板原因

- **大单砸盘**：主力或机构在涨停价位集中卖出
- **跟风不足**：封单量太小，稍有卖压即打开
- **情绪退潮**：整体市场走弱，连锁反应导致炸板
- **利好兑现**：消息刺激涨停后资金选择获利了结

## 4. 打板策略与风险

### 4.1 打板含义

"打板"指在股票涨停时买入（排队挂涨停价买入），博弈次日继续上涨。

### 4.2 常见打板方式

| 方式 | 描述 | 风险 |
|------|------|------|
| 扫板 | 涨停瞬间扫单买入 | 高（可能炸板） |
| 排板 | 涨停后挂单排队 | 中（可能排不到） |
| 半路板 | 股价冲板途中买入 | 极高（可能冲高回落） |

### 4.3 风险提示

- **炸板风险**：买入后涨停打开，当日即被套
- **次日低开**：涨停次日低开是常态（尤其尾盘涨停）
- **流动性风险**：一字涨停无法买入，一字跌停无法卖出
- **监管风险**：频繁异动可能触发交易所问询函或停牌核查
- **情绪退潮风险**：市场情绪转弱时，高位连板股跌幅最大

> 打板属于高风险短线策略，不适合普通投资者。本技能仅做数据分析，不构成打板建议。

## 5. 常见涨停原因分类

| 分类 | 细分 | 示例 |
|------|------|------|
| 利好公告 | 业绩预增、重大合同、资产重组 | "公司中标XX亿项目" |
| 概念炒作 | 政策利好、热点题材 | "AI概念"、"低空经济" |
| 业绩超预期 | 季报/年报超预期 | "净利润同比增长300%" |
| 资金博弈 | 游资接力、机构扫货 | 无明显消息面但连续涨停 |
| 超跌反弹 | 前期大跌后技术性反弹 | 股价腰斩后的涨停 |
| 蹭热点 | 公司发公告蹭市场热点 | "公司涉及XX业务"互动平台回复 |
| 股东增持/回购 | 大股东增持或公司回购 | "控股股东拟增持1-2亿" |
| 摘帽/重组 | ST摘帽或重组成功 | "公司撤销退市风险警示" |

### 5.1 涨停原因分析方法

1. 查看涨停池数据中的"涨停原因"字段（akshare 数据通常包含）
2. 结合 web 搜索确认当日相关新闻/公告
3. 同板块多股涨停 → 题材/概念驱动
4. 单只股票涨停 → 个股利好驱动

## 6. 市场情绪指标

### 6.1 涨跌停情绪判断

| 指标 | 偏强信号 | 偏弱信号 |
|------|---------|---------|
| 涨停数 | > 80家 | < 30家 |
| 跌停数 | < 10家 | > 30家 |
| 涨跌停比 | 涨停数 > 跌停数×3 | 跌停数 > 涨停数 |
| 炸板率 | < 25% | > 50% |
| 连板高度 | 最高标 ≥ 5板 | 最高标仅2板 |
| 首板数量 | > 50只 | < 15只 |

> 以上阈值为经验值，仅供参考，需结合大盘走势综合判断。

### 6.2 情绪周期

```
冰点期 → 修复期 → 发酵期 → 高潮期 → 分歧期 → 退潮期 → 冰点期
```

- **冰点期**：涨停家数极少（<20），跌停多，无连板股
- **修复期**：涨停数回升，出现首板试探
- **发酵期**：连板梯队开始形成，首板成功率提高
- **高潮期**：涨停家数爆发（>100），连板高度上升
- **分歧期**：高标出现分歧（炸板），但新题材接力
- **退潮期**：连板批量断板，炸板率飙升

## 7. 数据局限性

| 局限 | 说明 |
|------|------|
| 盘中变化 | 涨停池盘中实时变化，收盘前数据不是最终结果 |
| 竞价涨停 | 集合竞价一字涨停股无法区分封单强弱 |
| 涨停原因 | 数据源标注的原因可能不完整或有延迟 |
| 北交所差异 | 北交所30%涨幅不叫"涨停"但可能被纳入统计 |
| 注册制新股 | 上市前5日无涨跌幅限制，不适用涨停分析 |
