# 产业链分析参考指南

## 热门产业链速查

### 1. 新能源车产业链
```
上游：锂矿/钴镍（天齐锂业、赣锋锂业）、正负极材料（容百科技、璞泰来）、隔膜（恩捷股份）、电解液（天赐材料）
中游：动力电池（宁德时代、比亚迪、亿纬锂能）、电机电控（汇川技术）、热管理（三花智控）
下游：整车（比亚迪、长城汽车、理想汽车）、充电桩（特锐德）、换电（奥动新能源-未上市）
```

### 2. 光伏产业链
```
上游：硅料（通威股份、大全能源）、坩埚/金刚线（欧晶科技、美畅股份）
中游：硅片（隆基绿能、TCL中环）、电池片（通威股份、爱旭股份）、组件（隆基绿能、晶澳科技、天合光能）
下游：电站运营（三峡能源）、EPC（阳光电源-逆变器）、储能（阳光电源、派能科技）
```

### 3. 半导体产业链
```
上游：设备（北方华创、中微公司）、材料（沪硅产业、雅克科技）、EDA（华大九天、概伦电子）
中游：设计（韦尔股份、卓胜微、兆易创新）、制造/代工（中芯国际、华虹半导体）、封测（长电科技、通富微电）
下游：消费电子（立讯精密）、汽车电子（斯达半导）、工业控制、通信（中兴通讯）
关键：EDA/光刻机/高端材料国产化率低，是卡脖子环节
```

### 4. AI / 算力产业链
```
上游：AI芯片（寒武纪、海光信息）、服务器（浪潮信息、中科曙光）、光模块（中际旭创、新易盛）、PCB（沪电股份）
中游：云计算/IDC（润泽科技、数据港）、大模型（百度-港股、科大讯飞）、AI框架
下游：智能驾驶（德赛西威）、AI+医疗、AI+教育、AI+金融、AIGC应用（万兴科技）
关键：高端GPU受限，国产AI芯片替代是核心主线
```

### 5. 医药/创新药产业链
```
上游：CXO/CDMO（药明康德、凯莱英）、原料药（华海药业）、医疗器械上游（迈瑞医疗）
中游：创新药研发（恒瑞医药、百济神州-港股）、仿制药（石药集团-港股）、中药（片仔癀、云南白药）
下游：医院（爱尔眼科）、药店（益丰药房、大参林）、医药流通（国药控股-港股）
关键：集采政策影响仿制药，创新药出海是新方向
```

### 6. 白酒产业链
```
上游：粮食（高粱/小麦）、包装材料、酿酒设备（行业非上市为主）
中游：酒企（贵州茅台、五粮液、泸州老窖、山西汾酒、洋河股份）— 核心环节
下游：经销商体系、电商渠道（1919-未上市）、餐饮/礼品消费
关键：品牌壁垒极高，利润集中在中游酒企，马太效应明显
```

### 7. 军工产业链
```
上游：特种材料（光威复材、西部超导）、元器件（振华科技、紫光国微）
中游：分系统/配套（中航光电、航天电器）、发动机（航发动力）
下游：主机厂（中航沈飞、中直股份、中航西飞）、导弹/卫星（航天彩虹）
关键：军品定价机制改革、十四五装备放量、军转民
```

## 产业链分析框架

### 一、结构梳理（必做）

1. **上中下游划分**
   - 上游：原材料、核心零部件、设备
   - 中游：制造、加工、集成
   - 下游：应用、终端、消费
   - 注意：部分公司横跨多个环节（如比亚迪覆盖电池+整车）

2. **各环节市场规模**
   - 全球 vs 中国市场规模（亿元/亿美元）
   - 近 3-5 年 CAGR
   - 未来 3-5 年预测增速

3. **各环节竞争格局**
   - CR3/CR5 集中度
   - 龙头公司市占率
   - 格局是集中化还是分散化趋势

### 二、竞争格局分析

1. **市占率**：各环节龙头公司的全球/国内市占率
2. **壁垒类型**：
   - 技术壁垒（专利、know-how、研发投入）
   - 规模壁垒（产能、成本优势）
   - 资源壁垒（矿产、牌照、客户认证）
   - 品牌壁垒（消费端）
3. **竞争态势**：价格战 vs 差异化竞争 vs 垄断

### 三、利润分配分析

1. **微笑曲线位置**：哪些环节处于微笑曲线两端（高附加值）
2. **毛利率对比**：各环节平均毛利率排序
3. **净利率对比**：考虑费用率后的真实盈利能力
4. **利润转移趋势**：上下游议价权变化

### 四、技术壁垒与国产化率

1. **国产化率现状**：各环节国产化率（%）
2. **卡脖子环节**：国产化率 < 30% 的关键环节
3. **替代进度**：已实现突破 vs 正在突破 vs 短期难突破
4. **政策支持**：大基金/专项补贴/税收优惠

### 五、投资机会判断

优先关注满足以下条件的环节：
- 壁垒高（护城河深）
- 国产替代空间大（天花板高）
- 行业增速快（beta 强）
- 龙头公司盈利能力强（alpha 强）
- 估值合理（安全边际）

## 如何找到产业链核心环节

1. **利润最厚**：毛利率 > 40% 的环节通常是产业链核心
2. **壁垒最高**：技术门槛高、认证周期长、客户粘性强
3. **不可替代**：缺少该环节整个产业链无法运转
4. **增速最快**：受益于行业趋势最明确的方向
5. **国产替代**：进口依赖度高但已有国产突破的环节，往往有戴维斯双击机会

## 常见陷阱提醒

- 产能过剩环节（如光伏硅片/电池片）利润可能被压缩
- 技术路线变更风险（如磷酸铁锂 vs 三元锂、HJT vs TOPCon）
- 政策依赖型环节（补贴退坡后盈利能力可能大幅下降）
- 上游资源品周期性强（锂价/硅料价格波动剧烈）
- 未上市公司可能是真正的行业龙头（如华为海思在芯片设计领域）
