# 输入字段建议

## 最低必需材料

- 产品事实与风险点
- 目标客群画像
- 渠道与沟通场景
- 禁止或敏感表述要求

## 可选增强材料

- 同类产品常见异议
- 市场背景
- 历史销售反馈

## 建议输入结构

~~~json
{
  "task_name": "销售话术助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
