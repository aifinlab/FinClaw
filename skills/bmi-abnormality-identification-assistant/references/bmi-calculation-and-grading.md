# BMI 计算与等级划分指引

本文件用于统一 BMI 计算、核验和异常等级识别口径。

## 一、基础计算规则

BMI 计算方式：

- BMI = 体重（kg） / 身高（m）的平方

如身高体重单位不明确，应优先提示“单位需核实”，不要直接计算。

## 二、常用等级划分

可采用以下常见审查口径：

- BMI < 18.5：偏瘦或低体重
- BMI 18.5 至 23.9：一般可视为正常范围
- BMI 24.0 至 27.9：超重
- BMI >= 28.0：肥胖

如团队另有内部分级口径，应优先按用户或机构口径执行。

## 三、核验口径

重点核验以下问题：

- BMI 结果与身高体重是否一致
- 身高体重单位是否可能录错
- 身高、体重数值是否明显极端
- 不同资料中的身高体重是否前后冲突

## 四、信息不足识别口径

以下情况应列为信息不足：

- 身高缺失
- 体重缺失
- 测量时间不清
- 单位不清
- 既往数据无法对比
