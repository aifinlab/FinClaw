# BMI 异常识别标准输出结构

本文件用于统一 BMI 异常识别输出结构，确保结果可直接用于核保审查、风险筛查和资料流转。

## 一、基本体征信息

建议字段：

- 被保人基本信息：
- 身高：
- 体重：
- BMI：
- 测量时间：
- 材料来源或文本类型：

## 二、BMI 异常识别结论摘要

建议输出 1 至 3 条：

- BMI 是否异常
- 是否存在体重波动或一致性问题
- 是否建议补问、补件或人工审核

## 三、BMI 计算与异常等级判断

重点覆盖：

- BMI 计算或核验结果
- 等级判断
- 数据完整度提示

## 四、体重变化与一致性梳理

建议覆盖：

- 既往记录是否一致
- 是否存在短期体重波动
- 是否存在录入或单位异常

## 五、相关健康风险线索识别

建议按风险主题组织：

- 肥胖相关风险
- 消瘦或营养风险
- 代谢风险
- 心血管风险

## 六、核保关注点与风险提示

写法建议采用“已知事实 + 风险提示/判断边界”。

## 七、建议补充核实的问题或资料

问题应能直接用于业务人员、核保人员继续补问。

## 八、后续处理建议

可组合输出以下动作：

- 建议补充说明
- 建议补件
- 建议重点人工审核
- 建议结合体检、病史、慢病资料或门诊记录进一步判断
- 当前可进入下一环节核保审查

必须避免写成正式核保结论。
