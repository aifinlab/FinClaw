# 输入字段建议

## 最低必需材料

- 规模、份额与申赎数据
- 净值和回撤表现
- 渠道热度或销售数据
- 阶段目标与运营策略

## 可选增强材料

- 客户反馈
- 竞品动态
- 营销活动记录

## 建议输入结构

~~~json
{
  "task_name": "生命周期监测助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
