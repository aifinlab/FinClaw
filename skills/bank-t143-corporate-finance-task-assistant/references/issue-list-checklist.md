# 问题清单质量检查（t143）

## 1) 每条问题必须四要素
- 问题本体：一句话说清楚哪里不一致/哪里缺证据
- 最小证据：最少要补哪些材料才能核验
- 影响：对还款来源/现金流/条款触发/风险分类影响是什么
- 行动项：责任人 + 截止时间 + 交付物 + 复查节点

## 2) 分级要一致
- 高（硬阻断）：不解决不能推进/必须升级专岗
- 中（关键缺口）：补强证据后可推进
- 低（一般补充）：合并处理，不影响主要判断

## 3) 避免“空话”
- 禁止：仅写“请补充说明”“请提供材料”而不说明要核验什么
- 推荐：每条材料都对应一个核验点（例如“验证回款是否来自真实客户”）
