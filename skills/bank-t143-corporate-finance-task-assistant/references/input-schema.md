# 输入结构（t143 授信问题清单）

```json
{
  "company": {"name": "示例公司"},
  "issue_context": {
    "stage": "准入/续作/贷后/重组",
    "time_window": "本次"
  },
  "issues": [
    {
      "category": "交易真实性",
      "severity": "高",
      "issue": "核心交易合同与回款不匹配，存在回流风险",
      "evidence": "仅有口述解释，缺少可核验证据",
      "impact": "影响还款来源真实性与风险分类",
      "owner": "客户经理A",
      "due_date": "2026-03-20",
      "required_materials": ["合同", "发票", "物流/验收", "资金流水"]
    }
  ]
}
```

严重程度建议：
- `高`：硬性阻断项，不解决不能推进
- `中`：关键证据缺口，补齐后可推进
- `低`：一般补充项，可合并处理
