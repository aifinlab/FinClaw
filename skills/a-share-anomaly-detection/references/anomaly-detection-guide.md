# 量价异常检测参考指南

## 统计方法
### Z-score检测
- Z = (x - μ_N) / σ_N，N通常取20-60日
- |Z| > 2：显著异常（约5%概率）
- |Z| > 3：极端异常（约0.3%概率）

### 箱线图法
- Q1, Q3 = 第25/75百分位数
- IQR = Q3 - Q1
- 异常值：< Q1 - 1.5×IQR 或 > Q3 + 1.5×IQR

## 异常类型
| 类型 | 特征 | 可能含义 |
|------|------|---------|
| 放量突涨 | 量+价同时Z>2 | 利好/游资介入 |
| 放量暴跌 | 量Z>2, 跌>5% | 利空/机构出逃 |
| 缩量涨停 | 量Z<-1, 涨停 | 一致看多/强势 |
| 天量滞涨 | 量Z>3, 涨<1% | 多空分歧/出货 |
| 尾盘急拉 | 尾盘占比>50% | 做盘/诱多 |
| 开盘跳空 | 缺口>3% | 重大消息 |

## A股常见异动模式
1. 业绩预告前异动（信息泄露）
2. 停牌前放量（内幕交易嫌疑）
3. 月末/季末拉升（基金净值管理）
4. 两融标的异常（杠杆资金推动）
