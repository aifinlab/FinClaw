# 输入字段建议

## 最低必需材料

- 原始公告全文
- 产品基础信息
- 涉及事项的时间线
- 历史类似事件或存量规则

## 可选增强材料

- 客户关注问题
- 渠道反馈
- 同类产品处理案例

## 建议输入结构

~~~json
{
  "task_name": "基金公告解读助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
