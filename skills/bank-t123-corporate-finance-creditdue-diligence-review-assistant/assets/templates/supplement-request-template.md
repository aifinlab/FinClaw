# 补充核验要求模板

## 一. 主体与治理

- [ ] 工商变更、章程、股权结构说明

## 二. 业务与交易

- [ ] 主要销售合同与补充协议
- [ ] 主要采购合同

## 三. 财务与债务

- [ ] 最近三年财报/审计报告附注

## 四. 增信与缓释

- [ ] 保证人财力证明
- [ ] 抵押物权属、估值和处置说明

## 五. 风险事项说明

- [ ] 涉诉处罚说明
- [ ] 关联交易和资金往来说明
- [ ] 历史逾期或续作情况说明
