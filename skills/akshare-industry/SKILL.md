## 功能列表

### 1. 行业列表
- **功能描述**: 获取申万行业分类列表
- **输入参数**: 无
- **输出格式**: Markdown表格
- **数据源**: AkShare-申万
- **数据时效**: 季度调整
- **使用示例**:
  ```bash
  python scripts/industry_list.py
  ```

### 2. 行业成分股
- **功能描述**: 获取特定行业的成分股列表
- **输入参数**: 行业名称或代码
- **输出格式**: Markdown表格
- **数据源**: AkShare
- **数据时效**: 日频
- **使用示例**:
  ```bash
  python scripts/industry_stocks.py 半导体
  python scripts/industry_stocks.py 801070  # 电力设备
  ```

### 3. 产业链分析
- **功能描述**: 分析产业链上下游关系
- **输入参数**: 产业链名称
- **输出格式**: Markdown报告
- **数据源**: AkShare
- **数据时效**: 定期更新
- **使用示例**:
  ```bash
  python scripts/industry_chain.py 新能源
  python scripts/industry_chain.py 半导体
  ```
