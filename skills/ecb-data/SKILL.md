# ecb-data

欧央行经济数据查询 Skill。

## 核心能力

- 欧元区基准利率（主要再融资利率、存款便利利率）
- 欧元区 CPI / 核心 CPI（HICP）
- 欧元区 GDP 增速
- 主要成员国经济数据（德/法/意/西）

## 适用场景

- "欧央行现在的利率是多少"
- "欧元区通胀数据"
- "欧洲经济怎么样"

## 相关 Skill

- `boj-data`：日央行数据
- `fred-data`：美联储数据
