## 功能列表

### 1. ESG评级查询
- **功能描述**: 查询个股ESG评级详情
- **输入参数**: 股票代码
- **输出格式**: Markdown报告（E/S/G分项得分+总评级）
- **数据源**: AkShare-华证/商道融绿
- **数据时效**: 季度更新
- **使用示例**:
  ```bash
  python scripts/esg_rating.py --code 600519
  python scripts/esg_rating.py --code 000001
  ```

- **数据时效**: 季度更新
- **使用示例**:

## ESG投资意义

| 维度 | 关注点 | 投资风险 |
|:---|:---|:---|
| **环境(E)** | 碳排放、污染、资源利用 | 环保处罚、碳税 |
| **社会(S)** | 员工权益、供应链、社区 | 舆情风险、罢工 |
| **治理(G)** | 董事会、信息披露、反腐败 | 治理丑闻、财务造假 |

### ESG投资策略
1. **负面筛选**: 排除ESG表现差的公司
2. **正面筛选**: 优选ESG评级高的公司
3. **主题投资**: 专注绿色、低碳、可持续主题
4. **ESG整合**: 将ESG因素纳入传统投资分析
