# akshare-esg

ESG评级与碳中和数据查询 Skill。提供A股市场ESG评级、碳排放数据、可持续发展分析等功能。

## 功能

- ESG评级查询
- ESG行业排行
- 碳中和相关股票筛选
- ESG评级变动追踪
- 可持续发展报告

## 使用示例

查询ESG评级：
```bash
python scripts/esg_rating.py --code 000001
```

ESG排行：
```bash
python scripts/esg_rank.py
```

碳中和股票：
```bash
python scripts/esg_carbon.py
```
