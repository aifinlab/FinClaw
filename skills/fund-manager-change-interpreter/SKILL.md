# 基金经理变更公告解读 Skill

## 数据来源

本 Skill 面向基金经理变更相关公告文本，支持多种输入形态，但不负责公告抓取与外部检索。

### 1. 输入文档类型

* **公告原文**：基金公司发布的正式公告文本
* **PDF文件**：基金公告 PDF 文档
* **网页正文**：公告页面提取后的正文内容
* **图片转写文本**：扫描件、截图经 OCR 后得到的文本
* **摘要内容**：用户手工整理的公告摘要或要点

### 2. 适用公告类型

* **基金经理离任公告**
* **基金经理增聘公告**
* **基金经理变更公告**
* **共同管理安排调整公告**
* **与经理变动相关的补充说明公告**

### 3. 覆盖信息范围

* **基金基础信息**：基金名称、基金代码、基金简称
* **人员变动信息**：离任经理、新任经理、共管经理
* **时间信息**：公告日期、生效日期、任职/离任日期
* **原因信息**：工作安排、个人原因、公司安排等公告表述
* **履历信息**：新任经理从业年限、过往岗位、现任职务等

### 4. 输入质量要求

* **文本完整**：尽量包含标题、正文、落款日期等关键部分
* **字段清晰**：经理姓名、基金名称、日期信息应可辨识
* **格式可读**：若来自 OCR，建议先清洗明显错字与断行问题
* **来源可信**：建议使用正式公告文本，避免二次转载摘要

> 说明：本 Skill 的能力边界是“解读已提供的公告内容”。若公告未披露关键信息，本 Skill 会明确标注“公告未披露”或“无法仅凭公告确认”。

## 功能

本 Skill 聚焦基金经理变更事件，提供结构化提取与投研视角分析。

### 1. 公告字段提取

* **基金信息识别**：提取基金名称、基金代码、基金简称
* **事件类型识别**：识别增聘、离任、增聘兼离任等情形
* **时间节点提取**：提取公告日期、生效日期、任职/离任日期
* **人员信息提取**：提取新任、离任及共同管理经理姓名
* **原因字段识别**：提取公告中披露的变更原因

### 2. 事件摘要生成

* **核心内容概括**：用简明语言总结变更事项
* **管理安排概括**：说明是否存在接续安排或共管安排
* **信息缺口提示**：指出公告中未披露的重要信息

### 3. 投研视角解读

* **管理稳定性分析**：评估产品管理连续性是否受影响
* **风格延续性分析**：分析投资框架延续或扰动的可能性
* **团队支持分析**：判断是否有平台化投研或共管缓冲
* **潜在影响判断**：形成偏正面 / 中性 / 偏负面的初步结论

### 4. 影响等级评估

* **高影响事件识别**：如核心经理离任、缺乏接替安排等
* **中影响事件识别**：如离任同时增聘、共管结构调整等
* **低影响事件识别**：如单纯增聘、辅助性共管补充等

### 5. 跟踪建议输出

* **持仓跟踪建议**：关注后续季报、定期报告中的配置变化
* **经理履历跟踪**：关注新任经理过往产品管理经验
* **公司层面跟踪**：关注基金公司投研团队稳定性与后续人事变动

### 6. 异常输入处理

* **非经理变更公告识别**：若文本并非经理变更公告，给出提示
* **信息残缺处理**：输出可识别字段，并标注缺失项
* **表述模糊处理**：对无法确认的事实保持审慎，不做虚构补全

## 注意事项与限制

### 1. 信息边界限制

* 仅基于用户提供的公告文本进行解读
* 对公告未披露信息不进行主观补全
* 不直接输出未经文本支持的确定性判断

### 2. 分析结论属性

* 输出属于**公告解读与投研初判**
* 不构成投资建议或收益承诺
* 影响判断应以审慎、克制表述为主

### 3. 文本质量影响

* OCR 错字、断行、表格错位可能影响提取准确性
* 若经理姓名、日期、基金名称缺失，分析完整性会下降
* 非正式摘要文本可能遗漏关键上下文

### 4. 事件解读局限

* 公告通常只披露变更事实，不一定披露深层原因
* 风格是否变化，往往需要结合持仓与定期报告进一步确认
* 新任经理实际影响大小，还受基金公司内部决策机制影响

### 5. 使用范围限制

* 本 Skill 主要面向**基金经理变更类公告**
* 对分红、清盘、合同修订、份额持有人大会等公告不作为最佳模板
* 若输入文本不属于经理变更公告，应提示任务不匹配

## 参考输出规范

建议输出时按以下结构组织结果：

### 1. 公告要点

* 基金名称
* 基金代码
* 公告日期
* 生效日期
* 事件类型
* 离任经理
* 新任经理
* 变更原因
* 共同管理安排

### 2. 事件摘要

* 用 2 至 3 句话概括核心变动
* 优先说明“谁离任、谁接任、是否共管”

### 3. 投研解读

* 管理稳定性
* 风格延续性
* 团队与平台支持
* 综合影响判断

### 4. 结论输出

* 影响等级：高 / 中 / 低
* 跟踪建议：1 至 3 条
* 局限说明：明确分析边界

## 适用场景

* **基金研究员**：快速阅读和初筛经理变更公告
* **财富管理人员**：辅助判断客户持有基金的管理层变动风险
* **内容运营人员**：生成标准化公告解读内容
* **内部风控/投顾支持**：沉淀经理变更事件标签与跟踪记录
