# 关键证据提炼指引

## 已确认事实
- 仅写有明确材料支持、且前后基本一致的事实。
- 如事实依赖单一来源且边界不清，可在事实后补一句“目前主要依据某材料，仍需进一步核实”。

## 关键证据
- 优先保留最能支持审核判断的证据，不重复堆砌同类材料。
- 每类证据可概括为“材料名称 + 能支持的关键点 + 完整性状态”。

## 待核实事项
- 对材料冲突、关键页缺失、归属不清、真实性疑点等单列整理。
- 对待核实事项尽量写明“为什么需要核实”。

## 压缩原则
- 重复病历、重复说明、低价值描述可合并归纳。
- 与审核判断关联弱的细节不必展开。
- 结论型表达应建立在证据基础上，不凭常识补足。
