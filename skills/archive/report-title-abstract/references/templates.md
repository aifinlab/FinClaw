# 研报标题与摘要输出模板

## 标题选项模板

```markdown
# 【[公司名] 研报标题选项】

## 选项 1（观点型）
[公司名]：[核心观点]

## 选项 2（事件型）
[公司名]：[事件] + [影响]

## 选项 3（数据型）
[公司名]：[关键数据] + [判断]

## 选项 4（预期差型）
[公司名]：[市场误解] + [我们的观点]

## 选项 5（估值型）
[公司名]：[估值状态] + [建议]

## 推荐
推荐选项 [X]，理由：[说明]
```

## 摘要模板

```markdown
# 【[公司名] 摘要】

## 核心观点
[100-150 字]

## 关键论据
• [论据 1 + 数据]
• [论据 2 + 数据]
• [论据 3 + 数据]

## 投资建议
- 评级：[买入/增持/中性/减持]
- 目标价：[XX 元]
- 上行空间：[XX%]

## 风险提示
- [风险 1]
- [风险 2]
```

## 标题禁忌

| 禁忌 | 示例 | 建议修改 |
|------|------|----------|
| 夸大其词 | "暴涨在即" | "有望上涨" |
| 绝对表述 | "一定翻倍" | "上行空间较大" |
| 空洞无物 | "深度分析" | 删除或具体化 |
