# 交易成本分析/TCA助手

## 数据获取

通过 cn-stock-data skill 获取数据：
- **成交记录**: 历史成交明细
- **行情数据**: 基准价格序列
- **费率数据**: 佣金/印花税/过户费

## 分析工作流

### Step 1: 显性成本
- 佣金：买卖双向，通常万1.5-万3
- 印花税：卖出单边千1(2023年减半至万5)
- 过户费：沪市万0.2(深市无)
- 总显性成本：单次买卖约0.05-0.1%

### Step 2: 隐性成本
- 市场冲击：大单推动价格的成本
- 时机成本：从决策到执行的价格变动
- 价差成本：买卖报价之间的差距
- 机会成本：未成交部分的潜在收益损失

### Step 3: TCA分析框架
- Implementation Shortfall = 纸面收益 - 实际收益
- IS分解 = 延迟成本 + 冲击成本 + 时机成本 + 机会成本
- 按时段/标的/策略分类统计
- 与行业基准(peer benchmark)对比

### Step 4: 成本优化建议
- 佣金谈判：大资金可谈到万1以下
- 算法执行：降低冲击成本
- 执行时段优化：选择流动性最好的时段
- 换手率控制：降低不必要的交易频率

### Step 5: 输出报告

## 输出格式

### formal 风格（研报级）
```
# 交易成本分析(TCA)报告

## 一、成本汇总
| 成本类型 | 金额 | 占比 |
|---------|------|------|

## 二、IS分解
[各成本成分明细]

## 三、对标分析
[与基准对比]

## 四、优化建议
```

### brief 风格（快速分析）
```
## TCA速览
- 总交易成本 0.18%/次
- 显性0.06% + 隐性0.12%
- 冲击成本占隐性成本65%
- 建议：使用VWAP算法可降低冲击30%
```
