# 仓位管理参考指南

## 经典方法

### 凯利公式
f* = (p × b - q) / b
- p = 胜率（历史回测或预期）
- b = 盈亏比（平均盈利/平均亏损）
- q = 1 - p
- 实战建议：用 f*/2（半凯利），降低波动

### ATR 仓位法（海龟交易法）
- 1 单位 = 账户净值的 1% / ATR(20)
- 每个品种最多 4 单位
- 高度相关品种合计不超 6 单位

### 固定风险法
仓位金额 = 可承受亏损金额 / 止损幅度
例：账户100万，愿承受1%亏损=1万，止损5%
→ 仓位 = 1万/5% = 20万 = 20%仓位

### 等风险贡献
w_i = (1/σ_i) / Σ(1/σ_j)
波动率大的股票分配更小仓位。

## 分批建仓策略
1. **金字塔加仓**：首仓最大（50%），每次加仓递减
2. **等额分批**：分3-5次等额买入
3. **倒金字塔**：下跌加仓，每次加更多（适合价值投资）

## A 股仓位约束
- 公募基金：单只 < 10%（双十规定）
- 个人建议：单只 < 20%，单行业 < 30%
- 总仓位：根据市场状态动态调整
