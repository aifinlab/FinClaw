# 责任分工矩阵

## 一、中介机构职责

### 1.1 保荐人/主承销商
- 牵头协调各中介机构
- 业务尽调与核查
- 财务数据分析
- 募投项目论证
- 申报材料制作
- 问询回复统筹

### 1.2 律师事务所
- 法律尽调
- 历史沿革核查
- 资产权属核查
- 重大合同审查
- 诉讼仲裁核查
- 出具法律意见

### 1.3 会计师事务所
- 财务审计
- 财务尽调
- 内控审计
- 财务数据核查
- 出具审计报告

### 1.4 评估机构
- 资产评估
- 出具评估报告
- 评估方法说明

## 二、问询回复责任分工

| 问题类型 | 主责方 | 配合方 |
|----------|--------|--------|
| 业务类 | 保荐人 | 发行人 |
| 财务类 | 会计师 | 保荐人、发行人 |
| 法律类 | 律师 | 保荐人、发行人 |
| 募投类 | 保荐人 | 发行人、可研机构 |
| 行业数据 | 保荐人 | 第三方机构 |
| 综合类 | 保荐人 | 律师、会计师、发行人 |

## 三、核查程序分工

### 3.1 客户/供应商访谈
| 角色 | 职责 |
|------|------|
| 保荐人 | 组织、主导访谈 |
| 律师 | 参与、法律事项核查 |
| 会计师 | 参与、财务事项核查 |

### 3.2 函证程序
| 角色 | 职责 |
|------|------|
| 会计师 | 执行函证程序 |
| 保荐人 | 独立函证、复核 |
| 律师 | 配合（如需要） |

### 3.3 实地走访
| 角色 | 职责 |
|------|------|
| 保荐人 | 组织、主导走访 |
| 律师 | 参与、法律事项核查 |
| 会计师 | 参与、财务事项核查 |

## 四、文件签字责任

| 文件类型 | 签字人员 |
|----------|----------|
| 招股说明书 | 保荐代表人、发行人董监高 |
| 法律意见书 | 签字律师、律所负责人 |
| 审计报告 | 签字会计师、事务所负责人 |
| 评估报告 | 签字评估师、评估机构负责人 |
| 问询回复 | 保荐代表人、其他中介（如需要） |

## 五、沟通协调机制

### 5.1 定期会议
- 周例会：项目进度汇报
- 专题会：重大问题讨论
- 协调会：中介机构协调

### 5.2 信息共享
- 工作群：日常沟通
- 邮件：正式沟通
- 共享盘：文件共享

### 5.3 问题升级
- 一般问题：项目组内部解决
- 重要问题：项目负责人协调
- 重大问题：机构管理层沟通
