# 结论口径指引

仅使用以下四类结论：

## 可能属于责任范围

适用情形：

- 已知事实与重疾定义匹配度较高。
- 当前未见明确免责或重大限制因素。
- 材料基本足以支持责任启动的初步判断。

写法要求：

- 同时写明仍需关注的边界，例如完整条款核验、多次赔付限制或等待期复核。

## 可能部分匹配但需进一步核验

适用情形：

- 疾病名称或部分定义条件已接近匹配，但关键严重程度、病理、分期分级、手术条件或结果条件仍待核实。
- 已知事实提示可能命中轻症、中症或重疾中的某一层级，但层级尚未完全明确。

写法要求：

- 说明哪些定义条件较可能满足，哪些条件仍缺关键证据。

## 可能不属于责任范围

适用情形：

- 已知事实与重疾定义明显不匹配。
- 已出现较强的等待期、既往症、免责、层级不符或责任模块不匹配信号。

写法要求：

- 说明这是基于当前信息的初步判断，不等同于最终拒赔结论。

## 暂无法判断

适用情形：

- 条款缺失、定义不清、确诊未明、病理待回报、关键材料不足或记录冲突。
- 结论高度依赖待补充的病理、影像、手术记录、分期分级、等待期信息或完整条款。

写法要求：

- 明确列出缺什么、为什么缺、补齐后应重点核验什么。
