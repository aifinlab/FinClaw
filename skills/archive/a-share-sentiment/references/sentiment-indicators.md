# A股市场情绪指标体系

## 一、8大情绪维度

| 维度 | 指标 | 乐观信号 | 悲观信号 | 权重 |
|------|------|---------|---------|------|
| 涨跌广度 | 涨跌比(上涨家数/下跌家数) | >2:1 | <1:2 | 15% |
| 极端行情 | 涨停数-跌停数 | 涨停>50 | 跌停>50 | 10% |
| 成交热度 | 两市成交额/20日均值 | >120% | <80% | 15% |
| 换手率 | 全市场平均换手率 | >3% | <1% | 10% |
| 外资态度 | 北向资金5日累计 | 净买入>100亿 | 净卖出>100亿 | 15% |
| 杠杆水平 | 融资余额5日变化 | 连续增长 | 连续下降 | 10% |
| 赚钱效应 | 近5日涨幅中位数 | >3% | <-3% | 15% |
| 新股表现 | 近期新股首日涨幅 | >100% | <50% | 10% |

## 二、评分计算方法

### 总体原则
- 每个维度独立评分 0-100 分
- 加权平均得到综合评分
- 综合评分 = Σ(维度评分 × 权重)

### 各维度评分映射

#### 涨跌比评分（权重15%）
| 涨跌比 | 评分 |
|--------|------|
| >3:1 | 100 |
| 2:1 | 75 |
| 1:1 | 50 |
| 1:2 | 25 |
| <1:3 | 0 |

中间值线性插值。

#### 极端行情评分（权重10%）
- 净涨停数 = 涨停数 - 跌停数
- 净涨停>80 → 100分
- 净涨停=50 → 80分
- 净涨停=0 → 50分
- 净跌停=50 → 20分
- 净跌停>80 → 0分

#### 成交热度评分（权重15%）
- 成交额比值 = 当日成交额 / 20日均值
- >150% → 100分
- 120% → 75分
- 100% → 50分
- 80% → 25分
- <60% → 0分

#### 换手率评分（权重10%）
- >4% → 100分
- 3% → 75分
- 2% → 50分
- 1% → 25分
- <0.5% → 0分

#### 北向资金评分（权重15%）
- 5日累计净买入 >150亿 → 100分
- 100亿 → 80分
- 0 → 50分
- -100亿 → 20分
- <-150亿 → 0分

#### 杠杆水平评分（权重10%）
- 5日融资余额连续增长且增幅>2% → 100分
- 连续增长 → 75分
- 基本持平 → 50分
- 连续下降 → 25分
- 连续下降且降幅>2% → 0分

#### 赚钱效应评分（权重15%）
- 近5日涨幅中位数 >5% → 100分
- 3% → 75分
- 0% → 50分
- -3% → 25分
- <-5% → 0分

#### 新股表现评分（权重10%）
- 近期新股首日涨幅 >150% → 100分
- 100% → 75分
- 50% → 50分
- 20% → 25分
- 破发 → 0分

## 四、情绪周期与操作含义

| 情绪区间 | 名称 | 历史含义 | 策略参考 |
|----------|------|---------|---------|
| 0-20 | 极度恐慌 | 往往对应重要底部区域，但不代表不会更低 | 逆向思维，分批关注低估值标的 |
| 20-40 | 偏悲观 | 市场信心不足，赚钱效应差 | 防御为主，关注高股息/低估值 |
| 40-60 | 中性 | 市场正常运转，多空相对均衡 | 正常配置，均衡策略 |
| 60-80 | 偏乐观 | 趋势向好，赚钱效应尚可 | 趋势跟随但需控制仓位 |
| 80-100 | 极度贪婪 | 往往对应阶段性顶部 | 保持警惕，逐步降低仓位 |

## 五、A股情绪特有指标

### 成交额关口
- **<7000亿**：地量水平，市场极度冷清
- **7000亿-1万亿**：偏冷淡
- **1万亿-1.3万亿**：正常活跃
- **1.3万亿-1.5万亿**：较为活跃
- **1.5万亿-2万亿**：火热
- **>2万亿**：极度亢奋

### 涨停板数量
- **<20家**：市场冰点
- **20-50家**：正常水平
- **50-100家**：活跃
- **>100家**：热度极高

### 北向资金单日净流入
- **>100亿**：强烈看多信号
- **50-100亿**：偏积极
- **±50亿**：正常波动
- **-50到-100亿**：偏消极
- **<-100亿**：强烈看空信号

### 新股表现
- 新股上市首日表现是散户情绪的温度计
- 首日涨幅普遍>100%：市场亢奋
- 频繁破发：市场极度悲观

## 六、注意事项

1. **结合使用**：情绪指标需要与技术面、基本面结合使用，单一情绪指标不构成投资建议
2. **结构性行情失真**：部分板块热+部分板块冷时，全市场情绪指标可能失真，需分板块分析
3. **政策敏感性**：政策驱动的情绪变化可能非常快，一个重磅政策可以一日扭转情绪
4. **节假日效应**：长假前后情绪波动大，需考虑资金回流/避险因素
5. **外部冲击**：海外市场大跌、地缘政治事件等可能导致情绪指标急剧变化
6. **数据滞后**：部分数据（如融资余额）有T+1延迟，需注意时效性
