# Global Listed Material Completeness Check Skill

## 数据来源

本 Skill 使用**公网可查的公开数据**，不依赖企业内部系统，默认以中国 A 股上市公司为目标对象，核心来源如下：

1. **国家法律法规数据库（全国人大）**
   - 用途：查询上位法与现行法律法规文本。
   - 典型规则来源：
     - 《中华人民共和国证券法》
   - 网址：`https://flk.npc.gov.cn/`

2. **中国证监会（CSRC）官网**
   - 用途：查询上市公司信息披露监管规则、内容与格式准则。
   - 典型规则来源：
     - 《上市公司信息披露管理办法（2025 修订，自 2025-07-01 起施行）》
     - 《公开发行证券的公司信息披露内容与格式准则第 2 号——年度报告的内容与格式（2021 年修订）》
     - 《公开发行证券的公司信息披露内容与格式准则第 3 号——半年度报告的内容与格式（2021 年修订）》
   - 网址：`https://www.csrc.gov.cn/`

3. **上海证券交易所（SSE）官网**
   - 用途：查询上市规则、信息披露业务办理规则、公告格式与备查文件要求。
   - 典型规则来源：
     - 《上海证券交易所上市公司自律监管指南第 1 号——公告格式（2025 年 8 月修订）》
     - 《上海证券交易所上市公司自律监管指引第 1 号——规范运作（2025 年修订）》
   - 网址：`https://www.sse.com.cn/`

4. **深圳证券交易所（SZSE）官网**
   - 用途：查询上市规则、公告格式、定期报告报送和备查文件要求。
   - 典型规则来源：
     - 《深圳证券交易所上市公司自律监管指南第 2 号——公告格式（2025 年修订）》
     - 《深圳证券交易所创业板股票上市规则（2025 年修订）》
     - 《深圳证券交易所上市公司自律监管指南第 1 号——业务办理》
   - 网址：`https://www.szse.cn/`

5. **巨潮资讯（CNINFO）**
   - 用途：抓取上市公司公开披露公告、定期报告、附件、决议公告等。
   - 典型使用：
     - 历史公告检索
     - 定期报告与配套公告的完整性校验
   - 网址：`https://www.cninfo.com.cn/`

> 说明：本 Skill 以**公开披露材料的“是否齐备”**为主，不替代律师、审计师、保荐机构、交易所或监管机关的正式法律意见。

## 功能

本 Skill 面向**某一个上市企业**，做“材料完整性校验”，重点校验两类问题：

### 1. 规则基线校验
将企业某一报告期、某一事项类型所应具备的公开材料，与公开规则要求进行比对，形成检查清单。当前内置：
- `annual_report`：年度报告材料包
- `semiannual_report`：半年度报告材料包

### 2. 公网公告检索与匹配
通过巨潮资讯查询企业公告标题与披露时间，识别是否已披露：
- 年度报告 / 半年度报告全文
- 摘要（如适用）
- 董事会决议公告
- 监事会决议公告
- 审计报告（年报场景）
- 内部控制相关文件（年报场景，按规则与公司情况提示）
- 利润分配预案相关公告（如标题中出现相关信号）
- 其他与定期报告常见配套材料相关的提示项

### 3. 风险输出
输出 3 类结论：
- **PASS**：核心公开材料齐备
- **WARN**：材料可能不完整，或仅匹配到弱证据
- **FAIL**：关键材料缺失

### 4. 可追溯报告
生成 JSON / Markdown 风格结果，列明：
- 公司代码、公司名称、年份、报告类型
- 命中的公告标题与发布时间
- 缺失项、疑似项、通过项
- 对应规则依据与来源链接

## 使用示例

### 示例 1：校验某上市公司 2025 年年度报告材料是否完整

```bash
python script/main.py   --company-code 300433   --company-name 蓝思科技   --report-type annual_report   --year 2025
```

### 示例 2：校验某上市公司 2025 年半年度报告材料是否完整

```bash
python script/main.py   --company-code 600519   --company-name 贵州茅台   --report-type semiannual_report   --year 2025
```

### 示例 3：输出到本地 JSON 文件

```bash
python script/main.py   --company-code 000001   --company-name 平安银行   --report-type annual_report   --year 2025   --output result.json
```

## 交易说明

1. 本 Skill 的“交易”含义是**执行校验流程**，不是证券买卖建议。
2. 建议使用流程：
   - 输入上市公司代码与名称；
   - 选择 `annual_report` 或 `semiannual_report`；
   - 自动抓取公网公告；
   - 基于证监会 / 交易所规则进行材料完整性比对；
   - 输出缺失项与疑点项。
3. 本 Skill **不构成法律意见、投资建议、审计结论或合规背书**。
4. 若企业存在特殊披露豁免、行业特殊规则、境内外双重上市差异、保密豁免、延迟披露情形，结果应由人工复核。
5. CNINFO、交易所页面结构可能调整；如页面变更导致抓取失败，应更新 `script/fetch_public_data.py` 中的解析逻辑或请求头。
