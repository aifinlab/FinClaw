# 一致性核查清单

## 一、财务数据一致性

### 1.1 报表间一致性
- [ ] 资产负债表与利润表勾稽关系正确
- [ ] 现金流量表与资产负债表勾稽关系正确
- [ ] 所有者权益变动表与其他报表一致

### 1.2 文件间一致性
- [ ] 招股书与审计报告数据一致
- [ ] 招股书与验资报告数据一致
- [ ] 招股书与评估报告数据一致
- [ ] 法律意见书与招股书数据一致

### 1.3 期间一致性
- [ ] 报告期各期数据连续
- [ ] 期初期末数据衔接
- [ ] 同比数据口径一致

## 二、业务数据一致性

### 2.1 产能数据
- [ ] 招股书与可研报告产能数据一致
- [ ] 产能与产量数据匹配
- [ ] 产能利用率计算正确

### 2.2 销售数据
- [ ] 销量与收入数据匹配
- [ ] 单价计算正确
- [ ] 客户数据与合同一致

### 2.3 采购数据
- [ ] 采购量与成本数据匹配
- [ ] 供应商数据与合同一致
- [ ] 采购单价合理

- [ ] 涉案金额一致
- [ ] 进展状态一致

## 七、常见问题

| 问题类型 | 常见表现 | 预防措施 |
|----------|----------|----------|
| 数据不一致 | 同一数据在不同文件数值不同 | 建立数据台账，统一来源 |
| 表述不一致 | 同一事项描述不同 | 统一表述模板 |
| 更新不及时 | 部分文件更新，部分未更新 | 建立变更通知机制 |
| 引用错误 | 引用过时数据或文件 | 建立引用核查清单 |
