# 条款类别划分框架

按以下顺序划分条款：

## 1. 责任条款

- 直接规定哪些保险事故、疾病、费用或给付责任在保障范围内。
- 常用于判断责任触发和给付前提。

## 2. 定义条款

- 对疾病名称、身故情形、意外标准、住院标准、手术标准、合理且必要等概念做定义。
- 常用于核对案件事实是否满足责任条款中的关键概念。

## 3. 免责条款

- 对不承担保险责任的情形进行规定。
- 常用于排查除外责任、争议点和拒赔风险。

## 4. 边界限制条款

- 对等待期、医院范围、免赔额、比例、限额、医保目录、自费项目、材料要求、多次赔付、分组和间隔期等做限制。
- 常用于判断赔付范围和审核边界。

## 5. 特别约定条款

- 主附险组合、个别约定、销售版本差异、特殊责任调整等内容。
- 常用于处理版本差异和个案特殊规则。
