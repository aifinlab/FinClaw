# Global Listed Clause Conflict Check

## 1. 数据来源

本 Skill 使用**公网可查数据**与**企业公开材料**，对某个上市企业的章程、治理制度、公告文本进行条款冲突校验。

主要数据来源：

1. **国家法律法规数据库 / 全国人大官网**
   - 《中华人民共和国公司法》
2. **中国证监会官网 / 证监会法规库**
   - 《上市公司章程指引（2025年修订）》
   - 与公司治理、信息披露、上市公司运作相关的公开规则与说明
3. **证券交易所官网**
   - 上海证券交易所上市公司自律监管规则
   - 深圳证券交易所上市公司自律监管规则
4. **企业公开材料**
   - 公司章程
   - 股东会议事规则、董事会议事规则、独立董事制度、关联交易制度、对外担保制度等
   - 定期报告、临时公告、修订公告
5. **用户提供的公开文本或公网链接**
   - txt / md / html / pdf 文件
   - 企业官网或交易所/巨潮资讯可访问链接

> 说明：本 Skill 默认只使用公开可访问内容，不接入企业内部非公开文档。

---

## 2. 功能

### 核心目标

对某个上市企业公开材料中的条款进行**法律法规冲突校验**，输出：
- 冲突项
- 疑似冲突项
- 缺失项
- 风险等级
- 法规依据
- 修订建议

### 当前支持的检查方向

1. **治理结构冲突检查**
   - 股东会、董事会、经理层、审计委员会/监事会之间的职权边界是否冲突
2. **法定代表人条款检查**
   - 法定代表人产生机制、变更机制、权限表述是否与现行规则冲突
3. **股东权利与表决机制检查**
   - 股东知情权、表决权、提案权、累积投票或特别决议表述是否明显冲突
4. **董事/高级管理人员义务检查**
   - 忠实义务、勤勉义务、关联关系披露、回避机制是否存在冲突或遗漏
5. **关联交易与对外担保检查**
   - 审议权限、回避要求、披露要求、授权口径是否矛盾
6. **利润分配与资本运作条款检查**
   - 现金分红、回购、减资、注册资本等相关表述是否与现行法规冲突
7. **信息披露责任检查**
   - 责任主体、审议程序、披露触发条件相关条款是否存在明显不一致
8. **内部文本互相冲突检查**
   - 例如《公司章程》与《董事会议事规则》之间出现权限或程序冲突

### 输出形式

默认输出 JSON，包括：
- `company_name`
- `risk_level`
- `summary`
- `findings`
- `sources`

## 3. 使用示例

### 安装依赖

```bash
pip install -r requirements.txt
```

### 示例 1：校验本地章程文本

```bash
python script/main.py \
  --company "某上市公司" \
  --material-file ./examples/company_charter.txt
```

### 示例 2：校验公网链接

```bash
python script/main.py \
  --company "某上市公司" \
  --material-url "https://example.com/company-charter.html"
```

### 示例 3：同时校验多个公开材料

```bash
python script/main.py \
  --company "某上市公司" \
  --material-file ./examples/company_charter.txt \
  --extra-file ./examples/board_rules.txt \
  --extra-url "https://example.com/guarantee-policy.html"
```

### 示例输出

```json
{
  "company_name": "某上市公司",
  "risk_level": "high",
  "summary": "发现 2 项明显冲突、3 项疑似冲突、1 项缺失项。",
  "findings": [
    {
      "rule_id": "RULE_AUDIT_OR_SUPERVISOR",
      "severity": "high",
      "title": "监督机构表述可能与现行章程指引不一致",
      "evidence": "材料中仍以旧式监事会必设表述为唯一机制",
```

## 4. 交易说明

1. 本 Skill 仅用于**公开材料的一致性与合规性辅助检查**。
2. 输出结果不构成法律意见、审计意见、投资建议或交易建议。
3. 对上市企业进行条款冲突校验时，应以：
   - 最新生效法律法规
   - 公司最新公告版本
   - 交易所最新规则
   - 公司实际审议与披露文件
   为最终判断依据。
4. 若用于证券投资、并购、尽调、投票、治理整改等场景，建议由律师、董办、证券事务、内控或合规团队复核。
5. Skill 对 PDF/HTML/文本抽取采用程序化解析，可能受网页结构、OCR 质量、公告版本差异影响，存在漏检与误检风险。
