# 贷款明细报送数据校验报告

## 校验规则说明

| 字段 | 规则 |
|------|------|
| 贷款编号 | 必填，格式 LN+10位数字 |
| 借款人名称 | 必填 |
| 证件号码 | 必填，18位身份证号 |
| 贷款金额（万元） | 必填，正数，保留2位小数 |
| 贷款余额（万元） | 必填，非负数，且 <= 贷款金额 |
| 发放日期 | 必填，格式 YYYY-MM-DD |
| 到期日期 | 必填，格式 YYYY-MM-DD，须晚于发放日期 |
| 利率（%） | 必填，正数 |
| 五级分类 | 必填，取值范围{正常, 关注, 次级, 可疑, 损失} |
| 担保方式 | 必填，取值范围{信用, 保证, 抵押, 质押, 组合} |
