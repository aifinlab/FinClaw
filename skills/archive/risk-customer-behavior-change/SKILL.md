# 风险客户行为变化分析助手

## 1. 业务问题

高风险客户的行为并非静态不变，其交易习惯、账户使用方式、渠道偏好和产品持有情况可能随时间发生显著变化。合规团队需要对这些变化进行持续监测，及时发现行为突变或渐变趋势。本技能提供多维度行为变化分析，辅助识别需要关注的行为转变。

## 2. 适用场景

- 高风险客户定期行为复核（季度/半年度）
- 风险等级调整前的行为变化评估
- 客户触发预警后的历史行为回溯分析
- 解除风险标签前的行为稳定性验证
- 关联客户群体的行为变化趋势比对

## 3. 输入

| 输入项 | 说明 | 是否必需 |
|--------|------|----------|
| 客户标识 | 客户号或账户号 | 必需 |
| 交易流水数据 | 覆盖分析窗口期的完整交易记录 | 必需 |
| 分析时间窗口 | 行为变化分析的起止时间和对比基准期 | 必需 |
| 账户信息 | 账户数量、类型、状态变化等 | 可选 |
| 渠道使用记录 | 各渠道的登录和操作记录 | 可选 |
| 产品持有信息 | 产品签约、赎回、变更记录 | 可选 |
| 客户风险等级 | 当前及历史风险评级 | 可选 |

## 4. 输出

| 输出项 | 说明 |
|--------|------|
| 行为变化摘要 | 各维度行为变化的核心结论 |
| 交易行为变化 | 交易频率、金额、对手方、时段等的变化趋势 |
| 账户行为变化 | 账户开销户、余额波动、账户用途变化等 |
| 渠道行为变化 | 渠道偏好迁移、新渠道启用等 |
| 产品行为变化 | 产品持有结构变化、新产品签约等 |
| 变化显著性标注 | 标注变化幅度显著的维度和时间节点 |

## 5. 默认分析维度

- **交易维度**：交易频率趋势、单笔金额变化、累计金额变化、交易对手方变化、交易时段迁移
- **账户维度**：新开/销户行为、余额均值变化、账户活跃度变化、多账户使用比例变化
- **渠道维度**：线上/线下渠道使用比例变化、新渠道首次使用、渠道切换频率变化
- **产品维度**：产品持有数量变化、高风险产品签约/退出、产品组合复杂度变化
- **时间维度**：行为突变时间节点、渐变趋势斜率、季节性模式变化

## 6. 默认分析框架

1. **基准期行为画像**：基于基准期数据，建立客户各维度行为画像
2. **观察期行为画像**：基于观察期数据，建立同口径行为画像
3. **逐维度变化计算**：对比基准期与观察期，计算各维度的变化方向和幅度
4. **变化显著性判断**：标注变化幅度超出阈值的维度
5. **变化模式归类**：区分突变型（短期剧烈变化）和渐变型（长期趋势性变化）
6. **多维度交叉分析**：检查多个维度是否同时发生协同变化
7. **结论先行输出**：先呈现行为变化核心结论，再展开各维度详情

## 7. 安全边界

- 不编造数据；不夸大结论；缺失信息标注"未获取"或"待核实"
- 不替代正式审批、正式风控、正式合规、正式报送、正式处置、正式责任认定、正式管理决策
- 不对交易或客户做"疑似洗钱""疑似欺诈"等定性表述
- 仅提供数据模式分析和复核参考
- 不替代正式合规复核、审计、调查或账户处置决策
- 不替代可疑交易报告（STR）的正式判定
- 行为变化分析仅反映数据层面的变动情况，不等同于风险等级变更依据
