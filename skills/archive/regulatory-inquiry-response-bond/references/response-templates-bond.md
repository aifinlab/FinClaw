# 债券问询回复模板

## 回复基本结构

```markdown
## 问题 [X]：[问题标题]

### 问题原文
[审核机构问询原文]

### 回复要点
1. [核心观点 1]
2. [核心观点 2]

### 回复正文

#### 一、[分论点 1]
[详细论述 + 数据支撑]

#### 二、[分论点 2]
[详细论述 + 数据支撑]

### 核查程序
1. [核查程序 1]
2. [核查程序 2]

### 核查结论
[明确结论]

### 附件材料
- [附件 1 名称]
```

## 债券回复特殊要点

### 1. 偿债能力论证

```
一、资产负债结构
[说明资产负债情况]

二、盈利能力
[说明盈利情况]

三、现金流状况
[说明现金流情况]

四、偿债指标
[列示偿债指标]

五、融资能力
[说明融资能力]
```

### 2. 信用风险揭示

```
一、风险因素
[披露风险因素]

二、风险缓释措施
[说明缓释措施]

三、投资者保护
[说明保护措施]
```

### 3. 募投项目论证

```
一、项目合规性
[说明合规情况]

二、投资估算
[说明投资情况]

三、经济效益
[说明效益情况]

四、风险分析
[说明风险情况]
```

## 回复注意事项

1. 突出偿债能力分析
2. 财务数据覆盖最近三年及一期
3. 关注表外负债和隐性债务
4. 与评级报告保持一致
5. 符合监管规则要求
