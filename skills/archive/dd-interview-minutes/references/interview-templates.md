# 访谈纪要模板库

## 模板一：高管访谈纪要

```markdown
# 访谈纪要

## 基本信息
| 项目 | 内容 |
|------|------|
| 项目名称 | [项目名称] |
| 访谈类型 | 高管访谈 |
| 访谈时间 | YYYY-MM-DD HH:MM-HH:MM |
| 访谈地点 | [地点] |
| 访谈人 | [姓名 1]、[姓名 2] |
| 被访谈人 | [姓名] |
| 被访谈人职务 | [职务] |
| 记录人 | [姓名] |

## 一、访谈目的
[说明本次访谈的主要目的和背景]

## 二、被访谈人简介
[被访谈人的教育背景、工作经历、在公司任职情况等]

## 三、访谈内容

### 3.1 公司战略与发展
**问**：请介绍公司的发展战略
**答**：[回答内容]

### 3.2 业务与经营
**问**：请说明公司的商业模式
**答**：[回答内容]

### 3.3 财务与业绩
**问**：请介绍公司的业绩情况
**答**：[回答内容]

## 四、关键信息摘要
- [核心观点 1]
- [核心观点 2]

## 五、待核实事项
| 序号 | 事项 | 核实方式 | 责任人 | 截止时间 |
|------|------|----------|--------|----------|
| 1 | [事项] | [方式] | [姓名] | [日期] |

## 六、签字确认
**访谈人签字**：____________    日期：____________
**被访谈人签字**：____________    日期：____________
```

## 模板二：客户访谈纪要

```markdown
# 客户访谈纪要

## 基本信息
[同上]

## 一、合作情况
**问**：何时开始与发行人合作
**答**：[回答内容]

## 二、采购情况
**问**：年采购金额及占比
**答**：[回答内容]

## 三、评价与意向
**问**：对产品质量和服务的评价
**答**：[回答内容]

## 四、访谈结论
[对本次访谈的整体评价]
```

## 模板三：供应商访谈纪要

结构同客户访谈，重点关注供应稳定性、定价机制、结算方式等。
