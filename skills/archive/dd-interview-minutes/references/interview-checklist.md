# 访谈准备清单

## 一、访谈前准备

### 1.1 资料准备
- [ ] 访谈提纲
- [ ] 被访谈人背景资料
- [ ] 公司业务资料
- [ ] 历史访谈记录（如有）

### 1.2 工具准备
- [ ] 录音设备
- [ ] 笔记本和笔
- [ ] 相机（现场访谈）
- [ ] 名片

### 1.3 预约确认
- [ ] 确认访谈时间
- [ ] 确认访谈地点
- [ ] 确认参会人员
- [ ] 发送访谈提纲

## 二、访谈中记录

### 2.1 基本信息
- [ ] 记录访谈时间、地点
- [ ] 记录参会人员
- [ ] 拍摄现场照片（如适用）
- [ ] 收集名片

### 2.2 内容记录
- [ ] 完整记录问答内容
- [ ] 标注关键信息
- [ ] 记录待核实事项
- [ ] 记录承诺事项

### 2.3 注意事项
- [ ] 保持客观中立
- [ ] 不诱导被访谈人
- [ ] 不泄露敏感信息
- [ ] 控制访谈时间

## 三、访谈后工作

### 3.1 纪要整理
- [ ] 及时整理访谈纪要
- [ ] 核对关键信息
- [ ] 标注待核实事项
- [ ] 形成正式文档

### 3.2 签字确认
- [ ] 请被访谈人签字确认
- [ ] 访谈人签字
- [ ] 记录人签字
- [ ] 归档保存

### 3.3 后续跟进
- [ ] 核实待核实事项
- [ ] 跟进承诺事项
- [ ] 更新尽调报告
- [ ] 归档访谈资料

## 四、访谈资料归档

### 4.1 归档内容
- [ ] 访谈纪要
- [ ] 访谈签到表
- [ ] 访谈照片
- [ ] 访谈录音（如适用）
- [ ] 被访谈人名片

### 4.2 归档要求
- [ ] 统一编号
- [ ] 分类存放
- [ ] 电子备份
- [ ] 保密管理
