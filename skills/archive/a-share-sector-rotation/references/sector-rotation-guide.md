# A股行业轮动参考指南

## 1. 修正版美林时钟（A股适配）

经典美林时钟基于美国经济周期，A股需修正：政策周期权重更大，信用周期（社融）领先经济周期。

| 阶段 | 宏观特征 | 优势行业 | 典型信号 |
|------|---------|---------|---------|
| **复苏** | PMI回升、社融放量、CPI低位、政策宽松 | 金融（银行/券商）、可选消费（白酒/家电）、地产链 | 社融增速拐点向上、降息降准落地 |
| **扩张** | PMI持续荣枯线上、企业盈利改善、信用扩张 | 科技（半导体/软件）、周期（有色/化工）、高端制造 | 工业增加值加速、PPI回升 |
| **过热** | CPI上行、PPI高位、产能利用率高 | 能源（煤炭/石油）、材料（钢铁/有色）、农业 | CPI>3%、PPI持续高位、央行收紧信号 |
| **衰退** | PMI跌破50、盈利下滑、信用收缩 | 公用事业、医药、必选消费（食品/农业）、高股息 | PMI连续下行、社融增速回落、避险情绪升温 |

**A股修正要点**：
- 政策底通常领先市场底1-3个月，复苏初期关注政策敏感型行业（券商/地产）
- 中国CPI波动相对温和，过热阶段更多看PPI和上游资源品
- 衰退期A股"喝酒吃药"特征明显（白酒+医药防御）
- 科技行业受产业政策影响大于经济周期，需单独考量

## 2. 大小盘风格判断

**核心指标**：沪深300 / 中证1000 比值（简称"大小盘比值"）

| 比值趋势 | 含义 | 背景 |
|----------|------|------|
| 持续上升 | 大盘股占优 | 经济下行/不确定性高，资金抱团龙头，流动性溢价 |
| 持续下降 | 小盘股占优 | 经济复苏/流动性充裕，风险偏好提升，题材活跃 |
| 横盘震荡 | 风格均衡 | 过渡期，不宜押注单一风格 |

**切换信号**：
- 比值偏离20日均线超过2%后回归 → 可能发生风格切换
- 大盘股连续跑赢超过3个月 → 关注小盘反扑机会
- 新增社融大幅超预期 → 通常利好小盘成长
- IPO/再融资收紧 → 利好存量小盘股

## 3. 价值成长风格判断

**核心指标**：上证50 / 创业板指 比值

| 比值趋势 | 含义 | 典型驱动 |
|----------|------|---------|
| 上升 | 价值占优 | 利率上行、经济复苏确认、高股息吸引力增强 |
| 下降 | 成长占优 | 利率下行、产业政策催化、科技创新周期启动 |

**A股规律**：
- 利率下行周期，成长风格通常占优（低利率→高久期资产受益）
- 经济强复苏确认后，价值风格通常占优（盈利驱动>估值驱动）
- 年初"春季躁动"偏成长题材，年末"估值切换"偏价值蓝筹

## 4. 行业动量 vs 均值回归

### 动量策略（追强）
- 买入近10-20日涨幅前5的行业，持有2-4周
- 适用场景：趋势行情、增量资金入场期
- 风险：高位接盘、行业拥挤度过高
- 加强信号：成交额同步放大、北向资金同向流入

### 均值回归策略（抄底）
- 买入近20日跌幅前5但基本面未恶化的行业
- 适用场景：震荡市、行业轮动加速期
- 风险：价值陷阱、基本面持续恶化
- 加强信号：行业PE分位数低于20%、出现放量止跌K线

### 选择依据
| 市场状态 | 推荐策略 | 原因 |
|----------|---------|------|
| 单边上涨 | 动量为主 | 强者恒强，追涨收益高 |
| 单边下跌 | 均谨慎 | 两种策略都容易亏损 |
| 宽幅震荡 | 均值回归 | 行业轮动快，低吸高抛有效 |
| 窄幅震荡 | 动量为主 | 突破方向的行业可能引领新趋势 |

## 5. A股行业轮动典型模式

### 三阶段模型：政策驱动 → 资金驱动 → 业绩验证

**第一阶段：政策驱动（1-2周）**
- 国务院/部委出台行业支持政策
- 相关行业指数快速拉升，游资主导
- 特征：涨停板多、换手率极高、龙头股辨识度强
- 操作：快进快出，抓政策第一波脉冲

**第二阶段：资金驱动（2-4周）**
- 机构资金开始介入，北向资金流入
- 行业内分化，真正受益标的开始跑赢
- 特征：成交额持续放大但换手率回落、龙头股走独立行情
- 操作：精选细分龙头，中期持有

**第三阶段：业绩验证（1-3个月）**
- 季报/半年报验证行业景气度
- 业绩兑现的继续上涨，证伪的快速回落
- 特征：行业内个股严重分化、业绩超预期股大涨
- 操作：跟踪业绩，去伪存真

### A股特色轮动路径
- **牛市初期**：券商 → 地产 → 银行 → 周期 → 科技 → 消费
- **牛市中后期**：题材概念 → 低价股 → 垃圾股（风险极高）
- **熊市**：医药/消费防御 → 高股息（银行/公用事业）→ 全面杀跌
- **政策市**：政策受益行业脉冲式上涨，持续性取决于政策力度

## 6. 关键宏观指标与行业映射

| 指标 | 方向 | 利好行业 | 利空行业 |
|------|------|---------|---------|
| PMI > 50且上升 | 经济扩张 | 周期/制造/可选消费 | 防御（公用/医药相对弱） |
| PMI < 50且下降 | 经济收缩 | 公用事业/医药/必选消费 | 周期/地产/材料 |
| 社融超预期 | 信用扩张 | 金融/地产/小盘成长 | 高股息防御（相对吸引力下降） |
| 社融不及预期 | 信用收缩 | 高股息/现金流好的行业 | 高杠杆/重资产行业 |
| CPI上行 | 通胀升温 | 食品饮料/农业/零售（提价逻辑） | 成本敏感型制造业 |
| PPI上行 | 工业通胀 | 上游资源（煤炭/有色/石油） | 中下游制造（成本挤压） |
| 利率下行 | 货币宽松 | 成长科技/地产/券商 | 银行（息差收窄） |
| 利率上行 | 货币收紧 | 银行/保险 | 高估值成长/地产 |
| 人民币升值 | 资本流入 | 航空/造纸（成本下降）/A股整体 | 出口型（纺织/家电出口） |
| 人民币贬值 | 资本流出 | 出口型/黄金 | 航空/进口依赖型 |

### 组合观察框架
同时观察2-3个指标的组合方向，比单一指标更可靠：
- **PMI↑ + 社融↑ + 利率↓** = 复苏确认 → 超配金融+可选消费
- **PMI↑ + PPI↑ + 利率↑** = 过热信号 → 超配资源+减配成长
- **PMI↓ + 社融↓ + 利率↑** = 滞胀风险 → 超配必选消费+现金
- **PMI↓ + 社融↑ + 利率↓** = 政策托底 → 超配政策受益行业（基建/新能源）
