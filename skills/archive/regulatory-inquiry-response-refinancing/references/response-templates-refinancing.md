# 再融资问询回复模板

## 回复基本结构

```markdown
## 问题 [X]：[问题标题]

### 问题原文
[交易所/证监会问询原文]

### 回复要点
1. [核心观点 1]
2. [核心观点 2]
3. [核心观点 3]

### 回复正文

#### 一、[分论点 1]
[详细论述 + 数据支撑]

#### 二、[分论点 2]
[详细论述 + 数据支撑]

### 核查程序
1. [核查程序 1]
2. [核查程序 2]

### 核查结论
[明确结论]

### 附件材料
- [附件 1 名称]
```

## 融资必要性回复模板

```
一、前次募集资金使用情况
[说明使用进度、效果]

二、本次融资必要性
1. 业务发展的资金需求
2. 优化财务结构的需要
3. 提升竞争力的需要

三、融资规模合理性
[详细测算过程]

四、融资时机适当性
[市场环境分析]
```

## 募投项目回复模板

```
一、项目基本情况
[项目名称、内容、投资额]

二、项目合规性
[立项、环评、用地等]

三、投资估算合理性
[测算依据、对比分析]

四、经济效益测算
[假设条件、测算过程、敏感性分析]

五、项目实施能力
[技术、人员、管理]

六、风险提示
[项目风险及应对]
```

## 摊薄影响回复模板

```
一、摊薄影响测算
[测算假设、过程、结果]

二、填补回报措施
1. 加强经营管理
2. 加快募投项目建设
3. 完善利润分配政策

三、相关承诺
1. 控股股东承诺
2. 董事/高管承诺
```

## 回复注意事项

1. 融资必要性论证要充分
2. 募投项目可行性要清晰
3. 财务测算要合理、假设要审慎
4. 摊薄影响分析要完整
5. 与前次披露要一致
6. 核查程序要充分
