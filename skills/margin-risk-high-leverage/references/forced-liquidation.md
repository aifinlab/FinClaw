# 强制平仓操作流程

## 一、平仓触发条件

### 1.1 维保比例平仓
| 触发条件 | 说明 | 平仓权限 |
|----------|------|----------|
| 维保比例<120% | 低于平仓线 | 可启动平仓 |
| 维保比例<130% 且 T+2 日未追保 | 警戒线且未追保 | 可启动平仓 |
| 维保比例<140% 且 T+5 日未追保 | 预警线且未追保 | 可启动平仓 |

### 1.2 合约到期平仓
| 触发条件 | 说明 | 平仓权限 |
|----------|------|----------|
| 合约到期未偿还 | 融资/融券合约到期 | 必须平仓 |
| 合约展期未获批 | 展期申请未通过 | 必须平仓 |

## 二、平仓前流程

### 2.1 通知流程
```
T 日：维保比例跌破平仓线
  ↓
发送平仓预告通知（短信 + 微信 + 电话）
  ↓
T+1 日：再次通知
  ↓
如仍未追保，准备平仓
  ↓
T+2 日：启动平仓程序
```

### 2.2 通知内容
| 通知类型 | 内容要点 | 通知方式 |
|----------|----------|----------|
| 首次通知 | 当前维保比例、需追加金额、截止日期 | 短信 + 微信 |
| 二次通知 | 强调平仓风险、最后期限 | 电话 + 短信 |
| 最后通知 | 明确平仓时间、平仓顺序 | 电话 + 书面 |

## 三、平仓执行流程

### 3.1 平仓顺序原则
1. **优先平仓证券**：
   - 高波动证券
   - 高集中度证券
   - 流动性好的证券
   - 盈利证券

2. **平仓顺序**：
   ```
   第一步：平仓融券负债（如有）
   第二步：平仓高波动证券
   第三步：平仓高集中度证券
   第四步：平仓其他证券
   ```

3. **平仓目标**：
   - 维保比例恢复至 150% 以上
   - 或清偿全部负债

### 3.2 平仓价格策略
| 策略 | 适用场景 | 说明 |
|------|----------|------|
| 市价委托 | 紧急情况 | 快速成交，价格不确定 |
| 限价委托 | 一般情况 | 价格确定，可能不成交 |
| 分批委托 | 大额平仓 | 减少市场冲击 |
| 算法交易 | 大额平仓 | 使用 VWAP/TWAP 等算法 |

## 四、平仓后流程

### 4.1 平仓结果通知
| 通知内容 | 说明 |
|----------|------|
| 平仓时间 | 平仓执行的时间 |
| 平仓证券 | 被平仓的证券列表 |
| 平仓数量 | 各证券平仓数量 |
| 平仓金额 | 各证券平仓金额 |
| 平仓价格 | 各证券平仓均价 |
| 平仓后维保比例 | 平仓后的维保比例 |
| 剩余持仓 | 平仓后剩余证券 |
| 剩余负债 | 平仓后剩余负债 |

## 五、特殊情况处理

### 5.1 无法平仓情形
| 情形 | 处理方式 |
|------|----------|
| 证券停牌 | 等待复牌后平仓，加强监控 |
| 证券跌停 | 继续委托，同时通知客户 |
| 流动性不足 | 分批平仓，延长平仓时间 |
| 司法冻结 | 与司法机关沟通，依法处理 |

### 5.2 客户投诉处理
| 投诉类型 | 处理方式 |
|----------|----------|
| 通知不到位 | 核查通知记录，如属实则道歉 |
| 平仓不及时 | 核查平仓记录，解释原因 |
| 平仓价格异议 | 解释市场情况，提供成交明细 |
| 平仓顺序异议 | 解释平仓原则，提供依据 |
