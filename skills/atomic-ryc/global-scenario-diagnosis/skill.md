# 企业场景诊断 Skill

## 数据来源
- 巨潮资讯公开公告检索平台
- 中国证监会公开法规与监管文书
- 上交所、深交所公开规则与监管动态
- 企业公开公告、定期报告、重大事项公告

## 功能
- 根据企业近期公开公告，对企业当前所处场景进行诊断
- 支持识别扩产/投资、融资/资本运作、经营承压、监管/合规、治理变动、诉讼/纠纷、产品/技术进展等场景
- 输出场景强度、支持信号数量和证据列表
- 可用于投研快筛、事件跟踪和企业经营态势监测

## 使用示例
```bash
python script/scenario_diagnosis.py --company 比亚迪
```

## 交易说明
- 本 Skill 只反映公开信息中的“场景线索密度”，不代表未来业绩、股价走势或监管结论。
- 诊断结果适合用于研究辅助，不应脱离财务、估值、产业链和市场数据单独使用。
- 重大投资决策应结合原始公告、财报和专业判断。
