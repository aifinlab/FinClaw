# 输入字段建议

## 最低必需材料

- 持仓产品与持有周期
- 收益、回撤和波动信息
- 近期市场或产品事件
- 客户主要问题

## 可选增强材料

- 归因结果
- 经理观点
- 历史沟通记录

## 建议输入结构

~~~json
{
  "task_name": "持仓问答助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
