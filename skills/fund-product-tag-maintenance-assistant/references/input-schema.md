# 输入字段建议

## 最低必需材料

- 产品基础资料
- 策略和持仓特征
- 现有标签体系
- 渠道或系统字段要求

## 可选增强材料

- 客户搜索词与高频问法
- 竞品标签
- 近期持仓变化

## 建议输入结构

~~~json
{
  "task_name": "产品标签维护助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
