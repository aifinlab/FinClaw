# 晨会纪要输出模板

## Style: formal（券商晨会格式）

```
# 晨会纪要 — YYYY年MM月DD日

## 摘要
[3-5 句话概括今日市场要点]

## 一、市场回顾

| 指数 | 收盘 | 涨跌幅 | 成交额（亿） |
|------|------|--------|-------------|
| 上证指数 | X,XXX.XX | +X.XX% | X,XXX |
| 深证成指 | XX,XXX.XX | +X.XX% | X,XXX |
| 创业板指 | X,XXX.XX | +X.XX% | X,XXX |

## 二、资金动向

**北向资金**: 净流入/流出 XX.XX 亿元（沪股通 XX.XX 亿，深股通 XX.XX 亿）

**行业资金流向**（如有数据）:
- 净流入前三: XX行业(+XX亿), XX行业(+XX亿), XX行业(+XX亿)
- 净流出前三: XX行业(-XX亿), XX行业(-XX亿), XX行业(-XX亿)

## 三、隔夜要闻

### 政策面
- [要闻1]（来源：XX）
- [要闻2]（来源：XX）

### 行业/公司
- [要闻3]（来源：XX）
- [要闻4]（来源：XX）

### 海外市场
- 美股: 道指 XX (+X.XX%), 纳指 XX (+X.XX%), 标普 XX (+X.XX%)
- [其他海外重要动态]

## 四、今日关注
- [关注点1: 重要经济数据发布/政策实施/个股事件]
- [关注点2]
- [关注点3]

---
*免责声明: 本纪要仅供内部参考，不构成投资建议。数据来源：东方财富、新浪财经、财联社。*
```

## Style: brief（个人投研笔记）

```
## 晨会 MM/DD

**大盘**: 沪指 X,XXX (+X.XX%), 深成指 XX,XXX (+X.XX%), 创业板 X,XXX (+X.XX%)
**北向**: 净流入 XX 亿
**成交**: 两市合计 X,XXX 亿

**要闻**:
1. [最重要的消息，一句话]
2. [第二重要]
3. [第三重要]

**今日看点**: [1-2 句个人判断/关注方向]
```
