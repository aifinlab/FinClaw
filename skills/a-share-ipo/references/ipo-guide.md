# A股 IPO 参考指南

## 1. IPO 流程概述

```
申报材料 → 受理 → 审核问询 → 上市委审议 → 证监会注册 → 询价定价 → 网上/网下申购 → 缴款验资 → 挂牌上市
```

关键时间节点：
- 受理到上市委审议：通常 3-6 个月（注册制下提速）
- 注册批文有效期：12 个月
- 询价到上市：约 2-3 周
- T日申购 → T+2日公布中签 → T+2日16:00前缴款

## 2. 各板块上市条件对比

| 条件 | 主板 | 科创板 | 创业板 | 北交所 |
|------|------|--------|--------|--------|
| 定位 | 大盘蓝筹 | 硬科技 | 成长型创新 | 创新型中小企 |
| 市值+财务标准 | 多套标准 | 5套标准 | 3套标准 | 4套标准 |
| 最低市值要求 | 不低于3亿 | 不低于10亿 | 不低于10亿 | 不低于2亿 |
| 盈利要求(标准一) | 近3年净利润均为正且累计≥1.5亿 | 近2年净利润均为正且累计≥5000万 | 近2年净利润均为正且累计≥5000万 | 近2年净利润均为正且累计≥1500万 |
| 研发投入(可替代盈利) | 不适用 | 近3年研发投入合计≥营收15% | 不适用 | 不适用 |
| 行业限制 | 无特殊限制 | 重点支持六大领域 | 支持创新企业 | 创新层挂牌满12个月 |
| 涨跌幅限制 | 上市首日44%，次日起±10% | 前5日不设限，之后±20% | 前5日不设限，之后±20% | 前5日不设限（30%临停），之后±30% |
| 投资者门槛 | 无 | 50万+2年 | 10万+2年 | 50万+2年 |

## 3. 打新资格与规则

### 3.1 市值要求
- **沪市新股**：T-2日前20个交易日日均持有沪市非限售A股市值 ≥ 1万元
  - 每1万元市值 → 1个申购单位（1000股）
- **深市新股**：T-2日前20个交易日日均持有深市非限售A股市值 ≥ 1万元
  - 每5000元市值 → 1个申购单位（500股）
- 沪深市值不互通，各算各的

### 3.2 申购限额
- 网上申购上限 = 网上发行总量 / 1000（取整）
- 一般占总发行量的千分之一
- 超出上限部分视为无效申购

### 3.3 缴款规则
- T+2日公布中签结果
- T+2日16:00前确保账户有足够资金（自动扣款）
- **弃购惩罚**：连续12个月内累计弃购 ≥ 3次 → 6个月内不得参与新股申购
- 弃购包括：未足额缴款、放弃认购

### 3.4 科创板/创业板/北交所额外要求
| 板块 | 资产门槛 | 交易经验 |
|------|----------|----------|
| 科创板 | ≥50万（证券资产） | ≥2年 |
| 创业板 | ≥10万（证券资产） | ≥2年 |
| 北交所 | ≥50万（证券资产） | ≥2年 |

## 4. 新股定价分析框架

### 4.1 发行PE法（最常用）
```
发行PE = 发行价 / 每股收益(EPS)
行业PE = 中证指数公司发布的行业近一个月静态市盈率

判断标准：
  发行PE < 行业PE × 0.8  → 明显低估，打新价值高
  发行PE ∈ [行业PE×0.8, 行业PE]  → 合理偏低，可参与
  发行PE ∈ [行业PE, 行业PE×1.2]  → 合理偏高，需看基本面
  发行PE > 行业PE × 1.2  → 偏高，破发风险大
```

### 4.2 可比公司法
1. 选取 3-5 家同行业、同规模上市公司
2. 计算可比公司 PE/PB/PS 中位数
3. 新股发行估值 vs 可比公司估值
4. 考虑新股流动性溢价/折价

### 4.3 现金流折现法（DCF）
- 适用于科技型/高成长企业
- 预测未来 5-10 年自由现金流
- 选取合理折现率（WACC）
- 实操中较少单独使用，常作为交叉验证

### 4.4 综合评级
| 评级 | 含义 | 操作建议 |
|------|------|----------|
| 强烈推荐申购 | 发行价明显低估，基本面优秀 | 顶格申购 |
| 建议申购 | 定价合理，基本面良好 | 正常申购 |
| 谨慎申购 | 定价偏高或基本面一般 | 小仓位或观望 |
| 不建议申购 | 定价过高，破发风险大 | 回避 |

## 5. 首日表现统计规律

### 5.1 核制（2023年之前）
- 主板新股首日涨幅上限44%（涨停）
- 连续涨停天数中位数：7-8天
- 开板日平均涨幅约150%-300%

### 5.2 注册制（2023年后全面注册制）
- 前5日无涨跌幅限制（科创板/创业板/北交所）
- 首日涨幅分布：
  - 涨幅 > 100%：约15-20%
  - 涨幅 30-100%：约30-35%
  - 涨幅 0-30%：约25-30%
  - **破发（跌破发行价）：约15-25%**
- 影响首日表现的关键因素：
  1. 发行PE vs 行业PE（最重要）
  2. 市场情绪（大盘走势）
  3. 行业热度
  4. 公司质地与稀缺性
  5. 发行规模（盘子大小）

### 5.3 破发高危特征
- 发行PE显著高于行业PE
- 大盘处于下跌趋势
- 所属行业近期表现低迷
- 发行规模过大（募资超50亿）
- 业绩增速放缓或下滑

## 6. 注册制新股投资注意事项

### 6.1 破发风险
- 全面注册制后，新股破发比例约15-25%
- 破发不等于不好，部分破发股后续修复
- 核心看：定价是否合理 + 公司基本面

### 6.2 流动性风险
- 上市首日交易活跃，但部分小盘股次日起流动性骤降
- 北交所新股流动性普遍弱于沪深
- 关注成交量变化，避免流动性陷阱

### 6.3 弃购后果
- 中签后未缴款视为弃购
- 12个月内弃购累计 ≥ 3次 → 被列入限制名单
- 限制期：6个月内不得参与网上新股申购
- 弃购次数按"投资者"计算，多账户合并计算

### 6.4 打新策略建议
1. **精选打新**：不再"逢新必打"，重点看发行PE与行业PE对比
2. **关注可比公司**：找同行业标杆做估值锚
3. **市场环境**：弱市中适当降低打新频率
4. **板块偏好**：科创板/创业板波动大但机会多，北交所需关注流动性
5. **中签后策略**：
   - 首日大涨（>50%）：考虑止盈
   - 首日小涨或平开：持有观察2-3日
   - 首日破发：评估基本面，决定止损或持有
