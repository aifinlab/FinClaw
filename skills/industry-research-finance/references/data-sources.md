# 金融行业数据源清单

## 官方数据

| 数据源 | 网址 | 数据类型 |
|--------|------|----------|
| 人民银行 | www.pbc.gov.cn | 货币政策、社融、利率 |
| 金融监管总局 | www.nrta.gov.cn | 银行业/保险业监管数据 |
| 证监会 | www.csrc.gov.cn | 证券市场数据 |
| 统计局 | www.stats.gov.cn | 宏观经济数据 |

## 行业协会

| 协会 | 数据类型 |
|------|----------|
| 银行业协会 | 行业数据 |
| 保险行业协会 | 保费收入数据 |
| 证券业协会 | 券商经营数据 |
| 基金业协会 | 基金业数据 |

## 上市公司数据

| 数据源 | 数据类型 |
|--------|----------|
| 同花顺 iFinD | 财务数据、估值数据 |
| 万得 Wind | 财务数据、估值 |
| 公司年报 | 详细财务指标 |
| 公司经营公告 | 月度经营数据 |
