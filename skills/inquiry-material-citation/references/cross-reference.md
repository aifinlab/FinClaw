# 交叉引用规范

## 一、引用原则

1. **准确性**：引用位置准确
2. **一致性**：引用内容一致
3. **完整性**：引用信息完整
4. **清晰性**：引用指向清晰

## 二、引用方式

### 2.1 章内引用
```
详见本章第 X 节
参见本节"XXX"部分
```

### 2.2 章间引用
```
详见第 X 章第 X 节
参见"XXX"章节
```

### 2.3 文件间引用
```
详见《招股说明书》第 X 章
参见《法律意见书》第 X 页
```

## 三、数据引用

### 3.1 同一文件内
```
如上表所示
见下表
参见图 X-X
```

### 3.2 跨文件引用
```
数据来源于审计报告（天健审 [2024]001 号）
参见法律意见书（国浩律师 [2024]001 号）
```

## 四、附件引用

### 4.1 正文引用附件
```
详见附件 X-X
参见附件 X-X：[附件名称]
```

### 4.2 附件间引用
```
参见附件 X-X
与附件 X-X 一致
```

## 五、常见问题

| 问题 | 表现 | 解决 |
|------|------|------|
| 引用错误 | 引用位置错误 | 仔细核对 |
| 引用遗漏 | 未引用相关章节 | 全面检查 |
| 引用不一致 | 前后引用矛盾 | 统一口径 |
| 引用模糊 | 指向不清晰 | 明确标注 |

- [ ] 文件间引用是否正确
- [ ] 数据引用是否准确
