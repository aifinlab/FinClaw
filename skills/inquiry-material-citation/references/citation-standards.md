# 引用格式标准

## 一、引用基本原则

1. **准确性**：引用内容准确无误
2. **完整性**：引用来源完整清晰
3. **规范性**：引用格式统一规范
4. **可追溯**：引用材料可追溯查证

## 二、内部文件引用

### 2.1 访谈记录
```
格式：访谈纪要 [编号]
示例：访谈纪要 BP-005
```

### 2.2 会议纪要
```
格式：会议纪要 [日期]
示例：会议纪要 2024-01-15
```

### 2.3 核查记录
```
格式：核查表 [编号]
示例：核查表 FP-023
```

### 2.4 内部报告
```
格式：报告名称 [版本]
示例：尽调报告 V2.0
```

## 三、外部文件引用

### 3.1 审计报告
```
格式：事务所 + 报告号
示例：天健审 [2024]001 号
```

### 3.2 法律意见
```
格式：律所 + 文号
示例：国浩律师 [2024]001 号
```

### 3.3 评估报告
```
格式：机构 + 报告号
示例：中企华评报字 [2024]001 号
```

### 3.4 行业报告
```
格式：机构 + 报告名
示例：弗若斯特《行业报告》
```

## 四、官方文件引用

### 4.1 营业执照
```
格式：公司 + 证照类型
示例：发行人营业执照
```

### 4.2 资质证书
```
格式：证书名称 + 编号
示例：高新技术企业证书 GR2023XXX
```

### 4.3 批复文件
```
格式：机关 + 文号
示例：证监会证监许可 [2024]X 号
```

### 4.4 合规证明
```
格式：机关 + 日期
示例：XX 局合规证明 2024-01
```

## 五、合同协议引用

### 5.1 业务合同
```
格式：合同名 + 对方 + 日期
示例：与 A 公司销售合同 2023-01
```

### 5.2 借款合同
```
格式：银行 + 合同号
示例：XX 银行借款合同 XXX 号
```

### 5.3 担保合同
```
格式：担保方 + 日期
示例：XX 公司担保合同 2023-06
```

## 六、正文引用格式

```
根据 [材料名称]（[引用编号]），[内容说明]。

示例：
根据天健会计师事务所出具的《审计报告》（天健审 [2024]001 号），
发行人 2023 年度营业收入为 XX 万元。

根据对发行人总经理 XX 的访谈记录（访谈纪要 BP-005），
发行人主要采用直销模式开拓市场。
```

## 七、附件引用格式

```
**附件 X-X：[附件名称]**
- 来源：[材料来源]
- 日期：[文件日期]
- 页码：[关键页码]
- 证明事项：[证明目的]
```

## 八、注意事项

1. 引用应准确、完整、可追溯
2. 格式应统一规范
3. 涉密信息应脱敏
4. 附件应清晰可读
5. 引用材料应在有效期内
