### 技术指标计算公式

#### 移动平均线 (MA)
```
MA(N) = (C1 + C2 + ... + CN) / N
其中 C = 收盘价，N = 周期
常用周期：5, 10, 20, 60, 120, 250
```

#### MACD
```
EMA(N) = 前日EMA × (N-1)/(N+1) + 今日收盘 × 2/(N+1)
DIF = EMA(12) - EMA(26)
DEA = DIF的9日EMA
MACD柱 = 2 × (DIF - DEA)

信号判断：
- 金叉：DIF 从下方上穿 DEA
- 死叉：DIF 从上方下穿 DEA
- 零轴：DIF > 0 为多头市场，DIF < 0 为空头市场
- 顶背离：价格创新高但 DIF 未创新高 → 见顶信号
- 底背离：价格创新低但 DIF 未创新低 → 见底信号
```

#### RSI (Relative Strength Index)
```
RS = N日内上涨日平均涨幅 / N日内下跌日平均跌幅
RSI = 100 - 100/(1+RS)
常用周期：6, 14

判断：
- RSI > 80：超买区，可能回调
- RSI < 20：超卖区，可能反弹
- RSI 50 为多空分界线
```

#### KDJ
```
RSV = (C - L9) / (H9 - L9) × 100
K = 2/3 × 前日K + 1/3 × RSV
D = 2/3 × 前日D + 1/3 × K
J = 3K - 2D

其中 L9 = 9日最低价，H9 = 9日最高价

判断：
- J > 100：超买
- J < 0：超卖
- K/D 金叉/死叉
```

#### 布林带 (BOLL)
```
中轨 = MA(20)
标准差 = STD(20日收盘价)
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差

判断：
- 价格触及上轨：短期压力
- 价格触及下轨：短期支撑
- 布林带缩口：变盘信号
- 布林带开口：趋势加速
```

#### 成交量指标
```
量比 = 当日成交量 / 过去5日平均成交量
- 量比 > 2：明显放量
- 量比 > 5：巨量
- 量比 < 0.5：明显缩量
```

### 量价关系矩阵
| | 放量 | 缩量 |
|--|------|------|
| **上涨** | 健康上涨，趋势可能延续 | 上涨动力不足，警惕回调 |
| **下跌** | 恐慌抛售或主力出货 | 下跌动能衰减，可能接近底部 |
| **横盘** | 多空分歧大，可能变盘 | 市场观望，等待方向选择 |

### A 股技术分析特殊注意事项
1. **涨跌停制度**：主板±10%、创业板/科创板±20%；涨停缩量说明多头强势，跌停缩量说明空头强势
2. **T+1制度**：当日买入次日才能卖出，影响短线交易策略
3. **集合竞价**：9:15-9:25 的竞价量价可以反映开盘情绪
4. **季节性规律**：春季行情（2-4月）、"五穷六绝七翻身"、年底机构调仓
5. **板块轮动**：A 股板块轮动特征明显，单一个股技术分析应结合板块走势
6. **政策市特征**：重大政策（如降准降息、产业政策）可能使技术形态失效
7. **北向资金**：结合 a-share-northbound 的资金流向数据可增强技术分析可信度
