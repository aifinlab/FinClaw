# A股周度复盘 (a-share-weekly-review)

## 数据获取

```bash
SCRIPTS="$SKILLS_ROOT/cn-stock-data/scripts"

# 大盘指数本周K线（将 MONDAY 替换为本周一日期 YYYYMMDD）
python "$SCRIPTS/cn_stock_data.py" kline --code SH000001,SZ399001,SZ399006 --freq daily --start MONDAY

# 实时行情（最新价/涨跌幅）
python "$SCRIPTS/cn_stock_data.py" quote --code SH000001,SZ399001,SZ399006

# 主力资金流向
python "$SCRIPTS/cn_stock_data.py" fund_flow --code SH000001
```

补充数据通过 web 搜索获取：板块涨跌排名、北向资金周净流入、融资余额、本周重要事件/政策。

## Workflow

1. **确定时间范围** — 本周一至周五（未收盘则标注"截至周X"）
2. **大盘回顾** — 三大指数周涨跌幅、日线走势特征（先扬后抑/震荡/单边等）、日均成交额及环比变化
3. **板块表现** — 涨幅前5/跌幅前5行业板块（web搜索）；概念题材亮点
4. **资金动向** — 主力资金净流入/出方向、北向资金周净额（web搜索）、融资余额变化（web搜索）
5. **重要事件梳理** — 本周影响市场的政策/公告/经济数据发布
6. **下周展望** — 关注的经济数据/事件日历、技术面关键支撑压力位

## 风格切换

默认 formal；用户说"简单总结/简要/brief"时用 brief。

| 维度 | formal | brief |
|------|--------|-------|
| 篇幅 | 2-3 页 | 半页 |
| 板块分析 | 完整行业涨跌榜 | Top 3 涨跌 |
| 个股 | 本周牛股/熊股各5只 | 仅提亮点 |
| 下周展望 | 多维度分析 | 1-2 句关键判断 |
| 免责声明 | 需要 | 不需要 |

## 关键规则

1. **事实优先** — 以数据和事件为主，避免过度主观解读
2. **完整交易周** — 数据必须覆盖已完成的所有交易日，未收盘标注截止日
3. **板块排名靠搜索** — cn-stock-data 不直接提供板块排名，必须 web 搜索补充
4. **环比标注** — 对比上周数据时标注变化方向（放量/缩量、净流入增减等）
5. **时间戳** — 报告顶部注明"复盘周期：MM.DD-MM.DD"
6. **模板参考** — 见 `references/weekly-review-template.md`
