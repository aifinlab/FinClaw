## 季节性规律提醒

在复盘中酌情引用，增加分析深度：

| 时段 | 规律 | 关注点 |
|------|------|--------|
| 1月 | 春季躁动/开门红 | 流动性宽松预期、机构调仓 |
| 2-3月 | 两会行情 | 政策预期、政府工作报告 |
| 4月 | 年报/一季报披露 | 业绩暴雷/超预期、ST风险 |
| 5月 | "Sell in May" | 外资季节性流出、业绩空窗期 |
| 6月 | 半年末流动性 | 银行考核、资金面紧张 |
| 7-8月 | 中报季 | 半年报业绩、高送转 |
| 9月 | 秋季行情 | 政策窗口期、稳增长 |
| 10月 | 国庆效应 | 节前避险/节后做多、三季报 |
| 11月 | MSCI调整 | 外资被动配置、权重股异动 |
| 12月 | 年末效应 | 机构排名冲刺、高低切换、税收筹划 |

### 特殊事件窗口
- **财报季**：1月年报预告→4月年报/一季报→7-8月中报→10月三季报
- **指数调整**：MSCI/富时/沪深300定期调整前后一周
- **期权/期货到期**：每月第三个周五（三巫日效应较弱但存在）
- **长假效应**：春节/国庆前一周防御、节后首周进攻倾向
