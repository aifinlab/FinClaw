# 风险控制措施库

## 一、预防措施

### 1.1 客户教育
- 风险揭示书
- 投资者教育材料
- 风险提示推送
- 培训讲座

### 1.2 制度约束
- 集中度限制规定
- 新开仓限制
- 保证金比例调整
- 折算率调整

### 1.3 系统控制
- 事前校验
- 事中监控
- 事后分析
- 自动预警

## 二、监控措施

### 2.1 日常监控
| 监控内容 | 监控频率 | 责任人 |
|----------|----------|--------|
| 持仓集中度 | 每日 | 风控人员 |
| 维保比例 | 实时 | 系统 |
| 证券波动率 | 每日 | 风控人员 |
| 客户交易行为 | 每日 | 客户经理 |

### 2.2 专项监控
- 高集中度客户专项监控
- 高波动证券专项监控
- 重大事件专项监控

### 2.3 压力测试
- 市场下跌情景测试
- 单一证券下跌测试
- 流动性压力测试

## 三、干预措施

### 3.1 软性干预
- 风险提示
- 电话沟通
- 面谈建议
- 投资建议

### 3.2 硬性干预
| 措施 | 适用场景 | 审批权限 |
|------|----------|----------|
| 限制新开仓 | 橙色预警 | 营业部负责人 |
| 提高保证金 | 橙色预警 | 分公司风控 |
| 下调折算率 | 橙色预警 | 分公司风控 |
| 强制减仓 | 红色预警 | 总部风控 |

### 3.3 强制措施
- 强制平仓
- 限制交易
- 收回授信
- 法律追偿

## 四、沟通措施

### 4.1 沟通渠道
- 电话
- 短信
- 微信
- 邮件
- 面谈

### 4.2 沟通要点
- 风险情况说明
- 可能后果告知
- 改善建议提供
- 客户反馈记录

### 4.3 沟通记录
| 字段 | 说明 |
|------|------|
| 沟通时间 | 沟通发生时间 |
| 沟通方式 | 电话/微信/面谈等 |
| 沟通人员 | 参与沟通的人员 |
| 沟通内容 | 沟通的主要内容 |
| 客户反馈 | 客户的回应和态度 |
| 后续计划 | 下一步工作计划 |

## 五、应急措施

### 5.1 应急预案
- 应急组织架构
- 应急处置流程
- 应急资源保障
- 应急沟通机制

### 5.2 应急场景
- 证券连续跌停
- 证券长期停牌
- 市场极端波动
- 客户失联

### 5.3 应急处置
1. 启动应急预案
2. 成立应急小组
3. 评估风险情况
4. 制定处置方案
5. 执行处置措施
6. 跟踪处置效果

## 六、措施效果评估

### 6.1 评估指标
- 风险指标改善情况
- 客户配合度
- 措施执行率
- 风险事件发生率

### 6.2 评估周期
- 每周评估短期效果
- 每月评估中期效果
- 每季评估长期效果

### 6.3 持续改进
- 收集反馈意见
- 分析问题原因
- 优化措施方案
- 更新制度流程
