# eastmoney-fund-skill

东方财富基金数据 JS 封装。

## 核心能力

- 基金列表与筛选
- 基金排行榜
- 基金净值查询
- 基金详情数据

## 数据源

- 东方财富 datacenter-web.eastmoney.com API

## 使用

```bash
node "$SKILLS_ROOT/eastmoney-fund-skill/index.js"
```

## 相关 Skill

- `eastmoney-fund-daily`：单只基金详情（天天基金源）
- `akshare-fund`：AkShare 基金数据接口
