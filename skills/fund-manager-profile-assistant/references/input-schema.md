# 输入字段建议

## 最低必需材料

- 经理履历与任职信息
- 代表产品和任职区间表现
- 投资框架和行业偏好资料
- 回撤、波动和风格稳定性信息

## 可选增强材料

- 公开访谈与路演纪要
- 团队协作与投研支持背景
- 经理变更事件

## 建议输入结构

~~~json
{
  "task_name": "基金经理画像助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
