# 龙虎榜分析参考指南

## 1. 龙虎榜上榜条件

根据沪深交易所规定，以下情况会登上龙虎榜：

| 条件 | 阈值 |
|------|------|
| 日涨跌幅偏离值 | >=+7% 或 <=-7% |
| 日换手率 | >=20% |
| 日振幅 | >=15% |
| 连续3个交易日涨跌幅偏离值累计 | >=+20% 或 <=-20% |

> 偏离值 = 个股涨跌幅 - 对应板块指数涨跌幅

## 2. 席位类型识别

### 2.1 机构专用席位

- 显示为"机构专用"（不披露具体名称）
- 代表公募基金、保险、社保、QFII 等机构资金
- 机构买入通常被视为中长线布局信号

### 2.3 沪股通/深股通专用席位

- 显示为"沪股通专用"或"深股通专用"
- 代表北向资金（外资），但无法区分具体机构
- 大额买入时关注度高

### 2.4 量化席位特征

- 单笔金额相对均匀
- 多次、多日出现在不同个股的龙虎榜
- 常见于中信证券、中金公司、华泰证券等大型券商总部席位

## 3. 分析维度模板

### 3.1 买卖力量对比

```
净买入额 = 买入总额 - 卖出总额
买入集中度 = 买一金额 / 买入总额  （集中度高=主力明确）
卖出集中度 = 卖一金额 / 卖出总额
买卖比 = 买入总额 / 卖出总额      （>1 偏多，<1 偏空）
```

### 3.2 机构参与度

```
机构席位数 / 总席位数（买卖合计最多10席）
机构净买入 = 机构买入总额 - 机构卖出总额
```

- 机构席位占比高 + 净买入 = 机构看好
- 机构卖出 + 游资买入 = 换手/博弈

### 3.3 游资一致性

- 多个知名游资同时买入同一只股 = 强共识信号
- 知名游资出现在卖出榜 = 获利了结/看空

## 4. 常见模式

### 4.1 接力模式

```
T日：游资A买入（打板/低吸）→ 上龙虎榜
T+1日：高开，游资B接力买入 → 再次上榜
T+2日：游资A卖出，可能继续或结束
```

特征：连续2-3日上榜，每日买卖席位轮换。

### 4.2 机构锁仓模式

```
T日：机构买入 → 上龙虎榜
T+N日：后续龙虎榜卖出席位无机构
```

特征：机构出现在买入榜但后续不在卖出榜，暗示中长线持有。

### 4.3 对倒模式（需警惕）

```
同一席位同时出现在买入榜和卖出榜
```

特征：可能是同一资金左手倒右手，制造成交量假象。需重点关注净额而非总额。

### 4.4 龙虎榜见光死

```
T日：上龙虎榜（通常伴随涨停）
T+1日：高开低走，收阴线
```

统计概率约 40-50%，尤其在以下情况更常见：
- 游资为主（无机构参与）
- 换手率极高（>25%）
- 题材炒作末期

## 5. 数据局限性

| 局限 | 说明 |
|------|------|
| 仅前5席位 | 只公布买卖各前5名，大量中小席位交易不可见 |
| 不反映全部交易 | 龙虎榜金额通常只占当日成交额的一小部分 |
| T+1 延迟 | 收盘后才公布，无法用于盘中决策 |
| 席位≠实际操盘方 | 同一席位可能有多个客户，席位归属仅供参考 |
| 机构匿名 | "机构专用"不披露具体是哪家基金/保险 |
| 历史席位变更 | 游资可能更换营业部，历史归属不一定准确 |
