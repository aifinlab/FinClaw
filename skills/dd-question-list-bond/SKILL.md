# 尽调问题清单助手（债券项目版）

## 触发场景

使用此技能当：
- 用户需要为债券发行项目生成尽调问题清单
- 用户要准备债券尽调访谈提纲
- 用户需要梳理发行人的信用风险点
- 用户要整理债券尽调工作底稿的问题框架
- 用户提到"债券尽调"、"公司债尽调"、"信用债 DD"、"债券访谈"等

## 核心工作流

### 1. 确认项目基本信息

首先向用户确认以下信息（如未提供）：
- **债券品种**：公司债/企业债/中票/短融/PPN/可转债
- **发行人类型**：产业类/城投类/金融类
- **发行方式**：公募/私募
- **增信措施**：无担保/保证担保/抵押质押担保
- **用户角色**：主承销商/联席承销商/律师/会计师/评级机构

### 2. 生成尽调问题框架

根据债券项目特点，按以下模块生成问题清单：

#### 2.1 发行人基本情况
- 历史沿革与股权变更
- 公司治理结构与内部控制
- 控股股东与实际控制人
- 重要子公司与参股公司
- 组织架构与人员配置

#### 2.2 经营与业务状况
- 主营业务与商业模式
- 行业地位与竞争优势
- 主要客户与供应商集中度
- 业务区域分布
- 经营资质与许可
- 环保、安全生产合规

#### 2.3 财务状况（核心）
- 资产负债结构与杠杆水平
- 收入构成与盈利稳定性
- 现金流状况（经营/投资/筹资）
- 偿债指标（流动比率、速动比率、利息保障倍数）
- 有息债务结构与期限分布
- 对外担保与或有负债
- 受限资产情况
- 关联交易与资金占用

#### 2.4 融资与信用记录
- 银行授信与使用情况
- 存量债券与到期分布
- 征信记录与违约历史
- 融资渠道与融资成本
- 债务滚续安排

#### 2.5 募投项目（如有）
- 项目合规性（立项、环评、用地）
- 投资估算与资金筹措
- 经济效益测算
- 项目进度与风险

#### 2.6 增信措施（如有）
- 担保人资质与代偿能力
- 抵质押资产权属与价值
- 增信措施法律效力

#### 2.7 特殊事项（根据发行人类型）
- **城投类**：政府支持、隐性债务、区域财政
- **房地产**：土储、销售去化、三道红线
- **产能过剩行业**：去产能政策、环保限产

### 3. 输出格式

按以下结构输出问题清单：

```markdown

## 项目概况
- 债券品种：
- 发行金额：
- 发行人类型：
- 增信措施：

## 一、发行人基本情况
...

## 二、经营与业务状况
...

## 三、财务状况（重点）
...

## 四、融资与信用记录
...

## 五、募投项目
...

## 六、增信措施
...

## 七、信用风险重点关注
### 7.1 偿债能力风险
- [具体风险点]

### 7.2 流动性风险
- [具体风险点]

### 7.3 经营波动风险
- [具体风险点]

## 附录：尽调资料清单
- [所需文件列表]
```

### 4. 监管要求对照

根据债券类型，确保覆盖监管要求的尽调内容：
- **公司债**：《公司债券发行与交易管理办法》
- **企业债**：《企业债券管理条例》
- **债务融资工具**：交易商协会自律规则

### 5. 评级视角补充

如用户需要配合评级工作，增加：
- 评级模型关键指标
- 同业对比数据需求
- 压力测试假设

## 注意事项

- 财务数据应覆盖最近三年及一期
- 重点关注现金流对债务的覆盖能力
- 对异常财务指标要深入追问原因
- 关注表外负债和隐性债务
- 尽调问题应与募集说明书披露相呼应
- 保留尽调工作底稿痕迹
