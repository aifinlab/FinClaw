# 输入字段建议

## 最低必需材料

- 产品基础信息与定位
- 净值、回撤、波动或收益数据
- 规模、份额、申赎和持有人结构
- 同类或基准对照数据

## 可选增强材料

- 渠道反馈与客户投诉
- 费率与分红安排
- 经理变更和重大事件说明

## 建议输入结构

~~~json
{
  "task_name": "基金产品体检助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
