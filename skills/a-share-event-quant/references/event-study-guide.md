# 事件研究法参考指南

## 方法论（MacKinlay 1997）

### 基本框架
1. 定义事件和事件窗口
2. 选择正常收益模型
3. 计算异常收益
4. 统计检验
5. 解释结果

### 时间线
```
|<--- 估计窗口 --->|<- 事件窗口 ->|
[-250]          [-11][-10]  [0]  [+10]
                     事件前  事件日  事件后
```

## 正常收益模型

### 市场模型（最常用）
- R_i,t = α_i + β_i × R_m,t + ε_i,t
- 在估计窗口内 OLS 估计 α, β
- 事件窗口内：AR_i,t = R_i,t - (α̂ + β̂ × R_m,t)

### 市场调整模型（简化版）
- AR_i,t = R_i,t - R_m,t
- 假设所有股票 β=1, α=0

### Fama-French 三因子模型
- R_i,t = α + β_m×MKT + β_s×SMB + β_h×HML + ε
- 更精确但需要三因子数据

## 异常收益计算

### AR（Abnormal Return）
- AR_i,t = R_i,t - E(R_i,t)

### CAR（Cumulative Abnormal Return）
- CAR_i(t1,t2) = Σ_{t=t1}^{t2} AR_i,t

### CAAR（Cumulative Average Abnormal Return）
- CAAR(t1,t2) = (1/N) × Σ_{i=1}^{N} CAR_i(t1,t2)

## 统计检验

### 参数检验（t 检验）
- t = CAAR / (σ_CAAR / √N)
- σ_CAAR 从估计窗口的残差估计

### 非参数检验
- 符号检验：正 AR 比例是否显著大于 50%
- 秩检验（Corrado rank test）：更稳健

## 常见 A 股事件效应

| 事件类型 | 典型 CAR[-1,+1] | 方向 | 持续性 |
|---------|----------------|------|--------|
| 业绩超预期 | +3% ~ +8% | 正 | 中（漂移20天） |
| 业绩低于预期 | -3% ~ -10% | 负 | 强（漂移30天） |
| 定增预案 | +2% ~ +5% | 正 | 短（5天内衰减） |
| 回购公告 | +1% ~ +3% | 正 | 中 |
| 高管增持 | +1% ~ +2% | 正 | 弱 |
| 高管减持 | -1% ~ -3% | 负 | 中 |
| 股权激励 | +2% ~ +4% | 正 | 中 |
| 并购公告 | +5% ~ +15% | 正 | 视标的而定 |
| ST 公告 | -10% ~ -20% | 负 | 强 |

## 注意事项
1. **信息泄露**：A 股市场信息提前泄露现象较普遍，事件前可能已有异常收益
2. **涨跌停**：连续涨跌停会导致 AR 在多日释放
3. **停牌**：停牌复牌后 AR 集中释放
4. **盘后公告**：大部分公告在收盘后发布，T+1 为首个反应日
