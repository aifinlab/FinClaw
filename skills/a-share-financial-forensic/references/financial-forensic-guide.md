# 财务异常检测参考指南
## Beneish M-Score
M = -4.84 + 0.92×DSRI + 0.528×GMI + 0.404×AQI + 0.892×SGI + 0.115×DEPI - 0.172×SGAI + 4.679×TATA - 0.327×LVGI

### 8个变量
| 变量 | 含义 | 异常方向 |
|------|------|---------|
| DSRI | 应收天数变化 | 上升=异常 |
| GMI | 毛利率变化 | 下降=异常 |
| AQI | 资产质量 | 上升=异常 |
| SGI | 营收增长 | 过高=异常 |
| DEPI | 折旧率变化 | 下降=异常 |
| SGAI | 管销费变化 | 上升=异常 |
| TATA | 应计/总资产 | 高=异常 |
| LVGI | 杠杆变化 | 上升=异常 |

## M-Score解读
- M > -1.78: 操纵嫌疑(高风险)
- -1.78 > M > -2.22: 灰色区域
- M < -2.22: 正常

## A股常见财务造假手法
1. 虚增营收（关联交易/体外循环）
2. 少计费用（资本化过度）
3. 虚增资产（商誉/存货）
4. 操纵减值时点
5. 提前/延后确认收入
