# 信用分析方法

## 一、信用分析框架

### 1.1 5C 分析法
| 要素 | 含义 | 分析内容 |
|------|------|----------|
| Character（品格） | 借款人信誉 | 历史记录、还款意愿 |
| Capacity（能力） | 还款能力 | 现金流、盈利能力 |
| Capital（资本） | 资本实力 | 净资产、杠杆水平 |
| Collateral（担保） | 担保措施 | 抵押、质押、保证 |
| Condition（环境） | 外部环境 | 行业、政策、经济 |

### 1.2 财务分析
- 偿债能力分析
- 盈利能力分析
- 营运能力分析
- 现金流分析

### 1.3 非财务分析
- 行业分析
- 竞争地位分析
- 管理层分析
- 公司治理分析

## 二、工商企业信用分析

### 2.1 行业风险
| 因素 | 分析要点 |
|------|----------|
| 行业周期 | 上升期/平稳期/下行期 |
| 行业政策 | 支持/中性/限制 |
| 竞争格局 | 垄断/寡头/竞争 |
| 进入壁垒 | 高/中/低 |

### 2.2 竞争地位
| 因素 | 分析要点 |
|------|----------|
| 市场份额 | 领先/跟随/落后 |
| 竞争优势 | 技术/品牌/渠道/成本 |
| 客户结构 | 分散/集中 |
| 供应商结构 | 分散/集中 |

### 2.3 财务风险
| 指标 | 关注点 |
|------|--------|
| 资产负债率 | 杠杆水平 |
| 利息保障倍数 | 付息能力 |
| 经营现金流 | 现金创造能力 |
| 盈利稳定性 | 盈利波动 |

## 三、城投平台信用分析

### 3.1 区域经济实力
| 因素 | 分析要点 |
|------|----------|
| GDP 规模 | 经济总量 |
| GDP 增速 | 经济增长 |
| 财政收入 | 财政实力 |
| 产业结构 | 经济质量 |

### 3.2 政府支持
| 因素 | 分析要点 |
|------|----------|
| 股权关系 | 控股/参股 |
| 业务关系 | 核心平台/一般平台 |
| 支持历史 | 支持记录 |
| 支持意愿 | 重要性程度 |

### 3.3 平台自身实力
| 因素 | 分析要点 |
|------|----------|
| 资产规模 | 资产总量 |
| 业务稳定性 | 业务持续性 |
| 财务风险 | 债务压力 |
| 融资能力 | 融资渠道 |

## 四、房地产企业信用分析

### 4.1 特殊指标
| 指标 | 含义 | 警戒线 |
|------|------|--------|
| 剔除预收款资产负债率 | (总负债 - 预收款)/(总资产 - 预收款) | 70% |
| 净负债率 | (有息负债 - 货币资金)/净资产 | 100% |
| 现金短债比 | 货币资金/短期有息负债 | 1 |

### 4.2 土储分析
- 土储规模
- 土储成本
- 土储区域分布
- 去化周期

### 4.3 销售分析
- 销售规模
- 销售增速
- 销售去化率
- 销售回款率

## 五、信用评级流程

5. 等级评定
6. 结果反馈
7. 报告出具
