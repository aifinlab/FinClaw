# 债券发行规则汇编

## 一、公司债规则

### 1.1 主要法规
- 《证券法》
- 《公司债券发行与交易管理办法》
- 《公司债券发行与交易管理办法》配套指引
- 交易所公司债券上市规则

### 1.2 发行条件
| 条件 | 要求 |
|------|------|
| 净资产 | 股份有限公司≥3000 万，有限责任公司≥6000 万 |
| 累计债券余额 | 不超过净资产的 40% |
| 盈利要求 | 最近三年平均可分配利润足以支付一年利息 |
| 资金用途 | 符合国家产业政策 |
| 利率 | 不超过国务院限定水平 |

### 1.3 信息披露
- 募集说明书
- 发行公告
- 定期报告（年报、半年报）
- 临时报告（重大事项）
- 跟踪评级报告

## 二、企业债规则

### 2.1 主要法规
- 《企业债券管理条例》
- 发改委企业债券相关规定

### 2.2 发行条件
| 条件 | 要求 |
|------|------|
| 净资产 | 股份有限公司≥3000 万，有限责任公司≥6000 万 |
| 累计债券余额 | 不超过净资产的 40% |
| 盈利要求 | 最近三年连续盈利 |
| 资金用途 | 符合国家产业政策、固定资产投资方向 |

## 三、债务融资工具规则

### 3.1 主要法规
- 交易商协会自律规则
- 非金融企业债务融资工具管理办法

### 3.2 主要产品
| 产品 | 期限 | 特点 |
|------|------|------|
| 短期融资券 (CP) | ≤1 年 | 短期资金周转 |
| 中期票据 (MTN) | 3-5 年 | 中期资金需求 |
| 定向工具 (PPN) | 灵活 | 私募发行 |
| 超短期融资券 (SCP) | ≤270 天 | 高信用等级 |

## 四、信息披露要求

### 4.1 发行前披露
- 募集说明书
- 审计报告
- 法律意见书
- 信用评级报告
- 担保文件（如有）

### 4.2 存续期披露
- 年度报告
- 半年度报告
- 季度财务报表（公开发行）
- 重大事项临时报告
- 跟踪评级报告

### 4.3 重大事项
| 事项 | 披露时限 |
|------|----------|
| 重大诉讼 | 知悉后 2 工作日 |
| 重大亏损 | 知悉后 2 工作日 |
| 债务违约 | 知悉后 1 工作日 |
| 评级调整 | 知悉后 1 工作日 |
| 重大资产处置 | 知悉后 2 工作日 |

## 五、投资者保护

### 5.1 债券持有人会议
- 召开情形
- 召集程序
- 表决机制
- 决议效力

### 5.2 受托管理人
- 任职资格
- 职责范围
- 利益冲突防范
- 履职要求

### 5.3 偿债保障措施
- 偿债专户
- 偿债准备金
- 担保措施
- 限制条款
