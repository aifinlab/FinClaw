# A股监管体系参考指南

## 一、监管层级

### 1. 中国证监会（最高监管机构）
- **职能**：证券市场统一监管、行政处罚、立案调查
- **监管手段**：行政处罚、市场禁入、立案调查、移送司法
- **影响力**：最高，证监会处罚通常意味着问题严重

### 2. 证券交易所（一线监管）
- **上交所**：主板 + 科创板
- **深交所**：主板 + 创业板
- **北交所**：创新层/基础层
- **职能**：日常监管、问询函/关注函/警示函、纪律处分
- **监管手段**：问询函、关注函、监管函、通报批评、公开谴责

### 3. 地方证监局
- **职能**：属地监管、现场检查、出具警示函
- **监管手段**：警示函、责令改正、行政监管措施
- **影响力**：中等，通常是现场检查发现问题后出具

## 二、监管措施等级（由轻到重）

### L1: 问询函（年报/半年报问询函）
- **性质**：常规监管，针对财报疑点的书面问询
- **触发条件**：业绩大幅波动、异常交易、信披不充分
- **影响**：轻微，需在规定时间内回复并公告
- **频率**：较高，每年年报季集中发出
- **投资提示**：问询本身不代表有问题，关注回复质量

### L2: 关注函
- **性质**：对特定事项的专项关注，比问询函更有针对性
- **触发条件**：股价异动、重大交易、媒体报道质疑
- **影响**：中低，说明监管层已关注到某个具体问题
- **投资提示**：需关注公司回复是否能消除疑虑

### L3: 警示函 / 监管谈话
- **性质**：已认定存在违规行为，但情节较轻
- **发出方**：通常由地方证监局出具
- **影响**：中等，记入诚信档案，影响公司/个人评价
- **投资提示**：说明问题已被确认，关注是否有后续升级

### L4: 行政监管措施
- **类型**：责令改正、限制业务活动、暂停业务、责令更换高管
- **性质**：强制性监管措施，要求公司采取具体整改行动
- **影响**：中高，可能实质影响公司运营
- **投资提示**：关注整改是否影响主营业务

### L5: 立案调查
- **性质**：正式启动调查程序，通常由证监会发起
- **触发条件**：涉嫌信息披露违法违规、内幕交易、操纵市场等
- **影响**：高，具有以下直接后果：
  - 调查期间不得发行股票（不能再融资）
  - 不得进行重大资产重组
  - 大股东不得减持
  - 投资者可准备民事索赔
- **投资提示**：立案调查是强烈负面信号，后续大概率有处罚

### L6: 行政处罚
- **类型**：罚款、没收违法所得、市场禁入
- **性质**：最严厉的行政监管手段
- **影响**：极高，可能触发：
  - 投资者民事索赔（虚假陈述赔偿）
  - 强制退市（重大违法退市）
  - 管理层更换
  - 公司声誉严重受损
- **投资提示**：处罚金额和内容决定后续影响，关注是否触发退市条件

## 三、常见问询/监管重点事项

### 1. 业绩变脸
- 上市当年/次年业绩大幅下滑
- 业绩预告与实际差异巨大
- 季度间利润波动异常
- **问询重点**：业绩变动原因、是否存在前期利润调节

### 2. 关联交易
- 与关联方发生大额交易
- 关联交易定价公允性
- 未充分披露的关联关系
- **问询重点**：交易必要性、定价依据、是否损害中小股东利益

### 3. 商誉减值
- 并购形成大额商誉
- 标的业绩承诺未完成
- 商誉减值测试假设不合理
- **问询重点**：减值测试具体参数、是否存在减值不充分

### 4. 募集资金使用
- 募资项目进度缓慢
- 频繁变更募资用途
- 大额募资补充流动资金
- **问询重点**：项目可行性、变更合理性

### 5. 大股东行为
- 大股东高比例质押
- 大股东资金占用
- 违规担保
- 大股东减持不合规
- **问询重点**：质押风险、资金流向、担保合规性

### 6. 信息披露规范性
- 未及时披露重大事项
- 选择性披露/误导性陈述
- 定期报告差错更正
- **问询重点**：信披制度执行情况、内控有效性

### 7. 财务真实性
- 收入确认政策异常
- 大额预付款/其他应收款
- 存货异常增长
- 在建工程长期不转固
- **问询重点**：会计处理合规性、是否存在财务造假

## 四、退市新规要点（2024年修订版）

### 强制退市类型

#### 1. 重大违法强制退市
- **欺诈发行**：IPO/再融资中虚假记载，骗取发行核准/注册
- **重大信披违法**：
  - 连续3年财务造假，累计虚增利润超5亿元且超过同期披露利润50%
  - 连续3年财务造假，累计虚增营收超20亿元且超过同期披露营收50%
- **危害公众安全**：涉及国家安全/公共安全/生态安全/生产安全，被责令停产停业

#### 2. 财务类退市
- **净利润 + 营收组合指标**：
  - 最近一年净利润为负且营业收入 < 3亿元 → *ST
  - 连续两年触发 → 终止上市
- **期末净资产为负**：
  - 最近一年期末净资产为负 → *ST
  - 连续两年 → 终止上市
- **审计意见**：
  - 被出具无法表示意见或否定意见 → *ST
  - 连续两年 → 终止上市

#### 3. 规范类退市
- 未在法定期限内披露年报/半年报
- 半数以上董事无法保证年报/半年报真实性
- 信息披露或规范运作存在重大缺陷且拒不改正
- 公司主要资产被查封/扣押/冻结

#### 4. 交易类退市
- **面值退市**：连续20个交易日收盘价 < 1元
- **市值退市**：连续20个交易日总市值 < 3亿元（主板5亿元）
- **股东人数**：连续20个交易日股东人数 < 400人
- **成交量/金额**：连续120个交易日累计成交量 < 500万股

### ST / *ST 制度
- **ST（特别处理）**：其他风险警示（治理缺陷、资金占用等）
- ***ST（退市风险警示）**：触发财务类退市指标
- ST股票涨跌幅限制：±5%（主板）、±20%（科创板/创业板）

## 五、历史典型案例

### 案例1：康美药业（600518）— 财务造假
- **事件**：2016-2018年累计虚增货币资金886亿元
- **监管路径**：证监会立案调查 → 行政处罚 → 移送司法
- **处罚**：公司罚款60万元（旧证券法上限），责任人罚款+市场禁入
- **后果**：*ST康美 → 债务重组 → 变更为ST康美
- **投资者赔偿**：中国首例证券集体诉讼，赔偿投资者约24.59亿元
- **教训**：大额货币资金与高负债并存是典型造假信号

### 案例2：乐视网（300104）— 信披违规+欺诈
- **事件**：2007-2016年财务造假，IPO阶段即存在欺诈
- **监管路径**：多次问询 → 立案调查 → 行政处罚 → 退市
- **处罚**：公司罚款2.4亿元，贾跃亭罚款2.4亿元+终身市场禁入
- **后果**：2020年退市
- **教训**：关联交易复杂、现金流与利润严重背离是危险信号

### 案例3：康得新（002450）— 虚增利润
- **事件**：2015-2018年虚增利润115.3亿元
- **监管路径**：年报被出具无法表示意见 → 立案调查 → 行政处罚 → 退市
- **处罚**：顶格处罚，责任人市场禁入
- **后果**：2021年退市
- **教训**：独立存管银行账户资金被大股东挪用，关注资金真实性

### 案例4：獐子岛（002069）— 业绩变脸
- **事件**："扇贝跑了"——2014、2017年大额存货异常核销
- **监管路径**：问询函 → 关注函 → 立案调查 → 行政处罚
- **处罚**：公司罚款60万元，原董事长罚款30万+终身市场禁入
- **后果**：多次ST，后更名为ST獐子岛
- **教训**：生物资产类公司存货真实性难以核实，属高风险品种

### 案例5：紫晶存储（688086）— 科创板首例退市
- **事件**：IPO阶段即存在财务造假，上市后持续造假
- **监管路径**：媒体质疑 → 交易所问询 → 立案调查 → 行政处罚 → 退市
- **处罚**：公司罚款3668万元，责任人分别处罚
- **后果**：2023年退市，科创板首例重大违法退市
- **教训**：注册制下仍需警惕IPO造假，关注保荐机构和审计机构质量
