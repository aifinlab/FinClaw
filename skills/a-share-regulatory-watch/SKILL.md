# A股监管追踪 / 处罚公告 / 问询函分析

## 数据源
- **Web搜索**: 证监会处罚公告、交易所问询函/关注函/警示函、监管动态新闻
- **cn-stock-data quote**: 实时行情（股价/市值/涨跌幅，用于评估监管事件对股价的冲击）
- **cn-stock-data kline**: K线数据（监管事件前后股价走势对比）
- **akshare**: `ak.stock_notice_report(symbol='CODE')` 获取上市公司公告（如可用）

## Workflow

### Step 1: 搜索监管动态
- 解析用户输入，识别股票代码/名称或监管事件关键词
- 通过 cn-stock-data 转换为标准代码（SH600519 格式）
- Web搜索近期监管动态，关键词组合：
  - `"{公司名}" 问询函 / 关注函 / 警示函`
  - `"{公司名}" 行政处罚 / 立案调查`
  - `"{公司名}" 证监会 / 交易所 / 证监局`
  - `"{公司名}" 违规 / 信息披露`
- 尝试通过 akshare 获取公司公告列表，筛选监管相关公告
- 若用户未指定个股，可搜索近期全市场重大监管事件

### Step 2: 监管措施严重性分级
对照 `references/regulatory-guide.md` 中的监管措施等级，对检索到的监管事件进行分级：

| 等级 | 措施类型 | 严重性 | 标记 |
|------|---------|--------|------|
| L1 | 问询函 | 低 | 📋 |
| L2 | 关注函 | 中低 | 📌 |
| L3 | 警示函 / 监管谈话 | 中 | ⚠️ |
| L4 | 行政监管措施（限制业务/责令改正） | 中高 | 🔶 |
| L5 | 立案调查 | 高 | 🔴 |
| L6 | 行政处罚（罚款/市场禁入） | 极高 | ⛔ |

标注每个事件的监管层级（证监会 > 交易所 > 地方证监局）

### Step 3: 监管内容分类分析
将监管事项归类并分析实质：

1. **财务类**：业绩变脸、财务造假、利润调节、资产减值不充分
2. **信息披露类**：信披违规、延迟披露、选择性披露、误导性陈述
3. **交易类**：内幕交易、操纵市场、违规减持、短线交易
4. **公司治理类**：关联交易、资金占用、违规担保、大股东侵占
5. **募资使用类**：募集资金挪用、变更用途、闲置

重点关注：
- 问询函的具体问题清单
- 公司回复函的质量（是否实质性回复）
- 是否有二次问询（说明首次回复不充分）
- 是否涉及年报审计意见

### Step 4: 影响评估
**股价影响分析**：
- 获取监管事件公告日前后K线数据
- 计算事件日涨跌幅、5日/20日累计影响
- 与同期大盘/行业对比，剥离市场因素

**基本面影响判断**：
- 是否影响公司持续经营能力
- 是否可能导致财务数据重述
- 是否触发退市风险警示（*ST）条件
- 是否影响再融资/并购重组资格

**退市风险评估**（参考退市新规）：
- 重大违法强制退市（欺诈发行/重大信披违法/危害公众安全）
- 财务类退市（连续亏损+营收不达标）
- 规范类退市（信披/规范运作严重缺陷）
- 交易类退市（面值退市/市值退市）

### Step 5: 输出报告

**brief 模式**（默认）：
- 监管事件摘要：类型 + 等级标记 + 日期
- 监管层级 + 事项分类
- 股价反应（事件日涨跌幅）
- 一句话风险判断 + 后续关注点

**formal 模式**：
- 标题：XX（代码）监管动态追踪报告
- 监管事件时间线（按时间倒序排列）
- 每个事件详细分析：
  - 监管机构 + 措施类型 + 严重性等级
  - 事项内容分类 + 核心问题
  - 公司回复情况（如有）
  - 股价影响量化分析
- 综合风险评估
  - 监管风险等级（低/中/高/极高）
  - 退市风险判断
  - 对基本面和估值的影响
- 后续跟踪建议
  - 需关注的时间节点（回复截止日/处罚听证/法院判决）
  - 相关风险提示

## 关键规则
1. **问询函 ≠ 利空确认**：问询函是常规监管手段，需看公司回复内容和质量再下结论
2. **立案调查是严重信号**：一旦被立案调查，后续大概率有处罚，且期间不能再融资/重组
3. **关注回复函质量**：敷衍回复、多次延期回复、二次问询都是负面信号
4. **监管趋严是大趋势**：注册制下信披监管持续加强，要用发展眼光看待
5. **数据不可用时如实说明，不编造监管信息**——监管信息的准确性直接影响投资决策
6. **本工具仅供参考，不构成投资建议**

## 注意事项
- 监管信息以官方公告为准，新闻报道可能有偏差
- 关注监管事件的最新进展，而非仅看初始公告
- 同一公司短期内多次被监管，风险显著叠加
- 行业监管政策变化（如地产/教培/互联网）需特别关注
