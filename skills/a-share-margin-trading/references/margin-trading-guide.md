# 融资融券分析参考指南

## 1. 核心指标定义

| 指标 | 定义 |
|------|------|
| **融资余额** | 投资者未偿还的融资买入金额 |
| **融资买入额** | 当日新增融资买入 |
| **融资偿还额** | 当日归还融资 |
| **融资净买入** | 融资买入额 - 融资偿还额 |
| **融券余额** | 融券卖出未偿还的金额 |
| **融券卖出量** | 当日新增融券卖出 |
| **两融余额** | 融资余额 + 融券余额 |
| **杠杆率** | 两融余额 / 流通市值 |

## 2. 市场信号框架

| 信号组合 | 含义 |
|----------|------|
| 融资余额持续上升 + 大盘上涨 | 趋势确认（正常加仓） |
| 融资余额持续上升 + 大盘滞涨 | 风险信号（高位加杠杆） |
| 融资余额快速下降 | 去杠杆/强平风险 |
| 融券余额突然上升 | 机构看空信号 |
| 两融余额占比 > 2.5% | 杠杆偏高 |
| 两融余额占比 < 1.5% | 杠杆偏低 |

## 3. 历史参考区间（A股）

| 阶段 | 融资余额水平 |
|------|-------------|
| 2015年牛市峰值 | ~2.27万亿 |
| 正常区间（2023-2025） | 1.4-1.8万亿 |
| 低迷区间 | <1.2万亿 |

## 4. 个股两融分析要点

- **融资余额占流通市值比 > 5%** = 高杠杆
- **连续5日融资净买入** = 强看多
- **连续5日融资净卖出** = 资金撤退
- **融券余额突增** = 可能有利空预期

## 5. 数据源说明

- **沪市数据**：上交所每日披露
- **深市数据**：深交所每日披露
- **akshare** 接口整合了两市数据
- 数据延迟 T+1，通常下午收盘后更新

## 6. 注意事项

- **两融门槛**：50万资产 + 6个月交易经验
- **强制平仓线**：维持担保比例 < 130%（券商可调整）
- **两融利率**：融资约 5-8%，融券约 8-10%
