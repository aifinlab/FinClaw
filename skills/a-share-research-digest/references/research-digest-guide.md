# 研报摘要参考指南

## 1. 券商评级体系对照表

卖方各家评级名称不统一，需标准化映射：

| 标准评级 | 含义 | 各券商常见叫法 |
|---------|------|---------------|
| **买入** | 强烈看好，预期涨幅>20% | 买入(中信/中金/华泰)、强烈推荐(国金)、强推(部分)、Buy |
| **增持** | 看好，预期涨幅10-20% | 增持(中信/华泰)、推荐(中金/招商/国泰君安)、优于大市(海通/申万)、Outperform |
| **中性** | 中性，预期涨幅±10% | 中性(中信/中金)、持有(华泰)、中性(国泰君安)、谨慎推荐(部分)、Neutral/Hold |
| **减持** | 看淡，预期跌幅>10% | 减持(中信)、回避(中金)、卖出(极少用)、Underperform/Sell |

**注意**：
- 同一评级名称在不同券商含义可能不同（如"推荐"在A券商=买入，B券商=增持）
- 需结合该券商的评级体系说明判断
- 实际操作中"减持"和"卖出"评级极其罕见（卖方偏多倾向）

## 2. 目标价分析方法

### 一致目标价计算
- **中位数**（推荐）：去除极端值影响，比均值更稳健
- **均值**：容易被极端乐观/悲观预测拉偏
- **区间**：报告最高/最低目标价，展示分歧程度

### 上行/下行空间
```
上行空间 = (一致目标价中位数 - 当前价) / 当前价 × 100%
```

### 目标价可信度判断
| 信号 | 可信度 |
|------|--------|
| 多家券商目标价集中（标准差小） | 较高 |
| 目标价分散（最高/最低差>50%） | 较低，存在重大分歧 |
| 目标价近期频繁下调 | 基本面可能恶化 |
| 目标价远高于历史估值中枢 | 需谨慎，可能过于乐观 |
| 仅1-2家覆盖 | 样本不足，参考价值有限 |

## 3. 研报类型分类

| 类型 | 特点 | 阅读价值 |
|------|------|---------|
| **首次覆盖（Initiation）** | 最全面，含商业模式/竞争格局/估值体系 | ★★★★★ 深度了解公司必读 |
| **深度报告（In-depth）** | 针对特定主题深入分析 | ★★★★★ 专题研究价值高 |
| **业绩点评（Earnings Review）** | 财报发布后的快评 | ★★★★ 了解业绩边际变化 |
| **跟踪报告（Update）** | 定期跟踪更新 | ★★★ 关注评级/目标价是否变化 |
| **行业报告（Sector）** | 行业层面分析 | ★★★★ 理解行业趋势 |
| **策略报告（Strategy）** | 宏观策略+行业配置 | ★★★ 关注大方向判断 |
| **调研纪要（Field Notes）** | 实地调研/电话会记录 | ★★★★ 一手信息，但可能有选择性披露 |

## 4. 卖方研报阅读要点

### 核心原则
1. **假设比结论重要** — 不要只看评级和目标价，要看背后的关键假设（收入增速、毛利率、费用率等），判断假设是否合理
2. **关注边际变化** — 评级从"买入"下调到"增持"比维持"增持"更有信息量；盈利预测的调整方向比绝对值更重要
3. **注意利益冲突** — 承销关系、持股关系可能影响客观性
4. **交叉验证** — 多家券商观点对比，关注分歧点而非共识点（共识往往已被定价）

### 信息提取优先级
1. 评级变化（上调/下调/首次覆盖）
2. 盈利预测调整（上调/下调幅度）
3. 核心假设和关键变量
4. 催化剂/风险事件时间表
5. 估值方法和目标价推导逻辑

## 5. 一致预期的偏差分析

### 卖方共识可能错误的信号
| 信号 | 说明 |
|------|------|
| **过度一致** | 所有券商评级买入/增持、目标价集中 → 可能已充分定价，缺乏上行空间 |
| **线性外推** | 基于近期高增长线性预测未来 → 忽略均值回归和周期性 |
| **忽略尾部风险** | 未充分考虑政策变化、技术替代等低概率高影响事件 |
| **锚定效应** | 目标价跟随股价调整而非基本面 → 股价涨了调高目标价 |
| **覆盖不足** | 热门赛道过度覆盖、传统行业覆盖不足 → 冷门股可能有预期差 |
| **滞后性** | 基本面拐点已现但预测尚未调整 → 关注高频数据交叉验证 |

### 预期差机会
- 卖方一致看多但股价持续下跌 → 可能有未被研报覆盖的利空
- 卖方覆盖少/评级中性但基本面改善 → 可能存在预期差机会
- 盈利预测集中上调后股价无反应 → 市场可能已提前定价

## 6. 常见卖方术语翻译

### 正面措辞的真实含义
| 术语 | 翻译 |
|------|------|
| "长期看好" | 短期不好说 |
| "维持推荐" | 没有新观点，但不想撤评级 |
| "估值合理" | 不便宜也不贵，没啥空间 |
| "业绩符合预期" | 没有惊喜 |
| "超预期" | 确实不错（少见的诚实表达） |
| "具备配置价值" | 跌够了，可以考虑 |

### 负面/中性措辞的真实含义
| 术语 | 翻译 |
|------|------|
| "短期承压" | 在跌 |
| "合理区间波动" | 不涨不跌 |
| "存在不确定性" | 有风险但不想直接说 |
| "等待催化剂" | 现在没有买入理由 |
| "建议观望" | 别买（卖方能说出的最看空的话） |
| "关注风险" | 可能要出事 |
| "阶段性调整" | 跌了一段时间 |
| "估值消化" | 跌了但我不想承认 |
| "结构性机会" | 大部分会跌，少数能涨 |

### 业绩相关
| 术语 | 翻译 |
|------|------|
| "业绩略低于预期" | 业绩差 |
| "业绩大幅低于预期" | 业绩暴雷 |
| "下调盈利预测但维持评级" | 基本面变差了但我不想打脸 |
| "上调盈利预测" | 之前预测太保守了 |
| "业绩拐点" | 最差的时候可能过去了（希望如此） |

### 估值相关
| 术语 | 翻译 |
|------|------|
| "估值切换" | 用明年的预期来合理化现在的价格 |
| "给予估值溢价" | 贵但我要找理由解释 |
| "安全边际充足" | 够便宜了 |
| "戴维斯双击" | 业绩涨+估值涨，最好的情况 |
| "戴维斯双杀" | 业绩跌+估值跌，最坏的情况 |
