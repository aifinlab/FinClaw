# 价值陷阱识别参考指南
## 价值陷阱五大信号
1. **盈利恶化**: 净利润YoY < 0连续2季度
2. **现金流差**: OCF/净利润 < 0.5 或 OCF < 0
3. **行业下行**: 行业指数MA120空头排列
4. **治理问题**: 审计师变更/大额商誉/频繁并购
5. **技术破位**: 股价 < MA250且持续创新低

## 真便宜 vs 价值陷阱
| 特征 | 真便宜 | 价值陷阱 |
|------|--------|---------|
| 盈利趋势 | 稳定或拐点 | 持续下滑 |
| 现金流 | OCF>NI | OCF恶化 |
| 行业 | 周期底部/稳定 | 结构性衰退 |
| 催化剂 | 可预见 | 看不到 |
| 股东行为 | 回购/增持 | 减持/质押 |

## A股常见价值陷阱行业
- 传统零售（被电商替代）
- 传统媒体（被新媒体替代）
- 产能过剩行业（周期底部假象）
