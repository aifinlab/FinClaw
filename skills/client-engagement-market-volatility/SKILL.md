# 客户陪伴内容助手 - 市场波动版

**核心定位**：在市场大幅波动时，向客户传递理性、专业、有温度的陪伴内容，稳定客户情绪，传递长期投资理念。

**适用场景**：市场单日涨跌>2%、连续波动、极端行情

---

## 市场波动分类

| 波动类型 | 触发条件 | 内容基调 |
|---------|---------|---------|
| 单日大跌 | 单日跌幅>2% | 安抚情绪 + 解读原因 + 长期理念 |
| 连续下跌 | 3 日累计>5% | 深度解读 + 历史参考 + 配置建议 |
| 单日大涨 | 单日涨幅>2% | 理性看待 + 风险提示 + 不追高 |
| 快速上涨 | 短期累计>10% | 提醒止盈 + 估值分析 + 长期视角 |
| 震荡行情 | 持续横盘震荡 | 保持耐心 + 定投建议 + 时间价值 |

---

## 内容框架

### 核心要素（必含）
1. **市场情况说明** - 客观描述发生了什么
2. **原因解读** - 简要说明下跌/上涨原因
3. **情绪安抚** - 认可客户感受，不回避波动
4. **长期理念** - 传递长期投资、价值投资理念
5. **行动建议** - 给出具体、可操作的建议

### 合规边界
- ❌ 不预测涨跌（"马上会反弹"）
- ❌ 不承诺收益（"肯定会涨回来"）
- ❌ 不制造恐慌（"赶紧跑"）
- ✅ 可以说明市场背景
- ✅ 可以传递长期理念
- ✅ 可以给出配置建议

## 输出模板

### 模板 A：单日大跌安抚（微信/朋友圈）
```
【市场波动提醒】

今日市场大幅调整，沪深 300 下跌 X%，主要受 [原因] 影响。

理解大家的焦虑，市场波动是常态，历史上类似情况多次出现。

长期来看，[核心理念：如"经济基本面未变"/"估值已处低位"]。

建议：
1. 保持冷静，不要盲目操作
2. 检视持仓，优质资产值得坚守
3. 有闲钱可考虑分批布局

市场波动期，我们与您一起坚守。
```

### 模板 B：连续下跌深度陪伴
```
【致投资者的一封信】

最近市场连续调整，很多朋友感到焦虑，这很正常。

📊 市场情况
- 近 X 日累计下跌 X%
- 主要受 [因素 1]、[因素 2] 影响
- 当前估值处于历史 [分位]

📚 历史参考
- 20XX 年类似情况：下跌 X% 后，X 个月内修复
- 20XX 年：[另一个历史案例]
- 结论：波动是暂时的，价值终会回归

💡 我们的建议
1. 检视持仓逻辑是否变化
2. 避免情绪化操作
3. 闲钱可分批布局优质资产
4. 做好资产配置，控制仓位

🌟 长期理念
投资是认知的变现，波动是收益的来源。
在别人恐惧时保持理性，时间会奖励耐心的人。

市场低迷期，我们与您同行。
```

### 模板 C：单日大涨提醒
```
【市场快速上涨提醒】

今日市场大幅上涨 X%，恭喜持仓的朋友！

理性看待：
- 上涨受 [原因] 推动
- 短期涨幅较大，注意波动风险
- 不建议追高加仓

建议：
1. 持仓的继续持有，享受上涨
2. 想加仓的等待回调
3. 浮盈较多的可考虑部分止盈

投资是长跑，不为一时涨跌所动。
```

### 模板 D：一对一沟通话术
```
【开场】
"XX 总，今天市场波动比较大，给您发个消息。"

【共情】
"知道您可能有些担心，这很正常，市场确实跌了不少。"

【解读】
"这次下跌主要是 [原因]，从基本面看 [分析]。"

【建议】
"我的建议是 [具体建议]，您看怎么样？"

【结尾】
"有任何问题随时联系我，我们一起应对。"
```

## 高频场景专项内容

### 场景 1：政策利空导致下跌
**内容要点**：
- 说明政策内容和影响范围
- 区分短期冲击 vs 长期影响
- 强调政策底 vs 市场底的时间差

### 场景 2：外围市场拖累
**内容要点**：
- 说明外围市场情况
- 区分情绪影响 vs 实质影响
- 强调 A 股独立逻辑

### 场景 3：经济数据不及预期
**内容要点**：
- 解读数据具体内容
- 说明是周期性 vs 结构性
- 强调长期经济韧性

### 场景 4：行业黑天鹅事件
**内容要点**：
- 说明事件影响范围
- 区分行业 vs 个股
- 给出持仓检视建议

---

## 内容创作要点

### 语言风格
- ✅ 简洁明了，避免专业术语
- ✅ 有温度，不机械
- ✅ 真诚，不回避问题
- ❌ 避免空洞口号
- ❌ 避免过度乐观/悲观

### 情绪管理
- 先共情，再讲理
- 认可客户感受，不否定
- 给出希望，但不承诺

### 发送时机
- 大跌当日 16:00-18:00（收盘后）
- 连续下跌第 3 天、第 5 天
- 重大事件发生后 2 小时内

---

## 使用建议

1. **及时响应** - 市场波动后尽快发送
2. **分层发送** - 根据客户风险等级调整内容
3. **渠道适配** - 朋友圈/群发/一对一内容差异化
4. **持续跟踪** - 发送后关注客户反馈
