# 结论口径指引

仅使用以下四类结论：

## 可能属于责任范围

适用情形：

- 已知身故事实与责任条款匹配度较高。
- 当前未见明确免责或重大限制因素。
- 材料基本足以支持责任启动的初步判断。

写法要求：

- 同时写明仍需关注的边界，例如完整条款复核、受益人资料核验或后续调查因素。

## 可能部分属于责任范围

适用情形：

- 身故责任本身可能触发，但案件同时涉及多责任模块、返还责任、附加责任边界或部分事实仍待拆分核验。
- 已知事实提示基础责任较可能成立，但部分附加责任、事故属性或免责边界仍需进一步核验。

写法要求：

- 说明哪些部分较可能支持，哪些部分仍存在限制或待核实。

## 可能不属于责任范围

适用情形：

- 已知事实与身故责任条款明显不匹配。
- 已出现较强的等待期、自杀条款、免责、责任模块不匹配或高风险行为限制信号。

写法要求：

- 说明这是基于当前信息的初步判断，不等同于最终拒赔结论。

## 暂无法判断

适用情形：

- 条款缺失、死因不清、事故性质待定、司法结论未明、关键材料不足或记录冲突。
- 结论高度依赖待补充的死亡证明、病历、事故认定书、公安证明、司法文书或完整条款。

写法要求：

- 明确列出缺什么、为什么缺、补齐后应重点核验什么。
