# 变更跟踪模板

## 模板一：版本变更说明

```markdown
# [文件名称] 版本变更说明

## 版本信息
| 项目 | 内容 |
|------|------|
| 文件名称 | [名称] |
| 当前版本 | V[X.X] |
| 上一版本 | V[X.X] |
| 变更日期 | YYYY-MM-DD |
| 变更人 | [姓名] |

## 变更原因
[说明变更的原因和背景]

## 变更内容

### 新增内容
- [新增内容 1]
- [新增内容 2]

### 删除内容
- [删除内容 1]
- [删除内容 2]

### 修改内容
| 章节/页码 | 修改类型 | 变更前 | 变更后 | 变更原因 |
|-----------|----------|--------|--------|----------|
| 第 X 章 | 修改 | [摘要] | [摘要] | [原因] |

## 影响范围
- [受影响的其他文件 1]
- [受影响的其他文件 2]

## 核查事项
- [ ] 数据一致性核查
- [ ] 交叉引用核查
- [ ] 格式规范核查

## 审批记录
| 环节 | 审批人 | 审批意见 | 审批日期 |
|------|--------|----------|----------|
| 编制 | [姓名] | - | [日期] |
| 复核 | [姓名] | [意见] | [日期] |
| 审核 | [姓名] | [意见] | [日期] |
```

## 模板二：变更汇总表

```markdown
# [项目名称] 材料变更汇总表

## 汇总信息
| 项目 | 内容 |
|------|------|
| 变更日期 | YYYY-MM-DD |
| 变更批次 | 第 X 批 |
| 变更文件数 | X 个 |

## 变更文件清单
| 序号 | 文件名称 | 版本号 | 变更类型 | 变更摘要 |
|------|----------|--------|----------|----------|
| 1 | [文件] | V[X.X] | 新增/修订 | [摘要] |
| 2 | [文件] | V[X.X] | 新增/修订 | [摘要] |
```

## 模板三：数据变更对照表

```markdown
# 数据变更对照表

| 数据项 | 上一版本 | 当前版本 | 变化 | 变更原因 |
|--------|----------|----------|------|----------|
| [数据] | [数值] | [数值] | [+/-X%] | [原因] |
```
