# 回撤分析参考指南

## 核心公式
- 回撤(t) = (P(t) - max(P[0:t])) / max(P[0:t])
- 最大回撤 = min(回撤序列)
- 回撤持续期 = 从高点到恢复高点的交易日数
- 水下时间 = 回撤开始到恢复的总时长

## 回撤分类
| 级别 | 回撤幅度 | A股典型触发 |
|------|---------|------------|
| 轻微 | 5-10% | 正常波动/技术调整 |
| 中度 | 10-20% | 政策变化/行业利空 |
| 重度 | 20-30% | 系统性风险/流动性危机 |
| 极端 | >30% | 金融危机/股灾 |

## Calmar 比率
Calmar = 年化收益率 / |最大回撤|
- > 1.0：优秀
- 0.5-1.0：良好
- < 0.5：风险收益比差
