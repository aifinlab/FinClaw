# 输入字段建议

## 最低必需材料

- 周度行情数据与指数表现
- 重要政策和事件清单
- 行业或风格表现
- 内部或外部观点素材

## 可选增强材料

- 跨资产表现
- 资金面和情绪指标
- 重点产品相关事件

## 建议输入结构

~~~json
{
  "task_name": "市场观点周报助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
