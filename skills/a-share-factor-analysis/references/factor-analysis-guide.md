# 单因子研究参考指南

## 核心指标

### IC（Information Coefficient）
- **定义**：因子值与下期股票收益的截面相关系数（通常用 Spearman Rank Correlation）
- **公式**：IC_t = RankCorr(Factor_t, Return_{t+1})
- **判断标准**：|IC| > 0.03 有效，> 0.05 较强，> 0.1 很强
- **频率**：通常按月计算

### IR（Information Ratio）
- **定义**：IC 的均值除以标准差，衡量因子的稳定性
- **公式**：IR = mean(IC) / std(IC)
- **判断标准**：|IR| > 0.5 优秀，> 1.0 极佳

### IC 衰减（IC Decay）
- 计算因子值与未来第 N 期收益的 IC
- IC 衰减越慢，因子换手率越低，交易成本越小

## 分组回测（Quintile Analysis）

1. 每期按因子值将股票等分为 5 组（Q1-Q5）
2. 计算每组等权/市值加权组合收益
3. 关键指标：
   - **多空收益** = Q1 - Q5（或 Q5 - Q1，取决于因子方向）
   - **单调性**：Q1 > Q2 > Q3 > Q4 > Q5（理想情况）
   - **多头超额**：Q1 vs 基准
   - **收益分布**：各组年化收益、夏普比率、最大回撤

## 因子换手率
- **定义**：相邻两期因子排名前 N% 股票池的更替比例
- **公式**：Turnover = 1 - |S_t ∩ S_{t-1}| / |S_t|
- **意义**：换手率过高导致交易成本侵蚀 Alpha

## 常见 A 股因子

### 价值因子
| 因子 | 公式 | 数据来源 |
|------|------|---------|
| EP | 净利润 / 总市值 | finance |
| BP | 净资产 / 总市值 | finance |
| SP | 营业收入 / 总市值 | finance |
| DP | 每股股利 / 股价 | finance + quote |
| CFOP | 经营现金流 / 总市值 | finance |

### 成长因子
| 因子 | 公式 |
|------|------|
| 营收增速 | (营收_t - 营收_{t-4}) / 营收_{t-4} |
| 净利润增速 | (净利润_t - 净利润_{t-4}) / |净利润_{t-4}| |
| ROE 变化 | ROE_t - ROE_{t-4} |

### 动量因子
| 因子 | 公式 |
|------|------|
| MOM_1M | 过去20交易日收益率 |
| MOM_3M | 过去60交易日收益率 |
| MOM_6M | 过去120交易日收益率 |
| MOM_12M | 过去240交易日收益率 |

### 质量因子
| 因子 | 公式 |

| 毛利率 | (营收 - 营业成本) / 营收 |

### 波动率/风险因子
| 因子 | 公式 |

| 特质波动率 | Fama-French 残差的标准差 |

### 流动性因子
| 因子 | 公式 |
|------|------|
| 换手率 | 日均换手率的N日均值 |
| Amihud 非流动性 | |收益率| / 成交金额 |
| 成交金额 | 日均成交金额的对数 |

## 因子预处理

1. **去极值**：MAD（中位数绝对偏差）法，阈值 ±3 MAD
2. **标准化**：Z-score = (x - mean) / std
3. **中性化**：对市值和行业进行回归取残差
4. **缺失值处理**：行业均值填充或剔除

## A 股特殊注意事项

1. **ST/退市股**：建议剔除 ST、*ST 股票
2. **次新股**：上市不满 60 个交易日的股票建议剔除
3. **涨跌停**：涨停/跌停日无法买入/卖出，需在回测中处理
4. **停牌**：停牌期间无法交易，复牌后可能有大幅跳空
5. **行业分类**：推荐使用申万一级行业分类（31 个行业）
