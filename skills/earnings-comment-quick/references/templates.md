# 财报快评（快讯版）输出模板

## 标准模板（100-200 字）

```markdown
【XX 公司财报快评】

核心判断：[超预期/符合预期/低于预期]

2024 年营收 XX 亿，同比 +XX%；归母净利 XX 亿，同比 +XX%。其中 Q4 营收 XX 亿，同比 +XX%。

亮点：[1 条核心亮点，如"毛利率提升 2pct"或"主业增长 50%"]

展望：[1 条展望/指引，如"2025 年指引增长 20%-30%"]

评级：[买入/增持/中性/减持]
```

## 示例

```
【贵州茅台财报快评】

核心判断：符合预期

2024 年营收 1000 亿，同比 +18%；归母净利 500 亿，同比 +20%。其中 Q4 营收 280 亿，同比 +15%。

亮点：直销渠道占比提升至 45%，毛利率同比 +1pct。

展望：2025 年指引营收增长 15%-20%。

评级：增持
```

## 预期判断标准

| 判断 | 标准 |
|------|------|
| 大超预期 | 利润增速 > 一致预期 20%+ |
| 超预期 | 利润增速 > 一致预期 10%-20% |
| 符合预期 | 利润增速在一致预期±10% 内 |
| 低于预期 | 利润增速 < 一致预期 -10% |
| 大低于预期 | 利润增速 < 一致预期 -20%+ |
