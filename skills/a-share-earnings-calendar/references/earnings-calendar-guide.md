# A股财报日历参考指南

## 一、财报披露制度

### 披露时间窗口
| 报告类型 | 报告期 | 披露截止日 | 密集期 |
|---------|--------|-----------|--------|
| 一季报 | 1-3月 | 4月30日 | 4月中下旬 |
| 半年报（中报） | 1-6月 | 8月31日 | 7月中旬-8月底 |
| 三季报 | 1-9月 | 10月31日 | 10月中下旬 |
| 年报 | 1-12月 | 次年4月30日 | 3-4月 |

### 披露规则要点
- 上市公司须在规定期限内披露定期报告，逾期未披露将被交易所问询/处罚
- 年报和半年报需经会计师事务所审计，一季报和三季报无需审计
- 公司可自主选择在窗口期内的具体披露日期，通常提前公告预约日期
- ST 公司、业绩大幅变动公司通常被要求较早披露

## 二、业绩预告 vs 业绩快报 vs 正式财报

### 三者对比
| 维度 | 业绩预告 | 业绩快报 | 正式财报 |
|------|---------|---------|---------|
| 披露时间 | 最早（报告期结束后1-2个月内） | 居中 | 最晚（截止日前） |
| 数据精度 | 给范围（上下限） | 具体数字但未审计 | 审计后最终数据 |
| 强制性 | 特定条件强制（净利润为负、扭亏、变动50%+） | 自愿 | 强制 |
| 可靠性 | ★★☆ | ★★★ | ★★★★★ |
| 修正风险 | 较高（正式财报可能偏离预告范围） | 中等 | 无（最终数据） |

### 业绩预告强制披露条件
- 净利润为负（亏损）
- 净利润与上年同期相比上升或下降 50% 以上
- 实现扭亏为盈

## 三、业绩预告类型定义

| 类型 | 定义 | 净利润同比变动 |
|------|------|---------------|
| **预增** | 净利润大幅增长 | > +50% |
| **略增** | 净利润小幅增长 | 0% ~ +50% |
| **续盈** | 继续盈利（去年也盈利） | 变动幅度不大 |
| **扭亏** | 去年亏损今年盈利 | 亏转盈 |
| **略减** | 净利润小幅下降 | -50% ~ 0% |
| **预减** | 净利润大幅下降 | < -50% |
| **首亏** | 去年盈利今年亏损 | 盈转亏 |
| **续亏** | 继续亏损（去年也亏损） | 持续亏损 |

### 投资含义
- **积极信号**：预增 > 扭亏 > 略增 > 续盈
- **消极信号**：首亏 > 预减 > 略减 > 续亏（续亏市场已有预期，边际冲击可能较小）

## 四、财报季投资日历

### 全年节奏
| 时间段 | 事件 | 投资关注点 |
|--------|------|-----------|
| 1月中旬-2月初 | 年报业绩预告密集期 | 业绩超预期个股、"业绩雷"规避 |
| 3月-4月 | 年报+一季报密集期 | 全年最重要的业绩窗口，年报分红方案 |
| 4月中下旬 | 一季报集中披露 | 验证全年增长趋势的第一个数据点 |
| 7月中旬 | 中报业绩预告开始 | 半年度业绩拐点信号 |
| 8月 | 中报密集期 | 基金调仓依据 |
| 10月 | 三季报密集期 | 全年业绩可预测性最高的时点 |
| 10月底-11月 | 三季报收官 | 开始展望来年，布局来年逻辑 |

### 财报季典型市场特征
- **业绩真空期**（5-6月、11-12月）：主题投资活跃，概念炒作多
- **业绩密集期**（1月下旬、4月、7-8月、10月）：基本面驱动，业绩分化加剧
- **年报季**（3-4月）：高送转、分红概念活跃

## 五、业绩超预期的判断方法

### 参照基准（按优先级）
1. **卖方一致预期**（Wind/iFind一致预期）— 最权威的参照
2. **前期业绩预告中值** — 预告给出上下限时取中值比较
3. **历史增速趋势** — 过去4-8个季度的增速趋势外推
4. **管理层指引** — 公司在投资者交流中给出的展望

### 超预期幅度判断
| 幅度 | 描述 | 典型市场反应 |
|------|------|-------------|
| > 20% | 大幅超预期 | 可能涨停或连续大涨 |
| 10-20% | 明显超预期 | 通常正面反应 |
| 5-10% | 小幅超预期 | 反应温和 |
| -5% ~ +5% | 符合预期 | 股价波动不大 |
| -10% ~ -5% | 小幅低于预期 | 温和调整 |
| < -10% | 大幅低于预期 | 可能跌停或连续下跌 |

## 六、业绩公告后股价反应模式

### 常见模式

**1. 超预期利好**
- 公告后1-3日大涨，可能涨停
- 资金抢筹明显，成交量放大
- 后续走势取决于是否已被市场提前预期

**2. 利好出尽**
- 业绩符合或略超预期，但股价前期已大涨
- 公告后冲高回落或直接下跌
- 典型特征：公告前连续上涨 + 公告日放量滞涨

**3. 不及预期（业绩雷）**
- 公告后可能直接跌停
- 机构集中出逃，连续数日卖出
- 严重时可能引发连续一字跌停

**4. 低于预期但边际改善**
- 绝对数据不好但环比改善、趋势向好
- 股价先跌后涨，市场解读为"最差时刻已过"
- 通常出现在周期股拐点

**5. 续亏但市场已充分预期**
- ST/困境反转类公司，亏损本身不意外
- 关注亏损收窄幅度、现金流改善等边际信号
- 股价反应取决于是否有改善迹象

### 股价反应观察窗口
- **T+0**（公告日/次日）：初始反应，受情绪驱动
- **T+3**：短期消化完毕，反映机构态度
- **T+5 到 T+10**：中期趋势形成，分析师开始更新报告
- **T+20**：可作为业绩事件对股价影响的最终判断基准
