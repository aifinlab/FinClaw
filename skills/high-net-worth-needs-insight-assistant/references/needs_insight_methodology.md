# 高净值客户需求洞察方法说明

本文件说明如何从客户事实、行为变化、市场环境和行业基准四个层面构建需求洞察。

## 一、分析基本原则

1. 先事实，后判断。
2. 先需求，后产品。
3. 先证据，后建议。
4. 先置信度，后结论强度。

## 二、四层分析法

### 1. 客户事实层
包括身份、年龄段、家庭结构、资产规模、产品持有情况、风险偏好。

### 2. 行为变化层
包括申购、赎回、提前支取、到期去向、咨询主题、联系频率变化。

### 3. 环境映射层
包括市场利率环境、权益市场波动、汇率变化、同业产品供给与服务趋势。

### 4. 需求推断层
将前述三层信息转译为：流动性、稳健增值、进取配置、保障、传承、跨境、整合、陪伴、服务升级等需求。
