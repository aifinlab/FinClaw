# 高风险交易聚类分析助手

## 1. 业务问题

经初筛或异常识别后产生的高风险交易候选数量仍然较大，合规团队需要将其按模式归类以提升复核效率。本技能对高风险交易候选进行行为模式聚类，识别相似交易路径，并发现跨簇关联，帮助合规人员从全局视角理解风险交易分布。

## 2. 适用场景

- 大批量初筛命中交易的模式归类和分组复核
- 跨账户、跨时段的相似交易路径识别
- 关联交易网络的簇间链路发现
- 专项排查中的交易模式聚类分析
- 定期风险交易回顾中的模式演变跟踪

## 3. 输入

| 输入项 | 说明 | 是否必需 |
|--------|------|----------|
| 高风险交易候选集 | 经初筛或异常识别后的交易数据 | 必需 |
| 交易特征字段 | 金额、频率、时段、对手方、渠道等聚类特征 | 必需 |
| 聚类参数 | 相似度阈值、最小簇规模等 | 可选（默认使用内置参数） |
| 账户关联信息 | 同名账户、关联企业、共同对手方等 | 可选 |
| 历史聚类结果 | 过往聚类分析输出，用于模式演变对比 | 可选 |

## 4. 输出

| 输出项 | 说明 |
|--------|------|
| 聚类结果 | 每个簇的交易列表、核心特征和簇标签 |
| 簇特征摘要 | 每个簇的模式描述（金额范围、时段分布、对手方特征等） |
| 跨簇关联分析 | 不同簇之间的共同账户、共同对手方、资金链路等关联 |
| 离群交易标注 | 未归入任何簇的离群交易及其特征 |
| 模式演变对比 | 与历史聚类结果的模式变化对比（如有历史数据） |

## 5. 默认分析维度

- **金额结构相似度**：交易金额分布、单笔金额区间、累计金额规模
- **时间行为相似度**：交易时段偏好、频率节奏、持续周期
- **对手方网络相似度**：对手方重合度、对手方行业/地域分布
- **渠道偏好相似度**：交易渠道组合、渠道切换模式
- **资金路径相似度**：资金流向链条、中转层级、最终流向
- **跨簇关联维度**：共同节点账户、资金桥接路径、时间序列关联

## 6. 默认分析框架

1. **特征提取**：从交易候选集中提取聚类所需的多维特征向量
2. **相似度计算**：计算交易间的多维相似度矩阵
3. **聚类执行**：按相似度阈值进行聚类分组，标记离群项
4. **簇特征刻画**：为每个簇生成模式摘要和核心特征描述
5. **跨簇关联发现**：检查不同簇之间是否存在共同账户、资金链路等关联
6. **结论先行输出**：先呈现簇数量、主要模式类型和跨簇关联数量，再展开明细
7. **复核建议排序**：按簇规模和跨簇关联程度排序，建议复核优先级

## 7. 安全边界

- 不编造数据；不夸大结论；缺失信息标注"未获取"或"待核实"
- 不替代正式审批、正式风控、正式合规、正式报送、正式处置、正式责任认定、正式管理决策
- 不对交易或客户做"疑似洗钱""疑似欺诈"等定性表述
- 仅提供数据模式分析和复核参考
- 不替代正式合规复核、审计、调查或账户处置决策
- 不替代可疑交易报告（STR）的正式判定
- 聚类结果反映数据层面的模式相似性，不等同于交易关联性的业务认定
