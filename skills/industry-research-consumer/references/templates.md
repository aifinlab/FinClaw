# 消费行业研究输出模板

## 模板 A：快速扫描（500-800 字）

适用于用户说"快速了解"、"扫描一下"

```markdown
# 【XX 行业快速扫描】

## 行业定义
[1-2 句话定义行业边界]

## 市场规模
- 2023 年规模：XXX 亿元
- 近 3 年 CAGR：XX%
- 预计 2025 年：XXX 亿元

## 竞争格局
- CR3：XX%
- 主要玩家：[公司 A]、[公司 B]、[公司 C]
- 竞争特点：[价格战/差异化/并购整合]

## 关键趋势
1. [趋势 1]
2. [趋势 2]

## 投资关注
- [关注方向/标的]
```

## 模板 B：标准分析（1500-3000 字）

适用于用户说"做一份分析"、"研究一下"（默认模板）

```markdown
# 【XX 行业分析报告】

## 1. 研究范围与假设
- 行业边界：[定义]
- 地域范围：[默认中国市场]
- 时间范围：[当前至未来 3 年]
- 研究视角：[市场 + 竞争 + 商业模式]

## 2. 行业定义与分类
- [必选消费/可选消费分类]
- [子行业划分]

## 3. 市场规模与增长驱动
- 当前规模
- 历史增速
- 未来预测
- 驱动因素

## 4. 产业链分析
- 上游：[原材料/供应商]
- 中游：[制造/品牌]
- 下游：[渠道/终端]
- 价值分配：[各环节毛利水平]

## 5. 竞争格局
- 市场份额（CR3/CR5）
- 主要玩家对比
- 竞争壁垒

## 6. 商业模式分析
- 品牌力：[认知度/美誉度/忠诚度]
- 渠道力：[线上/线下结构]
- 产品力：[大单品/迭代速度]
- 供应链：[成本/产能]

## 7. 消费者趋势
- 目标客群
- 消费频次
- 消费趋势（健康化/高端化/国潮）

## 8. 政策与监管
- 相关政策
- 监管环境

## 9. 风险因素
- [风险 1]
- [风险 2]

## 10. 结论与建议
- 核心判断
- 投资建议
```

## 模板 C：深度报告（3000 字+）

适用于用户说"写完整报告"、"投研报告"

在模板 B 基础上增加：
- 详细财务分析
- 估值分析
- 敏感性分析
- 附录（数据表格、图表）
