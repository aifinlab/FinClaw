# 消费行业数据源清单

## 官方数据

| 数据源 | 网址 | 数据类型 | 更新频率 |
|--------|------|----------|----------|
| 国家统计局 | www.stats.gov.cn | 社零总额、CPI、产量 | 月度 |
| 商务部 | www.mofcom.gov.cn | 消费政策、流通数据 | 不定期 |
| 海关总署 | www.customs.gov.cn | 进出口数据 | 月度 |

## 行业协会

| 协会 | 数据类型 |
|------|----------|
| 中国酒业协会 | 白酒/啤酒产量、价格 |
| 中国乳制品工业协会 | 乳制品产量、销售 |
| 中国家用电器协会 | 家电产量、出口 |
| 中国连锁经营协会 | 零售门店数据 |

## 第三方数据

| 机构 | 数据类型 | 备注 |
|------|----------|------|
| 欧睿 Euromonitor | 消费品类市场份额 | 付费 |
| 尼尔森 Nielsen | 零售渠道销售数据 | 付费 |
| 魔镜市场情报 | 电商销售数据 | 付费 |
| 凯度 Kantar | 消费者行为数据 | 付费 |

## 上市公司数据

| 数据源 | 数据类型 |
|--------|----------|
| 同花顺 iFinD | 财务数据、经营指标 |
| 万得 Wind | 财务数据、估值数据 |
| 公司年报 | 门店数、单店产出、会员数据 |
| 公司公告 | 经营简报、重大合同 |

## 常用数据调用示例

### 社会消费品零售总额
```
来源：国家统计局
频率：月度
指标：社零总额、同比增速、细分品类增速
```

### 上市公司门店数据
```
来源：公司年报/经营公告
指标：门店总数、净开店、单店产出、坪效
```

### 电商销售数据
```
来源：魔镜市场情报
指标：天猫/京东销售额、市占率、增速
```
