## 功能描述

财经新闻与公告Skill，提供热点财经新闻、个股新闻、公司公告、券商研报、业绩预告、新闻情感分析等功能。支持关键词监控，自动识别利好/利空信息。

## 功能列表

### 1. 热点财经新闻
- **功能描述**: 获取当前市场热点财经新闻
- **输入参数**: 无（获取最新热点）
- **输出格式**: Markdown列表（标题+摘要+来源）
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/news_hot.py
  ```

### 2. 个股新闻
- **功能描述**: 获取特定股票的相关新闻
- **输入参数**: 股票代码
- **输出格式**: Markdown列表
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/news_stock.py 600519
  python scripts/news_stock.py 300750
  ```

### 3. 公司公告查询
- **功能描述**: 查询上市公司公告
- **输入参数**: 股票代码、公告类型（可选）
- **输出格式**: Markdown列表（按时间倒序）
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/announcement.py 600519
  python scripts/announcement.py 300750 --type 业绩
  ```

### 4. 券商研报
- **功能描述**: 获取券商研究报告
- **输入参数**: 股票代码
- **输出格式**: Markdown表格（评级/目标价/分析师）
- **数据源**: AkShare-东方财富
- **数据时效**: 日频
- **使用示例**:
  ```bash
  python scripts/research_report.py 600519
  python scripts/research_report.py 300750
  ```

### 5. 新闻情感分析
- **功能描述**: 分析新闻标题/内容的情感倾向
- **输入参数**: 新闻文本或股票代码
- **输出格式**: 情感评分 + 关键词提取
- **数据源**: 本地NLP分析
- **数据时效**: 实时分析
- **使用示例**:
  ```bash
  python scripts/news_sentiment.py 600519
  python scripts/news_sentiment.py "茅台发布超预期业绩预告"
  ```

### 6. 关键词监控
- **功能描述**: 监控特定关键词的新闻/公告
- **输入参数**: 关键词
- **输出格式**: Markdown列表
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/news_monitor.py 回购
  python scripts/news_monitor.py 增持
  python scripts/news_monitor.py 中标
  ```

### 7. 业绩预告
- **功能描述**: 获取上市公司业绩预告
- **输入参数**: 无（全市场）或股票代码
- **输出格式**: Markdown表格
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/earnings_preview.py
  python scripts/earnings_preview.py 600519
  ```

## 情感分析规则

### 积极信号词
| 类别 | 关键词 |
|:---|:---|
| 业绩类 | 预增、超预期、扭亏、大增、创新高 |
| 资金类 | 增持、回购、员工持股、大资金入场 |
| 业务类 | 中标、大单、合同、突破、订单饱满 |
| 其他 | 利好、获批、认证、合作、扩张 |

### 消极信号词
| 类别 | 关键词 |
|:---|:---|
| 业绩类 | 预亏、预减、下滑、不及预期、亏损 |
| 资金类 | 减持、解禁、套现、质押、爆仓 |
| 风险类 | 处罚、调查、诉讼、违规、退市 |
| 其他 | 利空、停产、裁员、收缩、违约 |

### 情感评分
| 分数 | 情感倾向 | 解读 |
|:---:|:---:|:---|
| +0.8 ~ +1.0 | 强烈积极 | 重大利好，可能涨停 |
| +0.3 ~ +0.7 | 积极 | 利好消息，可能上涨 |
| -0.2 ~ +0.2 | 中性 | 无明显影响 |
| -0.7 ~ -0.3 | 消极 | 利空消息，可能下跌 |
| -1.0 ~ -0.8 | 强烈消极 | 重大利空，可能跌停 |
