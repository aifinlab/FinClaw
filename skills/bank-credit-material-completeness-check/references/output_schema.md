# 输出结构定义

建议输出对象至少包含以下字段：

```json
{
  "skill_name": "credit-material-completeness-check",
  "审核对象": {
    "企业名称": "",
    "统一社会信用代码": "",
    "授信品种": "",
    "申请金额": "",
    "担保方式": "",
    "审核阶段": ""
  },
  "材料台账": [
    {
      "材料名称": "",
      "材料类别": "",
      "是否已提供": "是/否/待确认",
      "材料日期": "",
      "签章情况": "完整/不完整/待核验",
      "有效性": "有效/过期/待核验",
      "清晰完整性": "完整/缺页/模糊/待核验",
      "一致性": "一致/不一致/待核验",
      "问题说明": "",
      "是否需补件": "是/否"
    }
  ],
  "缺件清单": [],
  "问题清单": [],
  "待核验事项": [],
  "补件建议": [],
  "审核结论": ""
}
```

如需输出为自然语言报告，可在上述结构基础上转写为正式审核意见。
