# 输入字段建议

## 最低必需材料

- 产品设计方案与定位
- 策略亮点与风险点
- 目标渠道与目标客群
- 已有披露材料或制度要求

## 可选增强材料

- 经理介绍
- 同类竞品差异
- 渠道常见问题

## 建议输入结构

~~~json
{
  "task_name": "新产品材料生成助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
