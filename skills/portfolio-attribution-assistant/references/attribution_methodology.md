# 组合归因方法说明

## 一、目标

组合归因的目标不是简单罗列涨跌，而是解释区间收益与超额收益的来源，拆清“配置做对了什么、选股做对了什么、
是否承担了额外风格风险”。

## 二、常见归因层级

### 1. 总收益层

- 组合区间收益
- 基准区间收益
- 超额收益

### 2. 行业层

- 行业配置贡献：组合相对基准在行业权重上的偏离带来的贡献
- 行业内选股贡献：在行业内部，组合持有标的收益与基准代表收益的差异带来的贡献
- 交互项：配置和选股同时发生偏离带来的影响

### 3. 个券层

- 单只股票对组合收益的直接贡献
- 单只股票对超额收益的近似贡献
- 集中收益来源识别

### 4. 风格层

- 大盘/中盘/小盘
- 成长/价值
- 周期/消费/科技/金融/红利/防御

## 三、推荐解释顺序

建议按照“先总后分、先中观后微观”的顺序解释：

1. 先讲组合和基准谁更好
2. 再讲超额主要来自哪些行业
3. 再讲行业内部靠哪些个券
4. 最后讲风格是否与既定产品定位一致

## 四、结果表达原则

- 行业层解释优先于个券轶事
- 数据充分时强调证据链
- 数据不足时强调假设边界
- 既看正贡献，也看负拖累
