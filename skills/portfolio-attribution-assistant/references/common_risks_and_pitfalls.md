# 常见问题与风险提示

## 一、常见误区

1. 把区间末持仓当作全区间平均持仓
2. 把行业涨得好直接等同于经理配置能力强
3. 把少数重仓股偶发收益当作稳定选股能力
4. 忽略现金仓位和仓位升降对收益的影响
5. 忽略交易时点和调仓路径

## 二、典型红旗

- 超额收益高度集中在一两个个券
- 主要超额来自基准外小票，持续性存疑
- 行业归因与个券归因口径不一致
- 分析区间过短，噪音远大于信号
- 行业分类跨表不统一

## 三、输出时应保守表达的情况

- 使用估算均值权重而非真实日频权重
- 组合存在大额申赎但未调整口径
- 行业数据只覆盖部分资产
- 跨市场资产口径不统一
