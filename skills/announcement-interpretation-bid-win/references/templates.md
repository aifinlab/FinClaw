# 中标公告解读输出模板

## 快速解读（200-300 字）

```markdown
【XX 公司中标公告解读】

中标金额：XX 亿，占上年度营收 XX%

项目类型：[工程/产品/服务]

招标方：[名称 + 资质]

项目周期：XX 个月/年

业绩影响：预计贡献营收 XX 亿，利润 XX 亿

投资判断：[利好程度 + 简要理由]
```

## 中标重要性判断标准

| 重要性 | 中标金额/营收 |
|--------|---------------|
| 重大中标 | >30% |
| 中等中标 | 10%-30% |
| 小中标 | <10% |
