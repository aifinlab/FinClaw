# 挽留方案框架

## 方案分层
- **A 类（刚性现金需求）**：优先满足现金需求，提出“部分赎回 + 保留核心仓位”。
- **B 类（情绪驱动）**：先解释波动与期限，再给“分批赎回/观望复盘”。
- **C 类（绩效不满）**：结合替代产品或策略转换，强调匹配度与风险控制。
- **D 类（流动性误解）**：解释封闭期/赎回成本，建议延后或改用更高流动性方案。

## 选项模板
- **部分赎回**：满足急需 + 保留收益潜力
- **分批赎回**：降低择时风险，保留观察窗口
- **产品转换**：风险等级更匹配/流动性更高
- **配置下调**：降低波动，设置复盘节点
- **先观望复盘**：明确时间点与触发条件
