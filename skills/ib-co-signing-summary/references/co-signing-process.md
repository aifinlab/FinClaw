# 会签流程说明

## 一、会签类型

### 1.1 立项会签
- 目的：决定是否承接项目
- 发起：业务部门
- 参与：质控、内核、合规、风控
- 时限：5 个工作日

### 1.2 内核会签
- 目的：决定是否申报
- 发起：项目组
- 参与：内核委员
- 时限：5 个工作日

### 1.3 申报会签
- 目的：决定是否提交
- 发起：项目组
- 参与：相关部门
- 时限：3 个工作日

## 二、会签流程

### 2.1 发起阶段
1. 填写会签申请表
2. 准备会签材料
3. 提交会签系统
4. 通知会签部门

### 2.2 会签阶段
1. 各部门审阅材料
2. 提出会签意见
3. 项目组回复意见
4. 形成会签结论

### 2.3 决议阶段
1. 汇总会签意见
2. 形成会签决议
3. 通知项目组
4. 归档会签材料

## 三、会签意见

### 3.1 意见类型
| 类型 | 含义 | 处理方式 |
|------|------|----------|
| 同意 | 无条件同意 | 直接进入下一环节 |
| 有条件同意 | 需满足特定条件 | 落实条件后确认 |
| 反对 | 不同意推进 | 需重新论证或终止 |
| 弃权 | 不发表意见 | 不计入表决 |

### 3.2 意见要求
- 明确具体
- 有理有据
- 可操作
- 有时限

## 四、会签时限

| 会签类型 | 会签时限 | 加急时限 |
|----------|----------|----------|
| 立项会签 | 5 个工作日 | 3 个工作日 |
| 内核会签 | 5 个工作日 | 3 个工作日 |
| 申报会签 | 3 个工作日 | 1 个工作日 |

## 五、会签决议

### 5.1 决议形成
- 简单多数：超过半数同意
- 绝对多数：超过 2/3 同意
- 一致同意：全部同意

### 5.2 决议类型
- 通过
- 有条件通过
- 不通过
- 暂缓表决

### 5.3 决议执行
- 项目组执行决议
- 落实附加条件
- 反馈执行情况
- 归档决议材料

## 六、特殊情况

### 6.1 会签延期
- 原因：材料不齐、问题复杂
- 程序：申请延期、批准延期
- 时限：一般不超过 5 个工作日

### 6.2 会签复议
- 原因：对决议有异议
- 程序：申请复议、重新会签
- 时限：收到决议后 5 个工作日内

### 6.3 会签豁免
- 原因：特殊情况
- 程序：申请豁免、批准豁免
- 权限：分管领导批准
