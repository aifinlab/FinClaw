# 输入字段建议

## 最低必需材料

- 客户风险测评结果
- 产品风险等级
- 制度口径或业务规则
- 客户问题或场景

## 可选增强材料

- 历史沟通记录
- 客户画像
- 升级处理流程

## 建议输入结构

~~~json
{
  "task_name": "风险测评解释助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
