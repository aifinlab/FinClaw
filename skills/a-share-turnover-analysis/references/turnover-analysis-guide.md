# 换手率分析参考指南
## 换手率 = 成交量 / 流通股本 × 100%

## 换手率水平参考
| 换手率 | 含义 |
|--------|------|
| <1% | 极度低迷/无人关注 |
| 1-3% | 正常/温和 |
| 3-7% | 活跃 |
| 7-15% | 高度活跃/关注 |
| >15% | 极端活跃/警惕 |

## 不同市值换手率中枢
- 大盘(>500亿): 0.5-2%
- 中盘(100-500亿): 1-4%
- 小盘(<100亿): 2-8%
- 次新股: 5-20%

## 换手率异常信号
1. 换手Z>2 + 涨停: 强势放量
2. 换手Z>3 + 收阴: 出货嫌疑
3. 连续缩量至Z<-1.5: 即将变盘
4. 底部温和放量(Z≈1): 资金介入
