## 功能列表

### 1. 科创板列表
- **功能描述**: 获取科创板股票列表
- **输入参数**: 无
- **输出格式**: Markdown表格
- **数据源**: AkShare-上交所
- **数据时效**: 日频
- **使用示例**:
  ```bash
  python scripts/star_list.py
  ```

### 2. 科创板行情
- **功能描述**: 获取科创板实时行情
- **输入参数**: 股票代码（可选）
