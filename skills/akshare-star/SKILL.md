# akshare-star

科创板与北交所专题查询 Skill。提供科创板、北交所股票的行情、新股申购、IPO数据等功能。

## 功能

- 科创板股票列表
- 科创板行情查询
- 北交所股票列表
- 北交所行情查询
- 新股申购日历

## 使用示例

查询科创板：
```bash
python scripts/star_list.py
```

查询北交所：
```bash
python scripts/bse_list.py
```

新股申购：
```bash
python/scripts/star_ipo.py
```

## 数据来源

- AkShare 科创板/北交所数据接口
