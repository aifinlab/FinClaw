# IPO 问询回复模板

## 回复基本结构

```markdown
## 问题 [X]：[问题标题]

### 问题原文
[交易所/证监会问询原文]

### 回复要点
1. [核心观点 1]
2. [核心观点 2]
3. [核心观点 3]

### 回复正文

#### 一、[分论点 1]
[详细论述 + 数据支撑]

#### 二、[分论点 2]
[详细论述 + 数据支撑]

#### 三、[分论点 3]
[详细论述 + 数据支撑]

### 核查程序
1. [核查程序 1]
2. [核查程序 2]
3. [核查程序 3]

### 核查结论
[明确结论]

### 附件材料
- [附件 1 名称]
- [附件 2 名称]
```

## 常见回复模式

### 模式一：解释说明型
适用于监管要求解释说明的问题

```
一、基本情况
[说明基本情况]

二、原因分析
[分析原因]

三、合理性论证
[论证合理性]

四、风险提示
[披露相关风险]
```

### 模式二：数据论证型
适用于需要数据支撑的问题

```
一、数据对比
[与同行业/历史数据对比]

二、原因分析
[分析差异原因]

三、趋势判断
[判断未来趋势]
```

### 模式三：核查验证型
适用于需要核查验证的问题

```
一、核查程序
[列出核查程序]

二、核查结果
[说明核查结果]

三、核查结论
[给出核查结论]
```

## 回复要点

1. **针对性**：直接回应问题，不回避
2. **完整性**：覆盖问题所有方面
3. **准确性**：数据准确、表述严谨
4. **一致性**：与招股书披露一致
