# 问询回复质量检查清单

## 一、内容检查

### 1.1 回复完整性
- [ ] 问题是否全部回应
- [ ] 是否遗漏子问题
- [ ] 是否有避重就轻

### 1.2 回复准确性
- [ ] 数据是否准确
- [ ] 数据来源是否注明
- [ ] 计算过程是否正确

### 1.3 回复一致性
- [ ] 与招股书披露是否一致
- [ ] 与其他回复是否一致
- [ ] 前后表述是否一致

## 二、论证检查

### 2.1 论证充分性
- [ ] 论证是否充分
- [ ] 逻辑是否清晰
- [ ] 因果关系是否明确

### 2.2 数据支撑
- [ ] 是否有数据支撑
- [ ] 数据是否具有说服力
- [ ] 对比分析是否充分

### 2.3 风险揭示
- [ ] 风险是否充分揭示
- [ ] 是否如实披露
- [ ] 是否避重就轻

## 三、核查检查

### 3.1 核查程序
- [ ] 核查程序是否充分
- [ ] 核查方法是否适当
- [ ] 核查范围是否完整

### 3.2 核查证据
- [ ] 核查证据是否充分
- [ ] 证据是否具有证明力
- [ ] 证据是否留存

### 3.3 核查结论
- [ ] 结论是否明确
- [ ] 结论是否有依据
- [ ] 结论是否恰当

## 四、格式检查

### 4.1 格式规范
- [ ] 格式是否符合要求
- [ ] 标题层级是否清晰
- [ ] 序号是否连续

### 4.2 附件材料
- [ ] 附件是否完整
- [ ] 附件是否清晰
- [ ] 附件是否与正文对应

### 4.3 文字校对
- [ ] 是否有错别字
- [ ] 是否有语病
- [ ] 标点是否正确

## 五、签字盖章

### 5.1 签字人员
- [ ] 保荐代表人签字
- [ ] 项目协办人签字
- [ ] 内核负责人签字

### 5.2 盖章
- [ ] 保荐机构盖章
- [ ] 其他中介盖章（如需要）
- [ ] 发行人盖章（如需要）

## 六、提交检查

### 6.1 提交材料
- [ ] 回复文件
- [ ] 附件材料
- [ ] 签字盖章页

### 6.2 提交时间
- [ ] 是否在截止日期前
- [ ] 是否预留审核时间
- [ ] 是否考虑系统故障

### 6.3 提交确认
- [ ] 提交成功确认
- [ ] 回执保存
- [ ] 提交记录归档
