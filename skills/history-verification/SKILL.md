# 历史沿革核验助手

## 功能概述

本技能用于核验公司历史沿革的完整性、合法合规性，确保历次股权变动、增资减资、改制重组等事项合法有效，信息披露准确完整。

## 适用场景

- IPO 申报材料准备阶段
- 历史沿革梳理与核查
- 律师工作报告撰写
- 反馈回复准备

## 核心核验内容

### 一、设立情况核验

| 核验项 | 核验要点 | 所需文件 |
|--------|----------|----------|
| 设立批准 | 批准机关、批准文号 | 设立批复 |
| 注册资本 | 出资金额、出资方式 | 验资报告 |
| 股权结构 | 股东名称、持股比例 | 公司章程 |
| 工商登记 | 登记时间、登记事项 | 营业执照 |

### 二、历次变更核验

#### 1. 股权变更

- [ ] 股权转让协议
- [ ] 股东会决议
- [ ] 工商变更登记
- [ ] 价款支付凭证
- [ ] 税务完税证明

#### 2. 增资扩股

- [ ] 增资协议
- [ ] 股东会决议
- [ ] 验资报告
- [ ] 工商变更登记

#### 3. 减资

- [ ] 减资决议
- [ ] 债权人公告
- [ ] 验资报告
- [ ] 工商变更登记

#### 4. 改制重组

- [ ] 改制方案
- [ ] 审计评估报告
- [ ] 国资批复（如适用）
- [ ] 工商变更登记

### 三、合法合规性核验

#### 1. 程序合规

- [ ] 内部决策程序完备
- [ ] 外部审批程序完备
- [ ] 工商登记及时

#### 2. 实质合规

- [ ] 出资真实足额
- [ ] 股权清晰无争议
- [ ] 不存在代持
- [ ] 不存在纠纷

#### 3. 特殊事项

- [ ] 国有股权管理合规
- [ ] 外资审批合规
- [ ] 股权激励合规
- [ ] 对赌协议清理

### 四、信息披露核验

- [ ] 招股书披露与事实一致
- [ ] 时间节点准确
- [ ] 股东名称准确
- [ ] 持股比例准确
- [ ] 变更原因说明充分

## 使用流程

### 步骤 1：收集历史沿革资料

收集公司设立至今的所有工商档案、决议文件、协议等

### 步骤 2：整理历史沿革时间线

按时间顺序整理历次变更事项

### 步骤 3：运行核验脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\7. 历史沿革核验助手"
python scripts/check_history.py --input <历史沿革数据> --output <核验结果路径>
```

### 步骤 4：查看核验结果

核验报告将列出：
- 已核验事项
- 存在问题
- 补正建议

### 步骤 5：整改完善

根据核验结果补充材料或完善披露

## 输出示例

```markdown
## 历史沿革核验报告

### 公司基本情况
- 公司名称：XX 股份有限公司
- 设立时间：2010-05-20
- 注册资本：10,000 万元
- 变更次数：8 次

### 核验概况
- 核验事项：8 项
- 核验通过：7 项
- 存在问题：1 项

### 存在问题

1. **2018 年股权转让**
   - 问题：缺少价款支付凭证
   - 风险：无法证明转让真实性
   - 建议：补充银行转账凭证或出具确认函

### 核验结论

历史沿革基本清晰，除上述问题外，历次变更合法有效。
```

## 常见风险点

1. **出资不实**：虚假出资、抽逃出资
2. **程序瑕疵**：缺少决议、审批
3. **股权代持**：未披露的代持关系
4. **国资流失**：国有股权转让未评估
5. **外资违规**：未经审批的外资进入
6. **对赌未清**：影响股权稳定的对赌协议
