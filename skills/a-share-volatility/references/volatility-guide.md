# 波动率分析参考指南

## 1. 历史波动率（Historical Volatility, HV）计算方法

### 1.1 Close-to-Close（收盘价波动率）
最基础的计算方法，仅使用收盘价：

```
r_t = ln(C_t / C_{t-1})           # 对数收益率
HV_N = std(r_t, window=N) * sqrt(252)  # N日滚动标准差，年化
```

- 优点：数据要求最低，只需收盘价
- 缺点：忽略了日内波动信息（最高价/最低价）
- 适用：最常用，通用性强

### 1.2 Parkinson 波动率（极差波动率）
利用日内最高价和最低价，捕捉更多波动信息：

```
HV_park = sqrt( 1/(4*N*ln(2)) * sum( ln(H_i/L_i)^2 ) ) * sqrt(252)
```

- 优点：效率是 Close-to-Close 的约 5 倍（相同精度需要更少数据）
- 缺点：假设连续交易（无跳空），A 股有涨跌停可能失真
- 适用：日内波动分析、短窗口 HV 估计

### 1.3 Yang-Zhang 波动率
综合开盘跳空波动（overnight）和日内波动（open-to-close），最准确：

```
sigma_YZ^2 = sigma_overnight^2 + k * sigma_open-to-close^2 + (1-k) * sigma_RS^2
```

其中：
- sigma_overnight = var(ln(O_t / C_{t-1}))  —— 隔夜跳空
- sigma_open-to-close = var(ln(C_t / O_t))  —— 日内波动
- sigma_RS = Parkinson/Rogers-Satchell 估计
- k = 0.34 / (1.34 + (N+1)/(N-1))

- 优点：对跳空和日内波动都有考虑，最小方差估计
- 缺点：需要完整 OHLC 数据
- 适用：精确波动率估计

## 2. 波动率锥（Volatility Cone）构建方法

### 2.1 概念
波动率锥将不同时间窗口的历史波动率分位数分布可视化，形成"锥形"结构（短窗口波动率分布宽、长窗口窄）。

### 2.2 构建步骤
1. 选择窗口集合：{5, 10, 20, 60, 120} 个交易日
2. 对每个窗口 N：
   a. 计算全历史的 N 日滚动 HV 序列
   b. 统计该序列的分位数：P5, P25, P50, P75, P95
3. 以窗口天数为 X 轴，波动率为 Y 轴，绘制分位线
4. 将当前各窗口 HV 值标注在图上

### 2.3 解读方法
- **当前 HV 在 P75 以上**：波动率处于历史偏高水平，均值回归力量倾向使其下降
- **当前 HV 在 P25-P75**：波动率处于正常区间
- **当前 HV 在 P25 以下**：波动率处于历史偏低水平，可能即将扩张
- **短窗口远高于长窗口**：近期波动率激增，可能是事件驱动
- **短窗口远低于长窗口**：近期异常平静，警惕突然波动

### 2.4 波动率锥的"锥形"特征
短窗口的分位数范围宽（P5 和 P95 差距大），长窗口的范围窄，形成类似锥体的形状。这是因为：
- 短窗口受单日极端波动影响大
- 长窗口有平滑效应，极端值被稀释

## 3. GARCH(1,1) 模型原理与参数解读

### 3.1 模型定义
GARCH (Generalized Autoregressive Conditional Heteroskedasticity)：

```
收益方程: r_t = mu + epsilon_t, epsilon_t = sigma_t * z_t, z_t ~ N(0,1)
方差方程: sigma_t^2 = omega + alpha * epsilon_{t-1}^2 + beta * sigma_{t-1}^2
```

### 3.2 参数含义
| 参数 | 含义 | 典型范围 | 解读 |
|------|------|---------|------|
| omega | 常数项（基础方差） | > 0 | 长期方差的"锚" |
| alpha | ARCH 项系数 | 0.03-0.15 | 对昨日冲击的敏感度 |
| beta | GARCH 项系数 | 0.80-0.95 | 波动率的持续性 |
| alpha+beta | 持续性参数 | < 1 | 越接近 1，波动率聚集越强 |

### 3.3 长期方差与半衰期
- 长期年化波动率 = sqrt(omega / (1 - alpha - beta) * 252)
- 波动率半衰期 = ln(2) / ln(alpha + beta)  （回归一半的天数）

### 3.4 参数解读示例
- alpha=0.05, beta=0.93：持续性 0.98，半衰期约 34 天，波动率聚集很强
- alpha=0.10, beta=0.85：持续性 0.95，半衰期约 14 天，对冲击敏感且消退较快

### 3.5 预测
N 步预测方差：
```
sigma_{t+N}^2 = VL + (alpha+beta)^N * (sigma_t^2 - VL)
```
其中 VL = omega / (1 - alpha - beta) 为长期方差。

预测值指数衰减收敛到长期方差。

## 4. 波动率聚集与均值回归特性

### 4.1 波动率聚集（Volatility Clustering）
- 高波动日之后倾向于跟随高波动日
- 低波动日之后倾向于跟随低波动日
- 原因：信息冲击的持续性、市场参与者的群体行为
- GARCH 模型的 beta 参数正是描述这一特性

### 4.2 均值回归（Mean Reversion）
- 波动率长期来看会回归到某个均值水平
- 极高的波动率不会永远持续，会逐渐回落
- 极低的波动率也不会永远持续，终将扩张
- 这是波动率锥分析的理论基础
- GARCH 模型中 alpha+beta < 1 保证了均值回归

### 4.3 非对称性（Leverage Effect）
- 下跌时波动率上升幅度 > 上涨时波动率上升幅度
- 即"坏消息"比"好消息"对波动率的影响更大
- 可用 GJR-GARCH 或 EGARCH 捕捉这一特性
- A 股特色：涨停板和跌停板制度使非对称性更为复杂

## 5. HV vs IV 含义与对比

### 5.1 历史波动率（HV）
- 基于历史价格数据计算
- 反映"过去"的实际波动水平
- 是"已实现"的波动率

### 5.2 隐含波动率（IV）
- 从期权市场价格反推
- 反映市场对"未来"波动的预期
- 包含了市场参与者的情绪和预期

### 5.3 HV-IV 价差分析
| 情况 | 含义 | 策略倾向 |
|------|------|---------|
| IV >> HV | 市场过度恐慌/期权偏贵 | 卖出期权（卖方策略） |
| IV << HV | 市场过度乐观/期权偏低 | 买入期权（买方策略） |
| IV ≈ HV | 期权定价合理 | 方向性策略 |

### 5.4 A 股 IV 数据来源
- 50ETF 期权（510050）：最活跃，IV 数据最可靠
- 300ETF 期权（510300/159919）：流动性较好
- 个股期权：覆盖面有限
- 中国波指（iVIX/中国VIX）：类似 VIX，反映整体市场 IV

## 6. 波动率交易策略简介

### 6.1 做多波动率
- 适用场景：波动率处于历史低位（P25 以下），预期波动率扩张
- 方式：买入跨式（Straddle）/宽跨式（Strangle）
- 风险：时间价值衰减（Theta 损耗）

### 6.2 做空波动率
- 适用场景：波动率处于历史高位（P75 以上），预期波动率收缩
- 方式：卖出跨式/宽跨式、铁蝶式（Iron Butterfly）
- 风险：尾部风险（极端行情可能导致巨额亏损）

### 6.3 波动率均值回归策略
- 当 HV 大幅偏离长期均值时，押注回归
- 结合波动率锥的分位数位置做决策
- 使用 GARCH 预测辅助判断回归速度

### 6.4 HV-IV 套利
- IV > HV 时：卖出期权，赚取波动率溢价
- IV < HV 时：买入期权，在波动中获利
- 需要持续监控 HV-IV 价差的变化

### 6.5 风险提示
- 波动率交易并非无风险套利
- 黑天鹅事件可使波动率瞬间飙升（卖方策略巨亏）
- A 股期权品种有限，流动性可能不足
- 涨跌停制度使得极端波动率被压缩，实际风险可能被低估
