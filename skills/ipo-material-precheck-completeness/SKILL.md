# IPO 材料初审助手 - 章节完整性版

## 功能概述

本技能用于检查招股说明书的章节完整性，确保所有必需章节均已包含，且各章节内容符合《公开发行证券的公司信息披露内容与格式准则》的要求。

## 适用场景

- 招股书撰写过程中的自查
- 保荐机构内核审查
- 申报材料提交前的最终检查
- 反馈回复后的更新检查

## 使用流程

### 步骤 1：准备招股书文本

将招股书转换为可分析的文本格式（TXT 或 Markdown）

### 步骤 2：运行检查脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\2. IPO 材料初审助手 - 章节完整性版"
python scripts/check_chapters.py --input <招股书文本路径> --output <检查结果路径>
```

### 步骤 3：查看检查结果

检查报告将显示：
- ✅ 已包含的章节
- ❌ 缺失的章节
- ⚠️ 内容可能不完整的章节

### 步骤 4：补充完善

根据检查结果补充缺失章节或完善内容

## 输出示例

```markdown
## 招股书章节完整性检查报告

### 检查概况
- 检查日期：2026-03-15
- 应包含章节：17 节

### 缺失章节

### 内容待完善章节

2. 第十三节 募集资金运用 - 缺少"项目效益分析"小节
```

## 注意事项

1. **章节顺序**：章节应按标准顺序排列
2. **章节标题**：标题应规范、准确
3. **交叉引用**：注意章节间的交叉引用是否准确
4. **内容深度**：各章节内容应达到披露要求深度
5. **更新同步**：财务数据更新时，相关章节应同步更新
