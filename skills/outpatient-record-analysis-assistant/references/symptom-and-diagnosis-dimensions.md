# 门诊资料主诉与诊断识别维度

本文件用于统一门诊资料解析时的主诉、现病史和诊断识别口径。

## 一、基本信息维度

优先核对以下信息是否齐备：

- 年龄
- 性别
- 就诊时间
- 医疗机构
- 就诊科室
- 文书类型

如以上任一项缺失且会影响判断，应在门诊基本信息或后续处理建议中说明。

## 二、核心事实提取口径

重点提取以下内容：

- 主诉
- 现病史
- 起病经过
- 初步诊断
- 临床印象
- 待排查诊断
- 伴随疾病

## 三、信息不足识别口径

以下情况应列为信息不足：

- 仅有症状描述，无明确诊断
- 有诊断线索但无医生判断
- 现病史不完整，病程长短不明
- 多次就诊但时间顺序不清
