# 输入字段建议

## 最低必需材料

- 持有产品和市场表现
- 客户画像与风险等级
- 近期波动原因
- 希望触达的场景

## 可选增强材料

- 历史触达记录
- 客户问题反馈
- 渠道模板

## 建议输入结构

~~~json
{
  "task_name": "客户陪伴内容助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
