# akshare-concept

概念板块与题材热点查询 Skill。提供A股市场概念板块行情、板块资金流向、龙头股分析等功能。

## 功能

- 概念板块行情查询
- 板块资金流向
- 概念龙头股分析
- 热点题材追踪
- 板块内个股排行

## 使用示例

查询概念板块：
```bash
python scripts/concept_list.py
```

查询板块行情：
```bash
python scripts/concept_quote.py --name 人工智能
```

查询板块个股：
```bash
python scripts/concept_stocks.py --name 芯片
```

## 数据来源

- AkShare 概念板块数据接口
