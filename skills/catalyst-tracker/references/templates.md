## 催化剂日历
| 时间 | 事件 | 类型 | 确定性 | 预期影响 |
|------|------|------|--------|----------|
| | | | | |

## 重点催化剂详解

### [催化剂 1]
- 时间：[具体日期]
- 内容：[事件描述]
- 市场预期：[市场普遍预期]
- 我们的预期：[我们的判断]
- 超预期条件：[什么算超预期]
- 跟踪方法：[如何跟踪]
- 应对策略：
  - 事前：[策略]
  - 事中：[策略]
  - 事后：[策略]

### [催化剂 2]
- [同上]

## 综合建议
- 仓位建议：[XX%]
- 操作建议：[买入/持有/卖出]
- 止损位：[XX 元]

## 风险提示
- [风险 1]
- [风险 2]
