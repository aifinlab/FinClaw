# 募资文件对比助手 - 法律意见书版

## 功能概述

本技能用于对比法律意见书中关于募投项目的相关表述与招股说明书的一致性，确保法律意见与信息披露一致。

## 适用场景

- 法律意见书撰写
- 申报材料提交前核对
- 招股书更新后的同步检查
- 反馈回复后的更新检查

## 使用流程

### 步骤 1：准备对比文件

收集招股书、法律意见书等文件

### 步骤 2：提取关键信息

从各文件中提取募投项目相关法律信息

### 步骤 3：运行对比脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\13. 募资文件对比助手 - 法律意见书版"
python scripts/compare_legal_opinion.py --input <数据文件> --output <对比结果路径>
```

### 步骤 4：查看对比结果

对比报告将列出：
- 一致的表述
- 存在差异的表述
- 差异原因及处理建议

### 步骤 5：核实修正

对差异项进行核实并修正
