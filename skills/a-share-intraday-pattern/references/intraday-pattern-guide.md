# 日内模式参考指南
## A股交易时段
| 时段 | 时间 | 特征 |
|------|------|------|
| 集合竞价 | 9:15-9:25 | 定价+情绪 |
| 开盘30分钟 | 9:30-10:00 | 高波动/高换手 |
| 早盘 | 10:00-11:30 | 趋势形成 |
| 午盘 | 13:00-14:30 | 相对平淡 |
| 尾盘30分钟 | 14:30-14:57 | 方向确认 |
| 尾盘竞价 | 14:57-15:00 | 收盘定价 |

## 成交量U型分布
- 开盘30分钟: 约15-20%日成交量
- 盘中: 约50-60%
- 尾盘30分钟: 约15-25%

## 日内策略含义
- 开盘跳空后30分钟未回补→日内趋势大概率延续
- 午后反转→'午后效应'，A股较常见
- 尾盘急拉（最后5分钟涨>1%）→次日高开概率大但需警惕
