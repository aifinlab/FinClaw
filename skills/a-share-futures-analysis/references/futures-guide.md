## 二、基差计算与解读

### 基差定义
```
基差 = 期货价格 - 现货指数点位
基差率 = 基差 / 现货指数 × 100%
年化基差率 = 基差率 / 剩余交易日 × 250 × 100%
```

### 基差状态解读
| 状态 | 条件 | 含义 | 市场情绪 |
|------|------|------|---------|
| **升水(contango)** | 基差 > 0 (期货 > 现货) | 市场预期上涨或反映资金成本 | 偏乐观 |
| **贴水(backwardation)** | 基差 < 0 (期货 < 现货) | 对冲压力大或看空情绪浓 | 偏悲观 |
| **平水** | 基差 ≈ 0 | 期现基本一致 | 中性 |

### 基差幅度参考（以IF为例）
| 年化基差率 | 程度 | 说明 |
|-----------|------|------|
| > +5% | 深度升水 | 较罕见，极度乐观或投机过热 |
| +2% ~ +5% | 适度升水 | 正常偏乐观，含资金成本 |
| -2% ~ +2% | 基本平水 | 市场平稳 |
| -5% ~ -2% | 适度贴水 | 对冲需求偏大 |
| < -5% | 深度贴水 | 恐慌或对冲需求极大 |

### 重要提示
- **贴水不等于一定下跌**：贴水反映的是当前情绪和对冲压力，不是方向预测
- **基差必然收敛**：到期日基差趋近于零，这是期货定价的铁律
- **IC/IM长期贴水**：这是A股特有现象，主要因为量化对冲基金大量做空IC/IM对冲，不完全代表看空

## 三、期限结构

### 正向结构 (Contango)
```
当月价格 < 下月价格 < 当季价格 < 下季价格
```
- 正常市场状态，远月包含更多时间价值和持有成本
- 反映市场对未来预期平稳或偏乐观

### 反向结构 (Backwardation)
```
当月价格 > 下月价格 > 当季价格 > 下季价格
```
- 近月贵远月便宜，反映强烈的近期看空压力
- 通常出现在市场恐慌或对冲需求爆发时

### 混合结构
- 近月贴水+远月升水：短期承压但中期预期改善
- 近月升水+远月贴水：短期乐观但中期有隐忧

### 结构变化的意义
- 贴水收窄（向平水靠拢）→ 悲观情绪缓解
- 贴水扩大 → 悲观情绪加剧
- 由升水转贴水 → 市场情绪拐点（变差）
- 由贴水转升水 → 市场情绪拐点（改善）

## 四、持仓分析方法

### 前20名席位多空持仓
中金所每日公布各品种前20名会员的持买仓（多头）和持卖仓（空头）数据。

**核心指标：**
```
净持仓 = 多头持仓 - 空头持仓
多空比 = 多头持仓 / 空头持仓
```

**解读逻辑：**
| 变化 | 含义 |
|------|------|
| 净多增加 | 机构主力加多，看涨意愿增强 |
| 净空增加 | 机构主力加空，看跌意愿增强 |
| 多空同增 | 分歧加大，波动可能放大 |
| 多空同减 | 观望情绪浓，等待方向选择 |

### 重点关注席位
- **中信期货/华泰期货**：通常代表大型机构方向
- **中金期货**：QFII/外资通道，代表外资态度
- **国泰君安/海通期货**：综合类大型席位

### 持仓与价格的配合
| 价格 | 持仓 | 解读 |
|------|------|------|
| 涨 | 增 | 多头主导，趋势健康 |
| 涨 | 减 | 空头平仓推动，持续性存疑 |
| 跌 | 增 | 空头主导，下跌趋势强 |
| 跌 | 减 | 多头止损出场，跌势或近尾声 |

## 五、交割日效应

### 交割日规则
- **交割日**：每月第三个周五（遇节假日顺延）
- **交割方式**：现金交割（以交割日现货指数最后2小时算术平均价为结算价）
- **最后交易日**：与交割日为同一天

### 交割日效应表现
1. **基差收敛**：交割日基差趋近于零
2. **波动放大**：交割日前1-2天及当天波动通常加大
3. **移仓换月**：交割前一周，主力合约逐步从当月转向下月
4. **量价异动**：现货市场可能出现异常波动（所谓"三巫日"效应在A股较弱）

### 2026年交割日历（参考）
每月第三个周五：1月16日、2月20日、3月20日、4月17日、5月15日、6月19日、7月17日、8月21日、9月18日、10月16日、11月20日、12月18日
（实际以中金所公告为准，遇节假日会调整）

## 六、保证金与杠杆

### 保证金比例（交易所标准，期货公司会上浮）
| 品种 | 交易所保证金 | 期货公司通常 | 实际杠杆 |
|------|------------|------------|---------|
| IF | 12% | 15%-18% | 约6-8倍 |
| IH | 12% | 15%-18% | 约6-8倍 |
| IC | 14% | 17%-20% | 约5-7倍 |
| IM | 15% | 18%-22% | 约5-6倍 |

### 一手合约价值（示例）
```
IF：假设沪深300为3800点 → 3800 × 300 = 114万元/手
IH：假设上证50为2600点 → 2600 × 300 = 78万元/手
IC：假设中证500为5500点 → 5500 × 200 = 110万元/手
IM：假设中证1000为5800点 → 5800 × 200 = 116万元/手
```

### 开户门槛
1. **资金要求**：申请前连续5个交易日保证金账户可用资金不低于50万元
2. **知识测试**：通过中金所知识测试（80分以上）
3. **交易经历**：10笔以上商品期货交易记录 或 仿真交易经历
4. **诚信要求**：无不良诚信记录

### 风险提示要点
- 杠杆交易，盈亏放大，可能亏损超过本金
- T+0交易，日内波动可能触发强制平仓
- 到期必须平仓或交割，不能像股票长期持有
- 涨跌停板限制：上一交易日结算价的 ±10%（首日为 ±20%）
- 熔断机制：涨跌达到 ±5% 暂停交易
