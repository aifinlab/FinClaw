# 聪明钱/主力行为识别参考指南 (Smart Money Guide)

## 一、核心指标详解

### 1. 大单净流入占比 (Large Order Flow Ratio)

A股市场中，大单通常定义为单笔成交金额 ≥ 20万元（部分数据源以50万为界）。
大单被认为代表机构或大户资金，是"聪明钱"的直接代理变量。

**计算方法**:
```
large_order_ratio = 大单净流入额 / 当日总成交额 × 100%
```

**解读规则**:
- > +5%: 主力强力流入，积极做多信号
- +2% ~ +5%: 温和流入，关注持续性
- -2% ~ +2%: 多空平衡，主力无明显方向
- -5% ~ -2%: 温和流出，需警惕
- < -5%: 主力大幅撤退，强烈卖出信号

**A股特殊性**:
- 涨停板时大单流入数据可能失真（排队封板导致成交集中）
- 尾盘异常大单需单独关注（可能是机构调仓或做收盘价）
- 连板股的大单数据参考意义有限

### 2. OBV (On Balance Volume / 能量潮)

OBV 由 Joseph Granville 于1963年提出，核心思想是"量在价先"。

**计算方法**:
```
if close[i] > close[i-1]:
    OBV[i] = OBV[i-1] + volume[i]
elif close[i] < close[i-1]:
    OBV[i] = OBV[i-1] - volume[i]
else:
    OBV[i] = OBV[i-1]
```

**关键用法**:
- **OBV 上升 + 价格上升**: 上涨趋势确认，量价配合良好
- **OBV 上升 + 价格下跌/横盘**: 底部吸筹信号（最重要的聪明钱信号之一）
- **OBV 下降 + 价格上升**: 顶部出货信号，上涨缺乏量能支撑
- **OBV 下降 + 价格下跌**: 下跌趋势确认

### 3. A/D Line (Accumulation/Distribution Line / 聚散线)

由 Marc Chaikin 开发，比 OBV 更精细，考虑了价格在日内的位置。

**计算方法**:
```
CLV (Close Location Value) = ((close - low) - (high - close)) / (high - low)
AD[i] = AD[i-1] + CLV × volume[i]
```

**解读**:
- CLV = +1: 收盘价 = 最高价（买方完全控制）
- CLV = -1: 收盘价 = 最低价（卖方完全控制）
- CLV ≈ 0: 收盘在中间位置
- AD 持续上升但价格横盘 → 隐性买入（主力吸筹）
- AD 持续下降但价格横盘 → 隐性卖出（主力出货）

### 4. MFI (Money Flow Index / 资金流量指标)

MFI 被称为"成交量加权的 RSI"，范围 0-100。

**计算方法**:
```
typical_price = (high + low + close) / 3
raw_money_flow = typical_price × volume
positive_flow = sum(raw_money_flow where TP > prev_TP, over N periods)
negative_flow = sum(raw_money_flow where TP < prev_TP, over N periods)
money_ratio = positive_flow / negative_flow
MFI = 100 - 100 / (1 + money_ratio)
```

默认周期 N = 14。

**解读**:
- MFI > 80: 超买区，资金可能过热（主力出货窗口）
- MFI < 20: 超卖区，资金枯竭（主力吸筹窗口）
- MFI 与价格背离: 重要的趋势反转信号

## 二、A股主力行为模式特征

### 吸筹 (Accumulation)

**经典特征**:
- 股价在低位窄幅横盘，波动率下降
- 成交量整体萎缩，但偶尔出现间歇性放量（试盘或收集筹码）
- 大单净流入持续为正，但金额不大（避免引起注意）
- OBV 缓慢上升而价格不动
- 下跌时缩量，小幅反弹时略有放量

**A股典型场景**:
- 业绩拐点前1-3个月
- 重大政策利好发布前（可能存在信息不对称）
- 行业景气度底部反转初期

### 出货 (Distribution)

**经典特征**:
- 股价在高位宽幅震荡，波动率增大
- 成交量维持高位或逐步放大
- 大单持续净流出，尤其是盘中拉高后的大单卖出
- OBV 开始走平或下降而价格仍在高位
- 利好消息频出但股价不再创新高

**A股典型场景**:
- 连续大涨后的高位横盘
- 业绩高点公布后
- 解禁期前后
- 市场情绪极度亢奋时

### 洗盘 (Washout / Shakeout)

**经典特征**:
- 股价急跌但很快收回（长下影线）
- 下跌时成交量小，说明抛压有限
- 大单流出金额占比小（主力没有真正在卖）
- 急跌后 OBV 未明显下降
- 跌破关键支撑后快速收回

**与出货的区别**:
- 洗盘: 缩量下跌 → 放量反弹，大单流出少
- 出货: 放量下跌 → 缩量反弹，大单持续流出

### 拉升 (Markup)

**经典特征**:
- 连续上涨或突破关键阻力位
- 持续放量，量价齐升
- 大单强劲流入，占比持续 > 5%
- OBV 加速上升
- MFI 进入高位但尚未极端超买

## 三、量价背离分析 (Volume-Price Divergence)

### 顶背离（看跌信号）
- 价格创新高，但成交量未创新高
- 价格创新高，但 OBV/AD 未创新高
- 价格创新高，但 MFI 未创新高
- 多重背离同时出现时信号更强

### 底背离（看涨信号）
- 价格创新低，但成交量萎缩（抛压衰竭）
- 价格创新低，但 OBV/AD 未创新低
- 价格创新低，但 MFI 未创新低

## 四、局限性与注意事项

### 数据层面
- 大单定义标准不统一（20万/50万/100万），跨平台对比需注意
- 资金流向数据本质上是对成交的事后分类，非实时买卖意图
- 部分数据源的大单数据存在延迟（T+1）

### 方法论层面
- 单一指标容易产生假信号，必须多指标交叉验证
- 主力行为模式并非非此即彼，可能同时存在多种模式
- 市场极端行情（连续涨停/跌停）下指标失效
- 量化模型无法捕捉突发事件（黑天鹅）

### A股特殊性
- T+1 制度下，当日买入无法卖出，影响资金流向解读
- 涨跌停板制度导致极端行情下量价关系失真
- 北向资金（陆股通）是另一个重要的"聪明钱"代理变量，建议结合使用
- 股指期货/期权对冲行为可能导致现货资金流向信号失真
- 量化交易占比提升，传统大单分析的有效性在下降

### 投资建议
- 本工具仅为量化辅助分析，不构成任何投资建议
- 应结合基本面、技术面、消息面综合判断
- 任何单一信号都不应作为买卖的唯一依据
- 严格控制仓位和风险管理
