# 沟通话术

## 一、电话沟通话术

### 1.1 首次预警通知

```
"您好，[客户姓名]，我是 XX 证券的 [姓名]。

您的信用账户当前维保比例为 X%，已经低于预警线 (140%)。

为了您的账户安全，建议您在 [日期] 前追加 [万元] 的担保物，或者主动减仓降低负债。

如果维保比例持续下降，可能会触发强制平仓，造成不必要的损失。

您看您打算采取什么措施？有什么我可以帮您的吗？"
```

### 1.2 追保通知

```
"您好，[客户姓名]，我是 XX 证券的 [姓名]。

您的信用账户当前维保比例为 X%，已经低于警戒线 (130%)，情况比较紧急。

请您务必于 [日期] 前追加 [万元] 的担保物，使维保比例恢复至 150% 以上。

您可以选择：
1. 追加现金
2. 追加证券
3. 偿还负债
4. 主动减仓

如果未按时追保，我司将启动强制平仓程序，由此产生的损失由您自行承担。

您现在有什么打算？我可以帮您分析一下。"
```

### 1.3 平仓预告

```
"您好，[客户姓名]，我是 XX 证券的 [姓名]。

您的信用账户当前维保比例为 X%，已经低于平仓线 (120%)。

经多次通知，您尚未追加担保物。我司将于 [日期] 启动强制平仓程序。

平仓后如果仍有负债，我司有权继续追偿。

您现在还有什么问题吗？"
```

## 二、微信沟通话术

### 2.1 预警通知

```
[客户姓名] 您好，您的信用账户维保比例当前为 X%，低于预警线。

当前情况：
- 总资产：[万元]
- 总负债：[万元]
- 维保比例：X%
- 需追加：[万元]

建议您尽快追加担保物或减仓，避免触发平仓。

详询：[客户经理姓名] [电话]
```

### 2.2 追保通知

```
[客户姓名] 您好，您的信用账户维保比例已低于警戒线，情况紧急！

当前维保比例：X%
警戒线：130%

需追加：[万元]

请您务必于截止日期前追加担保物，否则将启动平仓程序。

紧急联系：[姓名] [电话]
```

## 三、面谈沟通话术

### 3.1 开场

```
"[客户姓名] 您好，感谢您今天过来。

今天请您来主要是想跟您沟通一下您信用账户的风险情况。

当前您的维保比例是 X%，已经低于 [预警线/警戒线]。"
```

### 3.2 风险说明

```
"我给您看一下您当前的持仓情况...

您看，您的最大持仓是 [证券]，占比 X%，集中度比较高。

如果这个证券下跌 10%，您的维保比例会下降到 X%。

如果下跌 20%，可能会触发平仓线。"
```

### 3.3 方案建议

```
"针对您当前的情况，我们有几个建议：

1. 追加 [万元] 现金，这是最直接的方式
2. 追加 [证券]，按 X% 折算率，需要追加市值 [万元]
3. 主动减仓 [证券]，降低集中度
4. 偿还 [万元] 融资负债

您看您倾向于哪种方式？"
```

### 3.4 结束

```
"好的，那我们就这么定了。

您于 [日期] 前完成 [措施]。

我会持续跟进您的账户情况，如有任何问题随时联系我。

这是我的名片，有什么问题随时打电话给我。"
```

## 四、特殊情况处理

### 4.1 客户情绪激动

```
"我理解您现在的心情，市场波动确实让人焦虑。

我们共同的目标是保护好您的账户，避免不必要的损失。

让我们一起看看有什么可行的解决方案..."
```

### 4.2 客户表示困难

```
"我理解您目前的困难。

我们可以看看有没有其他方式，比如：
1. 部分追加，先缓解风险
2. 分批减仓，减少损失
3. 与家属商量，寻求帮助

您看哪种方式比较可行？"
```

### 4.3 客户拒绝配合

```
"我理解您的想法，但我需要提醒您：

如果维保比例持续低于平仓线，我司将有权启动强制平仓程序。

平仓价格可能不利于您，造成额外损失。

而且平仓后如果仍有负债，我司有权继续追偿。

请您慎重考虑。"
```
