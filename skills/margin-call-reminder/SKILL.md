# 追保提醒助手

## 触发场景

使用此技能当：
- 用户需要向客户发送追加担保物通知
- 用户要跟踪客户的追保执行情况
- 用户需要管理追保期限和截止时间
- 用户要记录与客户的追保沟通
- 用户提到"追保"、"追加担保物"、"维保比例不足"、"补仓"等

## 核心工作流

### 1. 确认客户基本信息

首先向用户确认以下信息：
- **客户姓名/代码**：[客户标识]

## 三、风险提示

### 3.1 未追保后果
如未在规定期限内追加担保物，可能导致：

1. **强制平仓**：我司有权启动强制平仓程序
2. **平仓损失**：平仓价格可能不利于客户，造成额外损失
3. **信用影响**：可能影响客户在我司的信用记录
4. **追偿权利**：如平仓后仍不足以偿还债务，我司有权继续追偿

### 3.2 平仓流程
1. 追保期限届满，维保比例仍低于警戒线
2. 发送平仓预告通知
3. 如仍未改善，启动强制平仓
4. 平仓顺序：[顺序说明]
5. 平仓目标：维保比例恢复至 150% 以上

### 3. 通知模板库

#### 3.1 首次通知（维保比例<140%）

```markdown
【XX 证券】维保比例预警通知

尊敬的 [客户姓名]：

您的信用账户当前维保比例为 X%，已低于预警线 (140%)。

当前持仓情况：
- 总资产：[万元]
- 总负债：[万元]
- 维保比例：X%

请您于 [日期] 前采取以下任一措施：
1. 追加担保物（现金或可充抵保证金证券）
2. 主动减仓降低负债
3. 偿还部分融资/融券负债

如维保比例持续低于警戒线 (130%)，我司将有权启动强制平仓程序。

详询您的客户经理：[姓名] [电话]

XX 证券股份有限公司
YYYY-MM-DD
```

### 4. 追保方式建议

| 客户情况 | 建议方式 | 优点 | 缺点 |
|----------|----------|------|------|
| 有闲置资金 | 追加现金 | 快速、简便 | 占用资金 |
| 持有其他证券 | 追加证券 | 不占用现金 | 需考虑折算率 |
| 持仓盈利 | 主动减仓 | 释放资金、降低风险 | 可能错失上涨 |
| 有还款能力 | 偿还负债 | 直接降低负债 | 占用资金 |

### 5. 沟通话术

#### 5.1 电话沟通话术

```
"您好，[客户姓名]，我是 XX 证券的 [姓名]。

您的信用账户当前维保比例为 X%，已经低于预警线 (140%)。

为了您的账户安全，建议您在 [日期] 前追加 [万元] 的担保物，或者主动减仓降低负债。

如果维保比例持续下降，可能会触发强制平仓，造成不必要的损失。

您看您打算采取什么措施？有什么我可以帮您的吗？"
```
