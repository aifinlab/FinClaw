# 输出结构建议

## 一、标准报告结构

1. 客户情况概述
2. 当前持仓与问题摘要
3. 核心波动/变化原因
4. 沟通主线建议
5. 推荐输出内容
   - 电话提纲
   - 微信文案
   - 短信文案
6. 后续跟进建议
7. 风险提示
8. 待补充信息

## 二、字段说明

- customer_segment：客户分层
- risk_profile：风险承受能力
- holding_summary：持仓摘要
- issue_summary：当前问题摘要
- communication_goal：沟通目标
- recommended_tone：推荐语气
- key_messages：核心表达点
- action_suggestions：动作建议
- risk_notice：风险提示
- missing_information：待补充信息
