# 投资逻辑提炼输出模板

## 标准模板（300-500 字）

```markdown
# 【[公司/行业] 投资逻辑】

## 核心论点
[一句话投资判断，50 字以内]

## 关键论据
1. [论据 1 + 数据支撑]
2. [论据 2 + 数据支撑]
3. [论据 3 + 数据支撑]

## 催化剂
- 近期（1-3 个月）：[催化剂 1]
- 中期（3-12 个月）：[催化剂 2]

## 风险因素
- [风险 1]
- [风险 2]

## 验证方法
- 跟踪指标：[指标 1]、[指标 2]
- 验证时间：[时间]
- 证伪条件：[什么情况下逻辑被证伪]

## 投资建议
- 评级：[买入/增持/中性]
- 目标价：[XX 元]
- 仓位建议：[XX%]
```

## 极简模板（100-200 字）

```
【[公司/行业] 投资逻辑】

论点：[一句话]

论据：①[论据 1] ②[论据 2] ③[论据 3]

催化剂：[时间 + 事件]

风险：[风险 1]、[风险 2]
```
