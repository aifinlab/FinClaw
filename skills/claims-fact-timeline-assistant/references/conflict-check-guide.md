# 时间冲突与疑点检查参考

## 常见冲突类型
- 同一事件在不同材料中的日期不一致。
- 首次就医时间早于出险时间或与事故说明明显不匹配。
- 出院时间早于住院时间、手术时间晚于出院时间等顺序倒置。
- 报案时间明显晚于关键事件，且中间缺少解释节点。
- 补件通知、补件完成、审核启动等系统节点顺序混乱。

## 常见缺口类型
- 缺少首次就医时间。
- 缺少报案时间或受理时间。
- 缺少补件完成时间或调查启动时间。
- 缺少结论形成或结案时间。

## 表达原则
- 能确认冲突的，直接写明冲突双方和来源。
- 不能确认取舍的，并列展示，不强行选择其一。
- 对异常延迟或空档，提示“建议核实原因”而非直接下判断。
