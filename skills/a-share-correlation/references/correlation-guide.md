# 相关性分析参考指南

## 一、皮尔逊相关系数解读

| 相关系数范围 | 强度等级 | 含义 | 投资启示 |
|-------------|---------|------|---------|
| 0.7 ~ 1.0 | 强正相关 | 两标的走势高度一致 | 持有两者无分散化效果，选其一即可 |
| 0.3 ~ 0.7 | 中等正相关 | 走势有一定同步性 | 有限分散化效果，需关注是否趋向强相关 |
| 0.0 ~ 0.3 | 弱正相关 | 走势较为独立 | 较好的分散化配对 |
| -0.3 ~ 0.0 | 弱负相关 | 走势略有反向 | 良好的分散化配对 |
| -0.7 ~ -0.3 | 中等负相关 | 走势有一定反向性 | 天然对冲效果，适合风险管理 |
| -1.0 ~ -0.7 | 强负相关 | 走势高度反向 | 强对冲工具，极少见于A股个股之间 |

**注意**：A股市场整体同涨同跌特征明显，个股间相关系数普遍偏高（0.3-0.7 区间居多），真正低相关或负相关的配对较少。

## 二、Beta 系数解读

| Beta 范围 | 类型 | 特征 | 典型标的 |
|-----------|------|------|---------|
| Beta > 1.5 | 高弹性/进攻型 | 涨跌幅显著大于大盘，牛市超额明显 | 科技成长股、小盘股、券商 |
| 1.0 < Beta < 1.5 | 偏进攻型 | 涨跌幅略大于大盘 | 消费电子、新能源 |
| 0.8 < Beta < 1.0 | 同步型 | 与大盘基本同步 | 沪深300成分股、大蓝筹 |
| 0.5 < Beta < 0.8 | 偏防御型 | 涨跌幅小于大盘，熊市抗跌 | 消费白马、公用事业 |
| Beta < 0.5 | 强防御型 | 与大盘关联度低 | 银行、高速公路、水电 |
| Beta < 0 | 反向型 | 与大盘反向运动，极罕见 | 部分对冲策略ETF |

**R-squared（决定系数）**：
- R^2 > 0.5：Beta 具有较强解释力，可信度高
- 0.3 < R^2 < 0.5：Beta 有一定参考价值
- R^2 < 0.3：Beta 可靠性低，个股走势主要由非市场因素驱动

## 三、滚动相关性的意义

### 为什么看滚动相关性？
- 静态相关系数是一个时间段的平均值，掩盖了相关性的时变特征
- 滚动相关性揭示关系是否稳定、何时发生结构性变化

### 如何解读滚动相关性？
| 现象 | 解读 | 操作建议 |
|------|------|---------|
| 相关性长期稳定在高位（>0.7） | 两标的存在持续的联动逻辑（如同行业/同概念） | 不适合同时重仓 |
| 相关性长期稳定在低位（<0.3） | 真正独立的标的，分散化效果稳定 | 理想的组合搭配 |
| 相关性近期快速上升 | 可能受共同事件驱动（政策/行业变化），分散化效果正在减弱 | 警惕集中风险 |
| 相关性近期快速下降 | 两标的正在"脱钩"，可能有差异化催化剂 | 关注背离原因 |
| 相关性大幅波动 | 关系不稳定，难以依赖 | 不宜基于相关性做长期配置 |

### 推荐窗口期
- 20 日（约 1 个月）：捕捉短期联动变化，噪音较大
- 60 日（约 3 个月）：平衡敏感度和稳定性，**推荐默认值**
- 120 日（约半年）：反映中期趋势，较为平滑
- 250 日（约 1 年）：长期结构性关系

## 四、分散化原理

### 组合波动率公式
对于两资产组合（权重 w1, w2）：
```
σ_portfolio = sqrt(w1^2 * σ1^2 + w2^2 * σ2^2 + 2 * w1 * w2 * ρ * σ1 * σ2)
```
- 当 ρ = 1 时：组合波动率 = 加权平均波动率（无分散化效果）
- 当 ρ = 0 时：组合波动率 < 加权平均波动率（分散化有效）
- 当 ρ = -1 时：理论上可构建零波动组合（完美对冲）

### A股分散化实践
- 跨行业配置：如"消费 + 科技 + 金融"通常比同行业内分散更有效
- 跨风格配置：价值 + 成长，大盘 + 小盘
- 跨资产配置：股票 + 债券/黄金/商品（需借助 ETF）
- 3-5 个低相关标的通常已能获得大部分分散化收益

## 五、相关性陷阱与常见误区

### 1. 极端行情下相关性趋 1
- 股灾、流动性危机时，所有资产同步下跌
- 正常时期的低相关性在危机中可能失效
- **启示**：分散化是"好天气"策略，极端风险需要对冲或现金

### 2. 相关不等于因果
- 高相关可能因为共同的第三方因素（如宏观经济）
- 例：A股和铜价高相关，不代表铜价驱动A股，而是都受经济预期驱动
- **启示**：分析联动关系时要追溯底层逻辑

### 3. 历史相关性不保证未来
- 行业格局变化、政策转向都可能打破历史相关性
- 例：光伏和锂电曾高度相关，政策分化后相关性可能下降
- **启示**：定期重新计算，关注基本面变化

### 4. 样本期选择偏差
- 不同时间段计算出的相关性可能差异很大
- 牛市/熊市/震荡市的相关性特征不同
- **启示**：至少使用 1 年数据，最好覆盖不同市场阶段

### 5. 停牌数据扭曲
- A股停牌制度导致部分股票数据不连续
- 停牌期间收益率为 0，人为压低相关性
- **启示**：计算前剔除停牌日数据
