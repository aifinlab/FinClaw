# 输入字段建议

## 最低必需材料

- 产品资料
- 申购赎回和业务规则
- 费用与时间节点
- 现有问答样本

## 可选增强材料

- 渠道差异规则
- 系统操作流程
- 客服反馈

## 建议输入结构

~~~json
{
  "task_name": "渠道FAQ助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
