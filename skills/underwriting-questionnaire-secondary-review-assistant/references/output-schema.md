# 复核结果标准输出结构

本文件用于统一核保问卷复核输出结构，确保结果可直接用于复核留痕、质量控制和核保流转。

## 一、复核对象基本信息

建议字段：

- 险种/产品：
- 被保人基本信息：
- 问卷类型：
- 初审结论：
- 复核材料来源或文本类型：

## 二、复核结论摘要

建议输出 1 至 3 条：

- 初审意见是否充分
- 是否存在明显遗漏或判断偏差
- 补件是否足以支撑下一步判断

## 三、初审意见复核

建议从三点组织：

- 初审已识别问题概况
- 初审判断合理性
- 初审遗漏或不充分点

## 四、复核发现的问题

建议按问题类别书写：

- 初审遗漏
- 判断不一致
- 补件不足
- 高风险未充分核查

## 五、补问与补件充分性检查

建议覆盖：

- 已补充内容是否覆盖关键疑点
- 当前仍缺哪些关键说明或资料
- 哪些材料不足以支持继续流转

## 六、高风险项复核与风险提示

推荐分为四段：

- 健康风险复核
- 职业风险复核
- 生活习惯风险复核
- 投保行为风险复核

写法建议采用“初审处理情况 + 复核判断 + 仍待确认点”。

## 七、建议进一步核实的问题

问题应能直接用于继续复核、前台补问或正式核保继续核实。

## 八、后续处理建议

可组合使用以下动作：

- 建议补充说明
- 建议追加补件
- 建议升级人工核查
- 建议进入正式核保判断
- 建议退回补问或重新审查

必须避免写成正式核保结论。

## 九、材料质量异常写法

如材料质量较差，可补充一句：

- 部分复核依据不完整，以下结论仅基于当前可确认内容整理。
