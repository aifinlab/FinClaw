# 关联交易扫描助手

## 功能概述

本技能用于扫描识别公司关联交易，核查关联方的完整性、关联交易的必要性、定价公允性及决策程序规范性。

## 适用场景

- IPO 申报前关联交易核查
- 关联方识别与披露
- 关联交易公允性分析
- 关联交易规范性检查

## 核心扫描内容

### 一、关联方识别

#### 1. 关联法人

- [ ] 控股股东、实际控制人控制的其他企业
- [ ] 持股 5% 以上股东
- [ ] 董事、监事、高管控制或担任董监高的企业
- [ ] 上述主体的近亲属控制的企业

#### 2. 关联自然人

- [ ] 控股股东、实际控制人
- [ ] 持股 5% 以上自然人股东
- [ ] 董事、监事、高管
- [ ] 上述人员的近亲属

#### 3. 潜在关联方

- [ ] 历史关联方（过去 12 个月）
- [ ] 特殊关系方（合营、联营等）
- [ ] 实质重于形式认定的关联方

### 二、关联交易类型

| 交易类型 | 常见形式 | 核查要点 |
|----------|----------|----------|
| 采购商品 | 向关联方采购原材料 | 定价公允性、必要性 |
| 销售商品 | 向关联方销售产品 | 定价公允性、真实性 |
| 提供劳务 | 向关联方提供服务 | 定价公允性 |
| 接受劳务 | 接受关联方服务 | 定价公允性、必要性 |
| 资产租赁 | 租赁关联方资产 | 租金公允性 |
| 资金拆借 | 关联方资金往来 | 利息公允性、合规性 |
| 担保 | 关联担保 | 必要性、反担保 |
| 资产转让 | 收购/出售关联方资产 | 定价公允性 |
| 股权交易 | 收购/出售关联方股权 | 定价公允性 |

### 三、核查要点

#### 1. 必要性

- [ ] 交易背景真实
- [ ] 具有商业合理性
- [ ] 无法由独立第三方替代

#### 2. 公允性

- [ ] 定价方法合理
- [ ] 价格与市场价可比
- [ ] 毛利率与非关联交易一致

#### 3. 规范性

- [ ] 履行决策程序
- [ ] 独立董事发表意见
- [ ] 信息披露充分

#### 4. 独立性影响

- [ ] 不影响业务独立性
- [ ] 关联交易占比合理
- [ ] 不存在依赖

### 四、量化分析

| 指标 | 计算公式 | 关注标准 |
|------|----------|----------|
| 关联采购占比 | 关联采购/总采购 | <30% |
| 关联销售占比 | 关联销售/总销售 | <30% |
| 关联资金占用 | 关联方应收余额 | 申报前清零 |
| 关联担保余额 | 对外担保余额 | 逐步减少 |

## 使用流程

### 步骤 1：收集关联交易数据

收集关联方清单、关联交易合同、财务数据等

### 步骤 2：运行扫描脚本

```bash
cd "C:\Users\51994\Desktop\skills\证券 2\9. 关联交易扫描助手"
python scripts/scan_related_transactions.py --input <交易数据> --output <扫描结果路径>
```

### 步骤 3：查看扫描结果

扫描报告将包含：
- 关联方清单
- 关联交易汇总
- 公允性分析
- 风险提示

### 步骤 4：规范整改

根据扫描结果规范关联交易
