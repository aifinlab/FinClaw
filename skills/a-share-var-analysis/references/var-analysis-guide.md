# VaR风险价值参考指南

## 三种计算方法

### 历史模拟法
1. 收集过去N日（如250日）收益率
2. 从小到大排序
3. 95% VaR = 第 N×5% 个收益率（负值）
4. 优点：不假设分布；缺点：依赖历史

### 参数法（方差-协方差法）
VaR(α) = μ - z_α × σ
- 95%: z = 1.645; 99%: z = 2.326
- 组合VaR: σ_p = √(w'Σw)
- 优点：计算快；缺点：正态假设

### Monte Carlo 模拟
1. 估计收益率分布参数（或用Copula）
2. 模拟10000+条路径
3. 排序取分位数
- 优点：灵活；缺点：计算量大

## CVaR (Expected Shortfall)
CVaR = E[L | L > VaR]
比VaR更保守，关注尾部平均损失。
CVaR满足次可加性（subadditivity），VaR不满足。

## A股VaR经验值
| 资产 | 日95%VaR | 日99%VaR |
|------|---------|---------|
| 沪深300 | -2.0% | -3.5% |
| 中证500 | -2.5% | -4.5% |
| 个股(均值) | -4% | -7% |
| 创业板 | -3.0% | -5.5% |

## VaR回测
- Kupiec检验：实际突破次数 vs 理论次数
- Christoffersen检验：突破的独立性
- 合格标准：突破比例在置信区间内
