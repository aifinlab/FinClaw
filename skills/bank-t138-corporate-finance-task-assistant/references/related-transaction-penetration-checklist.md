# 关联交易穿透检查清单（快用版）

## 1. 目标与范围

- 穿透目标：资金占用 / 利润转移 / 循环交易 / 虚增收入 / 掩盖债务
- 时间窗口：至少覆盖最近 12 个月或关键异常期

## 2. 交易归类（先抓大额与高频）

- 销售/采购（含代理、委托加工、委托销售）
- 资金拆借、代收代付、保证金代垫
- 资产买卖、股权转让、费用分摊
- 担保与或有事项（关联担保、反担保）

## 3. 合理性测试（每类交易至少回答一条）

- 商业目的：交易为什么必须发生？是否可被外部第三方替代？
- 定价：是否偏离可比？是否通过价格搬移利润？
- 结算与账期：是否存在超长账期、期末突击、长期挂账？
- 对手方能力：关联方是否具备履约能力与经营实质？

## 4. 穿透资金流与货物流

- 是否存在回流/循环/同日往返？
- 资金用途是否与合同匹配？
- 是否存在“先回款后开票/先开票后回款”异常链条？

## 5. 风险输出

- 红旗列表 + 对应证据缺口
- 核验要点（最短路径）
- 缓释条件（资金监管、限额、押品、退出计划）
