## A 股 DCF 估值实操指南

### WACC 参数参考

- 来源：中国 10 年期国债收益率
- 近年范围：2.3% - 3.5%

### FCFF 计算详解

```
方法一（从利润表出发）:
FCFF = 净利润 + 利息费用×(1-T) + 折旧摊销 - 资本开支 - 营运资金增加

方法二（从 EBIT 出发，推荐）:
FCFF = EBIT × (1-T) + 折旧摊销 - 资本开支 - 营运资金增加

cn-stock-data 可直接获取:
- 净利润: basic_eps × 总股本（或从利润表推算）
- 折旧摊销: 需从 EBITDA - EBIT 推算，或 web 搜索年报现金流量表
- 资本开支: 需从 web 搜索年报获取
- 营运资金: 流动资产 - 流动负债的变动

实操建议:
- cn-stock-data 的 finance 数据适合做趋势分析和比率计算
- 具体的折旧/capex/营运资金明细通常需要 web 搜索年报补充
```

### 营运资金简化估算

```
营运资金 ≈ (应收账款 + 存货) - 应付账款
营运资金变动 = 本期营运资金 - 上期营运资金

简化法（当详细数据不可得时）:
营运资金增加 ≈ 收入增量 × 历史营运资金/收入比率
```

### 终值合理性检查

- 终值占总价值的比例：40%-70% 为合理，>75% 说明预测期过短或增长假设过保守
- 永续增长率 g：不应超过 GDP 长期增速（2%-3%），通常取 2.5%
- 退出倍数法：参考当前行业 EV/EBITDA 中位数，而非历史高点
- 两种方法算出的终值应在 ±20% 以内，否则需检查假设一致性

### 常见错误

1. **双重计税**：用税后利润又减税 → EBIT×(1-T) 或净利润+利息×(1-T)，不能混用
2. **忽略营运资金**：高增长公司的营运资金需求可能吃掉大量现金流
3. **Beta 取历史值不调整**：A 股 Beta 波动大，建议用 2-3 年周 Beta，并参考行业中位数
4. **永续增长率过高**：g > 3% 在 A 股几乎没有道理
5. **忽略少数股东权益**：从 EV 到股权价值要减去少数股东权益
6. **不区分经营性和非经营性资产**：持有大量投资性房产或金融资产的公司，需单独估值

### 不适用 DCF 的公司类型

- 银行/保险/券商（用 PB 或 DDM/PEV）
- 早期亏损公司（无正 FCFF）
- 强周期公司（难以预测现金流趋势）
- 地产公司（用 NAV 更合适）
