# 资本配置分析框架参考指南

## 一、Warren Buffett 的资本配置5种选择

企业经营产生自由现金流后，管理层面临5种选择（按 Buffett 的优先级排序）：

### 1. 再投资于现有业务（Reinvest in Existing Operations）
- **适用条件**：ROIC > WACC，且有足够的增长空间
- **形式**：资本开支(Capex)、研发投入(R&D)、营运资金扩张
- **判断标准**：增量 ROIC 是否持续高于资本成本
- **A股常见问题**：盲目扩产导致产能过剩（如光伏、面板行业周期性扩产）

### 2. 并购（Acquisitions）
- **适用条件**：标的与主业协同、价格合理、整合能力强
- **Buffett 原则**：只在价格合理时收购优质企业，不参与竞购
- **判断标准**：并购后 ROIC 是否提升、商誉是否过高
- **A股常见问题**：高溢价跨界并购（对赌期后业绩变脸 → 商誉暴雷）

### 3. 分红（Dividends）
- **适用条件**：ROIC < WACC，或公司进入成熟期，缺乏高回报投资机会
- **优势**：给予股东灵活选择权（再投资或消费）
- **劣势**：双重征税（企业所得税 + 个人所得税），A股持股<1年税率20%
- **A股特色**：监管鼓励分红（分红不达标可能影响再融资资格）

### 4. 回购（Share Buybacks）
- **适用条件**：股价低于内在价值时回购最优
- **优势**：避免分红的双重征税，集中度提升
- **关键区分**：
  - **注销式回购**：真正减少股本，提升每股价值 → 真回购
  - **库存股/激励用途**：不减少股本，本质是员工薪酬 → 假回购
- **A股特色**：2018年《公司法》修订后回购更便利，但注销比例仍低

### 5. 偿还债务（Debt Repayment）
- **适用条件**：杠杆过高、利息负担重、或经济下行周期
- **判断标准**：净负债/EBITDA > 3x 时应优先降杠杆
- **A股特色**：民企融资成本高，降杠杆的边际收益更高

## 二、ROIC（投入资本回报率）分析

### 计算方法
```
ROIC = NOPAT / Invested Capital
     = 税后经营利润 / 投入资本
     = EBIT × (1 - 税率) / (股东权益 + 有息负债 - 现金)
```

### WACC（加权平均资本成本）
```
WACC = E/(E+D) × Ke + D/(E+D) × Kd × (1-T)
```
- A股股权成本(Ke)一般取 8%-12%（无风险利率 + 风险溢价）
- 简化估算：多数 A 股公司 WACC 在 8%-10% 区间

### ROIC 分析要点
- **ROIC > WACC → 创造价值**：值得再投资
- **ROIC < WACC → 毁灭价值**：应分红/回购
- **增量 ROIC**：近3年新增资本的回报率（更反映当前投资效率）
- **ROIC 趋势**：持续下滑说明增长空间在收窄

## 三、并购与商誉风险

### 商誉产生
```
商誉 = 收购价格 - 标的净资产公允价值
```

### A股商誉风险指标
| 风险等级 | 商誉/净资产 | 说明 |
|---------|-----------|------|
| 低风险 | < 10% | 正常水平 |
| 中风险 | 10%-30% | 需关注并购标的业绩 |
| 高风险 | 30%-50% | 商誉减值可能显著影响利润 |
| 极高风险 | > 50% | 一旦减值可能导致亏损 |

### 并购质量评估清单
1. **战略匹配度**：产业链上下游协同 > 横向整合 > 跨界并购
2. **对价合理性**：标的 PE/PS 与行业对比
3. **业绩承诺**：对赌期限（3年居多）、完成率、到期后风险
4. **整合进展**：管理团队留任、客户流失率、协同效应实现情况
5. **历史记录**：管理层过往并购的成功率

### A股并购雷区
- 跨界并购追热点（如传统行业收购游戏/影视公司）
- 对赌到期后业绩断崖
- 关联方资产注入（利益输送风险）
- 借壳上市后的整合问题

## 四、A股特色再融资工具

### 定向增发（定增）
- **折价发行**：发行价一般为市价的80%（锁价定增可更低）
- **锁定期**：竞价定增6个月，锁价定增18个月
- **稀释效应**：增发股数/原总股本 = 摊薄比例
- **评估重点**：
  - 募资用途是否合理（项目投资 vs 补充流动资金 vs 偿债）
  - 大股东是否参与（参与 = 看好 vs 不参与 = 信号？）
  - 发行价格折扣率

### 配股
- **全体股东按持股比例认购**：不认购则被摊薄
- **A股实务**：多数公司优先选择定增而非配股
- **股东影响**：被动跟投或接受摊薄

### 可转债
- **兼具债券和期权特征**
- **转股摊薄**：转股价 → 转股后增加的股本
- **评估重点**：转股价 vs 当前股价、下修条款、强赎条款

## 五、好的 vs 差的资本配置案例

### 好的资本配置特征
- 高 ROIC 时积极再投资，低 ROIC 时积极分红/回购
- 并购聚焦主业、价格合理、整合成功
- 分红稳定增长，不因短期波动中断
- 股价低估时回购并注销
- 资产负债率保持在合理区间
- **典型案例**：贵州茅台（高分红+适度资本开支）、海螺水泥（高分红+低杠杆）、美的集团（并购整合+持续回购注销）

### 差的资本配置特征
- ROIC 低于 WACC 仍大举扩张
- 高溢价跨界并购，商誉堆积
- 分红吝啬但管理层薪酬丰厚
- 回购不注销，实际是股权激励
- 频繁再融资（定增/可转债）摊薄股东
- 账上堆积大量现金但不分红（代理问题）
- **典型案例**：乐视网（跨界扩张失控）、暴风集团（高溢价并购+商誉暴雷）

## 六、行业生命周期与资本配置匹配

| 生命周期 | 特征 | 最优资本配置 | A股典型行业 |
|---------|------|-----------|-----------|
| **导入期** | 高增长+亏损 | 全力再投资+融资 | 创新药、AI应用 |
| **成长期** | 高增长+盈利 | Capex+研发为主 | 新能源、半导体 |
| **成熟期** | 稳定增长+高盈利 | 分红+回购为主 | 白酒、银行 |
| **衰退期** | 负增长 | 偿债+转型+高分红 | 传统煤炭（但周期反转例外） |

## 七、资本配置评级框架

| 评级 | 标准 |
|------|------|
| **优秀(A)** | ROIC持续>WACC+5%，分红+回购率>50%，无重大商誉减值，资本结构稳健 |
| **良好(B)** | ROIC>WACC，分红稳定，并购整体成功，杠杆合理 |
| **一般(C)** | ROIC≈WACC，分红不稳定，有零星并购减值，偶有再融资 |
| **较差(D)** | ROIC<WACC仍扩张，商誉占比高，频繁再融资，分红吝啬 |
| **极差(E)** | 持续毁灭价值，商誉暴雷，大股东掏空，资不抵债风险 |
