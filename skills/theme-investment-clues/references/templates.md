# 主题投资线索输出模板

## 标准模板

```markdown
# 【[主题名称] 投资线索】

## 主题定义
[主题内涵 + 外延，100-200 字]

## 催化因素
### 政策催化
- [政策 1]：[时间 + 内容]
- [政策 2]：[时间 + 内容]

### 事件催化
- [事件 1]：[时间 + 内容]
- [事件 2]：[时间 + 内容]

## 受益标的
### 核心受益（弹性最大）
| 标的 | 逻辑 | 市值 | 弹性 |
|------|------|------|------|
| | | | |

### 间接受益
| 标的 | 逻辑 | 市值 | 弹性 |
|------|------|------|------|
| | | | |

## 投资节奏
- 当前阶段：[萌芽/发酵/高潮/退潮]
- 介入时机：[建议]
- 退出信号：[信号 1]、[信号 2]

## 风险提示
- [风险 1]
- [风险 2]
- [风险 3]

## 跟踪指标
- [指标 1]
- [指标 2]
```

## 快速模板

```
【[主题名称] 速览】

主题：[一句话定义]

催化：[时间 + 事件]

标的：①[核心标的 1] ②[核心标的 2] ③[核心标的 3]

节奏：[当前阶段 + 建议]

风险：[主要风险]
```
