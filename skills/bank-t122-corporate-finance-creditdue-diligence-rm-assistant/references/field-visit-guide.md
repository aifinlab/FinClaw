# 现场尽调指引

客户经理到现场的核心目标不是“走流程打卡”，而是验证企业经营是否真实、业务闭环是否存在、客户陈述是否与现场一致。

## 一. 到场前准备

- 先列明本次最想核验的 3 至 5 个问题
- 带上合同、发票、流水、财报中的关键疑点清单
- 提前确认要见的人：实控人、总经理、财务负责人、业务负责人、仓储或生产负责人

## 二. 到场后优先看什么

### 1. 场地真实性

- 办公地点、厂房、仓库、门店是否与资料一致
- 招牌、门牌、租赁关系、产权情况是否清楚
- 是否存在明显临时布置或借场地展示的迹象

### 2. 经营状态

- 员工数量、到岗率、现场忙碌程度是否与收入规模相称
- 生产线是否开动，设备是否具备使用痕迹
- 仓库库存、原材料、成品周转是否正常
- 门店客流、货架陈列、系统订单记录是否自然

### 3. 单据匹配

- 合同、发票、出库单、物流单、回款记录能否抽样对应
- 库存品名、批次、数量是否与台账大体一致
- 设备铭牌、车辆、仓单、货权资料是否能核对

### 4. 人员访谈一致性

- 管理层与财务负责人对融资用途是否说法一致
- 业务负责人对主要客户、账期、毛利、回款是否说得上来
- 现场人员描述是否与材料中的经营逻辑一致

## 三. 应重点拍照或记录的内容

- 企业门头、办公区、生产区、仓储区
- 关键设备、库存、产品、出入库记录
- 可公开展示的合同样本、排产信息、发货记录
- 现场异常点，例如停工、空仓、设备闲置、货物老旧

## 四. 高风险现场信号

- 场地空置、人员明显不足、设备长期闲置
- 库存与报表差异很大，无法解释
- 现场主体与申请主体不一致
- 客户拒绝查看关键区域或拒绝配合抽样核对
- 管理层、财务和业务负责人说法多次冲突

## 五. 离场前确认

- 哪些问题已核实，哪些仍需补证据
- 后续需补哪些材料，谁负责、何时提供
- 是否需要安排二次现场核查或电话回访
