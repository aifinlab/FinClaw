# 配对交易参考指南

## 协整理论

### 平稳性与协整
- **I(0)**：平稳序列，均值和方差不随时间变化
- **I(1)**：一阶差分后平稳（大多数股价序列）
- **协整**：两个 I(1) 序列的某个线性组合是 I(0)

### Engle-Granger 两步法
1. 对 P1 和 P2 做 OLS 回归：P1_t = α + β × P2_t + ε_t
2. 对残差 ε_t 做 ADF 检验
3. 若残差平稳（p < 0.05），则 P1 和 P2 协整
4. β 即为对冲比率（hedge ratio）

### Johansen 检验
- 适用于多变量协整检验
- 输出：协整秩（rank）、特征值、迹统计量
- 优点：可同时检验多对关系

## 价差构建方法

### 方法 A：对数价格比
- spread = ln(P1) - ln(P2)
- 优点：无量纲，天然标准化
- 缺点：假设对冲比 = 1

### 方法 B：OLS 残差法
- spread = P1 - β × P2 - α（β 由回归得到）
- 优点：考虑了不同的对冲比率
- 缺点：β 可能随时间变化（需滚动估计）

### 方法 C：Kalman 滤波动态对冲比
- 实时更新 β，适应结构变化
- 计算复杂度较高

## Z-Score 交易信号

```
Z-score = (spread - μ) / σ
μ = rolling_mean(spread, window)
σ = rolling_std(spread, window)
```

| 信号 | 条件 | 操作 |
|------|------|------|
| 开多价差 | Z < -entry_threshold (-2) | 买 Stock1 + 卖 Stock2 |
| 开空价差 | Z > +entry_threshold (+2) | 卖 Stock1 + 买 Stock2 |
| 平仓 | |Z| < exit_threshold (0.5) | 反向平仓 |
| 止损 | |Z| > stop_threshold (3) | 强制平仓 |

## 半衰期（Half-life）

基于 Ornstein-Uhlenbeck 过程：
- dS = θ(μ - S)dt + σdW
- 半衰期 = ln(2) / θ
- 估计方法：对 ΔS_t = a + b × S_{t-1} + ε 做回归，半衰期 = -ln(2) / b
- 半衰期 < 20 天适合短线配对交易
- 半衰期 > 60 天不适合配对交易

## A 股配对交易注意事项

1. **T+1 约束**：买入当日不能卖出
   - 应对：建仓分两天完成，或接受 1 天敞口
2. **涨跌停**：
   - 一方涨停无法买入/卖出，对冲失效
   - 建议避开涨跌停频繁的小盘股
3. **停牌**：
   - 一方停牌期间完全无法对冲
   - 建议选择流动性好的大盘股
4. **融券限制**：
   - 做空需要融券，A 股融券券源有限且成本高（6-8%年化）
   - 替代方案：ETF 对冲、期货对冲
5. **交易成本**：
   - 佣金：万 2.5（双边）
   - 印花税：卖出千 1
   - 融券费：6-8% 年化
   - 滑点：大盘股 1-2bp，小盘股 5-10bp
6. **推荐配对池**：
   - 银行：工商银行 vs 建设银行
   - 白酒：贵州茅台 vs 五粮液
   - 保险：中国平安 vs 中国人寿
   - 券商：中信证券 vs 华泰证券
