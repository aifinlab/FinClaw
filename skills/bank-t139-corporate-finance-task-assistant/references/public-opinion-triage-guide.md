# 舆情事件分诊指南（初筛版）

## 1. 先问三个问题（决定是否升级）

- 是否存在“硬事件”：司法冻结/强制执行/监管处罚/重大事故/兑付违约？
- 是否影响现金流：停工、欠薪、关键客户流失、回款链条断裂是否可被佐证？
- 是否影响融资：抽贷、压降、续作受阻、保证金上升是否出现证据？

## 2. 信号归类（建议至少一类）

- 司法：诉讼、执行、冻结、失信、破产重整
- 监管：处罚、通报、行政约谈、环保/安监问题
- 经营：停工、欠薪、裁员、订单下滑、供应链断裂
- 债务：逾期、展期、兑付风险、票据违约

## 3. 分级维度（建议三维）

- 严重度：轻微/一般/重大/危机（是否触及硬事件与经营中断）
- 可信度：低/中/高（来源是否权威、是否可交叉验证）
- 扩散性：低/中/高（是否持续发酵、多平台扩散、是否波及同业）

输出纪律：

- 在未核验前，不把“传闻”写成“事实”；用“待核验”表达，并列出最短核验路径。
