# 舆情异常升级处置 Playbook（银行对公初筛）

## 1. 立即核验（0-24小时）

- 事件时间线与来源核验（至少两源交叉）
- 是否存在硬事件链接/截图（司法/监管/公告）
- 对存量敞口的影响：到期安排、担保方式、押品、保证金、关键条款

## 2. 业务动作（24-72小时）

- 客户沟通：事实核验、解释口径、资金安排、应急方案
- 资金与回款：关注回款归集、异常大额支付、受限资金变化
- 授信管理：必要时设置临时条件、提高监测频率、启动压降/退出评估

## 3. 升级条件（建议触发任一条即升级）

- 司法冻结/执行/破产重整等硬事件明确
- 经营中断（停工、供应链断裂）得到多源验证
- 同业抽贷/压降信号明确，或出现逾期欠息

## 4. 升级后建议协同

- 风控：事件定性与敞口评估、预案与风险缓释
- 法务：担保效力、追偿路径、诉讼保全建议
- 合规：信息披露与外部沟通纪律（避免不当表述）
