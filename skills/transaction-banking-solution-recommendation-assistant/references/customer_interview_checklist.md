# 客户访谈清单

## 一、基础信息
- 企业所属行业
- 是否集团型客户
- 经营区域是否跨省或跨境
- 主要销售与采购模式

## 二、收款流程
- 主要收款渠道有哪些
- 回款识别是否存在难点
- 对账是否依赖人工
- 是否需要虚拟账户或专属识别能力

## 三、付款流程
- 主要付款类型有哪些
- 是否有多级审批
- 是否涉及集中付款或共享财务中心

## 四、账户与资金管理
- 目前账户数量及开户行分布
- 是否需要资金归集
- 是否存在头寸调拨场景
- 是否需要集团统一可视化

## 五、系统与接口
- 是否部署 ERP、财务系统或司库系统
- 是否需要银企直联或 API
- 当前数据接口能力如何

## 六、跨境与票据
- 是否涉及外币收支
- 是否使用信用证、保函、托收等国际结算工具
- 是否有高频票据使用需求
