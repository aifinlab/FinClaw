# 交易银行产品能力矩阵

## 一、账户管理类
适用问题：
- 集团账户分散，权限管理复杂
- 多法人、多区域账户难以统一管理
- 需要提升账户可视化和集中控制能力

对应能力：
- 账户结构梳理与分层设计
- 母子账户管理
- 分级授权与审批控制
- 虚拟账户与内部核算支持
- 账户可视化查询与余额监控

## 二、收款管理类
适用问题：
- 回款识别难
- 多渠道收款对账效率低
- 应收回款链路不清

对应能力：
- 渠道收款整合
- 回款自动识别
- 账单匹配与自动对账
- 收款通知与应收跟踪

## 三、付款管理类
适用问题：
- 付款流程分散
- 多级审批效率低
- 付款控制弱、内控要求高

对应能力：
- 批量付款
- 付款审批流配置
- 代发代扣
- 付款限额与权限管理

## 四、资金管理类
适用问题：
- 资金沉淀分散
- 成员单位头寸管理弱
- 难以及时掌握集团整体流动性

对应能力：
- 资金归集
- 内部调拨
- 头寸管理
- 流动性监测
- 资金计划协同

## 五、票据与结算类
适用问题：
- 票据使用频繁但管理分散
- 需要提升票据使用效率和集中管理能力

对应能力：
- 票据签发
- 票据贴现
- 票据池
- 票据集中视图

## 六、供应链与场景金融类
适用问题：
- 核心企业链条长
- 上下游结算协同弱
- 需要把金融嵌入真实交易场景

对应能力：
- 链上结算支持
- 应收应付协同
- 交易闭环对接
- 场景化流程嵌入

## 七、跨境与国际结算类
适用问题：
- 存在进出口或境外经营
- 外币收支多
- 需要提升全球资金可视化

对应能力：
- 跨境收付
- 国际结算
- 汇兑管理
- 全球账户信息整合
