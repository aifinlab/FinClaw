## 功能列表

### 1. 指数行情
- **功能描述**: 获取指数的实时行情数据
- **输入参数**: 指数代码
- **输出格式**: Markdown表格
- **数据源**: AkShare-交易所
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/index_quote.py 000300  # 沪深300
  python scripts/index_quote.py 000001  # 上证指数
  python scripts/index_quote.py 399006  # 创业板指
  ```

### 2. 指数成分股
- **功能描述**: 获取指数的成份股列表及权重
- **输入参数**: 指数代码
- **输出格式**: Markdown表格
- **数据源**: AkShare-中证/交易所
- **数据时效**: 季度调整
- **使用示例**:
  ```bash
  python scripts/index_components.py 000300  # 沪深300成分股
  python scripts/index_components.py 000905  # 中证500成分股
  ```

### 3. 指数估值
- **功能描述**: 获取主要指数的估值数据（PE/PB/股息率）
- **输入参数**: 无（全市场主要指数）
- **输出格式**: Markdown表格
- **数据源**: AkShare-中证
- **数据时效**: 日频
- **使用示例**:
  ```bash
  python scripts/index_valuation.py
  ```
