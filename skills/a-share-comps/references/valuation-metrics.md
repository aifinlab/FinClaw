# A 股估值指标定义 & 行业特殊处理

## 核心估值指标

### PE (Price-to-Earnings, 市盈率)
- **计算**: 总市值 / 归母净利润 (TTM)
- **适用**: 盈利稳定的公司
- **不适用**: 亏损公司、周期底部公司
- **参考区间**: A 股全市场中位数约 25-35x; 沪深 300 约 12-18x

### PB (Price-to-Book, 市净率)
- **计算**: 总市值 / 归母净资产
- **适用**: 重资产行业（银行、地产、制造业）
- **参考区间**: 银行 0.4-1.0x; 制造业 1.0-3.0x

### PS (Price-to-Sales, 市销率)
- **计算**: 总市值 / 营业收入 (TTM)
- **适用**: 亏损但有收入的成长型公司（SaaS、生物医药早期）
- **参考区间**: 行业差异极大

### EV/EBITDA
- **计算**: (市值 + 有息负债 - 现金) / EBITDA
- **适用**: 跨国对比（消除税制差异）、高杠杆行业
- **A 股特殊**: EBITDA 需从利润表+现金流量表估算 = 营业利润 + 折旧摊销
- **参考区间**: 全行业 8-15x

### PEG (PE-to-Growth)
- **计算**: PE / 净利润增速(%)
- **适用**: 成长股估值合理性判断
- **参考**: < 1 低估, 1-2 合理, > 2 偏贵

### 股息率
- **计算**: 每股分红 / 股价
- **适用**: 高分红蓝筹股对比
- **A 股特殊**: 关注分红频率（年度 vs 半年度）和分红稳定性

## 行业特殊处理

### 银行业
| 主用指标 | 辅用指标 | 不用 |
|---------|---------|------|
| PB | 股息率 | PE（受拨备影响大） |
| 不良率 | 拨备覆盖率 | PS |
| 净息差(NIM) | 资本充足率 | EV/EBITDA |

- PB 是银行估值的核心，反映资产质量预期
- 不良率 + 拨备覆盖率 组合判断资产质量真实性
- A 股银行 PB 区间：0.4x（悲观）-1.2x（乐观）

### 保险业
| 主用指标 | 辅用指标 |
|---------|---------|
| PEV (内含价值倍数) | PB |
| 新业务价值增速 | 投资收益率 |

- PEV = 市值 / 内含价值（EV），A 股保险通常 0.5-1.5x
- 内含价值数据通常需从年报/半年报搜索获取

### 房地产
| 主用指标 | 辅用指标 |
|---------|---------|
| 净负债率 | PE |
| 合同负债/收入 | 土储面积 |
| NAV折价 | PB |

- 净负债率 = (有息负债 - 现金) / 净资产
- 合同负债是已售未结转的领先指标
- 行业下行期关注现金短债比

### 科技/软件
| 主用指标 | 辅用指标 |
|---------|---------|
| PS | PE（若盈利） |
| EV/Revenue | PEG |
| 研发/收入占比 | 收入增速 |

- 高增长亏损公司用 PS/EV/Revenue
- SaaS 类关注 ARR 增速（通常需搜索获取）

### 医药/生物科技
| 主用指标 | 辅用指标 |
|---------|---------|
| PE（创新药盈利后） | PS（创新药早期） |
| 管线估值（DCF） | PB（CXO/仿制药） |

- 创新药企用管线 DCF 或同类交易对标
- CXO 用 PE + 订单增速

### 消费/白酒
| 主用指标 | 辅用指标 |
|---------|---------|
| PE | 毛利率 |
| PEG | 预收款变化 |
| 品牌溢价（定性） | 经销商库存（定性） |

- 白酒 PE 通常 20-40x（龙头溢价）
- 预收款环比变化是景气度领先指标

### 新能源/制造业
| 主用指标 | 辅用指标 |
|---------|---------|
| PE（基于未来2年利润） | PB |
| 产能利用率 | 订单/收入比 |
| 技术路线占有率 | 毛利率趋势 |

- 周期性强，不宜用历史 PE 做简单对比
- 关注产能扩张节奏 vs 行业需求

## 统计方法

### 必算统计量
- **均值**: 受极端值影响大，仅作参考
- **中位数**: 最稳健的集中趋势度量，作为核心参考
- **标准差**: 衡量估值分散程度
- **百分位排名**: 标的在可比组中的位置

### 异常值处理
- PE 为负数（亏损）→ 标记 "N/A" 不纳入统计
- PE > 100x → 标记但纳入统计，可选剔除
- 其他指标若偏离中位数 > 3 倍标准差 → 标注异常值
