# 输入字段建议

## 最低必需材料

- 组合持仓明细
- 收益和回撤数据
- 目标配置约束
- 客户或账户风险偏好

## 可选增强材料

- 基准与同类对照
- 调仓记录
- 客户关注点

## 建议输入结构

~~~json
{
  "task_name": "投顾组合诊断助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
