# 核心竞争力分析输出模板

## 模板 A：快速分析（300-500 字）

```markdown
# 【XX 公司核心竞争力分析】

## 护城河类型
- 主要护城河：[品牌/成本/网络/切换/牌照]
- 护城河强度：[强/中/弱]

## 核心优势
1. [优势 1]
2. [优势 2]

## 可持续性判断
[短期/中期/长期判断]

## 投资风险
- [风险 1]
- [风险 2]
```

## 模板 C：深度分析（1500 字+）

在标准分析基础上增加：
- 护城河来源详细分析
- 历史验证（过去如何抵御竞争）
- 压力测试（极端情况下的护城河）
- 附录（数据表格、图表）
