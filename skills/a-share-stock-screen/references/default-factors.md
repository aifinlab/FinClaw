## 预设策略模板

### 策略 1: 经典价值（Warren Buffett 风格）
```
ROE > 15%
资产负债率 < 50%
毛利率 > 30%
净利润同比 > 10%
PE < 行业中位数
经营现金流/净利润 > 0.8
```
适合：寻找长期持有的优质低估值公司

### 策略 2: 高成长（Peter Lynch 风格）
```
营收同比 > 30%
净利润同比 > 30%
PEG < 1 (PE / 净利润增速)
ROE > 12%
```
适合：寻找高速增长但估值合理的公司

### 策略 3: 高股息防御
```
股息率 > 3%
PE < 15
资产负债率 < 50%
连续 3 年分红
ROE > 10%
```
适合：熊市防御、追求稳定现金回报

### 策略 4: 北向资金跟踪
```
近 20 日北向资金净买入额排名前 50
PE < 30
ROE > 10%
市值 > 100 亿
```
适合：跟踪外资动向、寻找机构认可标的

### 策略 5: 困境反转
```
近 1 季度净利润 > 0（已转正）
去年同期净利润 < 0（去年亏损）
营收同比 > 10%
资产负债率 < 70%
```
适合：寻找业绩拐点、困境反转机会

## 综合评分方法

当用户未指定策略时，使用等权多因子评分：
1. 对每个因子在全样本中算百分位排名（0-100）
2. 各因子等权加总
3. 按总分降序排列

得分 = (ROE 排名 + 净利润增速排名 + (100 - PE 排名) + (100 - 负债率排名)) / 4
