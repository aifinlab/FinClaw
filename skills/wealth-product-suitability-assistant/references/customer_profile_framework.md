# 客户画像分析框架

## 一、客户画像核心维度
1. 生命周期：年轻客群、中年家庭客群、临退休与退休客群
2. 收入稳定性：稳定、波动、季节性、经营性收入为主
3. 家庭责任：教育、住房、赡养、养老等负担
4. 流动性需求：高、中、低
5. 投资经验：无经验、基础、较丰富
6. 风险偏好：保守、稳健、平衡、进取
7. 配置现状：存款偏高、理财偏高、权益偏高、保障不足等

## 二、常见客户目标
- 现金管理
- 稳健增值
- 子女教育储备
- 养老储备
- 大额支出前的过渡配置
- 家庭资产平衡化配置

## 三、错配识别
- 风险错配
- 期限错配
- 流动性错配
- 配置过度集中
- 收益预期明显不切实际
