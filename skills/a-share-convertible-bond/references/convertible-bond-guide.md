# A股可转债分析参考指南

## 1. 核心估值指标

### 转股价值
- **公式**：转股价值 = 100 / 转股价 × 正股价
- **含义**：假设立即转股，100元面值转债可换取的股票市值
- **用途**：衡量转债的"股性"强弱

### 转股溢价率
- **公式**：转股溢价率 = (转债价格 - 转股价值) / 转股价值 × 100%
- **含义**：转债价格相对转股价值的溢价程度
- **判断标准**：
  - < 0%：折价，极罕见（套利空间）
  - 0-10%：低溢价，股性强，跟涨能力好
  - 10-30%：中等溢价，攻守兼备
  - > 30%：高溢价，债性为主，正股上涨传导弱

### 纯债价值
- **计算方法**：按同评级企业债收益率折现转债剩余存续期内的全部现金流（票息+到期赎回价）
- **简化处理**：当缺少精确利率曲线数据时，可按同评级、相近期限的企业债收益率近似折现
- **含义**：转债的"债底"，即不考虑转股权时的纯债务价值
- **典型范围**：AAA 评级纯债价值通常在 85-95 元，AA 评级在 80-90 元

### 纯债溢价率
- **公式**：纯债溢价率 = (转债价格 - 纯债价值) / 纯债价值 × 100%
- **含义**：市场为转股期权支付的溢价

### 到期收益率 YTM
- **计算方法**：以当前转债价格买入、持有到期按赎回价兑付，计算内部收益率
- **判断标准**：
  - > 3%：纯债属性极强，适合防御
  - 1-3%：有一定债底保护
  - 0-1%：债底保护弱
  - < 0%：价格已超过债底，纯靠转股期权支撑

### 双低值
- **公式**：双低值 = 转债价格 + 转股溢价率 × 100
- **判断标准**：
  - < 130：低估区间，攻守兼备
  - 130-150：合理区间
  - > 150：偏高估，需精选
- **优势**：综合考虑绝对价格和相对溢价，避免单一指标的偏差

## 2. 关键条款解读

### 转股条款
- **转股期**：发行之日起满6个月后的第一个交易日起可转股
- **转股价**：初始转股价在发行时确定，后续可能因送股/配股/增发等调整
- **转股价下修**：董事会/股东大会可主动下修转股价（需满足特定条件）

### 强赎条款（有条件赎回）
- **典型条件**：正股连续30个交易日中至少15日收盘价 ≥ 转股价的130%
- **触发后果**：发行人有权按面值+当期利息赎回，实质上迫使投资者转股或卖出
- **注意事项**：
  - 各转债具体条款可能不同（如20/30天，或15/30天）
  - 发行人可选择不行使强赎权（"不强赎"公告）
  - 强赎公告后通常有最后交易日和最后转股日

### 回售条款（有条件回售）
- **典型条件**：正股连续30个交易日收盘价低于当期转股价的70%
- **触发后果**：投资者有权按面值+利息将转债回售给发行人
- **含义**：保护投资者的安全网，也是发行人下修转股价的动力来源

### 下修条款（转股价向下修正）
- **触发条件**：正股价低于转股价一定比例（如85%或90%），持续一定天数
- **修正流程**：董事会提议 → 股东大会表决（2/3以上同意）
- **修正底线**：修正后的转股价不得低于股东大会召开日前20个交易日均价和前一日均价的较高者
- **投资意义**：下修提升转股价值，对转债持有者有利

## 3. 常用策略

### 双低策略
- **筛选条件**：双低值 < 130
- **逻辑**：低价格提供债底保护，低溢价率保证正股上涨时的跟涨弹性
- **适用场景**：震荡市、对方向判断不确定时
- **注意**：需排除信用风险高的标的（低评级/正股ST）

### 低溢价率策略
- **筛选条件**：转股溢价率 < 5%
- **逻辑**：转债价格接近转股价值，正股涨多少转债大致跟涨多少
- **适用场景**：看好正股上涨，用转债替代正股（T+0 + 有债底）
- **风险**：正股下跌时转债同步下跌，债底保护弱（因价格通常较高）

### 高YTM策略
- **筛选条件**：YTM > 2%
- **逻辑**：即使不转股，持有到期也有正收益
- **适用场景**：防御型配置，替代纯债获取更高收益+免费看涨期权
- **风险**：需确认信用资质，排除违约风险标的

### 强赎博弈
- **筛选条件**：正股接近强赎触发线（正股价/转股价 > 120%，且已有部分天数达标）
- **逻辑**：强赎触发后转债价格锁定在约130元，低于130元买入可获利
- **风险**：发行人可能选择不强赎；正股回落导致强赎条件未达成

### 下修博弈
- **筛选条件**：正股下跌接近回售线，发行人有下修动力
- **逻辑**：下修转股价 → 转股价值提升 → 转债价格上涨
- **判断要点**：
  - 发行人是否有下修意愿（到期压力、资金状况）
  - 大股东持股比例（稀释影响）
  - 历史下修记录

## 4. 风险等级分类

### 信用评级与风险
| 评级 | 风险等级 | 投资建议 |
|------|---------|---------|
| AAA | 极低 | 可放心配置，关注估值即可 |
| AA+ | 很低 | 主流配置品种 |
| AA | 低-中 | 需关注正股基本面变化 |
| AA- | 中 | 需重点评估违约风险，关注财务数据 |
| A+ 及以下 | 高 | 谨慎参与，需深入研究信用资质 |

### 风险预警信号
- 正股被 ST / *ST
- 正股连续大幅下跌
- 发行人财务数据恶化（营收下滑、亏损扩大、现金流紧张）
- 审计意见为"保留意见"或"无法表示意见"
- 转债成交量极低（日成交额 < 100万元）

## 5. A股可转债市场特色

### 市场概况
- 全市场约 500+ 只存量可转债
- 发行面值统一为 100 元
- 存续期一般 6 年（少数 5 年）

### 品种特征
- **大盘转债**（银行/券商/公用事业）：溢价率通常较低，流动性好，适合大资金
- **中盘转债**：攻守兼备，策略适用性最广
- **小盘转债**：投机性强，溢价率可能很高，波动大，流动性可能不足

### 交易规则
- **T+0 交易**：当日买入当日可卖出（与正股 T+1 不同）
- **无涨跌幅限制**：但有临时停牌机制
  - 涨跌超过 20%：停牌30分钟
  - 涨跌超过 30%：停牌至14:57
- **最小交易单位**：10张（1手）

### 税费
- 佣金：通常万分之几（各券商不同）
- 无印花税
- 利息税：持有期间利息收入需缴纳20%个人所得税

## 6. 数据局限性

### 纯债价值计算
- 精确计算需要国债/企业债利率曲线数据
- 简化方案：参考同评级、相近期限的企业债到期收益率作为折现率
- 不同机构计算的纯债价值可能存在差异（折现率选取不同）

### 实时数据
- efinance 提供的转债行情为盘中实时/收盘数据
- 转股价、条款信息可能存在更新延迟，需通过 web 搜索确认最新值
- 强赎计数天数需手动跟踪或通过公告确认

### 其他注意
- 部分转债可能已公告强赎但尚未摘牌，行情数据仍在更新
- 新发行转债可能暂无完整的转股溢价率等数据（尚未进入转股期）
- 退市转债的历史数据可能不完整
