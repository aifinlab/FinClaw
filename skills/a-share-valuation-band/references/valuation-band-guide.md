# 估值分位/历史估值区间分析参考指南

## 1. 分位数计算方法详解

### 百分位排名公式

```
百分位(P) = (低于当前值的观测数 / 有效观测总数) x 100%
```

**示例**：某股票当前 PE(TTM) = 25，近 3 年共 730 个有效交易日，其中 PE < 25 的交易日有 511 天：
```
百分位 = 511 / 730 x 100% = 70.0%
```
表示当前 PE 高于历史 70% 的时间，处于偏高估区间。

### 计算步骤

1. 收集历史 N 年每日收盘价对应的估值指标序列
2. 剔除无效数据（亏损导致 PE 为负、停牌无数据等）
3. 对有效序列排序
4. 统计低于当前值的观测数量
5. 计算百分位
6. 同时输出五数概括：min、Q1(25%)、median(50%)、Q3(75%)、max

### 关键统计量

| 统计量 | 含义 | 用途 |
|--------|------|------|
| 均值(mean) | 历史平均估值 | 判断中枢水平 |
| 中位数(median) | 历史中间估值 | 比均值更抗极端值 |
| 标准差(std) | 估值波动幅度 | 判断当前偏离程度 |
| 当前值/中位数 | 相对中枢比率 | >1 偏贵，<1 偏便宜 |

## 2. PE/PB/PS 各自适用场景

### PE(TTM) — 市盈率

- **适用**：盈利稳定的公司（消费、公用事业、白酒、医药龙头）
- **不适用**：
  - 亏损企业（PE 无意义）
  - 强周期行业（周期底部利润极低，PE 反而极高，形成"低PE陷阱"）
  - 高增长但未盈利的科技公司
- **注意**：一次性损益（资产处置/减值）会严重扭曲 TTM 利润

### PB — 市净率

- **适用**：重资产行业（银行、保险、钢铁、房地产）、周期股
- **不适用**：
  - 轻资产公司（互联网、软件，净资产不反映真实价值）
  - 商誉占比过高的公司（净资产含大量商誉，减值风险高）
- **优势**：不受短期利润波动影响，周期股估值首选

### PS — 市销率

- **适用**：高增长但未盈利或利润波动大的公司（SaaS、创新药、新能源早期）
- **不适用**：
  - 低毛利率行业（贸易、零售，收入大但利润薄，PS 失真）
  - 成熟稳定企业（应直接用 PE）
- **注意**：PS 不考虑成本结构差异，同 PS 的公司盈利能力可能天差地别

### EV/EBITDA

- **适用**：资本密集型行业、跨国比较（消除资本结构/税率差异）
- **不适用**：金融行业（EBITDA 对银行无意义）

### 行业 → 首选指标速查

| 行业 | 首选指标 | 辅助指标 |
|------|---------|---------|
| 银行/保险 | PB | ROE、不良率 |
| 白酒/消费 | PE(TTM) | PEG |
| 科技/SaaS | PS | 收入增速 |
| 钢铁/煤炭/化工 | PB | 产能利用率 |
| 医药（成熟） | PE(TTM) | PEG |
| 创新药（未盈利） | PS | 管线价值 |
| 房地产 | PB | NAV折价率 |
| 公用事业 | PE | 股息率 |

## 4. 行业估值分位特殊处理

### 银行业：以 PB 为核心

- 银行利润受拨备调节影响大，PE 可操控性强
- PB 结合 ROE 判断更可靠：PB < 0.6 且 ROE > 10% 通常为低估
- 分位计算时建议用 PB 而非 PE

### 科技/SaaS：以 PS 为核心

- 多数处于投入期，利润为负或波动极大
- PS 分位 + 收入增速综合判断
- 警惕：收入增速放缓时 PS 中枢会系统性下移

### 周期股：PB + 产能利用率

- **绝对不能**单纯看 PE 分位！
  - 周期底部：利润极低 → PE 极高 → PE分位显示"高估" → 实际是买入时机
  - 周期顶部：利润极高 → PE 极低 → PE分位显示"低估" → 实际是卖出时机
- 正确做法：
  1. 用 PB 分位判断资产端低估程度
  2. 结合行业产能利用率/库存周期判断周期位置
  3. PB 处于历史低位 + 产能利用率触底回升 = 最佳买入信号

### 高增长公司：PEG 辅助

- PE 分位可能持续偏高（因业绩高增长消化估值）
- 用 PEG = PE / 未来增速 辅助判断
- PEG < 1 即使 PE 分位较高也可能合理

## 5. 常见陷阱

### 陷阱一：利润暴涨导致 PE 骤降

- **现象**：某公司出售资产获得巨额一次性收益，TTM利润翻倍，PE 骤降至历史低位
- **误判**：PE 分位显示 5%，极度低估
- **真相**：一次性收益不可持续，扣非后 PE 仍然很高
- **应对**：同时看扣非 PE，或对一次性损益做调整

### 陷阱二：会计政策变更

- **现象**：新租赁准则(IFRS 16)实施后，航空/零售公司资产负债表膨胀
- **影响**：PB 系统性下降，与历史不可比
- **应对**：调整后对比，或缩短分位计算窗口至准则变更后

### 陷阱三：股本变化

- **现象**：大比例增发/回购后，每股收益变化但总利润不变
- **影响**：PE 可能失真
- **应对**：关注总市值/总利润 而非每股指标

### 陷阱四：行业估值中枢迁移

- **现象**：某行业因政策变化（如教培双减、互联网反垄断），估值中枢永久性下移
- **误判**：用 10 年分位显示"低估"，但新常态下中枢已变
- **应对**：缩短时间窗口（如仅用近 2-3 年），或对政策变化前后分段计算

### 陷阱五：指数成分股变化

- **现象**：指数定期调仓，纳入高估值新经济股、剔除低估值传统股
- **影响**：指数 PE 中枢被动抬升，与历史不完全可比
- **应对**：了解成分股调整历史，重大调仓时段做标注

## 6. 与其他估值方法的配合使用

### 估值分位 + DCF（a-share-dcf）

- 分位法告诉你"相对历史贵不贵"
- DCF 告诉你"绝对值值多少"
- 最佳组合：分位低估 + DCF 显示低于内在价值 = 双重确认

### 估值分位 + 可比公司（a-share-comps）

- 分位法是"纵向比较"（自己和自己历史比）
- Comps 是"横向比较"（和同行比）
- 配合使用避免个股特殊因素干扰

### 估值分位 + 技术分析（a-share-technical）

- 基本面低估 + 技术面底部形态 = 更好的买入时机
- 避免"估值低但趋势向下"的价值陷阱

### 估值分位 + 北向资金（a-share-northbound）

- 低估值 + 北向持续流入 = 聪明钱认可的低估机会
- 低估值 + 北向持续流出 = 可能有基本面隐忧

### 建议分析流程

1. **先看分位**：快速判断估值高低（本 skill）
2. **再做横比**：与同行对比确认（a-share-comps）
3. **深入估值**：DCF 计算内在价值（a-share-dcf）
4. **综合研判**：结合资金面/技术面确定操作策略
