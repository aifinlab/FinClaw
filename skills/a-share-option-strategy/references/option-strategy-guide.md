- ETF期权：当月、下月、随后两个季月（3/6/9/12月）

- 行权价间距：标的价格不同，间距为0.05/0.1/0.25元

### 基础策略（四大单腿策略）

#### 1. 买入认购（Long Call）
- **适用场景**：看涨标的
- **损益**：最大亏损 = 权利金；最大盈利 = 无限
- **盈亏平衡** = 行权价 + 权利金
- **Greeks**：Delta+, Gamma+, Theta-, Vega+
- **要点**：选择合适行权价和到期日，避免买入深度虚值短期合约

#### 2. 买入认沽（Long Put）
- **适用场景**：看跌标的
- **损益**：最大亏损 = 权利金；最大盈利 = 行权价 - 权利金
- **盈亏平衡** = 行权价 - 权利金
- **Greeks**：Delta-, Gamma+, Theta-, Vega+
- **要点**：常用于持仓保护（保险策略）

#### 3. 卖出认购（Short Call）
- **适用场景**：看不涨/温和看跌
- **损益**：最大盈利 = 权利金；最大亏损 = 无限
- **盈亏平衡** = 行权价 + 权利金
- **Greeks**：Delta-, Gamma-, Theta+, Vega-
- **风险**：裸卖风险无限，需缴纳保证金，有强平风险

#### 4. 卖出认沽（Short Put）
- **适用场景**：看不跌/温和看涨
- **损益**：最大盈利 = 权利金；最大亏损 = 行权价 - 权利金
- **盈亏平衡** = 行权价 - 权利金
- **Greeks**：Delta+, Gamma-, Theta+, Vega-
- **风险**：需缴纳保证金，极端下跌亏损巨大

### 持仓增强策略

#### 备兑策略（Covered Call）
- **构建**：持有标的ETF + 卖出认购
- **适用场景**：温和看涨/震荡市，增强持仓收益
- **损益**：最大盈利 = 行权价 - 持仓成本 + 权利金；下行风险 = 持仓亏损 - 权利金缓冲
- **要点**：选择虚值1-2档认购，到期月选近月（Theta衰减快）

#### 保险策略（Protective Put）
- **构建**：持有标的ETF + 买入认沽
- **适用场景**：持仓保护，担心下跌但不想卖出
- **损益**：最大亏损 = 持仓成本 - 行权价 + 权利金（有限）；上行盈利保留
- **要点**：相当于给持仓买保险，权利金是保险费

### 组合策略

#### 牛市认购价差（Bull Call Spread）
- **构建**：买入低行权价认购 + 卖出高行权价认购（同到期日）
- **适用场景**：温和看涨，降低买购成本
- **最大盈利** = 高行权价 - 低行权价 - 净权利金
- **最大亏损** = 净权利金
- **Greeks**：Delta+（有限），Theta 视行权价位置

#### 熊市认沽价差（Bear Put Spread）
- **构建**：买入高行权价认沽 + 卖出低行权价认沽（同到期日）
- **适用场景**：温和看跌，降低买沽成本
- **最大盈利** = 高行权价 - 低行权价 - 净权利金
- **最大亏损** = 净权利金

#### 跨式策略（Straddle）
- **买入跨式**：买入同行权价认购 + 买入同行权价认沽（同到期日）
  - 适用：预期大幅波动（方向不确定），低IV环境
  - 最大亏损 = 两份权利金之和
  - 盈亏平衡 = 行权价 +/- 总权利金
- **卖出跨式**：卖出同行权价认购 + 卖出同行权价认沽
  - 适用：预期窄幅震荡，高IV环境
  - 最大盈利 = 两份权利金之和
  - 风险：双向无限

#### 宽跨式策略（Strangle）
- **买入宽跨式**：买入虚值认购 + 买入虚值认沽（不同行权价，同到期日）
  - 适用：预期大幅波动，成本低于跨式
  - 需要更大波动才能盈利
- **卖出宽跨式**：卖出虚值认购 + 卖出虚值认沽
  - 适用：预期震荡，盈利区间比跨式更宽
  - 风险：双向无限

#### 蝶式策略（Butterfly Spread）
- **构建**：买1份低行权价认购 + 卖2份中间行权价认购 + 买1份高行权价认购
- **适用场景**：预期标的在某个窄幅区间
- **最大盈利** = 中间行权价 - 低行权价 - 净权利金（在中间行权价时达到）
- **最大亏损** = 净权利金
- **特点**：成本低、风险有限、收益有限

#### 铁鹰策略（Iron Condor）
- **构建**：卖出虚值认沽价差 + 卖出虚值认购价差
  - 即：买入深虚认沽 + 卖出虚值认沽 + 卖出虚值认购 + 买入深虚认购
- **适用场景**：预期震荡，高IV环境最佳
- **最大盈利** = 净收入权利金
- **最大亏损** = 价差宽度 - 净权利金
- **特点**：风险有限（区别于裸卖跨式/宽跨式）

### Greeks 解读

| Greek | 含义 | 买方 | 卖方 | 实际意义 |
|-------|------|------|------|---------|
| **Delta** | 标的变动1元，期权价格变动 | 认购+(0~1) 认沽-(-1~0) | 反向 | 方向风险，类似"等效持仓" |
| **Gamma** | 标的变动1元，Delta变动 | 正值(买方友好) | 负值(卖方不利) | Delta变化速度，平值期权Gamma最大 |
| **Theta** | 每过1天，期权价格变动 | 负值(时间损耗) | 正值(时间收益) | 时间衰减，临近到期加速，卖方赚Theta |
| **Vega** | IV变动1%，期权价格变动 | 正值(IV涨有利) | 负值(IV涨不利) | 波动率风险，远月期权Vega更大 |

**Delta 经验值**：
- 深度实值：Delta 接近 +1（认购）或 -1（认沽）
- 平值（ATM）：Delta 约 +0.5（认购）或 -0.5（认沽）
- 深度虚值：Delta 接近 0

**Gamma 特性**：
- 平值期权 Gamma 最大
- 临近到期，平值 Gamma 急剧放大（Gamma Risk）
- 卖方临近到期需警惕平值合约的 Gamma 风险

**Theta 特性**：
- 时间衰减非线性，最后30天加速衰减
- 平值期权 Theta 绝对值最大
- 卖方主要盈利来源

### 隐含波动率（IV）判断框架

#### IV 百分位
```
IV百分位 = 过去N个交易日中 IV 低于当前值的天数占比
常用 N = 252（一年）
```

| IV 百分位 | 环境 | 策略倾向 |
|-----------|------|---------|
| > 80% | 高IV | 优先卖权策略（收取高权利金）：卖出跨式/宽跨式/铁鹰/备兑 |
| 50%-80% | 中高IV | 价差策略（降低Vega暴露）：牛市/熊市价差 |
| 20%-50% | 中低IV | 灵活选择，价差/单腿买权均可 |
| < 20% | 低IV | 优先买权策略（付出低成本）：买入跨式/宽跨式/单腿买权 |

#### IV vs HV 对比
- **IV > HV**：期权定价偏贵（隐含波动率溢价），卖方有统计优势
- **IV < HV**：期权定价偏便宜，买方有统计优势
- **IV/HV 比值 > 1.2**：显著偏贵
- **IV/HV 比值 < 0.8**：显著偏便宜

### 风险提示

1. **卖方风险无限**：裸卖认购理论亏损无限，裸卖认沽最大亏损为标的跌至零。务必使用价差策略限制风险或严格止损
2. **保证金与强平**：卖出期权需缴纳保证金，标的大幅波动时保证金不足将被强制平仓
3. **时间价值归零**：买方持有到期，虚值期权价值归零，实际盈利需覆盖权利金成本
4. **流动性风险**：深度虚值/深度实值合约流动性差，买卖价差大，可能无法以合理价格平仓
5. **参与门槛**：期权开户要求：
   - 申请开户前20个交易日日均资产不低于50万元
   - 通过交易所期权知识测试（不低于80分）
   - 具备融资融券或期货交易经验
   - 分级管理：一级（备兑/保险）→ 二级（买入开仓）→ 三级（卖出开仓）
6. **A股期权特色**：
   - 欧式行权（到期日行权），不可提前行权
   - T+0交易（期权当日买入可当日卖出，但标的ETF仍为T+1）
   - 涨跌幅限制：期权无涨跌停，但标的ETF有涨跌停
   - 合约调整：标的分红/送股时，行权价和合约单位会相应调整
