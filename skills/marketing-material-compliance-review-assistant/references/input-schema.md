# 输入字段建议

## 最低必需材料

- 原始材料文本
- 适用渠道与客群
- 制度或合规要求
- 版本信息

## 可选增强材料

- 历史整改意见
- 相关产品资料
- 敏感词库

## 建议输入结构

~~~json
{
  "task_name": "宣传材料合规审查助手",
  "analysis_scope": "",
  "facts": [],
  "metrics": {},
  "risks": [],
  "questions": [],
  "notes": [],
  "attachments": []
}
~~~

## 缺失信息处理

- 若最低必需材料不完整，先输出「缺失信息清单」
- 对推断性内容标记「初步判断」或「待核验」
- 不得用猜测填补关键事实缺口
