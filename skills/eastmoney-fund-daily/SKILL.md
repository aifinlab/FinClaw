# eastmoney-fund-daily

天天基金网基金详情数据 JS 封装。

## 核心能力

- 基金实时估值查询
- 历史净值数据
- 持仓结构（股票/债券/行业）
- 基金经理信息

## 数据源

- 天天基金网 fund.eastmoney.com/pingzhongdata API

## 使用

```bash
node "$SKILLS_ROOT/eastmoney-fund-daily/index.js" --fund 110011
```
