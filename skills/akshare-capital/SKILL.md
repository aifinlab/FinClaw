## 功能描述

资金流向Skill，提供个股资金流向、板块资金流向、主力资金排行、北向资金（陆股通）明细等数据。追踪超大单/大单/中单/小单资金动向，识别主力建仓/出货信号。

## 资金流向分类

| 类型 | 定义 | 代表意义 |
|:---|:---|:---|
| **超大单** | >100万股或>100万元 | 机构主力、大资金建仓/出货 |
| **大单** | 20-100万股或20-100万元 | 大资金、基金调仓 |
| **中单** | 4-20万股或4-20万元 | 中等资金、私募/大户 |
| **小单** | <4万股或<4万元 | 散户资金 |

## 功能列表

### 1. 个股资金流向
- **功能描述**: 获取特定股票的资金流向明细
- **输入参数**: 股票代码
- **输出格式**: Markdown表格（超大单/大单/中单/小单净流入）
- **数据源**: AkShare-东方财富
- **数据时效**: 实时（交易时间内）
- **使用示例**:
  ```bash
  python scripts/capital_flow.py 600519
  python scripts/capital_flow.py 300750
  ```

### 2. 板块资金流向
- **功能描述**: 获取各行业板块的资金流向排行
- **输入参数**: 无（全板块）或板块名称
- **输出格式**: Markdown表格（净流入排行）
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/capital_board.py
  python scripts/capital_board.py --limit 20
  ```

### 3. 北向资金明细
- **功能描述**: 获取北向资金（陆股通）当日买卖明细
- **输入参数**: 无
- **输出格式**: Markdown表格（沪股通/深股通净流入、个股买卖）
- **数据源**: AkShare-交易所
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/capital_north.py
  ```

### 4. 主力资金排行
- **功能描述**: 获取主力资金净流入/流出排行
- **输入参数**: 市场（sh/sz/all）、排序方式
- **输出格式**: Markdown表格
- **数据源**: AkShare-东方财富
- **数据时效**: 实时
- **使用示例**:
  ```bash
  python scripts/capital_main.py
  python scripts/capital_main.py --market sh --limit 30
  ```

## 资金流向解读

### 净流入/流出
| 情形 | 超大单 | 大单 | 中单 | 小单 | 解读 |
|:---|:---:|:---:|:---:|:---:|:---|
| **主力建仓** | +++ | ++ | - | --- | 机构买入，散户卖出 |
| **主力出货** | --- | -- | + | +++ | 机构卖出，散户接盘 |
| **一致看多** | +++ | ++ | + | ++ | 各资金集体流入 |
| **一致看空** | --- | -- | - | -- | 各资金集体流出 |
| **震荡洗盘** | +- | -+ | +- | +- | 资金分歧，震荡为主 |

### 北向资金信号
| 北向资金 | 信号 | 意义 |
|:---|:---:|:---|
| 连续3日净流入 >50亿 | 🟢 强烈看多 | 外资大幅加仓 |
| 连续3日净流出 >50亿 | 🔴 强烈看空 | 外资大幅减仓 |
| 单日净流入 >100亿 | 🟢 积极信号 | 外资抢筹 |
| 单日净流出 >100亿 | 🔴 警示信号 | 外资逃离 |
| 小幅波动 ±20亿 | ⚪ 中性 | 正常波动 |
