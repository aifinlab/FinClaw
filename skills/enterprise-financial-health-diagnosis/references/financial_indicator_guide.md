# 财务指标口径说明

## 一、偿债能力指标
### 1. 资产负债率
计算方式：总负债 / 总资产  
解读要点：用于衡量企业整体杠杆水平。数值越高，通常说明债务负担越重，但需结合行业特征判断。

### 2. 流动比率
计算方式：流动资产 / 流动负债  
解读要点：反映短期偿债能力。若持续下降，可能意味着短期资金压力上升。

### 3. 速动比率
计算方式：（流动资产 - 存货 - 预付款项） / 流动负债  
解读要点：排除存货等变现速度较慢项目后，评估短期偿债缓冲。

### 4. 利息保障倍数
计算方式：息税前利润 / 利息支出  
解读要点：衡量利润覆盖利息支出的能力。若持续偏低，债务承受能力较弱。

## 二、盈利能力指标
### 1. 毛利率
计算方式：（营业收入 - 营业成本） / 营业收入  
解读要点：反映主营业务盈利空间，应关注波动趋势及与行业水平的差异。

### 2. 净利率
计算方式：净利润 / 营业收入  
解读要点：反映企业最终盈利效率，但需注意是否受非经常性损益影响。

### 3. 总资产报酬水平
计算方式：净利润 / 平均总资产  
解读要点：反映资产使用效率及盈利能力。

## 三、营运能力指标
### 1. 应收账款周转率
计算方式：营业收入 / 平均应收账款  
解读要点：衡量回款效率。周转下降往往意味着信用销售压力加大或回款变慢。

### 2. 存货周转率
计算方式：营业成本 / 平均存货  
解读要点：衡量库存消化能力。周转下降可能意味着库存积压或销售承压。

### 3. 总资产周转率
计算方式：营业收入 / 平均总资产  
解读要点：衡量资产整体运营效率。

## 四、现金流指标
### 1. 经营活动现金流净额
解读要点：用于观察主营业务带来的真实现金创造能力。

### 2. 经营现金流净额 / 净利润
解读要点：判断利润含金量。若长期显著低于 1，需要关注收入确认、回款质量和营运资本占用。

### 3. 自由现金流
常用口径：经营活动现金流净额 - 资本性支出  
解读要点：反映企业在维持经营与投资后可支配的现金能力。

## 五、分析注意事项
- 所有指标必须结合趋势、行业、业务模式综合分析
- 不同行业的合理区间差异较大，不能机械套用单一阈值
- 对集团型企业，应区分合并口径与单体口径
- 对存在重大并购或资产重组的企业，应提示可比性受限
