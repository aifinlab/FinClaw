# 住院资料诊断与病程识别维度

本文件用于统一住院资料解析时的诊断、病程和关键事实识别口径。

## 一、基本信息维度

优先核对以下信息是否齐备：

- 年龄
- 性别
- 住院时间
- 出院时间
- 医疗机构
- 入院科室
- 文书类型

如以上任一项缺失且会影响判断，应在住院基本信息或后续处理建议中说明。

## 二、核心诊断提取口径

重点提取以下内容：

- 入院诊断
- 出院诊断
- 术前术后诊断
- 并发症诊断
- 伴随疾病

## 三、住院经过提取口径

重点提取以下内容：

- 入院原因
- 主要症状
- 关键检查结果
- 重要病情变化
- 出院结论

## 四、信息不足识别口径

以下情况应列为信息不足：

- 仅有住院事实，无明确诊断
- 有手术描述但无术后结论
- 有出院记录但无恢复状态
- 不同文书诊断不一致
