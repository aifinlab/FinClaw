# A股分红日历参考指南

## A股分红完整流程

一个分红方案从提出到投资者收到现金，经历以下阶段：

```
董事会预案 → 股东大会审议 → 实施公告 → 股权登记日 → 除权除息日 → 红利发放日
   ⚪           🟡              🟢           T-1            T            T+N
```

### 各阶段详解

1. **董事会预案**：公司董事会提出利润分配预案，披露每10股派息/送股/转增方案。此阶段方案可能调整。
2. **股东大会审议**：股东大会投票通过分配方案。通过后方案基本确定，但极少数情况仍可能变更。
3. **实施公告**：公司发布分红实施公告，明确股权登记日、除权除息日、红利发放日等具体日期。
4. **股权登记日**（Record Date）：当日收盘时持有该股票的股东，有权获得此次分红。**登记日当天买入仍可享有分红权。**
5. **除权除息日**（Ex-Dividend Date）：股票价格进行除权除息调整。当日及之后买入的股票不再享有本次分红。
6. **红利发放日**（Payment Date）：现金红利到达股东证券账户。通常在除权除息日后1-3个工作日。

### 时间线示例
```
3月15日  董事会公告：10派5元
4月20日  股东大会通过
5月10日  实施公告：登记日5/20，除权日5/21，发放日5/22
5月20日  股权登记日（当天收盘持有 → 有分红权）
5月21日  除权除息日（开盘价自动下调）
5月22日  红利发放日（现金到账）
```

## 除权价计算

### 纯派息（无送转）
```
除权参考价 = 前收盘价 - 每股派息
```
示例：前收盘价30元，10派5元（每股0.5元）
→ 除权参考价 = 30 - 0.5 = 29.50元

### 派息 + 送转
```
除权参考价 = (前收盘价 - 每股派息) / (1 + 每股送转比例)
```
示例：前收盘价30元，10派5元送5股转5股
→ 每股派息 = 0.5元
→ 每股送转比例 = (5+5)/10 = 1
→ 除权参考价 = (30 - 0.5) / (1 + 1) = 14.75元

### 纯送转（无派息）
```
除权参考价 = 前收盘价 / (1 + 每股送转比例)
```

## 红利税政策（差异化征收）

A股对个人投资者实行差异化红利税政策，按**持股期限**征收：

| 持股期限 | 税率 | 实际到手 | 说明 |
|---------|------|---------|------|
| > 1年 | **免税** | 100% | 鼓励长期投资 |
| 1个月 ~ 1年 | **10%** | 90% | 中期持有 |
| ≤ 1个月 | **20%** | 80% | 短期交易 |

### 关键细节
- **持股起算日**：从买入日开始计算，到卖出日结束
- **税款扣缴时点**：**卖出时**才扣税，不是分红时。分红时先全额发放，卖出时根据持股期限补缴税款
- **FIFO原则**：多次买入同一只股票，按先进先出计算持股期限
- **适用范围**：仅针对个人投资者，机构投资者（基金、保险等）免征红利税
- **送转股不征税**：送股和转增不视为红利，不征收红利税（但送股在会计上来源于未分配利润，转增来源于资本公积）

### 计算示例
某股10派5元（每股0.5元），持有1000股：
- 分红时到账：1000 × 0.5 = 500元（全额发放）
- 若持股3个月后卖出：补缴 500 × 10% = 50元，实际到手450元
- 若持股2年后卖出：无需补缴，实际到手500元

## 分红 vs 送转 的本质区别

| 对比项 | 现金分红 | 送股 | 转增股 |
|--------|---------|------|--------|
| 本质 | 利润分配给股东 | 利润转为股本 | 资本公积转为股本 |
| 现金流 | 公司付出真金白银 | 无现金流出 | 无现金流出 |
| 来源 | 未分配利润 | 未分配利润 | 资本公积 |
| 对总资产 | 减少 | 不变 | 不变 |
| 对每股价值 | 不变（除权调整） | 稀释 | 稀释 |
| 投资者收益 | 获得现金 | 股数增加但股价下调 | 股数增加但股价下调 |
| 税务 | 按持股期限征税 | 不征税 | 不征税 |

**核心观点**：只有现金分红是真正的股东回报，送转本质上是"把一张100元换成两张50元"，总价值不变。

## 高送转风险提示

### 什么是高送转
通常指每10股送转合计≥5股的方案，如"10转10"、"10送5转5"。

### 高送转常见风险
1. **配合减持**：大股东/高管在高送转前后减持套现，利用市场炒作抬高股价
2. **业绩不匹配**：公司业绩平平甚至下滑，却推出高送转方案，动机可疑
3. **炒作透支**：高送转预期炒作后，除权日后股价可能长期贴权（不填权）
4. **监管关注**：交易所对高送转实施重点问询，要求说明合理性

### 监管规则（沪深交易所）
- 高送转方案需充分披露合理性
- 提议人不得在方案披露后6个月内减持
- 亏损公司不得高送转
- 净利润增速低于送转比例的，需充分说明理由

## 分红日历的投资应用

### 登记日前策略
- **抢权行情**：部分投资者在登记日前买入以享受分红，可能推高股价
- **风险**：短期持有税率20%，扣税后可能得不偿失

### 除权日策略
- **填权行情**：优质高息股除权后可能快速填权（股价回到除权前水平）
- **贴权风险**：基本面转弱的公司除权后可能持续下跌

### 长期持有策略
- 持股超1年免红利税，适合高股息策略投资者
- 关注连续多年稳定分红的公司（如银行、公用事业）
- 股息率 > 10年国债收益率时，具有配置价值
