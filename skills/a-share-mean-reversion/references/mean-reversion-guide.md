# 均值回归策略参考指南

## 理论基础

### Ornstein-Uhlenbeck 过程
- 均值回归随机过程：dX_t = θ(μ - X_t)dt + σdW_t
- θ: 回归速度（越大回归越快）
- μ: 长期均值
- σ: 波动率
- 半衰期 = ln(2) / θ

### Hurst 指数
- H < 0.5: 均值回归（反持续性）
- H = 0.5: 随机游走
- H > 0.5: 趋势性（持续性）
- 计算方法：R/S 分析、DFA、方差比检验

### 方差比检验（Variance Ratio Test）
- VR(q) = Var(q日收益) / (q × Var(1日收益))
- VR < 1: 均值回归
- VR = 1: 随机游走
- VR > 1: 趋势性

## 策略类型

### 1. 布林带策略（Bollinger Band）
- 中轨 = MA(N)，通常 N=20
- 上轨 = 中轨 + k×σ，通常 k=2
- 下轨 = 中轨 - k×σ
- 买入：价格触及下轨；卖出：价格触及上轨
- A 股改进：结合成交量确认（缩量触轨更可靠）

### 2. RSI 均值回归
- RSI < 30: 超卖，买入信号
- RSI > 70: 超买，卖出信号
- A 股经验：RSI(6) < 20 是强超卖信号

### 3. 估值均值回归
- PE/PB 低于历史 10% 分位：超跌
- PE/PB 高于历史 90% 分位：超涨
- 适用于周期稳定的蓝筹股

### 4. 行业相对回归
- 个股 vs 行业指数的相对强弱 RS
- RS 大幅偏离均值后回归
- 适用于行业内个股轮动

## 半衰期计算

```python
# 方法：AR(1) 回归
# ΔS_t = a + b × S_{t-1} + ε
# 半衰期 = -ln(2) / b
import numpy as np
spread_lag = spread.shift(1)
delta = spread - spread_lag
# OLS: delta ~ spread_lag
b = np.cov(delta[1:], spread_lag[1:])[0,1] / np.var(spread_lag[1:])
halflife = -np.log(2) / b
```

### 半衰期解读
| 半衰期（天） | 适合策略 | 换仓频率 |
|------------|---------|---------|
| < 5 | 高频/日内 | 每日 |
| 5-20 | 短线均值回归 | 每周 |
| 20-60 | 中线均值回归 | 每月 |
| > 60 | 不适合均值回归 | - |

## A 股特有模式

### 短期反转效应
- 过去 1-4 周涨幅最大的股票，未来 1-4 周倾向于跑输
- 统计显著，是 A 股最稳健的异象之一
- 原因：散户追涨杀跌 → 短期过度反应 → 回归

### 估值锚定
- A 股投资者有强烈的"锚定效应"
- 股价偏离前期平台/整数关口后容易回归
- 尤其在震荡市有效

### 涨跌停对回归的影响
- 连续涨停/跌停后，均值回归信号可能延迟
- 应等待涨跌停打开、成交量恢复正常后再入场
