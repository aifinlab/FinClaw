# 审核关注点与常见争议参考

## 1. 责任成立相关
- 事故真实性是否有充分支持。
- 疾病或事故经过是否能支持本次申请责任。
- 关键诊断、伤残、死亡、治疗结果是否有正式材料支撑。

## 2. 免责边界相关
- 是否涉及等待期、免责情形、除外责任或投保前既往问题。
- 是否存在自述与病历、事故资料不一致导致责任边界不清。
- 是否存在需进一步核查的高风险描述，但当前依据不足。

## 3. 因果关系与定义适配
- 事故与伤情、疾病与诊断、治疗与费用之间是否存在合理衔接。
- 申请责任是否与产品责任定义存在适配疑点。
- 伤残等级、重疾定义、身故原因等是否存在核心证据不足。

## 4. 申请资格与身份关系
- 申请人身份是否明确。
- 受益人、监护人、法定代理人或关系链条是否清楚。
- 收款账户与申请主体是否对应。

## 5. 材料与真实性风险
- 是否存在关键页缺失、截图不完整、签章不清、归属不明等问题。
- 是否存在前后矛盾、时间线断裂、金额不匹配或证据来源不明。
- 是否需要调查支持、补件复核或升级审核。

## 表达原则
- 能确认的争议点直接写明。
- 不能确认的内容标记为“需进一步确认”。
- 不把争议点直接写成最终赔付或拒赔结论。
