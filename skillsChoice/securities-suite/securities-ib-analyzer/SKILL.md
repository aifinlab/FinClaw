# 券商投行业务分析器

## 功能

- IPO承销数据分析
- 再融资（定增、配股、可转债）统计
- 并购重组财务顾问业务
- 债券承销规模
- 投行收入排名

## 数据源

- 证监会官网
- 沪深交易所
- AkShare新股数据
- Wind/同花顺承销数据

## 核心指标

| 指标 | 说明 |
|------|------|
| IPO承销金额 | 首发承销规模 |
| IPO承销数量 | 承销家数 |
| 再融资承销 | 定增+配股+可转债 |
| 债券承销 | 公司债+企业债+ABS |
| 市占率 | 承销额/市场总额 |
