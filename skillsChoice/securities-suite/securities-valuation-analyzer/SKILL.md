# 券商股估值分析器

## 功能

- PB/PE/PEG计算
- 同业估值对比
- 历史估值分位
- 估值修复潜力分析
- 券商板块整体估值

## 数据源

- 同花顺iFinD API
- AkShare实时行情

## 核心指标

| 指标 | 说明 | 参考范围 |
|------|------|----------|
| PB | 市净率 | 券商通常1.0-2.5x |
| PE | 市盈率 | 通常15-30x |
| PEG | PE/盈利增速 | <1为低估 |
