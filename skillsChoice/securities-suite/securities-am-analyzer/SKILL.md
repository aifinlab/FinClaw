# 券商资管业务分析器

## 功能

- 券商资管规模统计
- 主动管理规模占比
- 产品类型结构（集合/定向/专项）
- 资管收入排名
- 公募基金管理规模（对于有公募牌照的券商）

## 数据源

- 中国证券投资基金业协会
- 中证协
- AkShare基金数据

## 核心指标

| 指标 | 说明 |
|------|------|
| 资管总规模 | 受托管理资金总额 |
| 主动管理规模 | 主动管理类产品规模 |
| 集合资管 | 面向合格投资者发行的资管计划 |
| 定向资管 | 单一客户委托的资管计划 |
| 专项资管 | 资产证券化等专项计划 |
