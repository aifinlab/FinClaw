# 证券行业政策分析器

## 功能

- 证监会最新政策追踪
- 创新业务政策解读
- 行业改革动态
- 政策影响评估
- 监管处罚信息

## 数据源

- 中国证监会官网
- 沪深交易所
- 证券业协会
- AkShare财经新闻

## 政策类型

| 类型 | 说明 |
|------|------|
| 注册制改革 | IPO发行制度 |
| 科创板/创业板 | 多层次资本市场 |
| 北交所 | 服务创新型中小企业 |
| 两融标的扩容 | 信用交易范围 |
| 衍生品创新 | 期权、期货品种 |
