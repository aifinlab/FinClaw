# 券商两融业务分析器

## 功能

- 融资融券余额统计
- 融资买入/偿还分析
- 融券卖出/偿还分析
- 担保品分析
- 维保比例监控

## 数据源

- 沪深交易所
- AkShare两融数据

## 核心指标

| 指标 | 说明 |
|------|------|
| 融资余额 | 融资未偿还金额 |
| 融券余额 | 融券未偿还金额 |
| 两融余额 | 融资+融券余额 |
| 维保比例 | 担保物价值/融资融券债务 |
| 融资买入额 | 当日融资买入金额 |
