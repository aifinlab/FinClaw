# 券商经纪业务分析器

## 功能

- 沪深两市成交额统计
- 券商经纪业务收入排名
- 佣金率趋势分析
- 市占率变化
- 新增投资者数量

## 数据源

- 沪深交易所
- 中证协
- AkShare市场数据

## 核心指标

| 指标 | 说明 |
|------|------|
| 日均成交额 | A股日均成交金额 |
| 经纪收入 | 代理买卖证券收入 |
| 佣金率 | 平均佣金费率 |
| 市占率 | 券商成交额占比 |
| 新增投资者 | 每月新开户数 |
