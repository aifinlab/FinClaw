# 券商自营业务分析器

## 功能

- 自营收入分析
- 投资收益率计算
- 持仓结构分析
- 衍生品投资情况
- 收益波动性分析

## 数据源

- Tushare财报数据
- AkShare财务数据
- 券商年报

## 核心指标

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 自营收入 | 投资收益+公允价值变动 | - |
| 自营收益率 | 自营收入/自营规模 | >5% |
| 收入占比 | 自营收入/总营收 | 20-40% |
| 收益波动 | 自营收入季度波动 | 越小越好 |
