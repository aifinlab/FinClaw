# 券商评级分析器

## 功能

- 证监会券商分类评级
- 评级变化追踪
- 不同级别券商统计
- 评级与业务资格关联
- 历史评级查询

## 数据源

- 中国证监会
- 中证协

## 评级分类

| 级别 | 说明 |
|------|------|
| AA | 最高级别，业务资格最全 |
| A | 优秀级别 |
| BBB | 良好级别 |
| BB | 一般级别 |
| B | 及格级别 |
| CCC/CC/C | 较差级别 |
| D | 最低级别 |
