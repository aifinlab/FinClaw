# 券商财务分析器

## 功能

- 资产负债表分析（总资产、净资产、杠杆率）
- 利润表分析（营收结构、净利润、各项业务收入）
- 收入结构分析（经纪/投行/资管/自营占比）
- 核心财务指标（ROE、ROA、杠杆率）
- 季度/年度对比分析
- 同业对比分析

## 数据源

- **同花顺iFinD API**: 实时行情、财务指标
- **Tushare Pro**: 详细财报数据（三大表）
- **AkShare**: 券商列表、基础数据

## 支持的券商

A股上市券商（主要50+家）：
- 头部券商：中信证券、华泰证券、海通证券、国泰君安、招商证券、广发证券、中国银河、中信建投
- 中型券商：东方证券、兴业证券、光大证券、国信证券、申万宏源、东方财富
- 特色券商：东方财富（互联网）、指南针（金融科技）

## 核心指标

| 指标 | 计算公式 | 优秀标准 |
|------|----------|----------|
| ROE | 净利润/平均净资产 | >8% |
| 杠杆率 | 总资产/净资产 | 3-5x |
| 经纪收入占比 | 经纪收入/总营收 | - |
| 投行收入占比 | 投行收入/总营收 | - |
| 资管收入占比 | 资管收入/总营收 | - |
| 自营收入占比 | 投资收益/总营收 | - |
