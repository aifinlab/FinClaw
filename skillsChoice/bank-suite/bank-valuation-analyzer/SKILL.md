# 银行股估值分析器

## 功能

- PB/PE/股息率计算
- 同业估值对比
- 历史估值分位
- 估值修复潜力分析
- DDM股利折现估值

## 数据源

- 同花顺iFinD API
- AkShare实时行情
- Tushare历史数据

## 核心指标

| 指标 | 说明 | 参考范围 |
|------|------|----------|
| PB | 市净率 | 银行通常0.5-1.5x |
| PE | 市盈率 | 通常5-15x |
| 股息率 | 股息/股价 | 高股息策略关注>5% |
| PEG | PE/盈利增速 | <1为低估 |
