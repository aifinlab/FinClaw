# 银行信贷分析器

## 功能

- 贷款结构分析（对公/零售/票据）
- 行业投向分布
- 区域分布分析
- 客户集中度监控
- 信贷增速跟踪

## 数据源

- 央行金融机构信贷收支表
- Tushare银行财报
- AkShare信贷数据

## 贷款分类

| 类型 | 说明 |
|------|------|
| 对公贷款 | 企业贷款、事业单位贷款 |
| 零售贷款 | 个人住房贷款、消费贷、经营贷 |
| 票据融资 | 银行承兑汇票贴现 |
