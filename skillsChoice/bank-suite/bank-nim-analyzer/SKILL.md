# 银行净息差(NIM)分析器

## 功能

- 净息差(NIM)计算与跟踪
- 生息资产收益率分析
- 负债成本率监控
- 息差同业对比
- 息差趋势预测

## 数据源

- Tushare银行财务数据
- 同花顺iFinD API
- AkShare财报数据

## 核心公式

| 指标 | 计算公式 |
|------|----------|
| 净息差(NIM) | 净利息收入 / 平均生息资产 |
| 生息资产收益率 | 利息收入 / 平均生息资产 |
| 负债成本率 | 利息支出 / 平均计息负债 |
