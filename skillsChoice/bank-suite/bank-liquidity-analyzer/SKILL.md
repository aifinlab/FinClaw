# 银行流动性分析器

## 功能

- 流动性覆盖率(LCR)分析
- 净稳定资金比例(NSFR)分析
- 存贷比监控
- 期限错配分析
- 流动性压力测试

## 数据源

- AkShare银行财务数据
- 央行货币政策执行报告
- 银保监会监管统计数据

## 核心指标

| 指标 | 监管要求 | 说明 |
|------|----------|------|
| 流动性覆盖率(LCR) | ≥100% | 优质流动性资产/未来30日净现金流出 |
| 净稳定资金比例(NSFR) | ≥100% | 可用稳定资金/所需稳定资金 |
| 存贷比 | ≤75% | 贷款余额/存款余额 |
| 流动性缺口率 | - | 流动性缺口/到期表内外资产 |
