### Test 4: fund-attribution-analysis (收益归因)

#### 数据来源
| 数据类型 | 来源 | 说明 |
|:---|:---|:---|
| Brinson模型 | 学术标准 | Brinson, Hood & Beebower (1986) |
| 计算公式 | 数学公式 | 配置/选择/交互三效应 |
| 行业收益 | 用户输入/模拟 | 组合与基准收益数据 |

#### Brinson公式验证
```
组合收益: 8.60%
基准收益: 6.40%
超额收益: +2.20%

归因分解:
  配置效应 = Σ(Wp-Wb)×Rb = +0.40% ✅
  选择效应 = ΣWb×(Rp-Rb) = +1.30% ✅
  交互效应 = Σ(Wp-Wb)×(Rp-Rb) = +0.50% ✅

验证: 0.40% + 1.30% + 0.50% = 2.20% ≈ 超额收益 ✅
```

#### 多因子模型
```
因子定义 (Fama-French扩展):
  MKT: 市场因子 (市场风险暴露)
  HML: 价值因子 (高减低价值)
  SMB: 规模因子 (小减大规模)
  MOM: 动量因子 (过去12月动量)
  QUAL: 质量因子 (高ROE-低ROE)
  LOWV: 低波因子 (低波动-高波动)
```

#### 分红方式对比验证
```
参数: 本金10万, 年化10%, 分红率3%, 5年

现金分红方案:
  净值增长率 = 10% - 3% = 7%
  5年后净值 = 100000 × (1.07)⁵ = ¥140,255
  累计分红 = 100000 × 3% × 5 = ¥15,000
  总价值 = ¥155,255 ✅

红利再投资方案:
  5年后价值 = 100000 × (1.10)⁵ = ¥161,051 ✅

差额: ¥5,796 (再投资多赚3.73%)
```
