# 基金市场研究 (Fund Market Research)

基金市场全景分析与趋势研究工具，提供全面的基金市场数据和趋势洞察。

## 功能概述

- **市场规模统计**: 基金数量、总规模、各类别分布
- **新发基金追踪**: 新发行基金统计、募集情况、发行趋势
- **资金流向分析**: 申购/赎回数据、资金净流入/流出
- **热门板块追踪**: 行业主题基金热度、板块轮动
- **收益分布分析**: 基金收益率分布、业绩排名
- **市场情绪指标**: 基金发行热度、投资者情绪指数
- **基金发行日历**:  upcoming新发基金、募集期提醒

## 使用方法

### 命令行调用

```bash
# 市场概览
fund-market --overview

# 查看资金流向
fund-market --flow --period 30d

# 热门板块
fund-market --hot-sectors

# 发行日历
fund-market --calendar --days 30

# 收益分布
fund-market --returns-dist --type 股票型
```

### Python API

```python
from fund_market_research import FundMarketResearch

research = FundMarketResearch()

# 市场概览
overview = research.get_overview()

# 资金流向
flow = research.get_fund_flow(days=30)

# 热门板块
sectors = research.get_hot_sectors()
```

## 市场指标说明

### 市场规模指标

| 指标 | 说明 | 数据来源 |
|:---|:---|:---|
| **基金总数** | 全市场公募基金数量 | 中基协 |
| **管理规模** | 公募基金总规模（万亿） | 中基协 |
| **类别分布** | 股票/混合/债券/货币/QDII/指数 | 分类统计 |
| **规模变化** | 环比/同比规模变化 | 月度数据 |

### 资金流向指标

| 指标 | 说明 | 计算方式 |
|:---|:---|:---|
| **净流入** | 申购-赎回的净额 | 正=流入，负=流出 |
| **流入占比** | 流入金额/总规模 | 反映市场热度 |
| **分类流向** | 各类型基金资金流向 | 权益/固收分别统计 |
| **渠道分布** | 直销/银行/券商/第三方 | 销售渠道分析 |

### 热门板块追踪

| 板块 | 追踪指标 | 热度计算 |
|:---|:---|:---|
| **行业主题** | 科技/医药/消费/新能源 | 相关基金数量和规模 |
| **风格主题** | 价值/成长/红利 | 风格基金表现 |
| **区域主题** | 港股/美股/新兴市场 | QDII资金流向 |
| **策略主题** | 量化/ESG/FOF | 策略基金热度 |

## 注意事项

1. 市场数据基于公开信息，存在一定滞后性
2. 资金流向为估算数据，仅供参考
3. 市场情绪指数基于多因子综合计算
4. 基金发行日历信息以基金公司公告为准
5. 热门板块追踪基于基金持仓和申赎数据估算
