# 期货风险分析器

## 功能

- 历史波动率计算
- VaR风险价值计算
- 最大回撤分析
- 价格跳空分析
- 流动性风险评估

## 数据源

- AkShare期货历史数据

## 风险指标

| 指标 | 说明 |
|------|------|
| 波动率 | 收益率标准差 |
| VaR | 风险价值 |
| 最大回撤 | 历史最大亏损 |
| 夏普比率 | 收益风险比 |
