# 期货交割分析器

## 功能

- 交割月份提醒
- 交割成本计算
- 仓单数据查询
- 交割仓库信息
- 逼仓风险评估

## 交割要素

| 要素 | 说明 |
|------|------|
| 交割月份 | 合约到期月份 |
| 交割价格 | 交割结算价 |
| 交割成本 | 运输、检验、仓储费用 |
| 仓单 | 标准仓单数量 |
| 升贴水 | 地区、品质差异调整 |
