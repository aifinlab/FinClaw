# 期货持仓追踪器

## 功能

- 期货公司持仓排名
- 多头/空头持仓分析
- 持仓集中度计算
- 主力资金动向追踪
- 持仓变化信号

## 数据源

- 交易所持仓排名数据
- AkShare期货数据

## 持仓指标

| 指标 | 说明 |
|------|------|
| 前20名多头持仓 | 多头持仓总量 |
| 前20名空头持仓 | 空头持仓总量 |
| 净持仓 | 多头 - 空头 |
| 持仓集中度 | 前20名持仓/总持仓 |
