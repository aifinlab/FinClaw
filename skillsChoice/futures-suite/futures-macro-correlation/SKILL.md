# 期货宏观相关性分析器

## 功能

- 期货与PMI相关性分析
- 期货与CPI/PPI关系
- 期货与利率走势
- 期货与汇率联动
- 经济周期定位

## 宏观指标

| 指标 | 影响品种 |
|------|----------|
| PMI | 工业品、金属 |
| CPI | 农产品、贵金属 |
| 利率 | 国债期货、金融期货 |
| 汇率 | 进出口相关品种 |
