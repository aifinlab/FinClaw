# 期货市场概览分析器

## 功能

- 期货市场整体成交持仓统计
- 四大交易所数据（上期所、大商所、郑商所、中金所）
- 各品种活跃度排名
- 成交额、成交量趋势
- 市场资金流向分析

## 数据源

- **AkShare**: 期货行情、成交持仓数据
- **上期所**: 有色金属、黑色金属、能源化工、贵金属
- **大商所**: 农产品、化工品、黑色系
- **郑商所**: 农产品、化工品、PTA等
- **中金所**: 股指期货、国债期货
- **上期能源**: 原油期货、期权

## 核心指标

| 指标 | 说明 |
|------|------|
| 成交量 | 当日成交手数 |
| 持仓量 | 未平仓合约数量 |
| 成交额 | 成交金额 |
| 换手率 | 成交量/持仓量 |
| 主力合约 | 成交量最大的合约 |

## 依赖

- akshare >= 1.10.0
- pandas >= 1.3.0
- requests >= 2.28.0
