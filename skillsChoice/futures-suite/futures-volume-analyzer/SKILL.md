# 期货成交量持仓分析器

## 功能

- 成交量趋势分析
- 持仓量变化分析
- 量价关系分析
- 资金流向判断
- 主力合约识别

## 数据源

- AkShare期货数据
- 交易所公开数据

## 核心指标

| 指标 | 说明 |
|------|------|
| 成交量变化率 | 当日成交量/前日成交量 |
| 持仓量变化率 | 当日持仓量/前日持仓量 |
| 量价背离 | 价格与成交量背离信号 |
| 仓差 | 当日持仓量变化 |
