# 期货保证金计算器

## 功能

- 交易所保证金率查询
- 期货公司保证金率设置
- 开仓资金计算
- 维持保证金监控
- 不同品种保证金对比

## 数据源

- 各交易所官方保证金标准
- AkShare期货数据

## 核心指标

| 指标 | 说明 |
|------|------|
| 交易所保证金率 | 交易所要求的最低保证金比例 |
| 期货公司保证金率 | 期货公司实际收取的保证金比例 |
| 合约价值 | 合约价格 × 合约乘数 |
| 保证金金额 | 合约价值 × 保证金率 |
