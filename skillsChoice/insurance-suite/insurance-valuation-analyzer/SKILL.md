# 保险股估值分析器

## 功能

- PEV估值分析（内含价值倍数）
- PB估值分析
- PE估值分析
- 同业估值对比
- 历史估值分位

## 核心指标

| 指标 | 说明 |
|------|------|
| PEV | 股价/每股内含价值 |
| PB | 市净率 |
| PE | 市盈率 |
| 内含价值 | 保险公司清算价值+有效业务价值 |
