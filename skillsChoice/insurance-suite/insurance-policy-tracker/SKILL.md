# 保险行业政策追踪器

## 功能

- 最新监管政策追踪（自动从多数据源获取）
- 保险法修订动态
- 监管处罚信息
- 行业改革措施
- 政策影响分析
- 政策搜索功能
- 自动更新机制（每日自动更新）

## 数据源

| 数据源 | 类型 | 更新频率 |
|--------|------|----------|
| 东方财富API | 公开API | 实时 |
| 新浪财经API | 公开API | 实时 |
| 金融监管总局公开信息 | 官方信息 | 定期 |
| 本地缓存 | 缓存文件 | 24小时有效期 |

## 使用方法

```bash
# 获取最新政策（默认）
python scripts/main.py

# 获取重点改革
python scripts/main.py --reforms

# 搜索政策（关键词）
python scripts/main.py --search "养老金"

# 强制更新（忽略缓存）
python scripts/main.py --force

# 执行自动更新
python scripts/main.py --update
```

## 政策类型

| 类型 | 说明 |
|------|------|
| 监管规定 | 偿付能力、资金运用等 |
| 业务规则 | 产品、销售、理赔等 |
| 改革政策 | 税优健康险、养老险等 |

## 自动更新机制

- 缓存有效期：24小时
- 自动更新：每日自动从API获取最新数据
- 失败回退：网络失败时使用本地模板数据
- 缓存位置：`data/policy_cache.json`

## 数据说明

本工具优先从东方财富、新浪财经等公开API获取保险行业动态信息，当网络连接失败时，将使用基于金融监管总局公开信息的模板数据作为回退方案。

## 依赖

```bash
pip install requests python-dateutil
```
