# 财产险业务分析器

## 功能

- 财险市场规模分析
- 车险业务分析
- 非车险业务分析
- 承保利润分析
- 综合成本率监控

## 核心指标

| 指标 | 说明 |
|------|------|
| 综合成本率 | 赔付率+费用率，<100%盈利 |
| 赔付率 | 赔付支出/保费收入 |
| 费用率 | 费用支出/保费收入 |
| 承保利润 | 保费收入-赔付-费用 |
| 车险占比 | 车险保费/总保费 |
