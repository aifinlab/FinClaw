# 保险市场概览分析器

## 功能

- 保险行业保费收入统计
- 人身险/财产险结构分析
- 赔付支出分析
- 保险资金运用余额
- 行业增长趋势

## 数据源

- 国家金融监督管理总局
- AkShare保险行业数据
- 中国保险行业协会

## 核心指标

| 指标 | 说明 |
|------|------|
| 保费收入 | 保险公司收取的保费总额 |
| 赔付支出 | 保险赔付金额 |
| 保险密度 | 人均保费收入 |
| 保险深度 | 保费收入/GDP |
| 资金运用余额 | 保险资金投资规模 |

## 依赖

- akshare >= 1.10.0
- pandas >= 1.3.0
