# 寿险业务分析器

## 功能

- 寿险市场规模分析
- 新单保费追踪
- 代理人渠道分析
- 新业务价值分析
- 产品结构调整

## 核心指标

| 指标 | 说明 |
|------|------|
| 新单保费 | 新签保单保费 |
| 新业务价值 | NBV，反映业务质量 |
| 代理人数量 | 销售队伍规模 |
| 人均产能 | 代理人平均保费 |
| 续期保费 | 续保保费收入 |
