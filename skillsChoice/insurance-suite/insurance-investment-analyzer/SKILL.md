# 保险资金运用分析器

## 功能

- 保险资金配置结构
- 投资收益分析
- 资产配置变化
- 另类投资分析
- 利率敏感性分析

## 资金配置类别

| 类别 | 说明 |
|------|------|
| 银行存款 | 流动性资产 |
| 债券 | 固定收益类 |
| 股票 | 权益类投资 |
| 另类投资 | 非标、不动产等 |

## 核心指标

| 指标 | 说明 |
|------|------|
| 投资收益率 | 总投资收益/平均投资资产 |
| 综合投资收益率 | 包含浮盈浮亏 |
| 资产配置比例 | 各类资产占比 |
